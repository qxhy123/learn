# 从零到高阶的概率论与数理统计教程（深度学习应用版）

## 项目简介

本教程旨在为学习者提供一套系统、完整的概率论与数理统计学习资源，从最基础的随机事件与概率概念出发，循序渐进地覆盖随机变量、常见分布、极限定理、参数估计、假设检验等高阶主题。

**本教程的独特之处**：每章都包含「深度学习应用」部分，展示该章数学概念在人工智能和机器学习领域的实际应用，配有可运行的 Python/PyTorch 代码示例。

---

## 目标受众

- 大学本科生、研究生，首次系统学习概率论与数理统计
- 机器学习/深度学习初学者，希望理解 AI 的数学基础
- 计算机科学、数据科学、人工智能等专业的在校学生
- 备考研究生入学考试（考研数学）的学习者
- 希望回顾或巩固概率统计知识的工程师和研究者

## 学习路径建议

### 路径一：快速入门（约 4-6 周）

适合有一定高中数学基础、希望快速掌握概率统计核心内容的学习者：

1. 学习第1-3章（概率基础）
2. 重点学习第4-6章（随机变量）
3. 学习第7-8章（常见分布）
4. 学习第16章（点估计）和第22章（信息论）

### 路径二：系统学习（约 3-4 个月）

适合大学新生或备考研究生的学习者：

1. 按章节顺序从头到尾完整学习（第1-24章）
2. 每章学完后完成配套练习题
3. 重点强化第10-12章（极限定理）和第16-18章（参数估计）

### 路径三：深度学习导向（约 2-3 个月）

适合希望理解 AI 数学基础的工程师：

1. 快速浏览第1-6章，建立基础概念
2. 重点学习每章的「深度学习应用」部分
3. 深入学习第16章（MLE与交叉熵）和第18章（贝叶斯估计）
4. 精读第22章（信息论）和第23章（蒙特卡洛方法）

## 前置要求

学习本教程需要以下数学基础：

- **代数基础**：多项式运算、方程求解、不等式
- **函数概念**：函数的定义、图像、基本性质
- **基础微积分**：导数、积分的基本概念（第5章及后续章节需要）
- **线性代数基础**（可选）：向量、矩阵的基本运算（第6章及后续章节有帮助）

如果上述基础知识有所欠缺，建议先学习本项目的配套教程：
- [微积分教程](../calculus-tutorial/README.md)
- [线性代数教程](../linear-algebra-tutorial/README.md)

## 如何使用本教程

1. **按需选择章节**：根据自身基础和学习目标，选择合适的学习路径
2. **先看概念，再看例题**：每节先理解概念定义与定理，再跟随例题实操
3. **运行代码示例**：每章的深度学习应用部分都有可运行的代码，建议动手实践
4. **完成练习题**：每章末尾有练习题，建议独立完成后再查看答案
5. **使用速查表**：学习过程中可随时参考附录公式速查表
