# 前言：如何使用本教程

## 为什么学习概率论与数理统计

概率论与数理统计是现代科学的基础工具，它研究随机现象的规律性，为我们提供了在不确定性中做出理性决策的数学框架。无论你是学习纯数学、应用数学，还是计算机科学、物理学、生物学、经济学，概率统计都是不可或缺的工具。

特别是在人工智能和深度学习时代，概率统计的重要性更加凸显：

- **神经网络的训练是概率优化问题**：损失函数本质是似然函数的负对数
- **深度学习充满随机性**：权重初始化、Dropout、数据增强、SGD都涉及随机采样
- **生成模型基于概率分布**：VAE、GAN、扩散模型都在学习数据的概率分布
- **不确定性量化**：贝叶斯神经网络、MC Dropout提供预测的置信度

本教程不仅教授概率统计的经典理论，更在每一章展示这些概念如何在深度学习中发挥作用。

## 本教程的结构

全书共分为**八个部分**，**24章**，按照由浅入深的顺序编排：

### 第一部分：概率基础（第1-3章）
建立随机事件和概率的基本概念，这是后续所有内容的基石。

### 第二部分：随机变量（第4-6章）
学习如何用数学语言描述随机现象，理解期望、方差等核心概念。

### 第三部分：常见分布（第7-9章）
掌握常用的离散分布和连续分布，理解它们的性质和应用场景。

### 第四部分：极限定理（第10-12章）
学习大数定律和中心极限定理，理解随机现象的渐近规律。

### 第五部分：统计基础（第13-15章）
从概率论过渡到数理统计，学习统计量和抽样分布。

### 第六部分：参数估计（第16-18章）
掌握点估计、区间估计和贝叶斯估计的方法和理论。

### 第七部分：假设检验（第19-21章）
学习如何用统计方法做出科学决策，理解显著性检验的原理。

### 第八部分：高级主题（第22-24章）
学习信息论、蒙特卡洛方法和概率图模型，这些是深度学习中最常用的高级工具。

## 每章的结构

每一章都遵循统一的结构，方便你系统学习：

1. **学习目标**：明确本章要掌握的知识点和技能
2. **正文内容**：定义、定理、证明和例题，配有直观解释
3. **本章小结**：提炼核心要点，便于复习
4. **深度学习应用**：展示本章数学概念在AI中的实际应用，配有代码示例
5. **练习题**：5道精选题目，巩固所学
6. **练习答案**：详细的解题过程（以折叠块形式呈现）

## 符号约定

本教程使用以下符号约定（详见[符号说明](./appendix/notation-guide.md)）：

| 符号 | 含义 |
|------|------|
| $\Omega$ | 样本空间 |
| $A, B, C$ | 事件 |
| $P(A)$ | 事件 $A$ 的概率 |
| $P(A \mid B)$ | 在 $B$ 发生条件下 $A$ 的条件概率 |
| $X, Y, Z$ | 随机变量 |
| $E[X]$ 或 $\mu$ | 期望 |
| $\text{Var}(X)$ 或 $\sigma^2$ | 方差 |
| $f(x)$ | 概率密度函数 (PDF) |
| $F(x)$ | 累积分布函数 (CDF) |
| $p(x)$ | 概率质量函数 (PMF) |
| $\mathcal{N}(\mu, \sigma^2)$ | 正态分布 |

## 学习建议

### 对于初学者

1. **按顺序学习**：第一部分到第三部分的内容是后续的基础，不要跳过
2. **多做计算**：概率统计需要大量的计算练习，熟练度很重要
3. **理解直觉**：每个概念都要建立直观理解，不要只记公式
4. **运行代码**：动手运行每章的代码示例，加深理解

### 对于有基础的学习者

1. **可以跳读**：如果已经掌握基础，可以直接从第四部分开始
2. **关注深度学习应用**：这是本教程的特色，即使熟悉概率统计也值得一读
3. **做练习题**：检验自己的理解是否到位

### 对于深度学习从业者

1. **重点章节**：第16章（MLE）、第18章（贝叶斯估计）、第22章（信息论）、第23章（蒙特卡洛）
2. **代码优先**：可以先看代码示例，再回顾数学原理
3. **结合实践**：将所学应用到自己的项目中
