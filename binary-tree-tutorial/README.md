# 二叉树从零到高阶完全教程

> 从基础概念到竞赛级算法，系统掌握二叉树的一切

## 教程结构

```
binary-tree-tutorial/
├── part1-foundations/       # 第一部分：基础概念
│   ├── 01-introduction.md       # 什么是二叉树
│   ├── 02-terminology.md        # 基本术语与性质
│   ├── 03-types.md              # 二叉树的分类
│   └── 04-representation.md    # 存储与表示方法
│
├── part2-traversals/        # 第二部分：遍历算法
│   ├── 01-dfs-recursive.md      # 深度优先遍历（递归）
│   ├── 02-dfs-iterative.md      # 深度优先遍历（迭代）
│   ├── 03-bfs.md                # 广度优先遍历 BFS
│   └── 04-morris.md             # Morris 遍历（O(1)空间）
│
├── part3-bst/               # 第三部分：二叉搜索树
│   ├── 01-bst-concepts.md       # BST 概念与性质
│   ├── 02-bst-operations.md     # BST 增删查改
│   └── 03-bst-applications.md  # BST 应用与变体
│
├── part4-balanced-trees/    # 第四部分：平衡树
│   ├── 01-avl.md                # AVL 树
│   ├── 02-red-black.md          # 红黑树
│   └── 03-btree.md              # B 树与 B+ 树
│
└── part5-advanced/          # 第五部分：高级专题
    ├── 01-segment-tree.md       # 线段树
    ├── 02-fenwick-tree.md       # 树状数组（BIT）
    ├── 03-trie.md               # 字典树（Trie）
    ├── 04-tree-dp.md            # 树形动态规划
    └── 05-lca.md                # 最近公共祖先（LCA）
```

## 学习路径

### 入门路径（1-2周）
1. Part1 全部 → Part2 第1-3章 → Part3 第1-2章

### 进阶路径（再1-2周）
2. Part2 第4章 → Part3 第3章 → Part4 第1章

### 竞赛路径（再2-3周）
3. Part4 全部 → Part5 全部

## 代码语言

所有示例使用 **Python**，关键部分提供 **C++** 对照实现。

## 前置知识

- 基本编程能力（函数、递归、类）
- 了解时间/空间复杂度分析
- 基础数据结构（数组、链表）
