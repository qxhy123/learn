# 从零到高阶的动态规划教程

## 项目简介

本教程旨在为学习者提供一套系统、完整的动态规划（Dynamic Programming，DP）学习资源。从最基础的递归思想出发，循序渐进地覆盖记忆化搜索、状态转移方程设计、经典DP模型，最终深入到区间DP、树形DP、状压DP、数位DP以及各种高级优化技巧。

**本教程的独特之处**：每章都包含「核心思维模型」部分，帮助你建立解题直觉，而不仅仅是记忆解法模板。所有代码示例均使用Python实现，附带详细的复杂度分析。

---

## 目标受众

- 算法初学者，希望从零开始系统掌握动态规划
- 有一定编程经验，刷LeetCode遇到DP题目就懵的开发者
- 准备算法面试（大厂）的候选人
- ACM/OI竞赛选手，希望系统整理DP知识体系
- 希望深入理解DP数学本质的研究者

## 如何使用本教程

每章结构如下：
1. **问题引入** - 从直觉出发，发现问题
2. **核心思维模型** - 建立解题框架
3. **代码实现** - 从暴力到优化，逐步推进
4. **复杂度分析** - 时间与空间的权衡
5. **变式与扩展** - 举一反三
6. **LeetCode题目推荐** - 即学即练
