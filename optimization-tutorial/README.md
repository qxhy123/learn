# 从零到高阶的最优化理论教程（深度学习应用版）

## 项目简介

本教程旨在为学习者提供一套系统、完整的最优化理论学习资源，从最基础的数学准备知识出发，循序渐进地覆盖无约束优化、约束优化、凸优化、数值方法、随机优化等核心主题。

**本教程的独特之处**：每章都包含「深度学习应用」部分，展示该章数学概念在人工智能和机器学习领域的实际应用，配有可运行的 Python/PyTorch 代码示例。

---

## 目标受众

- 大学本科生、研究生，首次系统学习最优化理论
- 机器学习/深度学习研究者，希望理解优化算法的数学原理
- 计算机科学、应用数学、运筹学等专业的在校学生
- 备考研究生入学考试的学习者
- 希望深入理解深度学习优化器原理的工程师

## 学习路径建议

### 路径一：快速入门（约 4-6 周）

适合有线性代数和微积分基础、希望快速掌握优化核心内容的学习者：

1. 学习第1-3章（数学基础）
2. 重点学习第4-6章（无约束优化）
3. 学习第16-18章（随机优化）
4. 学习第19章（损失曲面分析）

### 路径二：系统学习（约 3-4 个月）

适合大学新生或备考研究生的学习者：

1. 按章节顺序从头到尾完整学习（第1-24章）
2. 每章学完后完成配套练习题
3. 重点强化第7-9章（约束优化）和第10-12章（凸优化）

### 路径三：深度学习导向（约 2-3 个月）

适合希望理解深度学习优化原理的工程师：

1. 快速浏览第1-3章，建立基础概念
2. 重点学习第5章（梯度下降）和第16-18章（随机优化）
3. 深入学习第19-21章（深度学习优化专题）
4. 精读第22-24章（高级主题）

## 前置要求

学习本教程需要以下数学基础：

- **线性代数**：向量、矩阵运算、特征值分解、正定矩阵
- **微积分**：多元函数求导、链式法则、Taylor展开
- **概率统计**（可选）：期望、方差、随机变量（第6部分需要）

如果上述基础知识有所欠缺，建议先学习本项目的配套教程：
- [线性代数教程](../linear-algebra-tutorial/README.md)
- [微积分教程](../calculus-tutorial/README.md)
- [概率论与数理统计教程](../probability-statistics-tutorial/README.md)

## 如何使用本教程

1. **按需选择章节**：根据自身基础和学习目标，选择合适的学习路径
2. **先看概念，再看例题**：每节先理解概念定义与定理，再跟随例题实操
3. **运行代码示例**：每章的深度学习应用部分都有可运行的代码，建议动手实践
4. **完成练习题**：每章末尾有练习题，建议独立完成后再查看答案
5. **使用速查表**：学习过程中可随时参考附录公式速查表
