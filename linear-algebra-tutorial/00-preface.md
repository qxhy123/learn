# 前言：如何使用本教程

## 为什么学习线性代数

线性代数是现代数学的基础分支之一，它研究向量、向量空间、线性变换以及与之相关的方程组。无论你是学习纯数学、应用数学，还是计算机科学、物理学、工程学、经济学，线性代数都是不可或缺的工具。

特别是在人工智能和深度学习时代，线性代数的重要性更加凸显：

- **神经网络的本质是矩阵运算**：每一层的前向传播都是矩阵乘法加激活函数
- **优化算法依赖线性代数**：梯度、Hessian矩阵、特征值分析是理解优化的关键
- **降维与特征提取**：PCA、SVD等技术直接来源于线性代数理论
- **注意力机制**：Transformer中的自注意力本质是矩阵的相似度计算

本教程不仅教授线性代数的经典理论，更在每一章展示这些概念如何在深度学习中发挥作用。

## 本教程的结构

全书共分为**八个部分**，**24章**，按照由浅入深的顺序编排：

### 第一部分：基础（第1-3章）
建立向量和矩阵的基本概念，这是后续所有内容的基石。

### 第二部分：线性方程组（第4-6章）
学习如何求解线性方程组，理解解的存在性和唯一性，掌握高斯消元法和逆矩阵。

### 第三部分：行列式（第7-8章）
介绍行列式的定义、计算方法和几何意义，理解其在判断矩阵可逆性中的作用。

### 第四部分：向量空间（第9-12章）
进入抽象代数的领域，学习向量空间、线性相关性、基与维数、子空间等核心概念。

### 第五部分：线性变换（第13-15章）
理解线性映射的本质，掌握线性变换与矩阵的对应关系，学习基变换和相似矩阵。

### 第六部分：内积空间（第16-18章）
引入内积、范数和正交性的概念，学习正交化方法和最小二乘法。

### 第七部分：特征值与特征向量（第19-21章）
深入研究矩阵的特征值和特征向量，掌握对角化技术和谱定理。

### 第八部分：高级主题（第22-24章）
学习奇异值分解（SVD）、二次型和矩阵微积分，这些是深度学习中最常用的高级工具。

## 每章的结构

每一章都遵循统一的结构，方便你系统学习：

1. **学习目标**：明确本章要掌握的知识点和技能
2. **正文内容**：定义、定理、证明和例题，配有几何直观解释
3. **本章小结**：提炼核心要点，便于复习
4. **深度学习应用**：展示本章数学概念在AI中的实际应用，配有代码示例
5. **练习题**：5道精选题目，巩固所学
6. **练习答案**：详细的解题过程（以折叠块形式呈现）

## 符号约定

本教程使用以下符号约定（详见[符号说明](./appendix/notation-guide.md)）：

| 符号 | 含义 |
|------|------|
| $\mathbf{v}, \mathbf{u}, \mathbf{w}$ | 向量（粗体小写字母） |
| $A, B, C$ | 矩阵（大写字母） |
| $\mathbb{R}^n$ | $n$ 维实向量空间 |
| $\mathbb{R}^{m \times n}$ | $m \times n$ 实矩阵空间 |
| $A^T$ | 矩阵 $A$ 的转置 |
| $A^{-1}$ | 矩阵 $A$ 的逆 |
| $\det(A)$ 或 $|A|$ | 矩阵 $A$ 的行列式 |
| $\text{rank}(A)$ | 矩阵 $A$ 的秩 |
| $\lambda$ | 特征值 |
| $\|\mathbf{v}\|$ | 向量 $\mathbf{v}$ 的范数 |
| $\langle \mathbf{u}, \mathbf{v} \rangle$ | 向量 $\mathbf{u}$ 和 $\mathbf{v}$ 的内积 |

## 学习建议

### 对于初学者

1. **按顺序学习**：第一部分到第三部分的内容是后续的基础，不要跳过
2. **多做计算**：线性代数需要大量的矩阵运算练习，熟练度很重要
3. **画图理解**：二维和三维的例子要画图，建立几何直觉
4. **运行代码**：动手运行每章的代码示例，加深理解

### 对于有基础的学习者

1. **可以跳读**：如果已经掌握基础，可以直接从第四部分开始
2. **关注深度学习应用**：这是本教程的特色，即使熟悉线性代数也值得一读
3. **做练习题**：检验自己的理解是否到位

### 对于深度学习从业者

1. **重点章节**：第16章（内积）、第19章（特征值）、第22章（SVD）、第24章（矩阵微积分）
2. **代码优先**：可以先看代码示例，再回顾数学原理
3. **结合实践**：将所学应用到自己的项目中
