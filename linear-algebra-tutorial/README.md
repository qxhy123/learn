# 从零到高阶的线性代数教程（深度学习应用版）

## 项目简介

本教程旨在为学习者提供一套系统、完整的线性代数学习资源，从最基础的向量与矩阵概念出发，循序渐进地覆盖线性方程组、向量空间、线性变换、特征值与特征向量、奇异值分解等高阶主题。

**本教程的独特之处**：每章都包含「深度学习应用」部分，展示该章数学概念在人工智能和机器学习领域的实际应用，配有可运行的 Python/PyTorch 代码示例。

---

## 目标受众

- 大学本科生、研究生，首次系统学习线性代数
- 机器学习/深度学习初学者，希望理解 AI 的数学基础
- 计算机科学、数据科学、人工智能等专业的在校学生
- 备考研究生入学考试（考研数学）的学习者
- 希望回顾或巩固线性代数知识的工程师和研究者

## 学习路径建议

### 路径一：快速入门（约 4-6 周）

适合有一定高中数学基础、希望快速掌握线性代数核心内容的学习者：

1. 学习第1-3章（向量与矩阵基础）
2. 重点学习第4-6章（线性方程组与逆矩阵）
3. 学习第19章（特征值与特征向量）
4. 学习第22章（SVD）

### 路径二：系统学习（约 3-4 个月）

适合大学新生或备考研究生的学习者：

1. 按章节顺序从头到尾完整学习（第1-24章）
2. 每章学完后完成配套练习题
3. 重点强化第9-12章（向量空间）和第19-22章（特征值与SVD）

### 路径三：深度学习导向（约 2-3 个月）

适合希望理解 AI 数学基础的工程师：

1. 快速浏览第1-6章，建立基础概念
2. 重点学习每章的「深度学习应用」部分
3. 深入学习第16-18章（内积与最小二乘）
4. 精读第22章（SVD）和第24章（矩阵微积分）

## 前置要求

学习本教程需要以下数学基础：

- **代数基础**：多项式运算、方程求解、不等式
- **函数概念**：函数的定义、图像、基本性质
- **基础微积分**（可选）：导数、积分的基本概念（第24章需要）

如果上述基础知识有所欠缺，建议先学习本项目的[微积分教程](../calculus-tutorial/README.md)。

## 如何使用本教程

1. **按需选择章节**：根据自身基础和学习目标，选择合适的学习路径
2. **先看概念，再看例题**：每节先理解概念定义与定理，再跟随例题实操
3. **运行代码示例**：每章的深度学习应用部分都有可运行的代码，建议动手实践
4. **完成练习题**：每章末尾有练习题，建议独立完成后再查看答案
5. **使用速查表**：学习过程中可随时参考附录公式速查表
