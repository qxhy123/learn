# 附录C：符号说明

## 角与度量

| 符号 | 含义 |
|------|------|
| $\theta, \alpha, \beta, \gamma$ | 角度变量 |
| $°$ | 角度制单位（如 $90°$） |
| rad | 弧度制单位（常省略），$\pi$ rad $= 180°$ |
| $s = r\theta$ | 弧长公式（$\theta$ 为弧度） |
| $S = \frac{1}{2}r^2\theta$ | 扇形面积公式（$\theta$ 为弧度） |

## 三角函数

| 符号 | 含义 | 定义域限制 |
|------|------|-----------|
| $\sin x$ | 正弦 | $\mathbb{R}$ |
| $\cos x$ | 余弦 | $\mathbb{R}$ |
| $\tan x$ | 正切 | $x \neq \frac{\pi}{2}+k\pi$ |
| $\cot x$ | 余切 | $x \neq k\pi$ |
| $\sec x$ | 正割 ($1/\cos x$) | $x \neq \frac{\pi}{2}+k\pi$ |
| $\csc x$ | 余割 ($1/\sin x$) | $x \neq k\pi$ |

## 反三角函数

| 符号 | 含义 | 定义域 | 值域 |
|------|------|--------|------|
| $\arcsin x$ 或 $\sin^{-1}x$ | 反正弦 | $[-1,1]$ | $[-\pi/2,\pi/2]$ |
| $\arccos x$ 或 $\cos^{-1}x$ | 反余弦 | $[-1,1]$ | $[0,\pi]$ |
| $\arctan x$ 或 $\tan^{-1}x$ | 反正切 | $\mathbb{R}$ | $(-\pi/2,\pi/2)$ |

**注意**：$\sin^{-1}x$ 表示反正弦，不是 $(\sin x)^{-1} = \csc x$。

## 三角形

| 符号 | 含义 |
|------|------|
| $a, b, c$ | 三角形三边长 |
| $A, B, C$ | 三边对应的角 |
| $R$ | 外接圆半径 |
| $r$ | 内切圆半径 |
| $s = (a+b+c)/2$ | 半周长 |
| $S$ 或 $\Delta$ | 三角形面积 |

## 图像参数

| 符号 | 含义 |
|------|------|
| $A$ | 振幅（$y = A\sin(\omega x + \varphi) + k$ 中） |
| $\omega$ | 角频率，$T = 2\pi/\omega$ |
| $T$ | 周期 |
| $\varphi$ | 初相位 |
| $f = 1/T$ | 频率 |

## 复数与欧拉

| 符号 | 含义 |
|------|------|
| $i$ | 虚数单位，$i^2 = -1$ |
| $z = a + bi$ | 复数代数形式 |
| $z = re^{i\theta}$ | 复数极形式 |
| $\|z\| = r$ | 复数模 |
| $\arg z = \theta$ | 辐角 |
| $\bar{z} = a - bi$ | 共轭复数 |
| $\omega_k = e^{i2k\pi/n}$ | $n$ 次单位根 |

## 约定

- 默认使用弧度制，除非特别说明角度制
- 三角方程的一般解默认写成 $k\in\mathbb{Z}$ 的形式
- 反三角函数的主值按通常数学约定处理
- $\sin^2 x$ 表示 $(\sin x)^2$，不是 $\sin(\sin x)$
