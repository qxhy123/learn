# 附录B：符号说明

| 符号 | 含义 |
|------|------|
| $\theta, \alpha, \beta$ | 角或弧度变量 |
| $\sin, \cos, \tan$ | 三角函数 |
| $\csc, \sec, \cot$ | 三角函数的倒数函数 |
| $\arcsin, \arccos, \arctan$ | 反三角函数 |
| $T$ | 周期 |
| $A$ | 振幅（视上下文而定） |
| $\varphi$ | 相位或相位差 |
| $\omega$ | 角频率 |
| $i$ | 虚数单位，满足 $i^2=-1$ |
| $e^{ix}$ | 复指数形式 |
| $r,\theta$ | 极坐标中的模长与极角 |

## 约定

- 默认使用弧度制，除非特别说明角度制。
- 三角方程的一般解默认写成 $k\in\mathbb{Z}$ 的形式。
- 反三角函数的主值按通常数学约定处理。
