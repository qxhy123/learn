# 附录B：单位圆速查

## 1. 常用角

| 角度 | 弧度 | $\sin$ | $\cos$ | $\tan$ |
|------|------|--------|--------|--------|
| $0^\circ$ | $0$ | $0$ | $1$ | $0$ |
| $30^\circ$ | $\pi/6$ | $1/2$ | $\sqrt3/2$ | $\sqrt3/3$ |
| $45^\circ$ | $\pi/4$ | $\sqrt2/2$ | $\sqrt2/2$ | $1$ |
| $60^\circ$ | $\pi/3$ | $\sqrt3/2$ | $1/2$ | $\sqrt3$ |
| $90^\circ$ | $\pi/2$ | $1$ | $0$ | 无定义 |

## 2. 象限符号

| 象限 | $\sin$ | $\cos$ | $\tan$ |
|------|--------|--------|--------|
| I | + | + | + |
| II | + | - | - |
| III | - | - | + |
| IV | - | + | - |

## 3. 使用方法

1. 先把角归约到 $[0,2\pi)$。
2. 找参考角。
3. 用特殊角表取绝对值。
4. 用象限确定符号。
