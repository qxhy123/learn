# 从零到高阶的三角函数教程

## 项目简介

本教程是一套系统的三角函数学习资源，从角、弧度、单位圆开始，逐步覆盖六个三角函数、恒等式、图像、方程不等式、三角形几何、反三角与解析技巧，并进一步进入复数欧拉公式、傅里叶思想、信号、相量与旋转矩阵。

本教程强调三条主线：**旋转、投影、周期**。围绕这三条主线，三角函数不再是一堆孤立公式，而是一套连接几何、代数、分析和工程应用的语言。

## 学习路径建议

1. **基础路线**：第1-9章，建立单位圆、图像和恒等式基础。
2. **解题强化**：第10-18章，训练方程、不等式、几何、三角代换和参数问题。
3. **高阶应用**：第19-24章，理解复数、傅里叶、信号和旋转矩阵中的三角函数。

---

## 教程特色

- 从零开始，但不止于公式背诵
- 每章保留详细例题、推导和图像/结构分析
- 覆盖从初等三角到欧拉公式、傅里叶和信号应用
- 附录提供公式、单位圆、符号和练习答案
