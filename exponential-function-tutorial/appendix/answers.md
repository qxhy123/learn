## 使用建议

本附录不是完整详解手册，而是帮助你快速判断“这类题应该回看哪里”的导航页。

如果你做题时卡住了，优先先问自己：

- 我卡在代数变形、图像理解，还是建模翻译？
- 我是不知道公式，还是不知道为什么该选这个方法？
- 这道题更像是在比较图像、反求参数、分析长期趋势，还是切换到复指数/矩阵语言？

带着这些问题回看对应章节，复习效率会高很多。

## 复习回链建议

| 题型 / 卡点 | 优先回看 |
|------|------|
| 指数运算律老出错、总把加法也套成幂公式 | 第3章 |
| 图像看不出渐近线、锚点、平移和翻折 | 第4章 |
| 指数位置上的未知数不会解 | 第5章、第7章 |
| 不会判断不等式方向或参数边界 | 第8章 |
| 递推、等比、复利和离散增长混在一起 | 第9章 |
| 不理解 $e$ 为什么会出现 | 第10章 |
| 不清楚指数和对数谁增长更快 | 第11章 |
| 求导、凸性、切线不等式之间没连起来 | 第12章、第19章 |
| 看不懂 $y'=ky$、积分因子或指数积分模式 | 第13章 |
| 不会做小量近似、误差判断或级数桥接 | 第14章 |
| 复指数、极形式、单位根结构模糊 | 第15章 |
| 不清楚矩阵指数如何描述系统演化 | 第17章 |
| 不知道 Laplace / Fourier 为什么也会出现指数 | 第18章 |
| 综合题不知道该先选哪种语言 | 第20章 |
