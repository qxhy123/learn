# 符号说明

本附录收录本教程中最常见的符号与表达，方便快速查阅。

---

## 一、函数与集合

| 符号 | 含义 |
|------|------|
| $a^x$ | 以 $a$ 为底的指数函数（通常要求 $a>0,\ a\neq1$） |
| $e^x$ | 自然指数函数 |
| $\ln x$ | 自然对数 |
| $\log_a x$ | 以 $a$ 为底的对数（要求真数 $x>0$） |
| $\mathbb{R}$ | 实数集 |
| $\mathbb{C}$ | 复数集 |

---

## 二、指数相关

| 符号 | 含义 |
|------|------|
| $a>1$ | 指数函数递增的常见情形 |
| $0<a<1$ | 指数函数递减的常见情形 |
| $a^0=1$ | 零指数规则 |
| $a^{-n}=1/a^n$ | 负指数规则 |
| $a^{m/n}$ | 分数指数（实数范围下要检查定义条件） |
| $q$ | 离散模型中的比例因子 / 公比 |
| $r$ | 增长率或名义利率，需结合语境判断是“每步”还是“每年” |
| $k$ | 连续模型中的增长 / 衰减常数，单位随时间尺度而定 |

---

## 三、微积分相关

| 符号 | 含义 |
|------|------|
| $\dfrac{d}{dx}$ | 对 $x$ 求导 |
| $\int$ | 积分符号 |
| $C$ | 积分常数 |
| $\lim$ | 极限符号 |
| $\ll$ | 渐近上“增长得更慢” |

---

## 四、复数与推广结构

| 符号 | 含义 |
|------|------|
| $i$ | 虚数单位，满足 $i^2=-1$ |
| $e^{ix}$ | 复指数，描述旋转与振荡 |
| $z=re^{i\theta}$ | 复数极形式，其中 $r$ 为模长，$\theta$ 为辐角 |
| $\sinh,\cosh,\tanh$ | 双曲正弦、双曲余弦、双曲正切 |
| $e^A$ | 矩阵指数 |
| $\operatorname{LSE}(x_1,\dots,x_n)$ | log-sum-exp，常写作 $\log(e^{x_1}+\cdots+e^{x_n})$ |

---

## 五、常见模型表达

| 表达 | 含义 |
|------|------|
| $y=Ce^{kt}$ | 指数增长 / 衰减模型 |
| $y'=ky$ | 变化率与自身成正比 |
| $A=P(1+\frac{r}{m})^{mt}$ | 离散复利模型 |
| $T_d$ | 倍增时间 |
| $T_{1/2}$ | 半衰期 |
| $\Pr(X\ge t)$ | 尾概率 |
