# 符号说明

## 一、基本记号

| 符号 | 含义 |
|------|------|
| $\log_a x$ | 以 $a$ 为底、$x$ 的对数 |
| $a$ | 底数 |
| $x$ | 真数 |
| $\ln x$ | 自然对数，$\log_e x$ |
| $\lg x$ | 常用对数，$\log_{10}x$ |

---

## 二、条件与集合

| 符号 | 含义 |
|------|------|
| $a>0,\ a\neq1$ | 对数底数合法条件 |
| $x>0$ | 真数合法条件 |
| $\mathbb R$ | 实数集 |
| $\mathbb C$ | 复数集 |
| $\mathrm{Dom}$ | 定义域 |
| $\mathrm{Ran}$ | 值域 |

---

## 三、微积分相关

| 符号 | 含义 |
|------|------|
| $\dfrac{d}{dx}$ | 对 $x$ 求导 |
| $\int$ | 积分符号 |
| $C$ | 积分常数 |
| $\lim$ | 极限 |
| $\ll$ | 渐近上增长更慢 |

---

## 四、信息与概率

| 符号 | 含义 |
|------|------|
| $P(A)$ | 事件 $A$ 的概率 |
| $I(A)$ | 事件的信息量 |
| $H(X)$ | 熵 |
| $H(p,q)$ | 交叉熵 |
| $\ell(\theta)$ | 对数似然 |

---

## 五、复对数相关

| 符号 | 含义 |
|------|------|
| $z=re^{i\theta}$ | 复数极形式 |
| $\theta$ | 辐角 |
| $\mathrm{Arg}(z)$ | 主辐角 |
| $\mathrm{Log}\,z$ | 主值对数 |
| $k\in\mathbb Z$ | 复对数分支编号 |
