# 练习答案导航

本文件提供各章练习题的快速索引，用于复习和回看。扩写后的各章练习题更强调“结构解释、条件检查、易错辨析、迁移应用与综合复述”，建议先独立完成，再回看正文。

---

## 使用建议

- 先完成本章基础题，再做解释题、易错题和迁移题
- 回看答案前，先检查自己是否写出了底数与真数条件
- 方程题重点回看“是否验根”，参数题重点回看“是否验范围”
- 新增题目多数不是机械计算题，而是帮助你复述结构、比较方法和连接前后章节
