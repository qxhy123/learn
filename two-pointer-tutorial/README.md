# 从零到高阶的双指针教程

## 项目简介

本教程旨在为学习者提供一套系统、完整的双指针（Two Pointers）学习资源。从最基础的"两端夹逼"思想出发，循序渐进地覆盖对撞指针、同向双指针、快慢指针、滑动窗口，最终深入到区间合并、矩阵双指针、多指针分区等高阶技巧。

**本教程的独特之处**：每章都包含「核心思维模型」部分，帮助你建立解题直觉——双指针不是一种算法，而是一种思维框架。所有代码示例均使用Python实现，附带详细的复杂度分析和边界条件讨论。

---

## 目标受众

- 算法初学者，希望从零开始系统掌握双指针技巧
- 有一定编程经验，刷LeetCode遇到数组/字符串题目不知如何优化的开发者
- 准备算法面试（大厂）的候选人
- 希望深入理解滑动窗口、快慢指针等变体的进阶学习者
- ACM/OI 竞赛选手，希望系统整理双指针知识体系

## 如何使用本教程

每章结构如下：
1. **问题引入** - 从直觉出发，发现暴力解法的瓶颈
2. **核心思维模型** - 建立双指针解题框架
3. **代码实现** - 从暴力到优化，逐步推进
4. **复杂度分析** - 时间与空间的权衡
5. **边界条件与陷阱** - 工程实现中的细节
6. **变式与扩展** - 举一反三
7. **LeetCode题目推荐** - 即学即练
