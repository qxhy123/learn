# 第13章：て形的万能应用

> 「て形を制する者は日本語を制する」——掌握了て形，就掌握了日语的半壁江山。
> て形不仅仅是连接动词的工具，它是日语语法体系中最具生命力的形态，承载着时态、体态、情感与逻辑关系的丰富表达。

---

## 13.1 て形回顾与在语法体系中的核心地位

て形（Te-form，连用形2）是日语动词变化中使用频率最高的形态之一。在学习了基础的て形构成规则之后，本章将深入探讨て形所能延伸出的多元语法结构。

### て形的构成快速回顾

| 动词类型 | 代表例 | て形 |
|---------|--------|------|
| 五段动词（う结尾） | 買う | 買って |
| 五段动词（く结尾） | 書く | 書いて |
| 五段动词（ぐ结尾） | 泳ぐ | 泳いで |
| 五段动词（す结尾） | 話す | 話して |
| 五段动词（つ结尾） | 持つ | 持って |
| 五段动词（ぬ结尾） | 死ぬ | 死んで |
| 五段动词（ぶ结尾） | 遊ぶ | 遊んで |
| 五段动词（む结尾） | 飲む | 飲んで |
| 五段动词（る结尾） | 切る | 切って |
| 一段动词 | 食べる | 食べて |
| 一段动词 | 見る | 見て |
| 不规则：する | する | して |
| 不规则：来る | 来る | 来て |

**注**：行く的て形是いって（促音便，而非いいて），是例外中的例外，需要单独记忆。

### て形在语法体系中的地位

て形是日语语法的"枢纽"，它能够：

1. **连接动作**（顺序、方式、原因）
2. **附接补助动词**，形成时态和体态（ている、てある、ておく等）
3. **构成敬语**（てください、ていただく等）
4. **表达情感色彩**（てしまう、てほしい等）

## 13.2 ～ている的多重含义

`～ている` 是て形最核心的应用之一，其含义远比字面的"正在做"复杂得多。理解它的关键在于**动词的动作特性**。

### 含义一：动作进行中（動作の進行）

适用于**持续性动词**（表示持续状态的动作）：

```
今、ご飯を食べています。
（我现在正在吃饭。）

彼女は今、電話をかけています。
（她现在正在打电话。）

会議中です。部長は今、話しています。
（正在开会。部长现在正在讲话。）
```

### 含义二：结果状态持续（結果の状態）

适用于**瞬间性动词**（动作在一瞬间完成，但结果持续存在）：

```
窓が開いています。
（窗户是开着的。）← 不是"正在打开"，而是"处于打开状态"

電気がついています。
（灯开着。）

彼は結婚しています。
（他结婚了。）← 不是"正在结婚"，而是"处于已婚状态"
```

### 含义三：习惯/反复动作（習慣・繰り返し）

结合时间副词表达习惯性行为：

```
毎朝、公園を走っています。
（我每天早上都在公园跑步。）

彼女は毎日日本語を勉強しています。
（她每天都在学日语。）

私はあの会社で働いています。
（我在那家公司上班。）← 表示职业/长期状态
```

### 含义四：经历状态（経験・現在の状態）

表示某种经历所带来的当前状态：

```
あの映画、もう見ています。
（那部电影我已经看过了。）

彼のことはよく知っています。
（他的事我很了解。）

日本には三回来ています。
（我来过日本三次了。）
```

### 判断标准：动词的瞬间性 vs 持续性

这是理解`～ている`多重含义的核心：

| 动词类型 | 定义 | 例子 | ～ている的含义 |
|---------|------|------|--------------|
| 持続動詞（持续动词） | 动作有持续过程 | 食べる、走る、読む | 动作正在进行中 |
| 瞬間動詞（瞬间动词） | 动作瞬间完成 | 死ぬ、結婚する、着く | 结果状态持续中 |
| 状態動詞（状态动词） | 本身就是状态 | ある、いる、できる | 通常不用ている |

**判断方法**：问自己"这个动作有没有持续的过程？"

```
【持续动词的例】
書いている → 正在写（写的过程在持续）
泳いでいる → 正在游泳（游的过程在持续）

【瞬间动词的例】
結婚している → 处于已婚状态（结婚这个动作是瞬间的）
死んでいる → 处于死亡状态（死亡是瞬间发生的）
着いている → 已经到达（到达是瞬间的）
```

**难点辨析：**

```
【電気が消えている】
消える（熄灭）是瞬间动词，所以：
「電気が消えている」= 灯处于熄灭状态（不是"正在熄灭"）

【花が咲いている】
咲く（开花）可以被视为过程的结果，所以：
「花が咲いている」= 花开着（处于开放状态）

【知っている vs 分かっている】
「知っている」= 知道（知る是瞬间动词，て形=状态）
「分かっている」= 懂得、明白（同理）
```

## 13.3 ～てある（他动词的结果状态）

`～てある` 与 `～ている` 的区别是中级学习者最容易混淆的语法点之一。

### 构成与核心语义

`～てある` **只能接他动词**，表示：**某人（通常省略）有意识地做了某个动作，其结果状态现在仍存在**。强调的是"为了某个目的而预先做好的状态"。

### 核心对比：てある vs ている

| 表达 | 例句 | 含义 | 强调点 |
|------|------|------|--------|
| てある | 窓が開けてある | 窗户被（有意）开着 | 人为行为的结果状态 |
| ている | 窓が開いている | 窗户是开着的 | 单纯描述当前状态 |

```
【てある 的例句】
黒板に予定が書いてある。
（黑板上（有人）写了计划。）← 强调是有人写的

壁に絵が飾ってある。
（墙上装饰着画。）← 强调是有人装饰的

もう予約してあります。
（已经（为此）预约好了。）← 强调是预先做的准备

冷蔵庫にビールが冷やしてある。
（冰箱里（为你）冰了啤酒。）← 强调是为某目的准备的
```

### 助词的差异

```
てある句中：
・他動詞 + てある → 主語は「物」（が/は を使う）
  窓が開けてある。（窗被开着）
  ご飯が作ってある。（饭被做好了）

ている句中：
・自動詞 + ている → 主語は「物」
  窓が開いている。（窗是开的）

ている句中：
・他動詞 + ている → 主語は「人」
  彼が窓を開けている。（他在开窗）
```

### 语用层面的差异

```
「もう食事の準備ができている」← 客观陈述：准备完成了
「もう食事の準備をしてある」← 主观强调：（我）已经准备好了

后者暗含"我已经提前为你做了准备"的恩惠意识。
```

## 13.4 ～ておく（事先准备/放置）

`～ておく` 表示**提前做好某事以备将来之用**，或**暂时放任某种状态**。

### 核心语义

1. **预先准备**：提前做好，以应对未来的需要
2. **暂时放置**：暂且不管，维持现状

```
【预先准备】
試験の前に、よく勉強しておきます。
（考试前我要好好预习。）

お客さんが来る前に、部屋を掃除しておいた。
（客人来之前，提前打扫了房间。）

明日早いので、今夜服を準備しておこう。
（明天要早起，今晚先把衣服准备好吧。）

旅行前に、パスポートをコピーしておいてください。
（旅行前请先复印一下护照。）

【暂时放置/暂且不管】
その問題は、今は置いておきましょう。
（那个问题，现在先放一放吧。）

面倒だから、洗い物はそのままにしておこう。
（太麻烦了，碗先就那样放着吧。）
```

### 口语缩约形

在口语中，`～ておく` 常缩约为 `～とく`：

```
準備しておく → 準備しとく
買っておく → 買っとく
食べておく → 食べとく
```

## 13.5 ～てしまう（完了/遗憾）

`～てしまう` 是充满情感色彩的语法形式，表达**动作的完全完成**，并通常附带**遗憾、意外、不可挽回**等情绪。

### 核心语义

| 用法 | 含义 | 情感色彩 |
|------|------|---------|
| 完了 | 动作彻底完成 | 中性或轻微强调 |
| 遗憾/失态 | 做了不该做的事 | 负面情绪 |
| 意外 | 出乎意料地发生了 | 惊讶、无奈 |

```
【完全完成（无遗憾）】
この本、もう読んでしまいました。
（这本书我已经读完了。）

仕事が終わってしまったので、帰っていいですか。
（工作已经全部做完了，可以回去了吗？）

【遗憾/后悔】
大切なものを失くしてしまった。
（把重要的东西弄丢了。）← 遗憾

また遅刻してしまいました。申し訳ありません。
（又迟到了，真的很抱歉。）← 自责

お腹が痛くて、授業中にトイレに行ってしまった。
（肚子痛，课上不得不去了厕所。）← 失态

【意外/不可控】
傘を忘れてきてしまった。
（竟然把雨伞忘来了。）← 意外

財布をどこかに置き忘れてしまったようだ。
（好像把钱包落在什么地方了。）← 茫然无奈
```

### 口语缩约形：ちゃう / じゃう

这是日语口语中极为常见的缩约：

| 原形 | 缩约形 | 规则 |
|------|--------|------|
| ～てしまう | ～ちゃう | て→ちゃ |
| ～でしまう | ～じゃう | で→じゃ |
| ～てしまった | ～ちゃった | てしまった→ちゃった |
| ～でしまった | ～じゃった | でしまった→じゃった |

```
食べてしまう → 食べちゃう
飲んでしまう → 飲んじゃう
忘れてしまった → 忘れちゃった
死んでしまった → 死んじゃった

【口语对话例】
A：あ、ケーキ全部食べちゃったの？
   （啊，蛋糕全吃掉了？）
B：ごめん、おいしくてつい食べちゃった。
   （对不起，太好吃了忍不住吃掉了。）
```

## 13.6 ～てみる（尝试做）

`～てみる` 表示**尝试做某事，看看结果如何**。来自 `見る`（看），引申为"做一下试试看"。

### 核心语义与用法

```
一度やってみます。
（我试试看。）

この料理、食べてみてください。
（请尝一下这道菜。）

田中さんに聞いてみたらどうですか。
（去问一下田中先生怎么样？）

新しいカフェに行ってみたいです。
（我想去新咖啡馆看看。）

その映画、面白そうだから見てみよう。
（那部电影好像很有趣，去看看吧。）

試しにこのボタンを押してみてください。
（试着按一下这个按钮。）
```

### てみる 与 てみたら

`てみたら` 表示"试了之后（发现……）"，通常接意外或发现的结果：

```
食べてみたら、すごくおいしかった。
（吃了之后，发现超级好吃。）

行ってみたら、もう閉まっていた。
（去了之后，发现已经关门了。）
```

## 13.7 ～ていく / ～てくる（变化的方向）

`～ていく` 和 `～てくる` 是て形与方向动词的结合，表达**动作或变化的方向**——是远离说话者，还是靠近说话者。

### 空间移动层面

| 表达 | 含义 | 方向 |
|------|------|------|
| ～ていく | 边做边离去 | 远离说话者 |
| ～てくる | 边做边来 | 靠近说话者 |

```
【～ていく：离开】
弁当を持っていきます。
（带着便当出去。）← 便当和人一起离开

傘を持っていってください。
（请带上雨伞（出去）。）

歌いながら出ていった。
（边唱歌边出去了。）

【～てくる：前来】
お土産を持ってきました。
（带来了礼物。）← 礼物和人一起来了

コーヒーを買ってきてもらえますか。
（能帮我去买杯咖啡来吗？）

ちょっと見てきます。
（去看一下（然后回来）。）← 暗含将会回来
```

### 时间变化层面（重要！）

这是 `ていく/てくる` 更深层的用法，表示**随时间的变化趋势**：

| 表达 | 含义 | 时间方向 |
|------|------|---------|
| ～ていく | 从现在开始，朝未来方向变化 | 现在→将来 |
| ～てくる | 从过去到现在，一直持续或逐渐发展 | 过去→现在 |

```
【～ていく：向未来的变化】
これからも日本語を勉強していきます。
（今后也会继续学日语的。）← 将来持续的意志

地球の温度はこれからも上がっていくだろう。
（地球温度今后还会继续升高吧。）← 未来趋势

少子化が進んでいくと、問題が増える。
（随着少子化的发展，问题会增多。）← 未来发展

【～てくる：从过去到现在的变化】
最近、日本語が上手になってきた。
（最近日语越来越好了。）← 到现在为止的进步

空が暗くなってきた。
（天开始变暗了。）← 渐渐发生的变化

この10年で、テクノロジーが大きく変わってきた。
（这10年来，科技发生了巨大的变化。）← 积累到现在的变化

お腹がすいてきた。
（肚子开始饿了。）← 状态的出现/到来
```

### 综合对比

```
【大切にしていく】（将来也会好好珍惜）→ 未来方向
【大切にしてきた】（一直以来都很珍惜）→ 过去到现在

【増えていく】（会继续增加）→ 未来趋势
【増えてきた】（已经在增加了）→ 过去到现在的变化
```

## 13.8 ～てほしい vs ～たい

这两个表达都与"愿望"相关，但主语完全不同：

| 表达 | 主语 | 含义 |
|------|------|------|
| ～たい | 我（说话者） | 我想做～ |
| ～てほしい | 你/他（第三方） | 我希望你/他做～ |

```
【～たい：自己的愿望】
日本に行きたい。（我想去日本。）
もっと勉強したい。（我想更努力地学习。）
ラーメンを食べたい。（我想吃拉面。）

【～てほしい：希望别人做】
もっと早く来てほしい。（我希望你早点来。）
彼に正直に話してほしい。（我希望他能诚实地说。）
あなたにそこにいてほしい。（我希望你在那里。）

【否定形式】
来てほしくない。（不希望你来。）
知られてほしくない。（不希望被知道。）
```

### 敬体与礼貌度

```
～てほしい → 直接，用于亲近的人
～ていただきたい → 礼貌，希望对方（上位者）做
～てもらいたい → 中等礼貌
```

## 13.9 て形的连接功能

て形除了接补助动词，还直接承担**连接功能**，在复合句中起到关键的逻辑连接作用。

### 连接功能一：动作顺序（时间先后）

```
起きて、顔を洗って、朝ごはんを食べた。
（起床，然后洗脸，然后吃了早饭。）

図書館に行って、本を借りてきました。
（去了图书馆，借了书回来。）
```

### 连接功能二：原因/理由（因果关系）

```
風邪を引いて、学校を休みました。
（因为感冒，所以请假了。）

嬉しくて、涙が出てきた。
（因为太高兴了，眼泪流了出来。）

忙しくて、食事も満足にできない。
（因为太忙，连好好吃饭都做不到。）
```

**注意**：て形表原因时，通常表示自然导致、客观原因，不用于刻意的理由说明（那种情况用から/ので）。

### 连接功能三：并列（同时描述）

```
彼は歌がうまくて、ダンスも得意だ。
（他歌唱得好，舞也跳得好。）

この町は静かで、空気もきれいだ。
（这个城镇安静，空气也清新。）← 形容词て形
```

### 连接功能四：方式/手段（怎么做）

```
走って学校に行く。（跑步去学校。）
笑って答えた。（笑着回答了。）
電車に乗って、駅まで行きます。（坐电车去车站。）
```

## 13.10 て形相关表达总结对比表

| 表达形式 | 核心含义 | 关键特征 | 常见误用 |
|---------|---------|---------|---------|
| ～ている | 进行/状态/习惯 | 根据动词类型决定含义 | 用错动词类型 |
| ～てある | 有意为之的结果状态 | 只接他动词，暗含恩惠 | 与ている混用 |
| ～ておく | 预先准备/暂时放置 | 强调预见性/目的性 | 当作单纯完了 |
| ～てしまう | 完全完成/遗憾意外 | 情感色彩丰富 | 忽视情感层次 |
| ～てみる | 尝试看看 | 带有好奇/试验意味 | 与ようとする混淆 |
| ～ていく | 向未来/远处变化 | 方向：远离/未来 | 与てくる混淆 |
| ～てくる | 向现在/近处变化 | 方向：靠近/过去到现在 | 与ていく混淆 |
| ～てほしい | 希望他人做 | 主语是对方 | 与たい混淆 |
| て（连接） | 顺序/原因/并列/方式 | 不同语境含义不同 | 忽视逻辑关系 |

## 练习题

### 练习一：选择正确的て形表达

请从括号中选择正确的选项：

1. 明日の発表のために、資料を（準備しておいた / 準備してみた）。
2. 財布を忘れて（きた / いった）ので、コンビニで下ろした。
3. 彼は毎朝6時に起き（ている / てある）。
4. ドアが開け（ている / てある）。誰かが準備してくれたんだ。
5. この薬、試し（てみて / ておいて）ください。効くかもしれません。

### 练习二：用适当形式填空

1. 日本語が上手に______（なる / ていく / てくる）ためには、毎日練習が必要だ。
2. 最近、だんだん寒く______（なっていく / なってきた）。
3. 大事なメモは、なくさないようにスマホに______（撮っておいた / 撮ってしまった）。
4. こんなに難しい漢字を全部______（覚えてしまった / 覚えていった）のは素晴らしい。

## 本章小结

本章系统梳理了て形的多元应用体系：

| 核心概念 | 要点 |
|---------|------|
| ～ている的多义性 | 由动词类型（持续性/瞬间性）决定含义 |
| ～てある vs ている | 前者强调有意为之的结果，只接他动词 |
| ～ておく | 预见性的准备行为，口语缩约为とく |
| ～てしまう | 完了+情感色彩（遗憾/意外），口语缩约为ちゃう |
| ～てみる | 尝试性行为，"做做看" |
| ～ていく/てくる | 空间方向（离开/到来）+ 时间方向（未来变化/过去到现在）|
| ～てほしい | 希望他人做，区别于自己的願望（たい）|
| て形的连接功能 | 顺序、原因、并列、方式——上下文决定逻辑关系 |

て形是日语语法中少有的"一形多能"结构。掌握它，不仅需要记忆各种形式，更需要培养对**动词内在特性**的敏感度，以及对**语境中情感流向**的直觉。建议将每种表达形式与具体的生活情景绑定，反复在真实语境中练习使用。
