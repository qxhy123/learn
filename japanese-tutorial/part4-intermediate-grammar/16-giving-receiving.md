# 第16章：授受表达体系

> 「日本語の授受表現は、単なる文法ではなく、人間関係の地図である」——日语的授受表达，不仅仅是语法，更是人际关系的地图。"恩惠流向谁"、"我是局内人还是局外人"、"对方地位高还是低"——这些问题在日语中都要通过授受动词精确地表达出来。

---

## 16.1 日语授受表达的文化背景

### 恩恵の意識（恩惠意识）

日语授受表达体系的深层基础是**恩惠意识**（おんけいのいしき）：一切互动都可以被理解为"恩惠的给予与接受"。当某人为你做了某件事，那不只是一个中性的行为，而是一份"恩惠"（恩恵/おんけい）的流动。

这种意识渗透在日常语言中：
- 「食べてあげる」不只是"我帮你吃"，而暗含"我施予你恩惠"
- 「食べてもらう」不只是"你帮我吃"，而暗含"我承受你的恩惠"
- 「食べてくれる」不只是"他帮我吃"，而暗含"他向我流动了恩惠"

### ウチとソト（内与外）

日本文化中有强烈的**ウチ（内/我方）**和**ソト（外/他方）**的区分：

| 概念 | 含义 | 语言影响 |
|------|------|---------|
| ウチ（内） | 自己、家人、同组织成员 | 视角以ウチ为中心 |
| ソト（外） | 外人、其他组织成员 | 向ウチ的视角流动 |

授受动词的方向性，本质上是以"说话者的ウチ"为中心构建的视角体系。

### 上下関係（上下关系）

日语授受体系还内嵌了**上下关系**：

```
上位者 ←→ 下位者的方向决定使用哪个敬语形式：
さしあげる（我→上位者）、いただく（上位者→我）、くださる（上位者→我）
```

## 16.2 物の授受（给与物品）

### 三个核心授受动词

日语表达"给予/接受物品"有三个基本动词，区别在于**方向**和**视角**：

| 动词 | 方向 | 视角 | 含义 |
|------|------|------|------|
| あげる | 我/他 → 你/他 | 给出方 | 给予（不朝向我） |
| もらう | 你/他 → 我 | 接受方 | 得到/收到 |
| くれる | 你/他 → 我/我方 | 接受方(我) | 给我（向我方流动） |

### あげる（给出，方向远离说话者）

方向：**说话者/其他人 → 他人（非自己）**

```
私は田中さんにプレゼントをあげた。
（我给了田中先生礼物。）← 我给别人

彼は彼女に花をあげた。
（他给了她花。）← 第三者间的给予

彼女は猫にエサをあげた。
（她给猫喂了猫粮。）← 给动物

【注意：不能用于"别人给我"】
✗ 田中さんは私にプレゼントをあげた。（方向错误）
○ 田中さんは私にプレゼントをくれた。（用くれる）
```

### もらう（收到，说话者接受）

方向：**他人 → 说话者/说话者一方**

```
私は田中さんにプレゼントをもらった。
（我从田中先生那里收到了礼物。）

私はお母さんに手紙をもらった。
（我收到了妈妈的信。）

彼女は彼にダイヤモンドをもらった。
（她从他那里得到了钻石。）← 第三者视角

【もらう的特殊形：～に vs ～から】
友達にもらった（从朋友那里收到）← 人，通常用に
先生からもらった（从老师那里收到）← に/からどちらも可
会社からもらった（从公司得到）← 机构/组织，用から更自然
```

### くれる（给我，方向朝向说话者）

方向：**他人 → 说话者/说话者一方（ウチ）**

```
田中さんは私にプレゼントをくれた。
（田中先生给了我礼物。）← 向"我"流动

お母さんはケーキを作ってくれた。
（妈妈给我做了蛋糕。）

友達が自分の傘をくれた。
（朋友把自己的伞给了我。）

【くれる的关键：ウチの意識】
田中さんは弟にプレゼントをくれた。
（田中先生给了我弟弟礼物。）← 弟弟=ウチ，所以用くれる
```

### 视角差异图解

```
【あげる の方向図】
 私/AさんBさん → Cさん（ソト）
       ↗
   あげる

【もらう の方向図】
 Aさん → 私（ウチ）
     ↘
   もらう

【くれる の方向図】
 Aさん → 私/ウチの人
     ↘
   くれる
```

### 三者的完整对比

同一个情景，三个动词的不同说法：

```
【場面：田中さんが私に本をくれた】

1. 田中さんは私に本をくれた。
   （田中先生给了我书。）← くれる，从他人角度看向我

2. 私は田中さんに本をもらった。
   （我从田中先生那里收到了书。）← もらう，从我角度看向来源

3. ✗ 私は田中さんに本をあげた。（错误，方向相反）
```

## 16.3 敬语形式

### 授受动词的敬语体系

| 基本形 | 谦让形（自分が下） | 尊敬形（对方が上） | 使用场合 |
|-------|----------------|----------------|---------|
| あげる | さしあげる | — | 我给上位者 |
| もらう | いただく | — | 我从上位者收 |
| くれる | — | くださる | 上位者给我 |
| やる | — | — | 给下位者/动植物 |

### さしあげる（谦让）

我给上位者时使用，表达我的谦逊：

```
先生にプレゼントをさしあげました。
（我给了老师礼物。）

社長にご報告をさしあげます。
（我向社长汇报。）

【注意：さしあげる可能显得过于刻意】
现代日语中，さしあげる有时给人过度客套的印象，
特别是在てさしあげる（帮您做）的用法中：

△ 荷物を持ってさしあげましょうか。（语感过于居高临下）
○ 荷物をお持ちしましょうか。（更自然的表达）
```

### いただく（谦让）

我从上位者收到时使用，我方姿态低：

```
先生に貴重なご意見をいただきました。
（从老师那里得到了宝贵的意见。）

社長からお言葉をいただきました。
（收到了社长的话/训示。）

お時間をいただき、ありがとうございます。
（感谢您给我时间。）← 常见的感谢用法
```

### くださる（尊敬）

上位者给我时使用，抬高对方：

```
先生がこの本をくださいました。
（老师给了我这本书。）

社長が直接ご連絡をくださいました。
（社长亲自联系了我。）

お声がけをくださり、ありがとうございます。
（感谢您的搭话。）
```

### やる（给下位者/动植物）

`やる` 是 `あげる` 的降格形，用于给比自己地位低的对象（弟妹、下属、动物、植物）：

```
弟に小遣いをやった。（给弟弟零花钱。）
犬にエサをやった。（给狗喂食了。）
花に水をやった。（给花浇水了。）
```

**现代日语注意**：在现代年轻人用语中，`やる` 对人使用有时被视为粗鲁，建议在正式场合或对人时改用 `あげる`。

## 16.4 ～てあげる / ～てもらう / ～てくれる（动作的授受）

### 核心概念：不仅是动作，而是恩惠的流动

将授受动词接在 `て形` 之后，就从"物品的给予"扩展到了**"动作（行为）的给予"**。本质是：**以某个动作为载体的恩惠流动**。

### ～てあげる（我为你做——我施予恩惠）

```
荷物を持ってあげましょうか。
（我帮你拿行李吧？）← 我向对方施予

宿題を手伝ってあげた。
（帮他做了作业。）← 我施予恩惠

困っている人を助けてあげた。
（帮助了有困难的人。）

彼女に料理を作ってあげた。
（为她做了料理。）
```

**文化警告**：`～てあげる` 在语气上暗含"我在施予你恩惠"，使用不当会有 **恩着せがましい（施恩姿态）** 的感觉。

```
【避免的说法】
△ 教えてあげましょうか。（我施恩般地教你吧？）← 居高临下感
○ よかったら、教えましょうか。（如果可以的话，要我教你吗？）← 更谦逊

△ 道を教えてあげた。（我大发慈悲地告诉他路了。）← 奇怪
○ 道を教えた。（告诉他路了。）← 更自然
```

### ～てもらう（请/让某人为我做——我接受恩惠）

这是日语中**最安全、最礼貌的请求方式**之一，强调"接受对方的好意"：

```
友達に宿題を手伝ってもらった。
（请朋友帮了我做作业。）← 我接受恩惠

医者に診てもらった。
（去看了医生。/请医生给我看了病。）

田中さんに翻訳してもらえますか。
（能请田中先生翻译一下吗？）← 礼貌请求

彼女に送ってもらいました。
（让她送我了。）
```

**与てくれる的转换**：

```
田中さんが教えてくれた。
（田中先生教了我。）← くれる：田中的行为向我流动

= 私は田中さんに教えてもらった。
（我得到了田中先生的教导。）← もらう：我接受田中的恩惠

【同一事实，不同视角】
・くれる → 田中先生是主语，强调他的行为
・もらう → "我"是主语，强调我的接受
```

### ～てくれる（某人为我做——恩惠向我流动）

`～てくれる` 是充满感谢与温暖语感的表达：

```
彼女が料理を作ってくれた。
（她为我做了饭。）← 温暖感，感激之情

友達がいつも支えてくれる。
（朋友总是支持我。）← 感谢、依赖

先生が分かりやすく説明してくれました。
（老师用通俗易懂的方式给我解释了。）

子供が笑ってくれると、疲れが吹き飛ぶ。
（孩子一笑，疲劳就消散了。）← 暖心场景
```

### 三者在同一场景中的全面转换

```
【場面：先生が私に日本語を教えた】

1. 先生は私に日本語を教えてくれた。
   （老师教了我日语。）← くれる，感谢老师

2. 私は先生に日本語を教えてもらった。
   （我请/让老师教了我日语。）← もらう，我的视角

3. 私は先生に日本語を教えていただいた。
   （我承蒙老师教导了日语。）← いただく，更敬语

4. 先生が日本語を教えてくださった。
   （老师给我教了日语。）← くださる，最高尊敬

【同一事实，四种表达的敬意梯度】
普通：くれた（くれる）← 感谢，对平辈
↓
礼貌：くださった（くださる）← 尊敬，对上位者
↓
谦让：教えてもらった（もらう）← 我的谦逊
↓
高敬：教えていただいた（いただく）← 最高谦让
```

## 16.5 授受表达的敬体形式

### ～ていただく / ～てくださる 在请求中的使用

| 表达 | 形式 | 敬意度 | 例 |
|------|------|-------|---|
| てください | 基本请求 | 普通 | 見てください |
| てくれない？ | 亲密请求 | 亲密（非正式） | 見てくれない？ |
| てくれますか | 普通问句请求 | 普通 | 見てくれますか |
| てくれませんか | 礼貌否定疑问 | 礼貌 | 見てくれませんか |
| てもらえますか | もらう的疑问 | 礼貌 | 見てもらえますか |
| てもらえませんか | もらう的否定疑问 | 较礼貌 | 見てもらえませんか |
| ていただけますか | いただく的疑问 | 礼貌（上位者） | 見ていただけますか |
| ていただけませんか | いただく的否定疑问 | 最礼貌 | 見ていただけませんか |

### ～ていただけませんか（最礼貌的请求方式）

这是日语中最礼貌、最常用于正式场合的请求表达：

```
少し待っていただけませんか。
（能否请您稍等一下？）

こちらにサインをしていただけませんか。
（能否请您在这里签名？）

ご確認していただけませんか。
（能否请您确认一下？）

お時間をいただけませんか。
（能否给我一些时间？）

もう一度説明していただけませんか。
（能否再解释一次？）
```

**礼貌度梯度示例（同一请求"帮我看一下"）**：

```
見て！（命令，粗鲁）
見て。（命令，亲密）
見てください。（请求，一般）
見てくれますか。（礼貌问句）
見てもらえますか。（更礼貌）
見ていただけますか。（对上位者，礼貌）
見ていただけませんか。（最礼貌，正式场合）
```

## 16.6 授受表达的否定形式与特殊用法

### ～てくれない？（亲密关系的请求）

这是亲密关系中非常自然的请求方式：

```
ちょっと手伝ってくれない？
（能帮我一下吗？）← 亲密，向上的语气

宿題、見せてくれない？
（让我看一下作业？）← 轻松的请求

一緒に来てくれない？
（能陪我一起来吗？）← 带着期待
```

### ～てもらいたい（希望对方做）

表达说话者希望对方采取某行动：

```
あなたに正直に話してもらいたい。
（我希望你能坦诚地说。）

もっと早く来てもらいたかった。
（我本希望你早点来的。）← 过去的希望/遗憾

彼には責任を取ってもらいたい。
（我希望他能负起责任来。）
```

### 否定形式的使用

```
【～てほしくない / ～てもらいたくない】
来てほしくない。（不希望你来。）
そんなことをしてもらいたくない。（不希望你做那种事。）

【～てくれない（不给我做）vs ～てくれないで（别为我做/不给……）】
どうして教えてくれないの？（你为什么不教我？）← 遗憾/抱怨
そんなことを言わないでくれ。（别说那种话。）← 请求不做某事
```

## 16.7 授受表达选择决策树

### 第一步：是物品授受还是动作授受？

```
【物品授受】→ あげる/もらう/くれる + 名詞
【動作授受】→ てあげる/てもらう/てくれる + 動詞
```

### 第二步：谁给谁？方向是什么？

```
我 → 他人（非我方）：あげる / てあげる
他人 → 我：もらう / てもらう（我的视角）
        くれる / てくれる（他人为主语，强调他的行为）
```

### 第三步：上下关系？

```
我 → 上位者：さしあげる / ていただく（接受时）
上位者 → 我：くださる / ていただく（我接受）、てくださる
平辈之间：あげる / もらう / くれる（基本形）
```

### 完整决策图

```
有恩惠流动?
├─ YES
│   ├─ 我给（施予）?
│   │   ├─ 给上位者 → さしあげる / てさしあげる
│   │   ├─ 给平辈/下位者 → あげる / てあげる
│   │   └─ 给动植物 → やる
│   ├─ 我收（接受）?
│   │   ├─ 从上位者收 → いただく / ていただく
│   │   └─ 从平辈/下位者收 → もらう / てもらう
│   └─ 他人给我/我方?
│       ├─ 上位者给我 → くださる / てくださる
│       └─ 平辈/下位者给我 → くれる / てくれる
└─ NO → 普通动词即可
```

## 16.8 授受表达常见错误与练习20题

### 常见错误汇总

**错误1：あげる用于"别人给我"**
```
✗ 田中さんは私に本をあげた。（方向错误）
○ 田中さんは私に本をくれた。
```

**错误2：くれる用于"我给别人"**
```
✗ 私は田中さんにプレゼントをくれた。
○ 私は田中さんにプレゼントをあげた。
```

**错误3：混淆もらう与くれる的主语**
```
✗ 田中さんは私に教えてもらった。（田中先生收到了我的教导？）
○ 私は田中さんに教えてもらった。（我接受了田中先生的教导）
○ 田中さんは私に教えてくれた。（田中先生给我教了）
```

**错误4：对上位者用てあげる**
```
✗ 先生の荷物を持ってあげます。（居高临下感）
○ 先生の荷物をお持ちします。
○ 先生の荷物を持ちましょうか。
```

**错误5：さしあげる被滥用**
```
✗ ご説明をさしあげます。（过度客套，现代日语少用）
○ ご説明します。（更自然）
○ ご説明いたします。（谦让语，更好）
```

### 综合练习20题

**1-5：从括号中选择正确的动词**

1. 私は弟に本を（あげた / くれた）。
2. 先生が私にアドバイスを（あげた / くれた）。
3. 誕生日に彼女にプレゼントを（もらった / くれた）。
4. お客様にお茶を（さしあげ / いただき）ます。
5. 社長から貴重なお言葉を（さしあげ / いただき）ました。

**6-10：填入正确的授受动词（てあげる/てもらう/てくれる）**

6. 友達が空港まで送っ（　　　）。とても助かった。
7. 田中さんに翻訳し（　　　）ことにした。
8. 困っている人を見かけたら、助け（　　　）と思います。
9. 先生に論文を読ん（　　　）ことができますか。
10. 彼女がずっと支え（　　　）から、頑張れた。

**11-15：改成正确的敬语形式**

11. 先生に教えてもらった。→（最高敬意の形に）
12. 社長にプレゼントをあげた。→（谦让形に）
13. 田中先生が手伝ってくれた。→（尊敬形に）
14. 確認してもらえますか。→（最も丁寧な形に）
15. 少し待ってください。→（上司への依頼に）

**16-20：翻译练习**

16. 我朋友教了我日语。（用てくれる）
17. 能请您稍等一下吗？（用ていただけませんか）
18. 我想请老师帮我检查一下论文。（用ていただく）
19. 他为我唱了一首歌。（用てくれる）
20. 我打算让她帮我做这件事。（用てもらう）

**练习答案（1-10）**

1. あげた　2. くれた　3. もらった　4. さしあげ　5. いただき
6. てくれた　7. てもらう　8. てあげたい　9. でもらう　10. てくれた

## 练习题（综合）

### 练习一：改错题

请找出并改正以下句子中的错误：

1. ✗ 先生は私に辞書を「あげた」。
2. ✗ 私は先生の荷物を持って「あげました」。（対先生）
3. ✗ 田中さんに教えて「くれました」か。（田中さん＝受け手）
4. ✗ 社長から直接ご連絡を「もらった」。（社長は上位者）

## 本章小结

日语授受表达体系是整个日语学习中最能体现**文化语言学**深度的领域：

| 核心动词 | 方向 | 含义 | 敬语对应 |
|---------|------|------|---------|
| あげる | I/他 → 他 | 给予（不朝我方） | さしあげる |
| もらう | 他 → I | 我收到 | いただく |
| くれる | 他 → I（向我方） | 他给我 | くださる |
| やる | I → 下位者 | 给下位/动植物 | — |

**三大动作授受形式的语感差异**：

| 形式 | 语感 | 适用场合 |
|------|------|---------|
| てあげる | 我施予恩惠 | 谨慎使用，避免居高临下感 |
| てもらう | 我接受恩惠 | 最安全的请求/感谢方式 |
| てくれる | 他的恩惠流向我 | 感谢、温暖，以他人为主语 |

**学习建议**：

1. 先牢固掌握基本三词：あげる/もらう/くれる的方向性——这是整个体系的基础
2. 理解"恩惠的流动方向"而非机械记忆——所有变形都遵循这个逻辑
3. 在实际对话中，`～てもらう` 系列是最安全的请求表达，多用少错
4. 敬语的使用要结合ウチ/ソト和上下关系——不能只看词汇，要看关系
5. 授受表达的文化背景（恩惠意识、ウチ/ソト）是理解这一体系的钥匙

掌握了授受表达，不仅仅是学会了语法，更是学会了**如何用日语感知人际关系、表达情感与立场**。这正是日语授受体系的魅力所在：语言即人际关系的镜子。
