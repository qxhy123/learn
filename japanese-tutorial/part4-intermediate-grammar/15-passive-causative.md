# 第15章：被动态与使役态

> 「日本語の受身は、英語の受身とは根本的に違う」——日语的被动态与英语被动态有着根本的不同。日语的被动，尤其是"迷惑の受身"，折射出日本人对自我与外界关系的独特感知：世界不只是客观的，它时时在影响"我"。

---

## 15.1 受身形（被动态）

### 构成规则

被动形（受身形）的构成规律性强，是日语变形中较为整齐的体系：

| 动词类型 | 变化规则 | 例（原形→受身形） |
|---------|---------|----------------|
| 五段动词 | あ段 + れる | 書く→書かれる、読む→読まれる、飲む→飲まれる |
| 一段动词 | 語幹 + られる | 食べる→食べられる、見る→見られる |
| する | される | する→される |
| 来る（くる） | 来られる（こられる） | 来る→来られる |

**五段动词受身形的系统展示：**

```
う行動詞：
買う → 買われる
書く → 書かれる
急ぐ → 急がれる
話す → 話される
持つ → 持たれる
死ぬ → 死なれる
呼ぶ → 呼ばれる
飲む → 飲まれる
切る → 切られる

一段動詞：
食べる → 食べられる
起きる → 起きられる
開ける → 開けられる
見る → 見られる
```

**注意**：一段动词的受身形（食べられる）与可能形（食べられる）形态完全相同，区分依靠语境。

### 三种被动类型

#### 类型一：直接被动（他動詞の受身）

最基本的被动，主语是动作的承受者：

```
【主動文】→ 【受身文】
先生が学生をほめた。→ 学生が先生にほめられた。
（老师表扬了学生。→ 学生被老师表扬了。）

警察が犯人を捕まえた。→ 犯人が警察に捕まえられた。
（警察逮捕了罪犯。→ 罪犯被警察逮捕了。）

みんなが彼の作品を評価した。→ 彼の作品がみんなに評価された。
（大家评价了他的作品。→ 他的作品被大家评价了。）
```

助词规则：
- 主动句的を → 被动句的が（主语）
- 主动句的が（施动者）→ 被动句的に

#### 类型二：间接被动 / 迷惑の受身（日语独有！）

这是日语被动态最具特色的用法，在其他语言中几乎找不到对应。**自动词也可以变成被动形**，主语受到某动作/事件的间接影响（通常是不利影响）。

核心逻辑：**"我（受害者）受到了某事的影响"**

```
【自動詞 + れる/られる（迷惑の受身）】

雨に降られた。
（我遭到了雨的袭击 / 被雨淋了。）
← 雨自己下（自动词），但"我"受害了

子供に泣かれた。
（孩子一直哭，让我很困扰。）
← 孩子哭（自动词），但说话人受害

電車の中で隣の人にタバコを吸われた。
（在电车里，被旁边的人抽烟（困扰了）。）
← 他人抽烟（他动词，但我是间接受害者）

隣の人に毎晩大きい音楽を聞かれた。
（每天晚上被邻居大声放音乐。）

友達に突然来られて、困った。
（朋友突然来了，让我很困扰。）

彼女に急に別れを告げられた。
（被女友突然分手了。）← 惊愕与受害感
```

**迷惑の受身的感情色彩对比：**

```
【普通陈述】雨が降った。（下雨了。）← 客观事实
【迷惑受身】雨に降られた。（被雨淋了。）← 我受害了

【普通陈述】子供が泣いた。（孩子哭了。）← 客观事实
【迷惑受身】子供に泣かれた。（孩子（不停）哭，让我头疼。）← 我受困扰
```

#### 类型三：所有者被动（部分被动）

主语是某个整体（人），动作施加在其身体部分或所有物上：

```
電車の中で足を踏まれた。
（在电车里脚被踩了。）
← 主语は「わたし」，踏まれたのは「足」

スリに財布を盗まれた。
（被扒手偷了钱包。）
← 主语は「わたし」，盗まれたのは「財布」

母に日記を読まれた。
（日记被妈妈看了。）← 侵犯隐私的受害感

彼女に手を握られた。
（被她握住了手。）← 可以是正面或负面
```

### 被动句的助词：に vs によって

| 助词 | 使用场合 | 例 |
|------|---------|---|
| に | 一般被动（人的行为，个人动作） | 先生に叱られた |
| によって | 正式文体，历史事件，被动性强的行为，可以是抽象力量 | 彼の発明によって世界が変わった |

```
【に 的例】
彼女に愛されている。（被她爱着。）
友達にケーキを食べられた。（蛋糕被朋友吃了。）

【によって の例】
この作品は田中氏によって書かれた。
（这部作品是由田中先生写的。）

法律によって禁止されている。
（被法律禁止。）← 抽象力量

地震によって多くの建物が倒壊した。
（由于地震，许多建筑倒塌了。）← 自然力量
```

### 被动在学术/新闻中的使用

日语被动态大量用于学术论文和新闻报道中，以达到**客观性**和**省略施动者**的效果：

```
【学術論文】
この研究では、以下の方法が用いられた。
（本研究中，采用了以下方法。）

データは三つのグループに分けられた。
（数据被分为三组。）

以上の結果から、次のことが明らかにされた。
（根据以上结果，以下事项得以明确。）

【ニュース】
首相が記者会見を開いた → 記者会見が首相によって開かれた。
昨日、銀行が強盗に襲われた。
（昨天，银行遭到了劫匪袭击。）
```

## 15.2 使役形（使役态）

使役形表达"让/使某人做某事"，是表达命令、许可、感情驱动的重要语法形式。

### 构成规则

| 动词类型 | 变化规则 | 例 |
|---------|---------|---|
| 五段动词 | あ段 + せる | 書く→書かせる、読む→読ませる |
| 一段动词 | 語幹 + させる | 食べる→食べさせる、見る→見させる |
| する | させる | する→させる |
| 来る | 来させる（こさせる） | 来る→来させる |

```
五段動詞の使役形：
書く → 書かせる
飲む → 飲ませる
行く → 行かせる
話す → 話させる
待つ → 待たせる
遊ぶ → 遊ばせる
切る → 切らせる

一段動詞の使役形：
食べる → 食べさせる
着る → 着させる
見る → 見させる
起きる → 起きさせる
```

### 三种使役含义

#### 含义一：强制使役（命令/强迫）

```
先生は生徒に作文を書かせた。
（老师让学生写了作文。）← 强制要求

親は子供に野菜を食べさせた。
（父母让孩子吃了蔬菜。）← 强迫

上司は部下を残業させた。
（上司让下属加班了。）← 权力关系
```

#### 含义二：许可使役（允许/放任）

```
子供を自由に遊ばせてあげた。
（让孩子自由地玩了。）← 允许

好きなことをさせてあげましょう。
（让他做喜欢的事吧。）← 给予自由

彼女の言う通りにさせておいた。
（随她说的做了。）← 放任
```

#### 含义三：感情使役（引发感情）

```
心配させてごめんね。
（让你担心了，对不起。）

彼女を泣かせてしまった。
（让她哭了。）← 无意引发的情感

あなたが笑わせてくれる。
（是你让我笑的。）← 正面情感

その知らせは彼を絶望させた。
（那个消息让他陷入了绝望。）
```

### 使役句的助词：を vs に

这是使役态中最重要的语法规则：

| 原动词类型 | 使役句中的助词 | 理由 |
|----------|-------------|------|
| 自動詞 | を | 自动词没有宾语，使役对象用を |
| 他動詞 | に | 他动词有自己的宾语（を），使役对象用に |

```
【自動詞 + を】
行く（自動詞）→ 子供を行かせる（让孩子去）
走る（自動詞）→ 選手を走らせる（让选手跑）
立つ（自動詞）→ 学生を立たせる（让学生站起来）
泣く（自動詞）→ 子供を泣かせた（让孩子哭了）

【他動詞 + に + を】
食べる（他動詞）→ 子供に野菜を食べさせる（让孩子吃蔬菜）
書く（他動詞）→ 学生に作文を書かせる（让学生写作文）
飲む（他動詞）→ 患者に薬を飲ませる（让患者服药）
読む（他動詞）→ 生徒に本を読ませる（让学生读书）
```

**记忆口诀**：
- 自动词：**を**（动作对象即为使役对象）
- 他动词：**に**（他动词已有を，腾出位置给を的宾语）

## 15.3 使役受身形（使役被动态）

使役受身形是日语语法中最能体现**被迫感**和**日本人的受害意识**的表达形式。它将使役（让别人做）与被动（被……）结合在一起：**"我被迫做了某事"**。

### 构成规则

```
使役形 → 受身形
（させる → させられる）

【一段動詞・する・来る】
食べさせる → 食べさせられる（被迫吃）
着させる → 着させられる（被迫穿）
させる → させられる（被迫做）
来させる → 来させられる（被迫来）

【五段動詞（通常形）】
書かせる → 書かせられる（被迫写）
飲ませる → 飲ませられる（被迫喝）
行かせる → 行かせられる（被迫去）
```

### 五段动词的口语缩约形

五段动词有一种特殊的**缩约形**，在口语中极为常见：

| 原形 | 完整使役受身形 | 缩约形 | 规则 |
|------|------------|-------|------|
| 飲む | 飲ませられる | 飲まされる | ませられる→まされる |
| 書く | 書かせられる | 書かされる | かせられる→かされる |
| 行く | 行かせられる | 行かされる | かせられる→かされる |
| 待つ | 待たせられる | 待たされる | たせられる→たされる |
| 話す | 話させられる | 話させられる（不缩约） | す行例外 |

**注意**：す行五段动词（話す等）不进行缩约，す行保持完整形式。

### 核心含义与场景

使役受身形的核心语感：**"我不愿意，但被迫/不得不做了某事"**，充满无奈、怨恨、疲惫的情感色彩。

```
【工作场景】
上司に毎日残業させられた。
（每天被上司迫着加班。）← 怨恨、无奈

会議で意味のない報告を何時間もさせられた。
（在会议上被迫做了好几个小时毫无意义的报告。）

部長の代わりに謝らされた。
（被迫代替部长去道歉了。）

【家庭/学校场景】
子供の頃、嫌いな野菜を食べさせられた。
（小时候被迫吃不喜欢的蔬菜。）

先生に廊下に立たされた。
（被老师罚站在走廊。）

親に毎日ピアノを練習させられた。
（每天被父母逼着练钢琴。）

【社会场景】
日本では、飲み会に参加させられることが多い。
（在日本，经常被迫参加酒会。）

知らない間に、重要な役割を引き受けさせられていた。
（在不知不觉间，被迫承担了重要的角色。）

長い列に並ばされた挙句、売り切れだった。
（排了长队之后，竟然卖光了。）← 双重受害

【情感类】
その映画に何度も泣かされた。
（那部电影让我哭了好多次。）← 被感动/被迫流泪

彼の話に笑わされた。
（被他的话逗笑了。）
```

## 15.4 受身/使役/使役受身的助词与语义对照总表

| 形式 | 助词型 | 含义 | 感情色彩 | 典型例 |
|------|-------|------|---------|-------|
| 受身（直接） | Aに Bされる | B被A做 | 中性/负面 | 先生に褒められた |
| 受身（迷惑） | Aに ～される | 因A的行为受害 | 负面 | 雨に降られた |
| 受身（所有者） | AにBをされる | B（所有物）被A做 | 负面 | 財布を盗まれた |
| 使役（強制） | AをB-させる | 迫使B做 | 权威感 | 子供を行かせた |
| 使役（許可） | AをB-させる | 允许B做 | 温和感 | 自由にさせた |
| 使役（感情） | AをB-させる | 使B产生情感 | 中性 | 泣かせた |
| 使役受身 | AにB-させられる | 被A迫使做B | 强烈负面 | 残業させられた |

## 15.5 敬语中的被动形（尊敬の受身）

日语中有一种特殊用法：**用被动形表达尊敬语**，称为"尊敬の受身"或"受身形の尊敬語"。

### 构成与含义

被动形 = 尊敬形（仅用于上位者的行为）：

```
先生が話された。（先生讲了。）← 非被动，是尊敬语
社長がいらっしゃられた。（不正确，双重敬语）
部長が出発された。（部长出发了。）← 尊敬语的受身形

【更多例子】
学長がご挨拶をされた。
（校长致了辞。）

先生がこの本を書かれた。
（这本书是先生写的。）

田中先生が研究を始められた。
（田中先生开始了研究。）
```

### 尊敬の受身 vs 尊敬専用語

| 敬意等级 | 形式 | 例 |
|---------|------|---|
| 中等尊敬 | 被动形 | 先生が話された |
| 高等尊敬 | 尊敬专用动词 | 先生がおっしゃった |
| 最高尊敬 | お～になる | 先生がお話しになった |

在正式场合，应优先选择尊敬专用动词或 `お～になる` 形式，被动形的尊敬语是最低一级的尊敬表达。

## 15.6 中国学习者的常见错误与辨析

### 常见错误1：混淆受身与可能形

一段动词的受身形和可能形形态完全相同：

```
食べられる → 受身（被吃）or 可能（能吃）？
見られる → 受身（被看）or 可能（能看）？

【辨析方法：看主语和语境】
ケーキが弟に食べられた。（蛋糕被弟弟吃了。）← 受身
このケーキは私には食べられない。（这个蛋糕我吃不下。）← 可能

目撃者に見られてしまった。（被目击者看到了。）← 受身
そこからは富士山が見られる。（从那里能看到富士山。）← 可能
```

### 常见错误2：使役助词を/に搞错

```
✗ 先生は生徒を本を読ませた。
○ 先生は生徒に本を読ませた。（他動詞なのでに）

✗ お母さんは子供に泣かせた。
○ お母さんは子供を泣かせた。（自動詞なのでを）
```

### 常见错误3：迷惑受身的概念缺失

```
✗ 雨が私を降った。（不存在）
○ 雨に降られた。（被雨淋了）

✗ 子供が私を泣いた。（不存在）
○ 子供に泣かれた。（孩子哭让我很困扰）
```

### 常见错误4：使役受身的助词

```
✗ 上司を残業させられた。
○ 上司に残業させられた。（被上司迫使加班，施令者用に）
```

## 练习题

### 练习一：变形练习

将以下句子改为受身形：

1. 先生が私をほめた。（→ 私が…）
2. 田中さんが窓を開けた。（→ 窓が…）
3. 突然、友達が家に来た。（→ 迷惑受身：私は友達に…）
4. 泥棒が財布を盗んだ。（→ 私は財布を…）

### 练习二：使役形练习

将以下句子改为使役形，注意助词选择：

1. 子供がご飯を食べる。（母親が…）
2. 社員が残業する。（社長が…）
3. 学生が教室を掃除する。（先生が…）
4. 犬が公園を走る。（私が…）

### 练习三：使役受身的练习

将以下情景用使役受身形表达（使用括号中的动词）：

1. 我被上司迫着写了两小时报告。（書く）
2. 小时候被父母迫着每天练小提琴。（練習する）
3. 被迫在众人面前发言，很紧张。（発表する）
4. 喝完了自己不喜欢的酒（被迫喝）。（飲む→缩约形）

### 练习四：综合辨析

以下句子各属于哪种类型（直接受身/迷惑受身/所有者受身/使役/使役受身）？请翻译并说明：

1. 雨に降られて、かばんがびしょびしょになった。
2. 先生は生徒に長い作文を書かせた。
3. 私は上司に毎週レポートを提出させられている。
4. 電車の中で知らない人に足を踏まれた。
5. 部長がこの件について話された。（是哪种特殊用法？）

## 本章小结

被动态与使役态构成日语语法体系中的两大"视角转换"机制：

| 形式 | 构成核心 | 语义核心 | 文化背景 |
|------|---------|---------|---------|
| 受身（直接） | あ段+れる/+られる | 主语承受动作 | 客观陈述/文体需要 |
| 受身（迷惑） | 自動詞も使える | 我受到影响（受害） | 日本人的被害意识 |
| 使役 | あ段+せる/+させる | 让/使他人做 | 权威关系的语言化 |
| 使役受身 | 使役形+受身形 | 被迫做某事 | 无奈与压迫感 |

**学习路径建议**：
1. 先牢固掌握受身形的构成——规律性强，是基础
2. 理解"迷惑の受身"——这是日语独有的语法现象，没有对应的中文或英文结构
3. 使役形的助词（を vs に）是考试高频考点，必须掌握
4. 使役受身形是体现语感的高级表达，注意五段动词的缩约形
5. 识别尊敬语中的被动形用法

被动态与使役态的核心，不仅是语法形式，更是**观察日本人如何感知自我与世界关系**的窗口。掌握了"迷惑の受身"和"使役受身"，就真正触摸到了日语表达的文化灵魂。
