# 第26章：各类文体写作

> 「文体は思想の衣装である」——文体是思想的衣裳。不同的写作目的需要不同的文体，掌握多种文体是日语写作进阶的核心。

---

## 引言

日语写作涵盖多种文体，从严谨的議論文到抒情的感想文，从正式的手紙到轻松的日記，每种文体都有其独特的结构与语言要求。本章将系统介绍日语写作中最常见的六种文体，通过结构分析与范文示例，帮助学习者掌握各类文体的写作技巧。

---

## 26.1 意見文（议论文/观点文）

意見文是日本高中、大学入学考试及日语能力考试中最重要的文体，要求写作者清晰表明立场并提供有力论据。

### 意見文的基本结构

```
問題提起（提出问题）
    ↓
自分の意見（表明立场）
    ↓
理由・根拠①（理由一）
    ↓
理由・根拠②（理由二）
    ↓
反論への対応（反驳对立观点）
    ↓
結論（重申结论）
```

### 主张表达句型

| 表达功能 | 句型 | 例句 |
|---------|------|------|
| 表明立场 | ～と考える | 環境保護が最優先だと考える。 |
| 应该做 | ～べきだ | 政府は対策を講じるべきだ。 |
| 委婉质疑 | ～のではないだろうか | 見直す必要があるのではないだろうか。 |
| 不应该做 | ～べきではない | 経済発展だけを優先すべきではない。 |
| 必须 | ～なければならない | 早急に行動しなければならない。 |
| 重要性 | ～が重要である | 教育への投資が重要である。 |
| 提出问题 | ～が問題となっている | 少子化問題が深刻化している。 |
| 现状陈述 | ～が現状だ | 対策が不十分なのが現状だ。 |

### 论据展开的层次结构

| 层次词 | 用法 | 例句 |
|-------|------|------|
| 第一に | 首先 | 第一に、経済的なメリットがある。 |
| 第二に | 其次 | 第二に、環境への影響が少ない。 |
| 第三に | 第三 | 第三に、社会的な公平性が保たれる。 |
| 最後に | 最后 | 最後に、長期的な視点で考えると～ |
| また | 另外 | また、健康面でのメリットもある。 |
| さらに | 更进一步 | さらに、国際的な協力も必要だ。 |

### 反论对应的表达

```
【承认对方观点但坚持己见】
確かに、～という意見もある。しかし、～
（固然，也有…的看法。但是，…）

【反驳对立观点】
しかし、この主張には問題がある。なぜなら～
（但是，这一主张存在问题。因为…）

【转弯但坚持立场】
もちろん、～面もあることは否定できない。
しかし、全体的に考えると、やはり～が重要である。
（当然，…的方面不可否认。但综合考虑，还是…更重要。）
```

### 意見文范文（约400字）

```
【题目：高校生のスマートフォン使用を制限すべきか】
（高中生应该限制使用智能手机吗？）

【起：問題提起】
近年、高校生のスマートフォン依存が社会問題となっている。授業中も
スマートフォンを使用し、学習に支障をきたすケースが増えている。
この問題に対し、学校はスマートフォンの使用を制限すべきか否か、
改めて考えたい。

【承：自分の意見＋理由】
私は、授業中のスマートフォン使用は制限すべきだと考える。
その理由として、第一に、集中力への悪影響が挙げられる。
カリフォルニア大学の研究によれば、スマートフォンの通知があるだけで
認知能力が低下することが示されている。第二に、睡眠の質の問題がある。
就寝前のスマートフォン使用はブルーライトの影響で睡眠を妨げ、
翌日の学習効率を下げる。

【転：反論への対応】
確かに、スマートフォンは学習ツールとしても有効であり、
辞書アプリや学習アプリを活用している生徒も多い。また、
過度な制限は生徒の自律性を損なうという意見もある。
しかし、学習目的の使用と娯楽目的の使用を区別した上で、
授業中は学習用途に限定するルールを設けることは可能である。

【結：結論】
以上のことから、スマートフォンの全面禁止ではなく、
使用目的とシーンに応じたルール作りが必要だと考える。
学校・生徒・保護者が協力して、スマートフォンと学習の
バランスを見つけることが、現代の教育現場に求められている。
（約390字）
```

**范文逐句解析：**

| 段落 | 功能 | 关键表达 |
|-----|------|---------|
| 起 | 提出问题，设定背景 | 「社会問題となっている」「改めて考えたい」 |
| 承 | 表明立场，提供两个论据 | 「第一に」「第二に」「～によれば」 |
| 転 | 承认对立观点，给予回应 | 「確かに～しかし」对比结构 |
| 結 | 总结立场，提出建议 | 「以上のことから」「～が必要だと考える」 |

## 26.2 説明文（说明文）

説明文要求客观地传达信息，避免主观判断，语言要准确、中立。

### 客观描述的语言选择

| 主观表达（避免） | 客观表达（使用） | 说明 |
|--------------|--------------|------|
| ～と思う | ～と考えられる | 推断 |
| ～はすばらしい | ～には～という特徴がある | 描述特征 |
| ～はひどい | ～には問題がある | 指出问题 |
| たぶん～だろう | ～の可能性がある | 可能性表达 |
| 絶対に～だ | ～であることが多い | 程度限定 |

### 数据与事实的引用方式

```
【引用统计数据】
総務省の調査（2023年）によると、日本のスマートフォン普及率は約90%である。
（根据总务省调查，2023年日本智能手机普及率约为90%。）

【引用研究结论】
東京大学の研究によれば、運動習慣は認知機能の向上に効果があることが
示されている。
（根据东京大学的研究，运动习惯对提升认知功能有效。）

【表述已知事实】
富士山は3776mの標高を誇り、日本最高峰として知られている。
（富士山海拔3776米，以日本最高峰而闻名。）
```

### 客观表述的核心句型

| 句型 | 含义 | 适用场合 |
|------|------|---------|
| ～とされている | 被认为是… | 引用通说 |
| ～と言われている | 据说… | 引用民间说法 |
| ～が報告されている | 有报告显示… | 引用研究报告 |
| ～が明らかになった | 已明确… | 研究结论 |
| ～と考えられる | 被认为… | 客观推断 |
| ～の可能性がある | 有…的可能性 | 谨慎推测 |
| ～傾向がある/見られる | 有…倾向 | 趋势描述 |
| ～に分類される | 被分类为… | 分类说明 |

### 説明文范文

```
【题目：日本の四季（日本的四季）】

日本は温帯気候に属し、春・夏・秋・冬の四季が明確に分かれることで
知られている。この気候的特徴は、日本の文化や生活習慣に深く影響を
与えてきた。

春（3月〜5月）は、気温が徐々に上昇し、桜（ソメイヨシノ）が全国で
開花する季節として知られている。気象庁のデータによれば、東京での
桜の開花時期は3月下旬から4月上旬であることが多い。この時期には
花見（お花見）という文化的行事が各地で行われる。

夏（6月〜8月）は高温多湿で、6月には梅雨（つゆ）と呼ばれる長雨の
季節があることが特徴だ。その後、7〜8月は猛暑となり、気温が35度を
超える「猛暑日」が増加する傾向が見られる。

秋（9月〜11月）は紅葉が各地の山々を彩る季節であり、観光客が増加する
ことが知られている。冬（12月〜2月）は、太平洋側では乾燥した晴天が
続く一方、日本海側では大量の雪が降ることが多い。

このように、日本の四季は変化に富んでおり、それぞれの季節に固有の
自然現象と文化的行事が存在する。（約350字）
```

## 26.3 感想文（读后感/观后感）

感想文是日本学校教育中最常见的写作类型，要求写作者结合个人体验表达真实感受。

### 感想文的基本结构

```
あらすじ（梗概/内容简介）
    ↓
印象に残った点（印象深刻的内容）
    ↓
自分との関連・共感点（与自身经历的联系）
    ↓
自分の考え・学んだこと（自己的想法/收获）
    ↓
まとめ（总结/未来展望）
```

### 感想表达核心句型

| 表达功能 | 句型 | 例句 |
|---------|------|------|
| 感动 | ～に感動した | 主人公の勇気に感動した。 |
| 感受 | ～と感じた | 読んでいて悲しいと感じた。 |
| 深思 | ～について考えさせられた | 友情の大切さについて考えさせられた。 |
| 共鸣 | ～に共感した | 主人公の孤独感に共感した。 |
| 学习 | ～を学んだ | 諦めない大切さを学んだ。 |
| 印象 | ～が印象に残った | このセリフが特に印象に残った。 |
| 联想 | ～を思い出した | 自分の経験を思い出した。 |
| 震撼 | ～に衝撃を受けた | ラストシーンに衝撃を受けた。 |
| 惊讶 | ～に驚いた | 事実を知って驚いた。 |
| 反思 | ～を反省させられた | 自分の行動を反省させられた。 |

### 感想文范文

```
【题目：夏目漱石「こころ」を読んで（读夏目漱石《心》有感）】

夏目漱石の「こころ」は、明治時代を背景に、人間の罪悪感と
友情・愛情の葛藤を描いた長編小説である。主人公「私」と謎めいた
「先生」との交流を軸に、人間の心の奥底にある孤独と秘密が
丁寧に描かれている。

この作品を読んで、最も印象に残ったのは「先生」の告白の部分だ。
親友への裏切りという取り返しのつかない過去を抱え、生涯をかけて
苦しみ続けた先生の姿に、深い悲しみを感じた。

「先生」の苦しみを読みながら、私は自分自身の経験を振り返った。
友人と些細なことで言い争い、謝るタイミングを失ったことがある。
あの時の後悔は今も消えていない。「先生」の苦しみは極端な例だが、
人は誰でも後悔を抱えているのではないだろうか。

この作品から、私は「誠実さ」と「勇気」の重要性を学んだ。
過ちを犯した時に正直に向き合う勇気こそが、のちの後悔を
少なくするのではないかと感じた。

「こころ」は百年以上前に書かれた作品だが、人間の心の本質は
変わらないと思わせる力がある。この作品は、私自身の生き方を
問い直す貴重な機会となった。（約380字）
```

## 26.4 物語・創作文（叙事文/创作文）

物語文（故事文）是日语写作中最具创造性的文体，需要掌握时间表达、场景描写与心理描写等技巧。

### 时间表达与场景描写

**时间推进的表达：**

| 时间表达 | 读法 | 用途 |
|---------|------|------|
| ある日 | あるひ | 某天（故事开头常用） |
| その翌日 | そのよくじつ | 第二天 |
| 数時間後 | すうじかんご | 数小时后 |
| しばらくして | しばらくして | 过了一会儿 |
| 気がつくと | きがつくと | 回过神来 |
| 夕暮れ時 | ゆうぐれどき | 傍晚时分 |
| それから数年が経った | — | 此后数年过去了 |
| 季節は変わり | — | 季节更迭 |

**场景描写的技法：**

```
【五感描写法 - 综合运用感官】

視覚（视觉）：
　朝霧に包まれた森は、幻想的な白に染まっていた。

聴覚（听觉）：
　遠くで川のせせらぎが聞こえ、鳥のさえずりが響いていた。

嗅覚（嗅觉）：
　雨上がりの土の匂いが、懐かしい記憶を呼び起こした。

触覚（触觉）：
　吹き抜ける風は、頬に冷たく心地よかった。

味覚（味觉）：
　母の作ったみそ汁は、いつも温かく、塩辛すぎるくらいだった。
```

### 会话文的写法（「」的使用）

```
【会话文格式规范】

基本格式：
「こんにちは」と田中さんが言った。
（「你好」，田中说。）

动词+会话内容：
田中さんは「早く行こう」と促した。

感情+动作：
「もう限界だ」と彼は呻くように言い、その場に座り込んだ。

省略动词（紧凑对话）：
「どこへ行くの？」
「図書館に」
「一人で？」
「うん、勉強したいから」

内心独白（自言自语）：
（どうして彼は来なかったのだろう）と私は思った。
或写为：
「どうして来なかったんだろう……」私は一人でつぶやいた。
```

### 心理描写的手法

| 手法 | 说明 | 例句 |
|-----|------|------|
| 直接描写 | 直接描写内心感受 | 胸が痛かった。彼女は悲しかった。 |
| 比喻描写 | 用比喻表达心情 | 心に穴が開いたようだった。 |
| 行为暗示 | 通过行为暗示心情 | 彼は黙ったまま窓の外を見つめた。 |
| 生理反应 | 通过身体反应表现心情 | 手が震え、足が動かなかった。 |
| 环境衬托 | 用环境氛围烘托心情 | 雨が激しく降り始め、彼の気持ちを映すようだった。 |
| 内心独白 | 展示内心的思考过程 | （なぜ、あの時何も言わなかったのだろう） |

```
【心理描写综合示例】

試験の結果を見た瞬間、田中は全身が固まったような気がした。
（试验结果的瞬间，田中感觉全身僵住了。——生理反应）

「……不合格」
その三文字が、現実のものとは思えなかった。
（"……不合格"，这三个字感觉不像真实的。——心理直白）

窓の外では、雨が静かに降り始めていた。
（窗外，雨悄悄地开始下了。——环境衬托）

（もっと頑張ればよかった）という後悔が、波のように押し寄せてきた。
（"早知道更努力了"，悔恨如浪涌来。——内心独白+比喻）
```

## 26.5 手紙文（书信）

日本书信有严格的格式规范，分为正式信（書き言葉を使った改まった手紙）和非正式信（友人・家族への手紙）。

### 正式书信的完整结构

```
【正式书信格式】

① 頭語（书信开头敬语）
   → 拝啓（はいけい）男女通用，最常用

② 時候の挨拶（时候问候）
   → 季节性问候语

③ 安否の挨拶（对对方问候）
   → ご清栄のこととお喜び申し上げます。

④ 自分の近況（自我近况）
   → 私どもも変わりなく過ごしております。

⑤ 本文（正文）
   → さて、このたびは～

⑥ 結びの挨拶（结束语）
   → どうかお体にお気をつけください。

⑦ 結語（结尾敬语）
   → 敬具（けいぐ）与拝啓对应

⑧ 後付け（日期与署名）
   → 令和○年○月○日
   → 山田太郎

⑨ 宛名（收件人）
   → 田中一郎様
```

### 時候の挨拶 - 12个月完整对照表

| 月份 | 候文 | 现代说法 | 含义 |
|-----|------|---------|------|
| 1月 | 厳寒の候 | 新春の候・初春の候 | 严寒时节 |
| 2月 | 余寒の候 | 立春の候 | 余寒时节 |
| 3月 | 早春の候 | 春分の候 | 早春时节 |
| 4月 | 春暖の候 | 陽春の候・花見の候 | 春暖时节 |
| 5月 | 薫風の候 | 新緑の候・立夏の候 | 薰风时节 |
| 6月 | 梅雨の候 | 向暑の候 | 梅雨时节 |
| 7月 | 盛夏の候 | 炎暑の候・大暑の候 | 盛夏时节 |
| 8月 | 残暑の候 | 晩夏の候 | 残暑时节 |
| 9月 | 初秋の候 | 新秋の候・涼秋の候 | 初秋时节 |
| 10月 | 秋冷の候 | 仲秋の候・紅葉の候 | 秋凉时节 |
| 11月 | 晩秋の候 | 深秋の候・向寒の候 | 晚秋时节 |
| 12月 | 師走の候 | 初冬の候・歳末の候 | 岁末时节 |

**师走（しわす）：** 12月的别称，意为"连师傅（僧侣）都在奔走"，形容年末忙碌。

### 頭語与結語的对应关系

| 頭語 | 結語 | 使用场合 |
|------|------|---------|
| 拝啓（はいけい） | 敬具（けいぐ） | 正式书信，最常用 |
| 謹啓（きんけい） | 謹言（きんげん） | 非常正式，对长辈 |
| 前略（ぜんりゃく） | 草々（そうそう） | 省略前文，较随意 |
| 急啓（きゅうけい） | 草々 | 紧急事项 |
| （无头语） | （无结语） | 友人等非正式书信 |

### 正式信与非正式信的区别对比

| 比较项目 | 正式信 | 非正式信（友人・家族） |
|---------|-------|-----------------|
| 頭語・結語 | 必须使用 | 不使用 |
| 時候の挨拶 | 必须（候文） | 可省略或用口语 |
| 敬称 | ～様 | ～へ、～ちゃん |
| 文体 | 書き言葉 | 話し言葉OK |
| 折り方 | 正式折叠方式 | 随意 |
| 封筒 | 专用信封 | 普通信封 |

### 年賀状（贺年卡）的写法

年賀状是日本新年最重要的传统习俗之一，通常在12月25日前投邮，1月1日送达。

```
【年賀状基本格式】

① 新年の挨拶（新年祝贺语）
   → 謹賀新年（きんがしんねん）最正式
   → あけましておめでとうございます 一般用
   → 賀正（がしょう）简略版（对晚辈）

② 旧年の感謝（对去年的感谢）
   → 旧年中は大変お世話になりました。

③ 新年の挨拶・願い（新年问候）
   → 本年もどうぞよろしくお願いいたします。

④ 一言添える（添加个人话语）
   → お近くにお越しの際はぜひお立ち寄りください。

⑤ 日付（日期）
   → 令和○年 元旦

【注意事项】
- 「元旦」只能写在1月1日，不能写1月2日后
- 「4」「9」等数字要避免（谐音为「死」「苦」）
- 年賀状不能写进黑框或使用深色墨水
- 不能使用一般邮票，需贴年賀切手（年贺邮票）
```

## 26.6 日記（日记）的写法与练习方法

日記是语言学习中最有效的写作练习方式之一，每天坚持书写能快速提升语言表达能力。

### 日記の基本结构

```
【日记写法示例】

令和6年4月10日（水）晴れ

今日は学校で日本語のスピーチコンテストがあった。
昨日まで何度も練習したのに、本番では緊張してしまい、
途中で言葉に詰まってしまった。

でも、先生が「よく頑張った」と言ってくれて、少し気持ちが楽になった。
クラスメートのマリさんのスピーチはとても上手で、流暢な日本語に
感動した。私ももっと練習しなければと思った。

明日からは毎日10分、声に出して読む練習を続けよう。
```

### 日语日记写作技巧

| 技巧 | 说明 | 例句 |
|------|------|------|
| 时间开头 | 从今天发生了什么开始 | 今日は～があった。 |
| 心情描写 | 记录情绪变化 | ～して、嬉しかった/悲しかった。 |
| 反思展望 | 记录学到的内容 | ～を学んだ/～しようと思った。 |
| 目标设定 | 写下明天的计划 | 明日は～をしよう。 |
| 疑问记录 | 记录不懂的日语表达 | 「～」という表現はどう使うのだろう。 |

### 日记用语速查表

| 天气 | 写法 | 日记常用表达 | 意思 |
|------|------|------------|------|
| 晴れ | はれ | 今日は～した | 今天做了… |
| 曇り | くもり | ～ていた | 一直在… |
| 雨 | あめ | ～てしまった | 不小心… |
| 雪 | ゆき | ～ようと思う | 打算… |
| 台風 | たいふう | ～かもしれない | 也许… |

**日记学习法的优点：**
1. 用真实经历写作，材料随手可得
2. 每天少量积累，避免心理负担
3. 可以复习，发现自己的错误模式
4. 日记内容私密，无需担心被人评价
5. 记录语言学习进步，增加成就感

### 练习一：意見文写作

以「大学生はアルバイトをすべきか（大学生应该打工吗？）」为题，运用本章学到的意見文结构，写一篇300字左右的文章，包含：

## 本章小结

| 文体 | 核心结构 | 关键表达 | 适用场合 |
|-----|---------|---------|---------|
| 意見文 | 问题提起→主张→理由→反论→结论 | ～べきだ/第一に～ | 考试・论述 |
| 説明文 | 概念→特征→数据→总结 | ～とされている | 介绍・报告 |
| 感想文 | 梗概→印象点→感受→收获→总结 | ～に感動した | 读后感 |
| 物語文 | 场景→发展→高潮→结尾 | 感官描写+心理描写 | 创作 |
| 手紙文 | 頭語→時候→安否→正文→結語 | 拝啓/敬具 | 书信 |
| 日記 | 日期→事件→感受→展望 | 今日は～した | 日常练习 |

**本章学习要点：**

1. 意見文是考试中最重要的文体，「確かに～しかし」结构是处理反论的关键。
2. 説明文要保持语言的客观性，善用「～とされている」等客观表述句型。
3. 感想文不仅要写"感动了"，更要写"为什么感动"和"联系自己的体验"。
4. 物語文的生命力在于细节——五感描写、心理描写让文章生动立体。
5. 手紙文的時候の挨拶是展示日语文化素养的重要环节，12个月的表达要熟记。
6. 每天坚持写日记是提升写作能力性价比最高的练习方法。

> 「練習なくして上達なし」——没有练习就没有进步。无论哪种文体，唯有大量写作实践，方能真正掌握日语写作之道。
