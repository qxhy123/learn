# 第25章：作文基础与段落结构

> 「文は人なり」——文章即人，写作是思想的镜子。掌握日语写作，不仅是语言技能的提升，更是逻辑思维与文化理解的深化。

---

## 引言

日语写作与中文、英文写作既有相似之处，也存在独特的规范与习惯。从稿纸的使用到标点符号的摆放，从话し言葉到書き言葉的转换，每一个细节都体现着日本书写文化的严谨性。本章将系统讲解日语写作的基础知识，帮助学习者建立扎实的写作规范意识。

---

## 25.1 日语写作的基本规范

### 原稿用紙（稿纸）的使用规则

**原稿用紙**（げんこうようし）是日本传统的作文专用格纸，通常为20×20格（400字）或20×10格（200字）。在日本的学校教育与文学创作中，原稿用纸至今仍广泛使用，掌握其规范是理解日语写作文化的基础。

**原稿用紙的基本格式：**

| 规则项目 | 具体规范 | 说明 |
|---------|---------|------|
| 段首空格 | 每段开头空1格 | 与中文习惯相同 |
| 句点（。）位置 | 格内靠右下角 | 不独占一格 |
| 读点（、）位置 | 格内靠右下角 | 不独占一格 |
| 行末标点 | 可以写在行末格内 | 避免独立成行 |
| 括号开（「） | 可在行末 | 但关括号）不在行首 |
| 数字写法 | 竖排用汉字数字 | 一、二、三… |
| 数字写法 | 横排用阿拉伯数字 | 1、2、3… |
| 外来语 | 用片假名书写 | コンピュータ等 |
| 省略号 | 「……」占2格 | 每个「…」占1格 |
| 破折号 | 「——」占2格 | 每个「—」占1格 |

**标点符号的格内位置：**

在原稿用纸中，标点符号的位置有严格规定：
- 句点「。」和读点「、」写在格子的右下角，与文字共占一格
- 感叹号「！」和问号「？」独占一格，其后通常空一格
- 引号「」占格规则：开括号写在格子右半部分，关括号写在格子左半部分

**特殊情况处理：**

```
【竖排稿纸中的特殊符号处理】

长音符「ー」（片假名中）：竖排时改写为「｜」
小写假名「っ」「ゃ」等：写在格内右上角
数字连续出现：两位数字可写在一格内（如「25」）
```

### 縦書き（竖排）vs 横書き（横排）

| 比较项目 | 縦書き（たてがき） | 横書き（よこがき） |
|---------|----------------|----------------|
| 书写方向 | 从上到下，从右到左 | 从左到右，从上到下 |
| 适用场合 | 文学作品、小说、信件、报纸 | 学术论文、商务文件、教科书 |
| 数字写法 | 汉字数字（一、二、三） | 阿拉伯数字（1、2、3） |
| 外来语 | 片假名竖排 | 片假名横排 |
| 符号差异 | 「」→竖排专用 | ""→有时用于英文 |
| 文化含义 | 传统、正式、文学性 | 现代、实用、国际化 |

现代日本的学校作文和考试写作多采用横書き，但了解縦書き的规范对于阅读日本文学原著和理解传统文化非常重要。

### 常体（だ/である体）vs 敬体（です/ます体）在写作中的选择

这是日语写作中最重要的文体选择之一，两种文体不可混用。

| 比较项目 | 常体（だ・である体） | 敬体（です・ます体） |
|---------|-----------------|-----------------|
| 别称 | 普通体、简体 | 丁寧体 |
| 用途 | 学术论文、日记、小说、报告 | 手紙、一般作文、学校作业 |
| 语气 | 客观、正式、有距离感 | 礼貌、亲切、有温度 |
| 结尾形式 | ～だ/～である/～した | ～です/～ます/～ました |
| 疑问句 | ～だろうか/～ではないか | ～でしょうか/～ませんか |
| 适合人群 | 专业写作者、学者 | 初学者、一般场合 |

**示例对比：**

```
【敬体】
私は毎朝6時に起きます。朝ごはんを食べてから、学校へ行きます。
授業は楽しいです。友達もたくさんできました。

【常体】
私は毎朝6時に起きる。朝ごはんを食べてから、学校へ行く。
授業は楽しい。友達もたくさんできた。

【である体（学術）】
本稿では、現代日本語における敬語表現の変化について考察する。
従来の研究においては、敬語は社会階層を反映するものであるとされてきた。
```

**重要原则：** 一篇文章中只能使用一种文体，绝不能混用。这是日语写作中最常见的错误之一。

### 書き言葉 vs 話し言葉 对照表

書き言葉（书面语）与話し言葉（口语）在日语中有明显区别，写作时必须使用書き言葉。

| 話し言葉（口语） | 書き言葉（书面语） | 含义 |
|--------------|--------------|------|
| じゃない | ではない | 不是… |
| だから | したがって／そのため | 因此 |
| でも | しかし／しかしながら | 但是 |
| やっぱり | やはり | 果然/还是 |
| ちょっと | 少し | 稍微 |
| すごく | 非常に／とても | 非常 |
| なんか | など | 等等 |
| けど | けれども／が | 但是 |
| いっぱい | たくさん | 很多 |
| ちゃんと | きちんと | 好好地 |
| どんどん | 次々と／順調に | 不断地 |
| もっと | さらに | 更加 |
| なんで | なぜ／どうして | 为什么 |
| だって | なぜなら／というのも | 因为 |
| ～って | ～と（いう） | 说…/叫… |
| ～たり～たりする | ～たり～たりする | （正式场合相同） |
| めっちゃ | 非常に／大変 | 非常 |
| うまく | うまく→適切に | 适当地 |
| 全然（+肯定） | （避免使用） | 完全（避免） |
| ～みたいな | ～のような | 像…一样 |
| ってことは | つまり／ということは | 也就是说 |
| ～とか | ～や～など | …和…等 |
| もちろん | もちろん→当然 | 当然 |
| ～じゃなくて | ～ではなく | 不是…而是 |
| 絶対 | 必ず | 必定 |
| ほんと(に) | 本当に | 真的 |
| わかんない | わからない | 不懂 |
| やつ | 人/もの | 那个人/东西 |
| ～してる | ～している | 正在做… |
| ちゃう | ～てしまう | 不小心做了 |
| ～かなと思って | ～と考え | 认为… |
| 気をつけて | 注意して | 注意 |
| ～なんだけど | ～のだが | 是…但 |
| あのね | （不使用） | 那个… |
| やばい | 大変だ/困難だ | 糟糕/厉害 |
| どっちにしろ | いずれにせよ | 无论如何 |

## 25.2 日语段落（段落/パラグラフ）的构成

日语的段落结构有其独特性，理解这一结构是写出逻辑清晰文章的关键。

### 话题句（トピックセンテンス）

每个段落的第一句，明确表达本段的中心主题。

```
【例】
現代の若者のコミュニケーション方法は大きく変化している。
→ 这句话就是话题句，明确了本段要讨论"当代年轻人的沟通方式发生了巨大变化"。
```

**话题句的特征：**
- 简洁明确，通常为一句话
- 包含段落的核心观点
- 后续内容均围绕此句展开

### 支撑句（サポーティングセンテンス）

用具体的事实、数据、例子来支撑和证明话题句的内容。

```
【例（接续上方话题句）】
スマートフォンの普及により、直接会って話すよりもSNSやメッセージアプリを
通じたやり取りが主流になっている。総務省の調査によると、20代の9割以上が
毎日スマートフォンでコミュニケーションをとっているという。
```

### 总结句

段落末尾，对本段内容进行总结，或与下一段形成过渡。

```
【例】
このように、デジタル技術の発展は若者のコミュニケーションのあり方を
根本から変えつつある。
```

### 日语段落与英语段落的差异：起承転結 vs Introduction-Body-Conclusion

| 比较维度 | 日语（起承転結） | 英语（Introduction-Body-Conclusion） |
|---------|--------------|-------------------------------------|
| 结论位置 | 文章末尾 | 开头即明确（先结论后论证） |
| 论证方式 | 渐进式、迂回式 | 直接式、演绎式 |
| 転（转折） | 日语特有，引入新视角 | 通常没有此结构 |
| 文化背景 | 含蓄、间接表达 | 直接、清晰表达 |
| 适用文化 | 东亚文化圈 | 欧美文化圈 |

## 25.3 起承転結（起承转合）结构

起承転結是日本传统的文章结构，源自中国诗歌的四段式结构，在日语作文中广泛运用。

### 起（き）：导入话题

引出文章主题，吸引读者兴趣，设定写作背景。

**常用句型：**
- 近年、～という問題が注目されている。（近年，…这一问题备受关注。）
- ～について、あなたはどう思うだろうか。（关于…，你是怎么想的？）
- 私は幼いころから～に興味を持っていた。（我从小就对…感兴趣。）

### 承（しょう）：展开论述

承接"起"的内容，进一步说明、展开、提供背景信息和论据。

**常用句型：**
- たとえば、～という事例がある。（例如，有…这样的案例。）
- ～によれば、～とのことだ。（根据…，据说…。）
- 具体的に言うと、～（具体来说，…）

### 転（てん）：转折/新视角

这是日语文章结构中最独特的部分，引入一个新的视角、反例或转折，使文章更有深度。

**常用句型：**
- しかし、一方で～という考え方もある。（但是，另一方面也有…的看法。）
- ところが、現実はそれほど単純ではない。（然而，现实并非那么简单。）
- これに対して、異なる意見も存在する。（对此，也存在不同的意见。）

### 結（けつ）：总结收束

总结全文观点，表明自己的立场，或展望未来。

**常用句型：**
- 以上のことを踏まえると、～と言えるだろう。（综上所述，可以说…。）
- このように考えると、～が重要であることがわかる。（这样思考的话，可以理解…的重要性。）
- 私は今後も～を続けていきたいと思う。（我今后也想继续…。）

### 范文分析

```
【题目：スマートフォンと現代生活（智能手机与现代生活）】

【起】
現代社会において、スマートフォンはなくてはならない存在となっている。
電車の中でも、カフェでも、多くの人が画面を見つめている光景は、
今や珍しくもない。

【承】
スマートフォンは私たちの生活を大きく便利にした。地図アプリのおかげで
道に迷うことが減り、オンラインショッピングで外出せずに買い物ができる。
また、SNSを通じて遠くにいる友人とも気軽に連絡が取れる。

【転】
しかし、便利さの一方で、問題も生じている。スマートフォンへの依存が
深まるにつれ、直接的な人間関係が希薄になるという指摘がある。
また、長時間の使用による視力低下や睡眠障害も報告されている。

【結】
スマートフォンは確かに現代生活に欠かせないツールである。しかし、
その使い方を意識的にコントロールすることが、充実した生活を送る
ために必要ではないだろうか。道具に使われるのではなく、道具を
使いこなす姿勢が、これからの時代に求められていると思う。
```

**范文逐段解析：**

| 段落 | 结构要素 | 功能 | 使用技法 |
|-----|---------|------|---------|
| 起 | 引入话题 | 描述现象，引发共鸣 | 具体场景描写 |
| 承 | 展开论述 | 列举智能手机的优点 | 三点式论据（地图/购物/SNS） |
| 転 | 引入转折 | 提出问题与负面影响 | しかし引导，对比手法 |
| 結 | 总结收束 | 提出建议，升华主题 | 反问句+哲理性结语 |

## 25.4 接续表达在写作中的运用

接续词（接続詞）是连接句子和段落的"粘合剂"，正确使用接续词能使文章逻辑更清晰。

### 顺接（前后顺承关系）

| 接续词 | 用法 | 例句 |
|-------|------|------|
| そして | 并列/顺序 | そして、次の日も練習を続けた。 |
| それから | 时间顺序 | 宿題をした。それから、テレビを見た。 |
| また | 追加/并列 | 読書も好きだ。また、映画鑑賞も楽しむ。 |
| さらに | 进一步递进 | さらに、研究を深めると～がわかった。 |
| その上 | 叠加补充 | 成績が上がった。その上、友達も増えた。 |
| しかも | 意外的追加 | 安くて美味しい。しかも、健康にもいい。 |

### 逆接（前后转折关系）

| 接续词 | 用法 | 例句 |
|-------|------|------|
| しかし | 一般转折 | 努力した。しかし、結果は出なかった。 |
| しかしながら | 书面语转折 | しかしながら、問題は残されている。 |
| ところが | 意外的转折 | 急いだ。ところが、電車は出たあとだった。 |
| 一方（で） | 对比/另一方面 | 一方で、反対意見もある。 |
| それでも | 尽管如此 | 困難だった。それでも、諦めなかった。 |
| とはいえ | 虽说如此 | 便利だ。とはいえ、問題もある。 |

### 换言（换一种说法）

| 接续词 | 用法 | 例句 |
|-------|------|------|
| つまり | 简而言之 | つまり、努力が大切だということだ。 |
| すなわち | 也就是说（书面语） | すなわち、この結果は～を示している。 |
| 要するに | 总而言之 | 要するに、準備不足だったのだ。 |
| 言い換えれば | 换言之 | 言い換えれば、時間の無駄だった。 |
| 別の言い方をすれば | 换一种说法 | 別の言い方をすれば、効率が悪い。 |

### 举例（引出例子）

| 接续词 | 用法 | 例句 |
|-------|------|------|
| 例えば | 例如 | 例えば、日本語の敬語は複雑だ。 |
| 具体的には | 具体而言 | 具体的には、次の3点が挙げられる。 |
| たとえば | 例如（平假名形式） | たとえば、東京の地下鉄は複雑だ。 |
| 特に | 特别是 | 特に、発音の問題が難しい。 |
| とりわけ | 尤其是（书面语） | とりわけ、漢字の習得が課題だ。 |

### 总结（做出总结）

| 接续词 | 用法 | 例句 |
|-------|------|------|
| 以上のように | 如上所述 | 以上のように、様々な問題がある。 |
| このように | 就这样/综上 | このように、解決策は複数ある。 |
| 結論として | 作为结论 | 結論として、～と言える。 |
| まとめると | 总结来说 | まとめると、重要な点は3つだ。 |
| 総じて | 总的来说 | 総じて、結果は良好であった。 |

## 25.5 日语标点符号系统

### 基本标点符号及使用规范

| 符号 | 名称 | 读法 | 使用场合 |
|-----|------|------|---------|
| 。 | 句点 | くてん | 句子结束，陈述句末尾 |
| 、 | 读点 | とうてん | 句中停顿，列举时使用 |
| ！ | 感叹号 | かんたんふ | 感叹/强调（正式文章少用） |
| ？ | 问号 | はてな・ぎもんふ | 疑问句（正式文章用か代替） |
| 「」 | 引号（单） | かぎかっこ | 对话、引用、强调词语 |
| 『』 | 引号（双） | にじゅうかぎかっこ | 书名、引号内再引用 |
| （）| 括号 | かっこ | 补充说明、注释 |
| ・ | 中点 | なかてん | 列举外来语或名词 |
| … | 省略号 | てんてんてん | 省略、余韵（用2个为…… |
| — | 破折号 | ダッシュ | 说明、强调（用2个为——） |
| ～ | 波浪号 | なみせん | 范围、从…到…（非正式） |
| 々 | 叠字符 | どうのじてん | 汉字重复（人々、様々等） |

### 引号的嵌套使用

```
【正确使用示例】
先生は「『源氏物語』は日本文学の最高傑作だ」とおっしゃった。
（外层用「」，内层书名用『』）

田中さんは「私の夢は「医者になること」だ」と言った。
→ 改为正确形式：
田中さんは「私の夢は医者になることだ」と言った。
（内层单词不需要再加「」）
```

### 常见标点错误

| 错误类型 | 错误示例 | 正确形式 | 说明 |
|---------|---------|---------|------|
| 误用问号 | 彼は来るか？ | 彼は来るか。 | 正式文章中か已表疑问 |
| 列举滥用 | りんご、みかん、と（バナナ） | りんご・みかん・バナナ | 外来语列举用・ |
| 引号混用 | 彼は"おはよう"と言った | 彼は「おはよう」と言った | 日文引号用「」 |
| 省略号不足 | 彼女は…行った | 彼女は……行った | 省略号用两个… |
| 感叹号过多 | 本当にすごい！！！ | 本当にすごい。 | 正式文章避免多个！ |

## 25.6 日语写作常见错误

### 话し言葉的混入

这是最常见的错误，将口语表达带入书面写作。

```
【错误示例 - 话し言葉混入】
でも、スマートフォンってすごく便利じゃないですか。
みんな使ってるし、なんかないと困る感じがします。

【修正后 - 書き言葉】
しかし、スマートフォンは非常に便利である。
多くの人が使用しており、それなしでは生活に支障をきたすと感じる人も多い。
```

### 主语的过度省略导致歧义

日语可以省略主语，但过度省略会造成理解困难。

```
【歧义示例】
田中さんは山田さんに会った。彼は彼に本を渡した。
（「彼」指的是田中还是山田？引起歧义）

【修正后】
田中さんは山田さんに会った。田中さんは山田さんに本を渡した。
（明确指出主语，消除歧义）
```

### 长句过多

日语写作提倡适当断句，过长的句子会使读者难以理解。

```
【过长句子示例】
近年、環境問題が深刻化する中で、多くの企業が持続可能な開発を
目指して様々な取り組みを行っており、特に再生可能エネルギーの
導入や廃棄物の削減、さらにはサプライチェーン全体での
カーボンニュートラルの実現を目指した活動が注目されている。

【修正后（断句处理）】
近年、環境問題が深刻化している。これに対応するため、多くの企業が
持続可能な開発を目指している。特に注目されているのは、再生可能
エネルギーの導入や廃棄物の削減だ。さらに、サプライチェーン全体での
カーボンニュートラルの実現を目指す動きも広がっている。
```

### 「と思います」的过度使用

过度依赖「と思います」会使文章显得不自信、缺乏论据支撑。

```
【错误示例 - 过度使用と思います】
環境問題は大切だと思います。もっと努力すべきだと思います。
リサイクルが重要だと思います。

【修正后 - 多样化表达】
環境問題は現代社会が直面する最重要課題の一つである。
データが示すように、CO2排出量の削減は緊急の課題だ。
リサイクルの推進は、その有効な手段として広く認識されている。
```

**替代「と思います」的书面语表达：**

| 表达方式 | 适用场合 | 例句 |
|---------|---------|------|
| ～と考える | 表明观点 | これが最善策だと考える。 |
| ～と言える | 客观判断 | 効果的な方法だと言える。 |
| ～ではないだろうか | 委婉质疑 | 見直す必要があるのではないだろうか。 |
| ～と考えられる | 客观推断 | 原因は複数あると考えられる。 |
| ～であろう | 推测 | 今後も増加するであろう。 |
| ～のではないかと思われる | 谨慎推测 | 解決可能なのではないかと思われる。 |

## 本章小结

| 知识点 | 核心内容 | 重要程度 |
|-------|---------|---------|
| 原稿用紙 | 标点位置、段首空格、数字写法 | ★★★ |
| 文体选择 | 常体/敬体不可混用 | ★★★★★ |
| 書き言葉 | 30+组口语→书面语对照 | ★★★★★ |
| 段落结构 | 话题句→支撑句→总结句 | ★★★★ |
| 起承転結 | 日语特有文章结构 | ★★★★ |
| 接续词 | 顺接/逆接/换言/举例/总结 | ★★★★ |
| 标点系统 | 「」『』的使用规范 | ★★★ |
| 常见错误 | 话し言葉混入、主语省略歧义等 | ★★★★ |

**本章学习要点：**

1. 日语写作有严格的规范体系，原稿用纸的使用规则体现了这种严谨性。
2. 常体与敬体的区别是日语写作最基础的选择，一旦确定不可混用。
3. 話し言葉与書き言葉的差异是中级以上学习者需要重点攻克的难关。
4. 起承転結是理解日语文章逻辑的钥匙，也是写出地道日语文章的框架。
5. 适当使用接续词，能极大提升文章的逻辑性和可读性。
6. 避免过度使用「と思います」，学会多样化的论断表达方式。

> 「上手な文章は、読者への思いやりから生まれる」——优秀的文章，源于对读者的体贴。写作时始终考虑读者的理解，才能写出真正有效的日语文章。
