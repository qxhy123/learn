## 22.2 汉字的音读系统深入

### 四种音读体系

日语汉字的音读来自不同历史时期传入中国的汉字读音：

| 音读类型 | 传入时期 | 来源地区 | 特点 |
|---------|---------|---------|------|
| 呉音（ごおん） | 5-6世纪 | 江南（吴地）方音 | 较古老，多用于佛教/传统词汇 |
| 漢音（かんおん） | 7-8世纪 | 唐代长安音 | 多用于学术/政治词汇 |
| 唐音（とうおん） | 宋-明代 | 宋、明代官话 | 较少，多见于禅宗/茶道词汇 |
| 慣用音（かんようおん） | — | 日本独自的读法 | 约定俗成，不对应任何汉语发音 |

### 同一汉字的不同音读使用场景

**「明」字的多种读法：**

| 读法 | 音读类型 | 使用场景 | 例词 |
|------|---------|---------|------|
| ミョウ | 呉音 | 佛教、明後日 | 明朝（みょうちょう）＝明天早晨（书面/正式） |
| メイ | 漢音 | 一般词汇 | 明暗（めいあん）、説明（せつめい） |
| ミン | 漢音变体 | 历史/地名 | 明朝（みんちょう）＝中国明朝 |
| あか/あき | 訓読 | — | 明るい（あかるい）、明らか（あきらか） |

**「行」字的多种读法：**

| 読法 | 例词 | 含义 |
|------|------|------|
| ギョウ（呉音） | 行列（ぎょうれつ）、修行（しゅぎょう） | 排列、修行 |
| コウ（漢音） | 行動（こうどう）、旅行（りょこう） | 行动、旅行 |
| アン（唐音） | 行灯（あんどん） | 灯笼（禅宗词汇） |
| い/ゆ/おこな | 訓読 | 去、往来、实施 |

### 音读规律：入声字的促音化

中古汉语中以-p、-t、-k结尾的入声字，在日语中通常变为促音（っ）：

| 规律 | 原始音 | 日语结果 | 例子 |
|------|-------|---------|------|
| -k → っ | 学（xué） | ガク → ガッ | 学校（がっこう） |
| -t → っ | 出（chū） | シュツ → シュッ | 出発（しゅっぱつ） |
| -p → っ | 立（lì） | リツ → リッ | 立場（たちば）※但複合时保留 |

**复合词中的促音连接规律（入声+k/s/t/p开头）：**

```
学（ガク）+ 校（コウ）→ 学校（がっこう）✓（k前促音化）
出（シュツ）+ 発（パツ）→ 出発（しゅっぱつ）✓（p前促音化）
一（イチ）+ 階（カイ）→ 一階（いっかい）✓（k前促音化）
```

### 連濁（れんだく）现象

两个词合成复合词时，后一个词的清音（か行/さ行/た行/は行）有时变为浊音（が行/ざ行/だ行/ば行）：

| 原词1 | 原词2 | 复合词 | 連濁 |
|------|------|-------|------|
| 花（はな） | 火（ひ） | 花火（はなび） | ひ→び |
| 山（やま） | 道（みち） | 山道（やまみち） | なし |
| 秋（あき） | 風（かぜ） | 秋風（あきかぜ） | なし |
| 鼻（はな） | 血（ち） | 鼻血（はなぢ） | ち→ぢ |
| 手（て） | 紙（かみ） | 手紙（てがみ） | か→が |
| 一（ひと） | 時（とき） | 一時（ひととき） | と→ど |

**連濁が起きない場合（ライマンの法則）：**
后一个词中已含有浊音时，连浊不发生（Lyman's Law）：

```
山（やま）+ 柿（かき）→ 山柿（やまがき）※連濁○（かき中无浊音）
山（やま）+ 蛙（かえる）→ 山蛙（やまがえる）※連濁○
山（やま）+ 城（しろ）→ 山城（やましろ）※連濁✗（しろ中無浊音但无濁音規則更複雑）
```

## 22.5 二字熟語の构成模式

掌握二字熟语的构成模式，可以快速推测未知词的含义：

### 并列型（A和B同类/近义）

```
山川（さんせん）：山和川 → 山川、自然
善悪（ぜんあく）：善和恶 → 善恶
左右（さゆう）：左和右 → 左右、操控
開閉（かいへい）：开和关 → 开关
```

### 修饰型（A修饰B）

```
高山（こうざん）：高い山 → 高山
新年（しんねん）：新しい年 → 新年
大国（たいこく）：大きい国 → 大国
多数（たすう）：数が多い → 多数
```

### 动宾型（A动作，B是宾语）

```
読書（どくしょ）：読む書物 → 读书
作文（さくぶん）：文を作る → 写作文
登山（とざん）：山に登る → 登山
着席（ちゃくせき）：席に着く → 就座
```

### 主谓型（A是主语，B是谓语/动作）

```
地震（じしん）：地が震える → 地震
日没（にちぼつ）：日が没する → 日落
国立（こくりつ）：国が立てる/国が立てた → 国立
自立（じりつ）：自らが立つ → 自立
```

### 否定型（否定前缀+本义）

```
不満（ふまん）：満足でない → 不满
無力（むりょく）：力がない → 无力
非常（ひじょう）：常でない → 非常
未来（みらい）：まだ来ていない → 未来
```

### 补充型（A是结果，B是手段/补充）

```
改善（かいぜん）：改めて善くする → 改善
増加（ぞうか）：加わって増える → 增加
発展（はってん）：展開して発展 → 发展
```

**推测练习：** 看到「過労」，利用规律推测：過（过量）+ 労（劳苦）→ 过度劳累（正确：过劳）

## 22.6 汉字学习的高效方法

| 人（にんべん） | 人 | 他・休・体・使・作 |
| 口（くちへん） | 嘴/言语 | 呼・叫・唱・咲・味 |

| 金（かねへん） | 金属/钱 | 銀・鉄・鋼・銭・鍵 |

### 音符（声旁）推测音读

带有相同声旁的汉字往往有相似的音读：

```
【青（セイ・ショウ）系】
青（セイ）→ 晴（セイ）・清（セイ）・請（セイ/ショウ）・情（ジョウ）・静（セイ）

【包（ホウ）系】
包（ホウ）→ 抱（ホウ）・泡（ホウ）・胞（ホウ）・飽（ホウ）・砲（ホウ）

【工（コウ）系】
工（コウ）→ 功（コウ）・空（クウ）・紅（コウ）・項（コウ）・攻（コウ）

【里（リ）系】
里（リ）→ 理（リ）・裏（リ）・離（リ）・璃（リ）
```

## 练习

### 练习一：同形異義語（同形异义词）辨别

将以下日语句子译成中文，注意日中不同的含义：

1. 「もう大丈夫です。心配しないでください。」
2. 「母から手紙が届きました。」
3. 「もっと勉強しなければならない。」
4. 「愛人と一緒に旅行した。」（注：日语中指情人，翻译时需说明语境）
5. 「汽車で東京まで行きました。」

### 练习二：四字熟語の穴埋め

根据语境在括号中填入适当的四字熟语：

1. 試験まで毎日10時間勉強した。まさに（　　　　）の努力だ。
   （选项：粉骨砕身/馬耳東風/一目瞭然）
2. 二人は初めて会ったのにすぐに（　　　　）して、
   長時間話し合った。
   （选项：意気投合/右往左往/試行錯誤）
3. 計画がうまくいかず、チーム全体が（　　　　）の状態になった。
   （选项：五里霧中/本末転倒/不言実行）

### 练习三：二字熟語の構成分析

分析以下二字熟语的构成模式（并列/修饰/动宾/主谓/否定）：

1. 登山（とざん）
2. 不安（ふあん）
3. 花火（はなび）
4. 地震（じしん）
5. 読書（どくしょ）
6. 高原（こうげん）

### 练习四：熟字訓読み

为以下汉字词标注正确的假名读法：

1. 今日（　　）に授業がある。
2. 昨日（　　）は雨だった。
3. 果物（　　）を買ってきた。
4. 眼鏡（　　）をかけている。
