# 第23章：文章类型分析

> 「文章を読むとは、著者の思考回路を追体験することである。」
> ——阅读文章，是重走作者思维路径的过程。

---

## 引言

日语阅读理解的核心困难并非单纯的词汇量不足，而是**不懂各类文体的行文逻辑**。说明文有其独特的论证结构，论说文有其接续词导航系统，随笔有其情感流动方式，新闻有其省略规则。

一旦掌握了各文体的"结构图谱"，即使遇到生词也能通过上下文推断；即使面对复杂长句，也能迅速定位主旨。这正是JLPT N2/N1阅读高分考生的核心能力。

本章将系统拆解日语六大文章类型，提供可操作的阅读策略，并针对JLPT考试题型给出实战技巧。

## 23.1 日语文章的基本分类

### 23.1.1 六大文章类型概览

| 文体类型 | 日语名称 | 核心特征 | JLPT出现频率 |
|----------|----------|----------|-------------|
| 说明文 | 説明文（せつめいぶん） | 客观介绍事物、现象、知识 | ★★★★★ |
| 论说文 | 論説文（ろんせつぶん） | 提出主张并用论据支撑 | ★★★★★ |
| 随笔 | 随筆（ずいひつ） | 个人体验+感想+思考 | ★★★★☆ |
| 小说 | 小説（しょうせつ） | 虚构人物/情节，心理描写 | ★★★☆☆ |
| 书信/邮件 | 手紙・メール | 格式化表达，礼仪性语言 | ★★☆☆☆ |
| 新闻 | ニュース・記事 | 简洁、事实导向、标题省略 | ★★★☆☆ |

### 23.1.2 文体识别快速法

拿到一篇文章，30秒内完成文体判断：

```
判断流程：
① 有没有「私は～思う/感じる」？ → 随筆 or 論説文
② 有没有人物名和对话「」？ → 小説
③ 全文用「～である/～だ」结尾且无第一人称？ → 説明文 or 論説文
④ 有没有「拝啓/敬具/よろしくお願いします」？ → 手紙・メール
⑤ 动词省略、体言止め为主？ → ニュース
⑥ 论点在末尾提出？ → 日本語的論説文の典型
```

## 23.2 説明文の阅读（JLPT阅读主力文体）

### 23.2.1 説明文的定义与特征

説明文是以**客观、中立的立场**介绍某一现象、概念、科学知识或社会现象的文章。JLPT阅读中约60%以上的文章属于此类。

**核心特征：**
- 不含作者强烈的个人主张
- 逻辑清晰，结构对称
- 多用定义句（「～とは～である」）
- 多用数据、比较、举例

### 23.2.2 説明文的标准结构：序論→本論→結論

```
説明文の典型構造：

【序論（はじめに）】
・話題の導入
・読者に問いを投げかける
・「なぜ～か」「～について考えてみよう」

【本論（なかほど）】
・定義の提示：「～とは、～のことである」
・事実・データの提示：「～によると」「～%が」
・具体例：「たとえば」「例を挙げると」
・比較・対比：「一方」「それに対して」
・メカニズム説明：「なぜなら」「その理由は」

【結論（おわりに）】
・まとめ：「このように」「以上から」
・再強調：「つまり」「要するに」
・展望・提言：「今後は」「～が望まれる」
```

### 23.2.3 关键信号词（シグナルワード）一览表

| 功能 | 日语信号词 | 解读提示 |
|------|-----------|----------|
| 定义 | ～とは、～というのは、～とは何か | 后面是核心概念，必读 |
| 举例 | たとえば、具体的には、一例として | 例子可略读，但要抓住它说明的要点 |
| 追加 | さらに、加えて、また、なお | 同方向信息叠加 |
| 转折 | しかし、ところが、だが、けれども | 后面是重点！方向改变 |
| 因果 | なぜなら、その理由は、したがって、そのため | 因→果关系，结论往往在「したがって」之后 |
| 总结 | つまり、要するに、すなわち、言い換えると | 主旨的高度概括，必须精读 |
| 对比 | 一方（で）、それに対して、反対に | 两侧信息都要把握 |
| 条件 | ～場合は、もし～ならば | 假设条件，注意范围 |
| 强调 | 特に、とりわけ、何より、重要なのは | 后接的内容是文章核心 |

### 23.2.4 主旨句（トピックセンテンス）的定位策略

```
主旨句定位三原则：

原则1：首段首句优先
説明文では多くの場合、第1段落の1文目か最終文が主旨を含む。

原则2：結論シグナル後を探す
「つまり」「要するに」「このように」の直後が主旨の再掲であることが多い。

原则3：繰り返される語句に注目
本文中で繰り返されるキーワードが主旨に直結する。

実例分析：
「現代社会において、人々は大量の情報に囲まれている。
　しかし、その情報の質は様々で、中には誤った内容も含まれる。
　したがって、情報リテラシーを身につけることが現代人に不可欠だ。」

→主旨：情報リテラシーの必要性（最終文・「したがって」後）
```

### 23.2.5 图表描述语言

| 图表变化 | 日语表现 |
|----------|----------|
| 增加 | 増加する、上昇する、高まる、伸びる |
| 减少 | 減少する、低下する、下がる、落ち込む |
| 维持不变 | 横ばい、推移する、変わらない |
| 急剧变化 | 急増/急減、急上昇/急落下、大幅に |
| 约为 | 約～、およそ～、～程度 |
| 占比 | ～を占める、～の割合を示す |
| 最高/最低 | 最多/最少、ピークに達する、底を打つ |

## 23.3 論説文の阅读

### 23.3.1 論説文与説明文的本质区别

| 対比項目 | 説明文 | 論説文 |
|----------|--------|--------|
| 目的 | 知識・情報の伝達 | 主張の説得・意見の提示 |
| 主語 | 事象・現象 | 著者（私は～と考える） |
| 文末 | 「～である」「～だ」（客観） | 「～べきだ」「～はずだ」（主観混じり） |
| 構造 | 事実→説明 | 問題提起→主張→根拠→結論 |
| 判断 | 読者に委ねる | 著者が結論を提示する |

### 23.3.2 論説文的典型结构

```
日本語論説文の特徴的な構造（末尾主張型）：

【問題提起】：「～という問題がある」「～はどうあるべきか」
　↓
【現状分析】：「現在、～という状況だ」「一般的に～とされている」
　↓
【反論・譲歩】：「確かに～という意見もある」「～という面も否定できない」
　↓
【主張】：「しかし、私は～と考える」「だが、真に重要なのは～だ」
　↓
【根拠】：「なぜなら～だからだ」「その理由として～が挙げられる」
　↓
【結論】：「以上のことから、～と言えよう」「したがって～すべきだ」

注意：日本語の論説文は主張が末尾に来ることが多い！
（英語の論文とは逆の構造パターン）
```

### 23.3.3 接続词导航系统

接続词（接续词）是论说文的"路标"，读懂接续词就能把握逻辑结构：

**順接系（顺向推进）：**
```
だから / したがって / それゆえ / よって
→ 前の内容を受けて結論・帰結を述べる
例：「データが示すように不足している。したがって、政策の見直しが必要だ。」
```

**逆接系（方向转折）：**
```
しかし / だが / ところが / けれども / とはいえ / もっとも
→ 前の内容を否定・制限する（直後が著者の真の主張であることが多い）
例：「確かに便利だ。しかし、プライバシーの問題は無視できない。」
```

**並列・対比系：**
```
一方（で） / それに対して / 反対に / 逆に
→ 異なる観点・立場の比較
例：「都市部では進んでいる。一方、地方では遅れが目立つ。」
```

**換言系（再叙述）：**
```
つまり / すなわち / 要するに / 言い換えると / 言うなれば
→ 直前の内容を簡潔に言い直す（ここが主旨の凝縮）
例：「データ処理能力が低下し、判断ミスが増加した。つまり、システムが限界に達した。」
```

**例示系：**
```
たとえば / 具体的には / 例えば / 一例として
→ 抽象的主張の具体化（読み流してよいが、主張を確認するために一読）
```

### 23.3.4 反驳与让步的识别（確かに～しかし型）

论说文中最重要的逻辑模式之一：

```
【让步→反驳模式】

確かに A。しかし、B。

= 「Aは認める。でも本当に重要なのはBだ」という構造

読み方のポイント：
① 「確かに/なるほど/たしかに/もちろん」→ 著者が一時的に反対意見を認める
② 「しかし/だが/けれども/とはいえ」→ 著者の本当の主張が始まる
③ Bの部分（しかし以降）が著者の主張

例文：
「確かに、SNSは人々の繋がりを促進する面がある。
　しかし、それは同時に孤独感を深めるという逆説をはらんでいる。」

→ 著者の主張：SNSが孤独感を深める（B部分）
　反論として認めた点：繋がりの促進（A部分）
```

### 23.3.5 结论位置的把握

```
日本語論説文で結論が来やすい場所：

① 最終段落（最も多い）
② 「したがって/以上から/このように」の直後
③ 問題提起に対する直接的な回答部分
④ 繰り返し言及されるキーフレーズの最終登場時

危険なパターン：
× 「筆者は第一段落で述べているように～」（日本語では主張は末尾が多い）
× 冒頭を主旨と決め打ちする（英語論文の癖を持ち込まない）
```

## 23.4 随筆の読み方

### 23.4.1 随筆の特徴

随筆（エッセイ）は、**個人の体験・観察をもとに感想や思索を自由に綴った文章**。論理的な証明より、感性・情緒・連想が重視される。

| 要素 | 特徴 |
|------|------|
| 視点 | 一人称（私）が中心 |
| 構造 | 体験談→感想→思索（非線形も多い） |
| 文体 | 口語的〜やや文語的（著者によって差が大きい） |
| テーマ | 日常の発見、記憶、社会への違和感など |
| 結論 | 明示的でないことも多い（余韻を残す） |

### 23.4.2 作者の感情・態度の判断法

随筆読解で問われる「筆者の気持ちとして最も適切なものを選べ」型の問題攻略：

```
感情判断のための語彙マップ：

【ポジティブ系】
嬉しい・喜ぶ・感動する・安堵する・誇らしい
うれしくなる・心が温まる・清々しい

【ネガティブ系】
悲しい・悔しい・後悔する・焦る・恥ずかしい
虚しい・複雑な気持ち・釈然としない

【複雑系（随筆で頻出）】
「複雑な気持ち」= ポジとネガが混在
「何とも言えない気持ち」= 言語化できない感覚
「違和感」= 何かおかしいという直感
「懐かしさ」= 過去への郷愁

判断の手順：
1. 「～と感じた」「～気がした」「～思った」を探す（直接表現）
2. 直接表現がなければ、行動・状況から推測
   例：「ため息をついた」→ 失望・疲弊・安堵のどれか、文脈で判断
3. 比喩表現の解釈（「心が重かった」→ 憂鬱・プレッシャー）
```

### 23.4.3 比喩と暗示の読み解き

```
随筆に多い比喩表現とその読み取り方：

直喩（～のように/～みたいに）：
「彼の言葉は氷のように冷たかった」
→ 感情的な冷淡さ・拒絶感を表す

隠喩（比喩語なしに同一視）：
「私の心は砂漠だった」
→ 乾いた、感情のない、孤独な状態

擬人法（物を人のように表現）：
「窓が悲しそうに濡れていた」
→ 語り手の悲しい心情の投影（感情移入）

象徴（ある物が別の概念を表す）：
「春が来るたびに、彼女のことを思い出す」
→ 春=再生・出会い・希望、あるいは特定の記憶のトリガー

読解問題への応用：
比喩の解釈問題では、比喩の「車体（tenor）」と「乗り物（vehicle）」を分離する：
「彼は岩のような意志を持っている」
→ 岩（vehicle）= 堅固・変わらない（の意味を取り出す）
→ 意志（tenor）に当てはめ：堅固で変わらない意志 ✓
```

## 23.5 ニュース日本語

### 23.5.1 新闻标题の省略规则

日本語のニュース見出し（ヘッドライン）は独特の文体を持つ。

| 省略ルール | 例 | 補完後 |
|-----------|-----|--------|
| 助詞の省略 | 「政府　対策発表」 | 政府が対策を発表した |
| 助動詞の省略 | 「消費税引き上げへ」 | 消費税が引き上げられる見通しだ |
| 動詞の省略（体言止め） | 「大地震　甚大な被害」 | 大地震で甚大な被害が発生した |
| 敬語の省略 | 「首相　訪米」 | 首相が訪米した/する |
| 完了のテンス省略 | 「A社　新製品を開発」 | A社が新製品を開発した |

### 23.5.2 体言止め（たいげんどめ）の機能

体言止めは**名詞・名詞句で文を終わらせる**表現で、ニュース見出しに多用される。

```
一般的な体言止めの効果：
① 簡潔さ：助詞・助動詞を省き、情報を圧縮
② 強調感：名詞で締めると情報が際立つ
③ 余韻：詩的・感情的な文章では余韻を残す

ニュース見出しの例：
「増税反対、1万人デモ」
→ 补全：増税反対を訴える1万人規模のデモが行われた

「地震　死者30人超に」
→ 补全：地震による死者が30人を超えた

「AI、医療現場へ本格導入」
→ 补全：AIが医療現場に本格的に導入される（見通しだ）
```

### 23.5.3 ニュース日本語特有の表現

| 表現 | 意味 | 例 |
|------|------|-----|
| ～の見通し | ～という予想・予測 | 「今夜から雨の見通し」 |
| ～に向け | ～を目標として | 「解決に向け協議」 |
| ～を受け | ～をきっかけとして | 「批判を受け辞任」 |
| ～との見方 | ～という見解 | 「回復しつつあるとの見方」 |
| ～めぐり | ～に関して対立・議論 | 「予算案をめぐり対立」 |
| ～に絡む | ～に関連した | 「汚職に絡む疑惑」 |
| 相次ぐ | 続けて起こる | 「事故が相次ぐ」 |
| 急ぐ | 急いでいる（状態） | 「対応を急ぐ政府」 |

### 23.5.4 NHK NEWS WEB Easy の活用

NHK NEWS WEB Easy（https://www3.nhk.or.jp/news/easy/）は外国人向けの簡易日本語ニュースサイト。

```
特徴：
・通常ニュースを平易な語彙・文法に書き直し
・難読漢字にルビ（ふりがな）付き
・重要語彙のポップアップ説明あり
・音声読み上げ機能あり

活用法：
① 同じニュースを通常版とEasy版で比較読みする
② Easy版で内容把握→通常版でニュース日本語の表現を学ぶ
③ 日本語能力の目標：N2合格者がEasyを自力で読めるレベル
　　　　　　　　　 N1レベルで通常ニュースを快適に読める

学習のヒント：
「通常版の見出し→Easy版の文章展開→通常版の全文」
という順で読むと、見出し省略の補完訓練になる。
```

## 23.6 JLPT阅读理解题型分析

### 23.6.1 JLPT読解セクションの構成（N2/N1基準）

| 問題種類 | テキスト長 | 問数 | 内容 |
|----------|-----------|------|------|
| 短文（内容理解・短） | 200字前後 | 各1問 | 短い説明文・メモ |
| 中文（内容理解・中） | 500字前後 | 各3問 | 論説文・随筆 |
| 長文（内容理解・長） | 900字以上 | 各4問 | 評論・論説 |
| 情報検索 | 700字前後 | 各2問 | 広告・案内・スケジュール |

### 23.6.2 时间分配と解题順序

```
N2読解推奨タイムスケジュール（総合で60分想定）：

情報検索（5分）→ 短文×4（8分）→ 中文×2（16分）→ 長文×1（15分）
残り時間でレビュー（16分）

ポイント：
・情報検索を最初に解く（スキャン型なので頭が疲れていないうちに）
・長文は最後（集中力が必要）
・「問題文を先に読む」技法：何を問われているか知ってから本文を読む
```

### 23.6.3 各题型解题技巧

**短文（内容理解）：**
```
攻略法：
① 全文1回精読
② 問題の選択肢と本文の該当箇所を1対1で照合
③ 「最も適切なもの」= 本文内容から導けるもの（推測は不可）

落とし穴：
× 一般常識や自分の意見で解かない
× 「書いてあること」と「書いてないこと」を混同しない
```

**中文・長文（内容理解）：**
```
攻略法：
① 問題文（質問）を先に読み、どの段落が関連するか予測
② 本文を段落ごとに読み、各段落の「小見出し（主旨）」をメモ
③ 筆者の主張を探す：「～と思う/考える」「～すべきだ」「～はずだ」
④ 選択肢の吟味：

消去法の手順：
Step 1：明らかに間違い（本文と矛盾）を消す
Step 2：本文に書いていない情報が含まれるものを消す
Step 3：部分的に正しいが全体として不正確なものを消す
Step 4：残ったものが答え（必ず本文で確認）
```

**情報検索（広告・案内読解）：**
```
攻略法：
① 問題文の条件を先にリストアップ
   例：「30歳・女性・土曜日・予算3000円以下」
② 本文の表・一覧から条件に合う項目を探す（スキャン読み）
③ 複数条件の組み合わせに注意（一つ満たしても他が×なら不可）
④ 「利用できるか」「申し込めるか」「割引対象か」などを確認

頻出のひっかけ：
・対象年齢・期間・曜日の条件ミス
・「以上・以下・超・未満」の数値境界
・「初回のみ」「会員限定」などの制限条件
```

### 23.6.4 よく出る問いかけのパターンと対応

| 問题文パターン | 日語 | 攻略方针 |
|--------------|------|----------|
| 主旨問い | 「この文章で筆者が最も言いたいことは何か」 | 結論部・換言表現の直後を探す |
| 理由問い | 「なぜ～と言っているか」 | 「なぜなら/から/ため/理由は」を探す |
| 指示語問い | 「この/その/あのXとは何か」 | 指示語の直前の名詞・文を探す |
| 態度問い | 「筆者の立場として正しいのはどれか」 | 譲歩（確かに）直後の主張側に注目 |
| 適切表現選択 | 「（　）に入る最も適切な言葉を選べ」 | 前後の接続関係（逆接か順接か）を確認 |

## 練習問題

### 問題1：接続詞の選択

以下の文章の（　）に入る最も適切な接続詞を選びなさい。

「SNSは情報共有の場として有効だ。（　　）、誤情報が拡散しやすいという問題もある。（　　）、リテラシー教育の充実が求められている。」

選択肢：① しかし・だから　② したがって・しかし　③ また・つまり

**答え：① しかし（逆接）・だから（因果）**

### 問題2：文体识别

次の文章の文体（説明文/論説文/随筆）を判断し、その理由を説明しなさい。

「去年の夏、久しぶりに故郷へ帰った。駅を出ると、変わらない山の形が迎えてくれた。なのに、どこか懐かしさよりも、静かな淋しさが胸に広がった。時間とはこういうものか、と思った。」

**答え：随筆。理由：一人称（「私」）の体験と感情（淋しさ）の記述、思索的な結び、非論理的な感性的展開。**

### 問題3：ニュース見出し展開

以下のニュース見出しを完全な文章に展開しなさい。

「政府　緊急経済対策を閣議決定　来月施行へ」

**答え：政府は緊急経済対策を閣議決定した。来月（に）施行される見通しだ。**

## 本章小結

| 文体 | 核心構造 | 読解戦略 |
|------|---------|----------|
| 説明文 | 序論→本論→結論 | シグナルワードで構造把握、「つまり」後が主旨 |
| 論説文 | 問題提起→根拠→主張（末尾型） | 「しかし」後の主張を追う、確かに～しかし型に注意 |
| 随筆 | 体験→感想→思索（非線形） | 感情語・比喩の解釈、作者態度の判断 |
| ニュース | 見出し（体言止め）＋本文 | 省略補完、スキャン型の速読 |
| JLPT | 短文/中文/長文/情報検索 | 問題文先読み、消去法、条件照合 |

文章の種類を見極めることで、読む速度と正確性は格段に上がる。文体識別は「地図を持って街を歩く」ようなもの——構造が見えれば、迷わない。

> 「読書とは、著者との対話である。相手の話し方を知ることが、理解の第一歩だ。」
