# 第5章：日语句子结构（SOV语序）

> 「文法は言語の骨格である。骨格なくして、言葉は立てない。」
> 语法是语言的骨架。没有骨架，语言便无法站立。
> ——语言学家箴言

---

## 引言

日语和中文同属东亚语言圈，字形上有诸多相似，但语法结构却存在根本性差异。中文学习者初学日语时，最大的障碍之一就是语序的颠倒感——动词跑到句子末尾去了，助词像幽灵一样贴在名词后面，主语经常消失不见。

理解日语句子结构，不是死记规则，而是要建立一套全新的语言直觉。本章将从三语对比出发，系统梳理日语的基本句型、句子变换规则、指示代词体系和疑问词系统，为后续语法学习打下坚实基础。

## 5.1 日语句子的基本语序（SOV vs SVO 三语对比）

### 什么是语序？

语序（word order）是指句子中主语（S）、谓语动词（V）、宾语（O）的排列顺序。这是区分不同语言语法类型的核心指标之一。

| 语言 | 语序类型 | 示例（"我吃饭"） |
|------|----------|-----------------|
| 中文 | SVO | 我（S）+ 吃（V）+ 饭（O） |
| 英语 | SVO | I（S）+ eat（V）+ rice（O） |
| 日语 | SOV | 私は（S）+ ご飯を（O）+ 食べます（V） |

### 三语完整对比表

| 中文 | 英语 | 日语 | 语序分析 |
|------|------|------|----------|
| 我读书 | I read books | 私は本を読みます | S+V+O → S+O+V |
| 他买了苹果 | He bought an apple | 彼はリンゴを買いました | S+V+O → S+O+V |
| 猫喜欢鱼 | The cat likes fish | 猫は魚が好きです | S+V+O → S+O+V |
| 我去学校 | I go to school | 私は学校に行きます | S+V+Loc → S+Loc+V |
| 她给我礼物 | She gave me a gift | 彼女は私にプレゼントをくれました | 双宾语→S+IO+O+V |

### SOV语序的深层含义

**动词后置原则**是日语最核心的语法特征：

1. **动词永远在句末**（从句除外）：无论句子多长，谓语动词总是压轴出现
2. **修饰语前置原则**：所有修饰成分（定语、状语）都必须放在被修饰成分之前
3. **助词标记语法关系**：由于语序相对自由，助词（は、が、を、に、で等）承担了标记各成分语法功能的责任

```
【例句分析】
昨日、私は図書館で友達と一緒に日本語の本を三冊読みました。
きのう  わたし  としょかん  ともだち  いっしょ  にほんご  ほん  さんさつ  よみました

时间状语 + 主题 + 地点状语 + 同伴状语 + 宾语修饰 + 宾语 + 数量 + 动词（过去）

译：昨天，我在图书馆和朋友一起读了三本日语书。
```

**关键发现**：在这个长句中，核心动词「読みました」出现在最末尾，所有信息都在动词前堆叠。

## 5.2 日语的四种基本句型

### 句型一：名词谓语句（AはBです）

**结构**：主题 は + 名词 + です

这是最基础的日语句型，相当于中文的"A是B"结构。

```
基本公式：
AはBです → A是B

わたしは がくせい です。
私  は  学生  です。
我   是  学生  （礼貌助动词）

→ 我是学生。
```

**更多例句**：

```
これは本です。         → 这是书。
田中さんは先生です。   → 田中先生是老师。
東京は日本の首都です。 → 东京是日本的首都。
あの人は私の友達です。 → 那个人是我的朋友。
```

**结构要点**：
- `は` 是主题标记助词，发音为「わ」（wa）
- `です` 是礼貌体系动词，相当于中文的"是"，但更准确说是礼貌断定助动词

### 句型二：形容词谓语句

日语有两类形容词，形成两种不同的谓语结构。

#### い形容词谓语句

```
基本公式：
AはB（い形容词）です

この映画は面白いです。    → 这部电影很有趣。
富士山は高いです。        → 富士山很高。
この料理はおいしいです。  → 这道菜很好吃。
彼女は若いです。          → 她很年轻。
```

**注意**：い形容词本身已包含谓语功能，`です`在此是礼貌成分，口语中可省略。

#### な形容词谓语句

```
基本公式：
AはB（な形容词）です
（な形容词词干 + です）

この部屋は静かです。       → 这个房间很安静。
彼女はきれいです。         → 她很漂亮。
この仕事は大変です。       → 这份工作很辛苦。
東京は便利です。           → 东京很便利。
```

**对比**：

| 类型 | 肯定现在 | 例子 |
|------|----------|------|
| い形容词 | ～い + です | 面白いです |
| な形容词 | 词干 + です | 静かです（不加な） |

### 句型三：动词谓语句（AはBをV）

**结构**：主题 は + 宾语 を + 动词（ます形）

```
基本公式：
AはBをVます

私は本を読みます。         → 我读书。
彼はコーヒーを飲みます。   → 他喝咖啡。
田中さんは音楽を聞きます。 → 田中先生听音乐。
母は料理を作ります。       → 妈妈做菜。
```

**无宾语动词句**（自动词句）：

```
鳥が飛びます。   → 鸟飞翔。
雨が降ります。   → 下雨了。
子供が笑います。 → 孩子在笑。
```

### 句型四：存在句（～がある/いる）

存在句表达"某处存在某物/某人"，是日语中非常重要的句型。

**核心区分**：
- `ある`（ある）→ 用于**无生命体**（物品、植物、事情等）
- `いる`（いる）→ 用于**有生命体**（人、动物等）

```
【物的存在】
机の上に本があります。     → 桌子上有书。
部屋の中に椅子があります。 → 房间里有椅子。
冷蔵庫にビールがあります。 → 冰箱里有啤酒。

【人/动物的存在】
教室に学生がいます。       → 教室里有学生。
庭に猫がいます。           → 庭院里有猫。
部屋にだれもいません。     → 房间里没有任何人。
```

**存在句的语序**：

```
【基本结构】
場所 + に + 物/人 + が + ある/いる

（OR 用"は"标记地点作主题）
机の上には本があります。   → 桌子上面嘛，有书。
```

| 存在类型 | 动词 | 中文对应 |
|----------|------|----------|
| 物・植物・概念 | ある | 有/存在 |
| 人・动物 | いる | 有/在 |
| 举办・存在（活动） | ある | 有（会议有、活动有） |

## 5.3 肯定/否定/疑问的变换

### です系列的变换

| 形式 | 日语 | 读音 | 中文 |
|------|------|------|------|
| 现在肯定 | ～です | desu | 是～ |
| 现在否定 | ～ではありません | dewa arimasen | 不是～ |
| 现在否定（口语） | ～じゃないです | ja nai desu | 不是～ |
| 现在否定（口语2） | ～じゃありません | ja arimasen | 不是～ |
| 过去肯定 | ～でした | deshita | 曾是～ |
| 过去否定 | ～ではありませんでした | dewa arimasen deshita | 曾不是～ |

```
【变换示例】
肯定：彼は学生です。           → 他是学生。
否定：彼は学生ではありません。  → 他不是学生。
口语否定：彼は学生じゃないです。→ 他不是学生。（更自然）
过去：彼は学生でした。         → 他曾是学生。
过去否定：彼は学生ではありませんでした。 → 他曾不是学生。
```

### ます系列的变换

| 形式 | 语尾 | 例（読む→） |
|------|------|------------|
| 现在/将来肯定 | ～ます | 読みます |
| 现在/将来否定 | ～ません | 読みません |
| 过去肯定 | ～ました | 読みました |
| 过去否定 | ～ませんでした | 読みませんでした |

```
【完整变换示例：食べる（吃）】
現在肯定：毎日ご飯を食べます。          → 每天吃饭。
現在否定：朝ご飯を食べません。          → 不吃早饭。
過去肯定：昨日すしを食べました。        → 昨天吃了寿司。
過去否定：昨日何も食べませんでした。    → 昨天什么都没吃。
```

### 疑问句的构成（か的用法）

日语疑问句的构成极为简洁：在陈述句末尾添加 `か`。

**书面/正式语体**：句末升调 + か（但书面日语实际上不需要升调）

```
【疑问句构成：直接加か】
田中さんは先生ですか。       → 田中先生是老师吗？
これは本ですか。             → 这是书吗？
毎日日本語を勉強しますか。   → 每天学日语吗？
```

**口语体的疑问**：

在非正式口语中，年轻人常用 `？`（上升语调）代替 `か`，或省略 `か`：

```
これ、本？（非正式）          → 这是书？
田中さん、先生？（非正式）    → 田中是老师？
```

**疑问词疑问句**：

包含疑问词（なに、だれ、いつ等）的疑问句，语调通常**不升调**（因为已有疑问词标记）：

```
これは何ですか。（↓平调）     → 这是什么？
あの人はだれですか。（↓平调） → 那个人是谁？
```

**疑问句回答规则**：

```
「～ですか」的回答：
肯定：はい、（そうです）。    → 是的。
否定：いいえ、～ではありません。→ 不，不是～。

「～ますか」的回答：
肯定：はい、～ます。          → 是的，～。
否定：いいえ、～ません。      → 不，不～。
```

## 5.4 指示代词体系（こそあど系統）

日语的指示代词系统以 **こ・そ・あ・ど** 四个音节为核心，形成完整的四维对立系统，称为「こそあど言葉」。

### 核心原则

| 前缀 | 指示范围 | 记忆口诀 |
|------|----------|----------|
| こ（ko）| 近指（靠近说话人） | **こ**ちら=这边（**这**） |
| そ（so）| 中指（靠近听话人/已知信息） | **そ**ちら=那边（**那**[近]） |
| あ（a）| 远指（远离双方/共知信息） | **あ**ちら=那边（**那**[远]） |
| ど（do）| 不定（疑问） | **ど**ちら=哪边（**哪**） |

### こそあど完整体系表

#### 1. 事物指示（这个/那个/哪个）

| 类别 | こ系 | そ系 | あ系 | ど系 |
|------|------|------|------|------|
| 独立指示代词 | これ | それ | あれ | どれ |
| 中文对应 | 这（个） | 那（个）[近] | 那（个）[远] | 哪（个） |

```
これは何ですか。               → 这是什么？
それは私の本です。             → 那是我的书。
あれは富士山です。             → 那（远处）是富士山。
どれが田中さんのですか。       → 哪个是田中先生的？
```

#### 2. 场所指示（这里/那里/哪里）

| 类别 | こ系 | そ系 | あ系 | ど系 |
|------|------|------|------|------|
| 场所代词 | ここ | そこ | あそこ | どこ |
| 中文对应 | 这里 | 那里[近] | 那里[远] | 哪里 |

```
ここはどこですか。             → 这里是哪里？
そこに座ってください。         → 请坐在那里。
あそこにコンビニがあります。   → 那边有便利店。
トイレはどこですか。           → 厕所在哪里？
```

#### 3. 连体词（这个～/那个～的）

| 类别 | こ系 | そ系 | あ系 | ど系 |
|------|------|------|------|------|
| 连体词 | この | その | あの | どの |
| 中文对应 | 这（个）～ | 那（个）～[近] | 那（个）～[远] | 哪（个）～ |

**关键区别**：连体词必须后接名词，不能单独使用！

```
この本は面白いです。           → 这本书很有趣。
その映画を見ましたか。         → 你看了那部电影吗？
あの山は何という山ですか。     → 那座山叫什么山？
どのかばんが田中さんのですか。 → 哪个包是田中先生的？
```

#### 4. 方式/状态指示（这样/那样/怎样）

| 类别 | こ系 | そ系 | あ系 | ど系 |
|------|------|------|------|------|
| 方式副词 | こう | そう | ああ | どう |
| 中文对应 | 这样 | 那样[近] | 那样[远] | 怎样 |

```
こう書いてください。           → 请这样写。
そうですね。                   → 是那样呢。（表示认同/思考）
ああ、なるほど。               → 哦，原来如此。
どうしましたか。               → 怎么了？发生什么了？
```

#### 5. 方向指示（这边/那边/哪边）

| 类别 | こ系 | そ系 | あ系 | ど系 |
|------|------|------|------|------|
| 方向代词 | こちら/こっち | そちら/そっち | あちら/あっち | どちら/どっち |
| 正式程度 | こちら（正式）| そちら（正式） | あちら（正式） | どちら（正式） |
| 口语 | こっち（口语） | そっち（口语） | あっち（口语） | どっち（口语） |

```
こちらへどうぞ。               → 请往这边来。（引导客人）
そちらのほうが安いです。       → 那边的更便宜。
あちらが受付です。             → 那边是接待处。
どちらが好きですか。           → 你喜欢哪边（哪个）？
```

### こそあど使用的特殊规则

**谈话场景中 そ系的特殊用法**：

当谈话双方共同知道某事（文脉上已提及），即使物理上不在附近，也用 `そ系`：

```
A：昨日、変な映画を見ました。 → 昨天看了部奇怪的电影。
B：その映画、どんな映画ですか。→ 那部电影是什么样的？
（「その」指代A刚提到的电影，虽然B没见过，但已进入对话语境）
```

## 5.5 疑问词系统

日语的疑问词体系丰富，以下是完整的疑问词分类表：

### 核心疑问词总表

| 疑问词 | 读音 | 中文 | 用法说明 |
|--------|------|------|----------|
| 何（なに/なん） | nani/nan | 什么 | 后接で、の等时读なん |
| 誰（だれ） | dare | 谁 | 敬语：どなた |
| いつ | itsu | 什么时候 | |
| どこ | doko | 哪里 | |
| なぜ/どうして | naze/doushite | 为什么 | どうして更口语 |
| なぜなら | nazenara | 因为 | 回答なぜ时 |
| いくら | ikura | 多少钱 | 价格 |
| いくつ | ikutsu | 几个/几岁 | 数量/年龄 |
| どう | dou | 怎样 | 方式/状态 |
| どんな | donna | 什么样的 | 性质描述 |
| どの | dono | 哪个（连体） | 后接名词 |
| どれ | dore | 哪个（独立） | 三者以上选择 |
| どちら/どっち | dochira/docchi | 哪边/哪个 | 二者选择 |

### 疑问词例句对照

```
何を食べますか。               → 你要吃什么？
誰が来ますか。                 → 谁来？
いつ日本に来ましたか。         → 你什么时候来日本的？
どこに住んでいますか。         → 你住在哪里？
なぜ日本語を勉強しますか。     → 为什么学日语？
このりんごはいくらですか。     → 这个苹果多少钱？
子供はいくつですか。           → 孩子几岁了？
日本語はどうですか。           → 日语怎么样？（感觉如何）
どんな映画が好きですか。       → 你喜欢什么样的电影？
```

### 疑问词 + でも（任意化）

疑问词加上 `でも` 变为"任何～都"的含义：

```
何でも食べます。               → 什么都吃。
誰でもできます。               → 谁都能做。
どこでも行きます。             → 哪里都去。
いつでもいいです。             → 什么时候都行。
```

### 疑问词 + も + 否定（全面否定）

```
何も食べません。               → 什么都不吃。
誰も来ません。                 → 谁都不来。
どこにも行きません。           → 哪里都不去。
何もわかりません。             → 什么都不明白。
```

## 5.6 主题与主语的区别（は vs が 初步理解）

`は` 和 `が` 的辨析是日语语法中最具挑战性的话题之一。本节先做初步介绍，详细辨析见第6章。

### 直觉区分

| 助词 | 功能 | 语感 |
|------|------|------|
| は | 主题标记 | "说到～的话..." / 对比强调 |
| が | 主语标记 | 自然发生 / 新信息 / 排他强调 |

### 基本对比

```
【は：主题/话题化】
象は鼻が長いです。
→ 说到大象，鼻子很长。（以"大象"为主题，陈述其特征）

【が：主语/新信息】
象が来た！
→ 大象来了！（"大象"是新出现的信息，引起注意）
```

```
【は：对比含义】
私は魚を食べます（が、肉は食べません）。
→ 我嘛，吃鱼（但肉不吃）。（暗示对比）

【が：排他/唯一】
私が魚を食べます（ほかの人じゃなくて）。
→ 是我吃鱼（不是别人）。（强调是"我"而非他人）
```

### 疑问句中的 は vs が

```
【疑问词作主语时，必须用が】
誰が来ましたか。    → 谁来了？（疑问词作主语→用が）
何が好きですか。    → 喜欢什么？（疑问词作主语→用が）

【已知信息作主题时，用は】
田中さんはどこに行きましたか。 → 田中先生去哪里了？
```

## 5.7 省略与语境依赖

日语是高度依赖语境的语言，主语、宾语甚至动词都可在语境明确时省略。

### 主语的省略

日语中，当主语从上下文可以推断时，几乎总是省略的：

```
A：田中さん、今日学校に来ましたか。
   → 田中先生，今天来学校了吗？

B：いいえ、来ませんでした。
   → 没有，没来。
   （省略了「田中さんは」——从问题可知谈论对象）
```

```
A：このケーキ、食べていいですか。
   → 这个蛋糕可以吃吗？
   （省略了「私は」和「このケーキを」的一部分）

B：どうぞ。
   → 请便。（整句话省略）
```

### 宾语的省略

```
A：この映画、見ましたか。 → 这部电影你看了吗？
B：はい、見ました。      → 看了。（省略宾语「この映画を」）
```

### 省略的文化背景

日语的省略现象不仅是语法特征，更体现了日本文化中"察言观色"（空気を読む）的重要性——说话者默认听话者能理解省略的内容，要求双方共享高度的语境信息。

## 练习

### 练习一：语序转换

将下列中文句子按照日语SOV语序排列单词（不要求完整翻译，只需排顺序）：

1. 我（私は）/ 读（読みます）/ 书（本を）
2. 她（彼女は）/ 喜欢（好きです）/ 音乐（音楽が）
3. 田中先生（田中さんは）/ 去（行きます）/ 学校（学校に）

### 练习二：句型选择

用正确的存在动词（ある/いる）填空：

1. 机の上に猫が（　　）ます。
2. 冷蔵庫に牛乳が（　　）ます。
3. 公園に子供が（　　）ます。
4. 庭に花が（　　）ます。

### 练习三：こそあど填空

根据情境选择正确的こそあど词：

1. （A拿着一本书）「（これ/それ/あれ）は面白い本です。」
2. （B拿着一本书）「A：（これ/その/あの）本はどこで買いましたか。」
3. （远处有座山）「（ここ/そこ/あそこ）に登りたいです。」

### 练习四：疑问句变换

将下列陈述句变成疑问句，并写出一个可能的否定回答：

1. 田中さんは先生です。→（疑问）→（否定回答）
2. 毎日日本語を勉強します。→（疑问）→（否定回答）

## 本章小结

| 知识点 | 核心内容 |
|--------|----------|
| 语序 | 日语为SOV，动词永远在句末 |
| 四种基本句型 | 名词句/形容词句/动词句/存在句 |
| 否定变换 | です→ではありません、ます→ません |
| 疑问句 | 句末加か（正式）或语调上升（口语） |
| こそあど | こ近/そ中/あ远/ど疑，涵盖事物/场所/方向/方式 |
| 疑问词 | なに/だれ/いつ/どこ/なぜ/いくら/いくつ/どう等 |
| は vs が | は=主题/对比，が=主语/新信息（初步） |
| 省略 | 语境明确时主语/宾语可省略 |
