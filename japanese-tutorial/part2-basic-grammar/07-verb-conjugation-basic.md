# 第7章：动词基础活用

> 「動詞の活用は日本語学習の山場であり、越えれば平野が広がる。」
> 动词活用是日语学习的山峰，越过去就是一片平原。
> ——日本语教育学者

---

## 引言

动词活用（動詞の活用，どうしのかつよう）是日语语法中最系统、最核心的部分。与中文动词"一字不变"的特性完全不同，日语动词根据时态、语气、接续关系不断变化形态。

但这并非混乱无序——日语动词活用有严格规律，只要掌握分类方法和变换规则，就能准确生成数十种活用形式。本章聚焦四种最基础的活用形：ます形、て形、ない形、た形。

---

## 7.1 日语动词的分类

### 三大类动词概览

日语动词分为三大类，分类依据是词尾形态和活用规则：

| 类别 | 日本传统称呼 | 英语学习者称呼 | 特征 |
|------|-------------|---------------|------|
| 一类动词 | 五段動詞（ごだんどうし） | う-Verb / Group 1 | 词尾为う段，活用跨五段 |
| 二类动词 | 一段動詞（いちだんどうし） | る-Verb / Group 2 | 词尾为る，去掉る接续 |
| 三类动词 | 変格動詞（へんかくどうし） | Irregular | する和くる两个特殊动词 |

---

### 一类动词（五段动词/う-Verb）

五段动词的词尾在五十音「う段」上变化（う/く/ぐ/す/つ/ぬ/ぶ/む/る），活用时在五个段（あ/い/う/え/お）之间切换。

**辨识特征**：
- 词尾不是 `る`；或词尾是 `る` 但前一个假名不在「い段」或「え段」

```
【典型五段动词】
書く（かく）    → 写
読む（よむ）    → 读
話す（はなす）  → 说、讲
待つ（まつ）    → 等待
飲む（のむ）    → 喝
買う（かう）    → 买
帰る（かえる）  → 回去（词尾る，但前音「え」不在い/え段？）
聞く（きく）    → 听
```

**注意**：`帰る（かえる）` 虽词尾是る，但它是五段动词，因为它不符合二类动词的完整条件（见下文）。

---

### 二类动词（一段动词/る-Verb）

二类动词的词尾为 `る`，且 `る` 前一个假名在**い段**（上一段）或**え段**（下一段）。

| 分类 | 特征 | 例 |
|------|------|----|
| 上一段动词 | 词尾为る，る前是い段假名 | 見る（み・る）、起きる（お・き・る）|
| 下一段动词 | 词尾为る，る前是え段假名 | 食べる（た・べ・る）、教える（お・し・え・る）|

```
【典型二类动词】
上一段（い段+る）：
見る（みる）      → 看
起きる（おきる）  → 起床
着る（きる）      → 穿（衣服）
いる              → 存在（人/动物）
借りる（かりる）  → 借

下一段（え段+る）：
食べる（たべる）  → 吃
教える（おしえる）→ 教
出る（でる）      → 出去
寝る（ねる）      → 睡觉
答える（こたえる）→ 回答
```

---

### 三类动词（变格动词）

只有两个特殊动词，必须单独记忆：

| 动词 | 读音 | 意思 | 类型 |
|------|------|------|------|
| する | suru | 做 | サ变（sa-irregular） |
| くる | kuru | 来 | カ变（ka-irregular） |

`する` 可与名词组合构成复合动词：
```
勉強する（べんきょうする）  → 学习
運動する（うんどうする）    → 运动
結婚する（けっこんする）    → 结婚
料理する（りょうりする）    → 做料理
掃除する（そうじする）      → 打扫
電話する（でんわする）      → 打电话
```

---

### 分类判断流程

```
【动词分类判断步骤】

第一步：词尾是不是「る」？
  ├─ 不是 → 一定是五段动词（一类）
  └─ 是 → 进入第二步

第二步：る前面的假名是不是在「い段」或「え段」？
  ├─ 是 → 几乎可以判断为二类动词（一段）
  └─ 不是（在あ/う/お段）→ 是五段动词

第三步：是否是する或くる？
  └─ 是 → 三类动词

【陷阱动词（形似二类，实为一类）】
帰る（かえる）  → 五段（回去）← 常见错误！
切る（きる）    → 五段（切）  ← 与着る(二类)混淆！
走る（はしる）  → 五段（跑）
入る（はいる）  → 五段（进入）
知る（しる）    → 五段（知道）
```

## 7.2 ます形（敬体形）

ます形是日语最基本的礼貌体动词形式，用于正式场合和礼貌表达。

### 变换规则

#### 一类动词（五段动词）：う段 → い段 + ます

五段动词的词尾从「う段」变换到「い段」，再接 `ます`：

| 词尾音 | 变换 | 例 |
|--------|------|----|
| う → い | 買う→買います | 買（か）います |
| く → き | 書く→書きます | 書（か）きます |
| ぐ → ぎ | 泳ぐ→泳ぎます | 泳（およ）ぎます |
| す → し | 話す→話します | 話（はな）します |
| つ → ち | 待つ→待ちます | 待（ま）ちます |
| ぬ → に | 死ぬ→死にます | 死（し）にます |
| ぶ → び | 遊ぶ→遊びます | 遊（あそ）びます |
| む → み | 読む→読みます | 読（よ）みます |
| る → り | 帰る→帰ります | 帰（かえ）ります |

#### 二类动词（一段动词）：去る + ます

```
食べる → 食べ + ます = 食べます    （吃）
見る   → 見  + ます = 見ます      （看）
起きる → 起き + ます = 起きます   （起床）
寝る   → 寝  + ます = 寝ます      （睡觉）
教える → 教え + ます = 教えます   （教）
```

#### 三类动词（不规则）

```
する  → します
くる  → きます（来ます）
```

---

### ます形的时态变换

| 时态 | 形式 | 例（食べる） | 中文 |
|------|------|-------------|------|
| 现在/将来肯定 | ～ます | 食べます | 吃/将吃 |
| 现在/将来否定 | ～ません | 食べません | 不吃 |
| 过去肯定 | ～ました | 食べました | 吃了 |
| 过去否定 | ～ませんでした | 食べませんでした | 没吃 |

```
【完整时态示例：書く（写）】
書きます。          → 写（习惯）/ 要写（将来）。
書きません。        → 不写。
書きました。        → 写了。
書きませんでした。  → 没写。
```

## 7.3 て形（连接形）

て形是日语最重要的活用形之一，用途极广。变换规则比ます形复杂，但有规律可循。

### 一类动词的て形（音变规则）

五段动词变て形时发生**音便（おんびん）**现象，分三种类型：

#### ① イ音便（い音便）

词尾 `く/ぐ` 变换时，中间音变为「い」：

| 词尾 | 变换 | 例 |
|------|------|----|
| く → いて | 書く → 書いて | 书写→书写て |
| ぐ → いで | 泳ぐ → 泳いで | 游泳→游泳て |

```
書く → 書いて（写）
聞く → 聞いて（听）
歩く → 歩いて（走路）
泳ぐ → 泳いで（游泳）
急ぐ → 急いで（赶紧）
```

#### ② 促音便（つまる音便）

词尾 `う/つ/る（五段）` 变换时，变为促音「っ」：

| 词尾 | 变换 | 例 |
|------|------|----|
| う → って | 買う → 買って | 买 |
| つ → って | 待つ → 待って | 等 |
| る（五段）→ って | 帰る → 帰って | 回去 |

```
買う → 買って（买）
待つ → 待って（等）
帰る → 帰って（回去）
作る → 作って（制作）
乗る → 乗って（乘坐）
```

#### ③ 撥音便（はねる音便）

词尾 `ぬ/ぶ/む` 变换时，变为撥音「ん」，且て形变为「で」：

| 词尾 | 变换 | 例 |
|------|------|----|
| ぬ → んで | 死ぬ → 死んで | 死 |
| ぶ → んで | 遊ぶ → 遊んで | 玩 |
| む → んで | 読む → 読んで | 读 |

```
死ぬ → 死んで（死）
遊ぶ → 遊んで（玩）
飲む → 飲んで（喝）
読む → 読んで（读）
飛ぶ → 飛んで（飞）
```

#### ④ す → して（例外简单）

```
話す → 話して（说）
貸す → 貸して（借出）
押す → 押して（推）
```

**特殊例外**：
```
行く → 行って（去）← 不遵循イ音便，是例外！
```

---

### 二类动词的て形

去掉 `る`，直接加 `て`：

```
食べる → 食べて（吃）
見る   → 見て  （看）
起きる → 起きて（起床）
寝る   → 寝て  （睡觉）
教える → 教えて（教）
```

### 三类动词的て形

```
する  → して
くる  → きて（来て）
```

---

### て形的基本用法

```
① ～てください（请做～）
窓を開けてください。         → 请开窗。
ゆっくり話してください。     → 请慢慢说。
ここに名前を書いてください。 → 请在这里写名字。

② ～ている（正在进行/状态持续）
今、ご飯を食べています。     → 现在正在吃饭。
雨が降っています。           → 正在下雨。
田中さんは結婚しています。   → 田中先生已婚（结婚状态）。

③ 连接两个动作（然后）
朝起きて、顔を洗います。    → 早上起床，然后洗脸。
コーヒーを飲んで、仕事をします。→ 喝完咖啡，再工作。

④ ～てもいいですか（可以做～吗）
写真を撮ってもいいですか。   → 可以拍照吗？
ここに座ってもいいですか。   → 可以坐这里吗？
```

## 7.4 ない形（否定形）

ない形是动词的否定基本形，也称为「未然形+ない」。

### 一类动词：う段 → あ段 + ない

| 词尾音 | 变换 | 例 |
|--------|------|----|
| う → わ | 買う → 買わない | 不买 |
| く → か | 書く → 書かない | 不写 |
| ぐ → が | 泳ぐ → 泳がない | 不游泳 |
| す → さ | 話す → 話さない | 不说 |
| つ → た | 待つ → 待たない | 不等 |
| ぬ → な | 死ぬ → 死なない | 不死 |
| ぶ → ば | 遊ぶ → 遊ばない | 不玩 |
| む → ま | 読む → 読まない | 不读 |
| る（五段）→ ら | 帰る → 帰らない | 不回去 |

**特殊注意**：词尾是 `う` 时，あ段不是「あ」，而是「わ」：
```
買う → 買わない（不是 買あない！）
言う → 言わない
```

### 二类动词：去る + ない

```
食べる → 食べない（不吃）
見る   → 見ない  （不看）
起きる → 起きない（不起床）
寝る   → 寝ない  （不睡觉）
```

### 三类动词

```
する  → しない
くる  → こない（来ない）
```

---

### ない形的重要用法

```
① ～ないでください（请不要做～）
ここで写真を撮らないでください。 → 请不要在这里拍照。
タバコを吸わないでください。     → 请不要吸烟。
走らないでください。             → 请不要跑。

② ～なければならない（必须做～）
宿題をしなければならない。       → 必须做作业。
薬を飲まなければなりません。     → 必须吃药。

③ ～なくてはいけない（必须做～，口语常用）
行かなくてはいけません。         → 必须去。
もっと練習しなくてはいけない。   → 必须多练习。

④ ～なくてもいい（不做也可以）
今日は来なくてもいいです。       → 今天不来也可以。
全部食べなくてもいいですよ。     → 不用全部吃完的。
```

## 7.5 た形（过去形）

**好消息**：た形和て形的变换规则完全对应——只需把 `て→た`、`で→だ` 即可！

### て形与た形的对应关系

| て形 | た形 | 例（て形→た形） |
|------|------|----------------|
| ～いて | ～いた | 書いて → 書いた |
| ～いで | ～いだ | 泳いで → 泳いだ |
| ～って | ～った | 買って → 買った |
| ～んで | ～んだ | 読んで → 読んだ |
| ～して | ～した | 話して → 話した |
| 食べて | 食べた | 食べて → 食べた |
| して  | した  | して → した |
| きて  | きた  | きて → きた |

```
【た形例句】
昨日、図書館で本を読んだ。       → 昨天在图书馆读了书。
朝ご飯を食べた。                 → 吃了早饭。
友達と映画を見た。               → 和朋友看了电影。
日本語を勉強した。               → 学了日语。
```

（`た形`是简体/普通体过去式，相当于ました的普通形式）

---

### た形的重要用法

```
① ～たことがある（有过做～的经验）
日本に行ったことがあります。     → 去过日本。
すしを食べたことがありますか。   → 吃过寿司吗？
富士山に登ったことがありません。 → 没爬过富士山。

② ～たり～たりする（列举代表性动作/交替动作）
週末は映画を見たり、買い物をしたりします。
→ 周末做看电影、购物等事情。

休みの日は本を読んだり、音楽を聞いたりします。
→ 休息日读读书、听听音乐之类的。

③ ～たあとで（做～之后）
ご飯を食べた後で、散歩します。  → 吃饭之后散步。
宿題をした後で、テレビを見ます。→ 做完作业之后看电视。

④ ～たほうがいい（最好做～）
薬を飲んだほうがいいですよ。    → 最好吃药哦。
早く寝たほうがいいです。        → 最好早点睡。
```

## 7.7 中国学习者的动词活用难点与记忆策略

### 难点1：五段动词词尾判断

**问题**：`切る（きる）` 和 `着る（きる）` 读音相同，类型不同。

| 动词 | 意思 | 类型 | ない形 |
|------|------|------|--------|
| 切る（きる） | 切割 | 五段 | 切らない |
| 着る（きる） | 穿（上衣） | 二段 | 着ない |

**记忆策略**：遇到可疑的る结尾动词，优先查字典确认，或通过ない形检验——五段动词ない形词干末音在あ行，二段在去掉る之后直接加ない。

### 难点2：い音便的例外（行く）

`行く` 的て形是 `行って`，而不是按规则的 `行いて`。这是唯一的例外，必须单独记忆。

**记忆口诀**：「行く」单独去，て形加って，不走い音便。

### 难点3：う结尾动词的ない形

```
買う → 買わない（不是買あない）
言う → 言わない
笑う → 笑わない
```
**记忆策略**：`う` 的あ段是 `わ`，不是 `あ`。可以用五十音表的あ行记忆：あ→わ的对应关系在助词は（わ）上也有体现。

### 难点4：て形的三种音便区分

**口诀记忆法**：

```
「くぐ」→ い音便（書いて、泳いで）
「うつる」→ 促音便（買って、待って、帰って）
「ぬぶむ」→ 撥音便（死んで、遊んで、読んで）
「す」→ して（話して）
特殊：「行く」→ 行って
```

### 难点5：ている的两种含义

```
【动作进行（动态动词）】
食べている → 正在吃（进行中）
走っている → 正在跑（进行中）

【状态持续（状态动词/结果状态）】
結婚している → 已婚（结婚的状态）
死んでいる   → 已死（死亡的状态）
着ている     → 穿着（穿上的状态）
```

### 记忆策略总结

| 方法 | 具体操作 |
|------|----------|
| 分组记忆 | 按词尾音（く/む/す等）分组，一起记て形 |
| 类比扩展 | 掌握一个词的活用后，找同类动词举一反三 |
| 例句记忆 | 把活用形嵌入真实例句，不孤立记忆词尾 |
| 音便口诀 | 「くぐ→い音便；うつる→促音便；ぬぶむ→撥音便」 |
| 大量输入 | 多听日语，让耳朵对活用形产生直觉 |

## 本章小结

| 活用形 | 功能 | 一类（五段）规则 | 二类（一段）规则 | 三类 |
|--------|------|-----------------|-----------------|------|
| ます形 | 礼貌体 | う段→い段+ます | 去る+ます | します/きます |
| て形 | 连接/请求/进行 | 三种音便+て/で | 去る+て | して/きて |
| ない形 | 否定 | う段→あ段+ない | 去る+ない | しない/こない |
| た形 | 过去/经验 | て形将て→た | 去る+た | した/きた |

动词活用是日语学习的硬骨头，但规律性极强。建议每天通过造句练习强化记忆，把活用形与真实使用场景绑定，才能真正内化。下一章进入形容词体系。
