# 第8章：形容词体系

> 「形容詞は言語の絵の具だ。色彩なくして、世界は描けない。」
> 形容词是语言的颜料。没有色彩，无法描绘世界。
> ——文学评论家

---

## 引言

如果说动词是日语句子的引擎，那么形容词就是句子的调色板。日语的形容词体系独具特色——它拥有两大截然不同的形容词类型，各自有完整的活用系统，并且这套活用系统与中文形容词"只需加程度副词"的简单模式形成鲜明对比。

更令人着迷的是，日语的い形容词本质上是一种"动词化的形容词"，自带时态变化能力；而な形容词更接近名词性描述，需要借助助动词完成活用。理解这两类形容词的本质差异，是掌握日语描述系统的关键。

---

## 8.1 日语形容词的两大类

### 形容词分类概览

| 类型 | 日文名称 | 英文称呼 | 特征 | 典型例 |
|------|----------|----------|------|--------|
| い形容词 | 形容詞（けいようし） | i-adjective | 词尾为い，自身活用 | 大きい、高い、おいしい |
| な形容词 | 形容動詞（けいようどうし） | na-adjective | 做定语时加な，借助だ活用 | きれいな、静かな、便利な |

---

### い形容词的典型特征

所有い形容词都以**い**结尾（辞书形），且这个「い」是活用语尾的一部分：

```
大きい（おおきい）  → 大
高い（たかい）      → 高/贵
安い（やすい）      → 便宜
面白い（おもしろい）→ 有趣
おいしい            → 好吃
新しい（あたらしい）→ 新的
古い（ふるい）      → 旧的
難しい（むずかしい）→ 难的
```

---

### な形容词的典型特征

な形容词的辞书形词尾**不是**い，做定语时词干后加な：

```
きれい（な）    → 漂亮的（きれいな部屋）
静か（な）      → 安静的（静かな図書館）
便利（な）      → 方便的（便利な場所）
有名（な）      → 著名的（有名な歌手）
好き（な）      → 喜欢的（好きな食べ物）
嫌い（な）      → 讨厌的（嫌いな科目）
上手（な）      → 擅长的（上手な料理）
下手（な）      → 不擅长的（下手な絵）
```

---

### 判断技巧：为什么「きれい」「有名」是な形容词？

这是许多学习者的困惑点。`きれい` 词尾是 `い`，看起来像い形容词，但它是な形容词！判断依据如下：

**规则**：如果一个词的词尾「い」在汉字写法中对应汉字读音（而非假名活用语尾），则为な形容词。

| 词 | 汉字写法 | 判断 |
|----|----------|------|
| きれい | 綺麗（き・れ・い） | い是汉字读音音节，非活用语尾→な形容词 |
| きらい | 嫌い（き・ら・い） | 同上→な形容词 |
| ゆうめい | 有名（ゆう・め・い） | い是汉字音节→な形容词 |
| おおきい | 大きい（おお・き・**い**） | 末尾い是活用语尾（去掉い=大き）→い形容词 |

**实用判断法**：
1. 将词尾い去掉，剩余部分是否还有意义且可单独存在？
   - `大きい` → `大き`（不能单独用）→ い是语尾 → い形容词
   - `きれい` → `きれ`（「綺麗」去掉い剩きれ/綺麗，词义单元保持）→ な形容词
2. 能否接续 `ではない`？
   - い形容词否定：大きくない（不用ではない）
   - な形容词否定：きれいではない / きれいじゃない

## 8.2 い形容词的活用

### 完整活用表

| 活用形 | 构成方式 | 例（大きい） | 例（高い） |
|--------|----------|-------------|-----------|
| 基本形（辞書形） | —— | 大きい | 高い |
| 肯定现在（礼貌） | ～い + です | 大きいです | 高いです |
| 否定现在 | ～く + ない（です） | 大きくないです | 高くないです |
| 否定现在（礼貌） | ～く + ありません | 大きくありません | 高くありません |
| 肯定过去 | ～かった（です） | 大きかったです | 高かったです |
| 否定过去 | ～くなかった（です） | 大きくなかったです | 高くなかったです |
| 连用形（接て） | ～くて | 大きくて | 高くて |
| 副词化 | ～く | 大きく | 高く |

### 详细说明

#### 肯定/否定变换

```
【否定：去い，加くない】
大きい → 大きくない（不大）
高い  → 高くない  （不高）
安い  → 安くない  （不便宜）
面白い → 面白くない（不有趣）
おいしい → おいしくない（不好吃）

【礼貌否定：去い，加くありません】
大きくありません    → 不大。（礼貌）
高くありません      → 不贵。（礼貌）
面白くありません    → 不有趣。（礼貌）
```

#### 过去/过去否定变换

```
【过去肯定：去い，加かった】
大きい    → 大きかった    → 曾经很大
高い      → 高かった      → 曾经很贵
難しい    → 難しかった    → 曾经很难
楽しい    → 楽しかった    → 曾经很快乐

【过去否定：去い，加くなかった】
大きくなかった    → 曾经不大
高くなかった      → 曾经不贵
難しくなかった    → 曾经不难
```

```
【完整时态示例：おいしい（好吃）】
おいしいです。           → 好吃。（现在）
おいしくないです。       → 不好吃。（现在否定）
おいしかったです。       → （之前）很好吃。（过去）
おいしくなかったです。   → （之前）不好吃。（过去否定）
```

#### 连用形（くて形）

连用形用于连接两个形容词，或表示原因：

```
【连接两个性质】
この部屋は広くて、明るいです。
→ 这个房间又宽又亮。

あの映画は面白くて、感動的でした。
→ 那部电影既有趣又感人。

【表示原因（て形表因果）】
頭が痛くて、学校を休みました。
→ 因为头痛，请了学校的假。

忙しくて、電話できませんでした。
→ 因为忙，没能打电话。
```

#### 副词化（く形）

い形容词去掉い，加く，变为副词：

```
大きい → 大きく（变大地）
早い  → 早く  （早地/快地）
上手い → 上手く（熟练地）
強い  → 強く  （强有力地）

大きく変わりました。     → 大幅度改变了。
早く起きてください。     → 请早点起床。
もっと上手く話せるように練習します。→ 练习以便能说得更流利。
```

## 8.3 な形容词的活用

### 完整活用表

| 活用形 | 构成方式 | 例（静かだ） | 例（きれいだ） |
|--------|----------|-------------|--------------|
| 基本形（辞書形） | 词干+だ | 静かだ | きれいだ |
| 肯定现在（礼貌） | 词干+です | 静かです | きれいです |
| 否定现在 | 词干+ではない/じゃない | 静かではない | きれいじゃない |
| 否定现在（礼貌） | 词干+ではありません | 静かではありません | きれいではありません |
| 肯定过去 | 词干+だった（でした） | 静かでした | きれいでした |
| 否定过去 | 词干+ではなかった | 静かではなかった | きれいじゃなかった |
| 连体形（定语） | 词干+な | 静かな部屋 | きれいな花 |
| 连用形（接て） | 词干+で | 静かで | きれいで |
| 副词化 | 词干+に | 静かに | きれいに |

### 详细说明

#### 肯定/否定变换

```
【否定：词干+ではない/じゃない】
静かです    → 静かではありません / 静かじゃないです
きれいです  → きれいではありません / きれいじゃないです
便利です    → 便利ではありません / 便利じゃないです
有名です    → 有名ではありません / 有名じゃないです

（ではない=正式；じゃない=口语）
```

#### 过去/过去否定变换

```
【过去肯定：词干+でした/だった】
静かでした。             → 曾经安静。
きれいでした。           → 曾经漂亮。
有名でした。             → 曾经有名。

【过去否定：词干+ではありませんでした/じゃなかった】
静かではありませんでした。 → 曾经不安静。
きれいじゃなかったです。  → 曾经不漂亮。（口语）
```

```
【完整时态示例：便利（方便）】
便利です。               → 方便。（现在）
便利ではありません。     → 不方便。（现在否定）
便利でした。             → 曾经方便。（过去）
便利ではありませんでした。→ 曾经不方便。（过去否定）
```

#### 连体形（定语位置用な）

```
静かな部屋が好きです。   → 喜欢安静的房间。
きれいな花ですね。       → 真漂亮的花。
有名な俳優が来ます。     → 著名演员来了。
便利な場所に住んでいます。→ 住在方便的地方。

【对比：连体形必须加な，谓语位置不加な】
この部屋は静かです。（谓语，不加な）→ 这个房间安静。
静かな部屋に住みたい。（定语，加な）→ 想住在安静的房间。
```

#### 连用形（て形用で连接）

な形容词的て形是「词干+で」：

```
【连接两个性质】
彼女はきれいで、頭もいいです。
→ 她又漂亮又聪明。

この町は静かで、住みやすいです。
→ 这个城市安静，宜居。

【表示原因】
病気で有名になりました。（不同语境）
不便で困っています。
→ 因为不方便而困扰。
```

#### 副词化（词干+に）

な形容词词干加に变为副词：

```
静かに話してください。   → 请安静地说话。
きれいに書いてください。 → 请写漂亮点。
上手に料理します。       → 烹饪得很好。
親切に教えてくれました。 → 亲切地教了我。
```

## 8.4 形容词的比较级与最高级

日语**没有**像英语那样通过词形变化表达比较（-er/-est），而是通过句型结构表达比较。

### 比较级：AはBより～です（A比B更…）

```
基本句型：
A は B より [形容词] です。
→ A比B更[形容词]。

東京はロサンゼルスより大きいです。
→ 东京比洛杉矶大。

日本語は英語より難しいですか。
→ 日语比英语难吗？

今日は昨日より暖かいです。
→ 今天比昨天暖和。

彼女は私より上手に歌います。
→ 她比我唱得更好。（副词比较）
```

### 二选一：AとBとどちらが～ですか

```
基本句型：
A と B と、どちらが [形容词] ですか。
→ A和B，哪个更[形容词]？

回答：
A のほうが B より [形容词] です。
→ A比B更[形容词]。

【对话示例】
Q：コーヒーと紅茶と、どちらが好きですか。
   → 咖啡和红茶，你更喜欢哪个？

A：コーヒーのほうが（紅茶より）好きです。
   → 咖啡更喜欢一些。

Q：日本と中国と、どちらが人口が多いですか。
   → 日本和中国，哪个人口多？

A：中国のほうが多いです。
   → 中国更多。
```

### 最高级：～の中で一番～です（…中最…）

```
基本句型：
[范围] の中で [比较对象] が 一番 [形容词] です。
→ 在[范围]中，[比较对象]最[形容词]。

クラスの中で田中さんが一番背が高いです。
→ 班级里田中先生最高。

季節の中で夏が一番好きです。
→ 四季中最喜欢夏天。

この店のメニューの中で何が一番おいしいですか。
→ 这家店菜单里哪道菜最好吃？

世界で一番高い山は何ですか。
→ 世界上最高的山是什么？（エベレスト/珠穆朗玛峰）
```

### 与英语比较级的对比

| 特征 | 英语 | 日语 |
|------|------|------|
| 比较级词形变化 | tall → taller → tallest | 高い → 高い（不变） |
| 表达方式 | 词缀变化（-er/-est） | 句型结构（より/一番） |
| 等级数量 | 原级/比较级/最高级 | 只有一种形式，靠句型区分 |

## 8.5 形容词做定语和谓语的区别

### 位置决定形式

| 使用位置 | い形容词 | な形容词 |
|----------|----------|----------|
| 谓语（句末） | 直接用い形 | 词干+です |
| 定语（名词前） | 直接用い形（不变） | 词干+な |

```
【い形容词：两个位置形式相同】
定语：大きい犬がいます。     → 有一只大狗。（大きい+犬）
谓语：この犬は大きいです。   → 这只狗很大。（大きい+です）

【な形容词：两个位置形式不同】
定语：静かな部屋があります。 → 有安静的房间。（静かな+部屋）
谓语：この部屋は静かです。   → 这个房间很安静。（静か+です，无な）
```

### 常见错误

```
× この部屋は静かなです。    （谓语位置不用な）
○ この部屋は静かです。

× 静か部屋に住みたい。      （定语位置必须加な）
○ 静かな部屋に住みたい。

× 大きな犬は（大きい的な形）（注意：大きな是个特殊形式，详见8.6）
```

## 8.6 特殊形容词

### いい / よい 的活用（最重要的特殊形容词）

`いい`（好）是日语中最常用的形容词，但它的活用形**不基于いい，而是基于よい**：

| 活用形 | 正确形式 | 错误形式（常见） |
|--------|----------|-----------------|
| 基本形 | いい / よい | — |
| 否定 | よくない | ~~いくない~~ |
| 过去 | よかった | ~~いかった~~ |
| 过去否定 | よくなかった | ~~いくなかった~~ |
| 副词化 | よく | ~~いく~~ |
| 连用形 | よくて | ~~いくて~~ |

```
【いい/よい的正确活用】
この映画はいいです。         → 这部电影不错。（基本形用いい）
この映画はよくないです。     → 这部电影不好。（否定用よく）
あの映画はよかったです。     → 那部电影很好。（过去用よかった）
よく食べますね。             → 吃得真多呢。（副词用よく）
よくわかります。             → 很明白。（副词）

天気がよくて、散歩しました。 → 天气很好，去散步了。（连用形）
```

**记忆口诀**：`いい`只在肯定现在基本形用，其他所有活用形换成`よ`替代`い`。

### 大きな / 小さな（特殊连体词用法）

`大きい` 和 `小さい` 在定语位置可以有两种形式：

| 形式 | 例 | 语感差异 |
|------|-----|---------|
| 大きい + 名词 | 大きい声 | 客观描述（声音很大） |
| 大きな + 名词 | 大きな声 | 印象/感受（好大一个声音） |

```
大きい問題です。  → 是个大问题。（客观）
大きな問題です。  → 是个大问题。（语感更感叹）
小さい子供がいます。→ 有个小孩子。
小さな子供がいます。→ 有个小孩子。（更口语/亲切）
```

**注意**：`大きな/小さな` 是**连体词**，只能修饰名词，不能用于谓语位置。

### 多い（おおい）和 少ない（すくない）的限制

`多い` 和 `少ない` 不能直接在名词前做定语（与中文"很多的人"结构不同）：

```
× 多い人が来ました。         （错误：不能直接做定语）
○ 人が多く来ました。         （副词化：来了很多人）
○ 多くの人が来ました。       （多く+の：正确定语形式）
○ たくさんの人が来ました。   （使用たくさん替代）

× 少ない時間しかありません。  （限制较多）
○ 時間が少ないです。         （做谓语使用）
○ わずかな時間しかありません。（用わずかな替代）
```

## 本章小结

| 对比项目 | い形容词 | な形容词 |
|----------|----------|----------|
| 词尾特征 | 必以い结尾（活用语尾） | 词尾通常不是活用い |
| 否定形 | ～くない / ～くありません | ～ではない / ～じゃない |
| 过去形 | ～かった | ～でした / ～だった |
| 定语形式 | 不变（直接+名词） | 词干+な |
| 连用形 | 词干+くて | 词干+で |
| 副词化 | 词干+く | 词干+に |
| 特殊形容词 | いい→よ系列活用 | — |

| 比较句型 | 结构 | 含义 |
|----------|------|------|
| 比较级 | AはBより～ | A比B更～ |
| 二选一 | AとBとどちらが～ | A和B哪个更～ |
| 最高级 | ～の中で一番～ | ～中最～ |

日语形容词体系的精妙之处在于：い形容词拥有完整的时态活用，本身就可以作为完整谓语使用（大きい！）；な形容词则通过と、で、に展现描述的灵活性。掌握这两套系统，你就能在日语中精准描绘这个世界的一切颜色。

完成第二部分（第5-8章）的学习，你已经掌握了日语语法的核心骨架：句子结构、助词系统、动词活用、形容词体系。第三部分将进入词类专题，深入讲解名词、副词、助动词等更精细的语法层面。
