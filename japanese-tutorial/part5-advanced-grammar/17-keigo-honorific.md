# 第17章：敬语体系（尊敬語・謙譲語・丁寧語）

> 「言葉は人なり」——语言即人格。日语敬语体系是日本文化中最精密的社会关系镜像，掌握它不仅是语言技能，更是理解日本社会结构的钥匙。

---

## 17.1 日语敬语的文化根基

### 上下関係（等级秩序）

日本社会自古以来以严格的等级秩序为基础。从平安时代的宫廷文化到江户时代的武士制度，再到现代企业文化，等级意识深刻嵌入语言体系。日语敬语正是这种社会结构的语言表达。

| 社会关系维度 | 影响敬语使用的因素 | 示例场景 |
|------------|-----------------|--------|
| 年龢（年齢） | 年长者使用敬语 | 后辈对前辈 |
| 地位（地位） | 职位低者使用敬语 | 下属对上司 |
| 社会关系（社会的立場） | 服务方使用敬语 | 店员对顾客 |
| 亲疏程度（親疏） | 陌生人之间使用敬语 | 初次见面 |

### ウチとソト（内与外）

日语敬语最独特的特征之一是"ウチ-ソト"（内-外）概念。

**ウチ（内）**：自己所属的群体——家人、同一公司的同事等
**ソト（外）**：自己以外的群体——对方、客户、外部人员等

```
关键规则：
- 对"ソト"（外部人）提及"ウチ"（内部）成员时，用谦让语
- 对"ソト"（外部人）谈及"ソト"的人时，用尊敬语

例：
× 「部長は今いらっしゃいます」（对外部人谈自己部长）
○ 「部長は今おります」（谦让语——部长是"ウチ"成员）
```

### 場面と相手（场合与对象）

敬语的选择不仅取决于"谁"，还取决于"在什么场合"。

| 场合（場面） | 相手（对象） | 推荐敬语程度 |
|-----------|-----------|------------|
| 正式商务场合 | 客户/上司 | 最高级敬语 |
| 公司内部 | 上司 | 中级敬语 |
| 公司内部 | 同等同事 | 普通丁寧語 |
| 日常生活 | 邻居/陌生人 | 基本丁寧語 |
| 私人场合 | 朋友/家人 | 普通体 |

## 17.2 敬语的分类体系

### 文化庁の五分類（文部科学省 2007年答申）

2007年日本文化厅将敬语重新规范为五大类：

```
敬語
├── 尊敬語（そんけいご）
│   └── 提升对方或对方相关人员的地位
├── 謙譲語Ⅰ（けんじょうごI）
│   └── 压低说话者，向对方行为对象表示敬意
├── 謙譲語Ⅱ（丁重語）（ていちょうご）
│   └── 压低说话者，向听话者表示礼貌（郑重表达）
├── 丁寧語（ていねいご）
│   └── 直接向听话者表示礼貌（です/ます）
└── 美化語（びかご）
    └── 通过美化表达体现教养（お/ご前缀）
```

### 三分类与五分类的关系

传统教学常用三分类，与五分类对照如下：

| 传统三分类 | 文化厅五分类 | 核心功能 |
|---------|-----------|--------|
| 尊敬語 | 尊敬語 | 提高对方地位 |
| 謙譲語 | 謙譲語Ⅰ + 謙譲語Ⅱ（丁重語） | 降低自己地位 |
| 丁寧語 | 丁寧語 + 美化語 | 礼貌表达 |

## 17.3 尊敬語（そんけいご）

尊敬語用于**提升对方或对方相关人物**的行为、状态、所有物的地位。

### お/ご + 動詞になる（最基本尊敬形式）

**构成：お + 和語動詞の連用形 + になる**

```
お読みになる（读 → 您读）
お書きになる（写 → 您写）
お使いになる（使用 → 您使用）
お持ちになる（拿 → 您拿）
お召し上がりになる（吃/喝 → 您享用）※ 双重敬语，见17.8
```

**构成：ご + 漢語動詞語幹 + になる**

```
ご利用になる（利用 → 您使用）
ご覧になる（看 → 您看）※ 特殊，直接使用
ご出席になる（出席 → 您出席）
ご参加になる（参加 → 您参加）
```

> **注意**：「お/ごになる」不能用于说话人自身的行为。

### 特殊尊敬动词表（最重要，必须记忆）

| 普通形 | 尊敬語 | 含义 |
|-------|-------|-----|
| いる | いらっしゃる | 在、存在 |
| 行く | いらっしゃる / おいでになる | 去 |
| 来る | いらっしゃる / おいでになる / お見えになる | 来 |
| 言う | おっしゃる | 说 |
| 食べる・飲む | 召し上がる | 吃、喝 |
| もらう | お受け取りになる | 收到 |
| する | なさる | 做 |
| 見る | ご覧になる | 看 |
| 知る | ご存知（だ） | 知道 |
| 思う | ～とおっしゃる / お考えになる | 认为 |
| 着る・履く | お召しになる | 穿 |
| 寝る | お休みになる | 睡觉 |
| 死ぬ | お亡くなりになる / 他界される | 去世 |
| 読む | お読みになる / 拝読なさる | 阅读 |
| 聞く | お聞きになる / お耳に入る | 听、问 |

```
使用例：
先生はいらっしゃいますか。（老师在吗？）
社長は何とおっしゃいましたか。（社长说了什么？）
お客様はもう召し上がりましたか。（顾客您已经用餐了吗？）
部長はご存知ですか。（部长您知道吗？）
```

### 受身形的尊敬用法（受動の尊敬語）

在日语中，被动形也可表达尊敬，语气比「お～になる」稍轻：

```
构成：動詞の受動形（られる/れる）

読む → 読まれる（您读）
書く → 書かれる（您写）
帰る → 帰られる（您回去）
出席する → 出席される（您出席）
考える → 考えられる（您考虑）
```

| 敬语程度 | 表达形式 | 例句 |
|---------|---------|-----|
| 最高 | お読みになる | 先生は本をお読みになっています |
| 高 | 読まれる | 先生は本を読まれています |
| 普通 | 読む | 先生は本を読んでいます |

### お/ご + 名詞/形容詞（尊敬的物品・状态表达）

```
お名前（您的名字）　　ご住所（您的地址）
お仕事（您的工作）　　ご家族（您的家人）
お体（您的身体）　　　ご意見（您的意见）
お忙しい（您很忙）　　ご健康（您的健康）
お元気（您的精神状态）ご多忙（您的繁忙）
```

> **规律**：和语词（大和言葉）前加「お」，汉语词（漢語）前加「ご」。例外：「お電話」「お食事」等常用汉语词也加「お」。

## 17.4 謙譲語（けんじょうご）

謙譲語用于**降低说话者（或说话者一方）**的行为、状态，从而间接提升对方地位。

### お/ご + 動詞する/いたす（基本谦让形式）

**构成：お + 和語動詞の連用形 + する/いたす**

```
お待ちする（等候您）　→ お待ちいたします（更礼貌）
お届けする（为您送达）→ お届けいたします
お持ちする（为您拿）　→ お持ちいたします
お教えする（告诉您）　→ お教えいたします
```

**构成：ご + 漢語動詞語幹 + する/いたす**

```
ご案内する（为您引路）→ ご案内いたします
ご連絡する（与您联系）→ ご連絡いたします
ご説明する（向您说明）→ ご説明いたします
ご報告する（向您汇报）→ ご報告いたします
```

### 特殊谦让动词表（必须记忆）

| 普通形 | 謙譲語 | 含义 |
|-------|-------|-----|
| いる | おる | 在、存在（謙譲語Ⅱ/丁重語） |
| 行く | 参る（まいる） | 去 |
| 来る | 参る（まいる） | 来 |
| 言う | 申す（もうす） | 说 |
| する | いたす | 做 |
| もらう | いただく | 得到、承蒙 |
| あげる | 差し上げる（さしあげる） | 给予（对上） |
| くれる | くださる | 给予（对方给我） |
| 見る | 拝見する（はいけんする） | 看、拜读 |
| 知る/思う | 存じる（ぞんじる） | 知道、认为 |
| 聞く・訪問する | 伺う（うかがう） | 听、询问、拜访 |
| 食べる・飲む | いただく | 吃、喝 |
| 借りる | 拝借する（はいしゃくする） | 借 |
| 読む | 拝読する（はいどくする） | 阅读 |
| 会う | お目にかかる | 见到 |
| 見せる | お目にかける / ご覧に入れる | 让您看 |
| 受け取る | 頂戴する（ちょうだいする） | 收到 |

```
使用例：
私はこの件を担当いたします田中と申します。
（我是负责此事的田中。）

先ほどのご説明を拝聴いたしました。
（刚才的说明我恭听了。）

明日3時にお伺いしてもよろしいでしょうか。
（明天3点我可以拜访您吗？）

お名刺を頂戴できますでしょうか。
（可以请您给我一张名片吗？）
```

### 謙譲語Ⅰ vs 謙譲語Ⅱ（丁重語）的关键区别

这是2007年文化厅新分类的重要区分：

| 分类 | 功能 | 代表词 | 例句 |
|-----|------|-------|-----|
| 謙譲語Ⅰ | 通过降低自己来尊敬行为对象 | 伺う/拝見する/申し上げる | お客様に資料をお持ちする |
| 謙譲語Ⅱ（丁重語） | 降低自己向听话者表示礼貌，不需要行为对象 | おる/参る/申す/いたす | 私はここにおります |

```
对比：
謙譲語Ⅰ：社長にご報告申し上げます。
（向社长汇报——行为对象是社长，社长是尊敬对象）

謙譲語Ⅱ：私は営業部の田中と申します。
（我叫营业部的田中——没有特定的尊敬对象，只是礼貌自我介绍）

謙譲語Ⅱ：只今参ります。
（我马上来——向听话者礼貌表达，非对特定人行敬）
```

## 17.5 丁寧語与美化語

### です/ます 体系（丁寧語的核心）

丁寧語不针对特定人物，而是整体提升话语礼貌程度：

| 形式 | 肯定 | 否定 | 过去 | 过去否定 |
|-----|------|------|------|--------|
| 動詞 | 食べます | 食べません | 食べました | 食べませんでした |
| 形容詞（い） | 高いです | 高くないです | 高かったです | 高くなかったです |
| 形容詞（な）/名詞 | 便利です | 便利ではありません | 便利でした | 便利ではありませんでした |

### 美化語（びかご）——教养的语言表达

美化語是通过在词汇前加「お/ご」来美化语言，体现说话者的教养，并非针对特定人物：

```
お水（水）　　　お茶（茶）　　　お酒（酒）
お花（花）　　　お菓子（点心）　お料理（料理）
ご飯（米饭/饭食）お天気（天气）　お化粧（化妆）
お風呂（浴室）　お金（钱）　　　お手洗い（洗手间）
```

> **注意**：美化語中的「お/ご」已固化，失去尊敬功能。例如「ご飯」已是日常词汇，不代表对任何人的尊敬。但对自己的行为说「お金をいただく」中的「お金」仍有美化效果。

### 美化語 vs 尊敬語中お/ご的区别

```
お名前（尊敬語）：您的名字 → 指对方所有物，表尊敬
お水（美化語）：水 → 仅美化，不指向特定人
ご住所（尊敬語）：您的地址 → 指对方地址，表尊敬
ご飯（美化語）：饭 → 已完全词汇化，无尊敬功能
```

## 17.6 敬語の使い分け（敬语场景使用指南）

### 职场敬语的三层关系

```
【对外部客户（取引先/お客様）】
・最高级敬语，谦让语用于自己和公司同事（包括上司）
・不能对客户使用谦让语

【对公司上司（上司）】
・敬语使用尊敬语
・谈及上司时对外人使用谦让语

【对公司同事（同僚）】
・视亲密程度，一般使用です/ます
・内部会议可用普通体
```

**对外部客户谈及内部上司的错误示例：**

```
× 社長が「来る」とおっしゃいました。（向外部人员谈内部社长不能用尊敬语）
○ 社長が「来る」と申しておりました。（用谦让语降低内部社长地位）

× 部長はいらっしゃいます。（向外部人员谈内部部长不能用尊敬语）
○ 部長はおります。（用丁重语表达内部人员存在）
```

### 电话敬语（電話の敬語）

```
【接电话（受電）】
「はい、○○会社でございます。」（您好，这里是○○公司。）
「少々お待ちくださいませ。」（请稍等。）
「ただいま担当者に代わります。」（我马上帮您转接负责人。）
「申し訳ございませんが、只今外出しております。」（非常抱歉，他/她现在不在。）
「よろしければ、ご伝言を承りましょうか。」（如果方便的话，要我转告吗？）

【打电话（発電）】
「お世話になっております。○○会社の田中と申します。」
（承蒙关照。我是○○公司的田中。）
「○○様はいらっしゃいますでしょうか。」（请问○○先生/女士在吗？）
「お取り込み中のところ、失礼いたします。」（打扰您在忙，非常抱歉。）
```

### 邮件敬语（メールの敬語）

```
【开头问候】
「お世話になっております。」（承蒙关照。——商务常用开场白）
「いつもご支援いただき、ありがとうございます。」（感谢您一贯的支持。）
「突然のご連絡、失礼いたします。」（突然联系，非常抱歉。）

【正文常用】
「ご確認いただけますでしょうか。」（可以请您确认一下吗？）
「ご検討のほど、よろしくお願いいたします。」（请您考虑，谢谢。）
「ご不明な点がございましたら、お気軽にお申し付けください。」
（如有不明之处，请随时告知。）

【结尾问候】
「何卒よろしくお願いいたします。」（请多关照。）
「以上、よろしくお願い申し上げます。」（以上，请多关照。）
「お返事をお待ち申し上げております。」（期待您的回复。）
```

### 二重敬語（过度敬语）与常见错误

**二重敬語**指在同一个词上重复叠加敬语，是错误用法：

```
【二重敬語的错误示例】
× おっしゃられました（おっしゃる本身是尊敬语，再加られる是叠加）
○ おっしゃいました

× おいでになられる（おいでになる已是尊敬语，再加られる）
○ おいでになる

× 拝見させていただく（拝見是谦让语，させていただく也是谦让，叠加）
○ 拝見いたします / 拝見します

× お召し上がりになられる
○ お召し上がりになる / 召し上がる
```

**其他常见敬语错误：**

```
× ご苦労様です（对上司使用——ご苦労様は目下の人への労い）
○ お疲れ様です（对上司使用的正确表达）

× どうぞ、いただいてください（いただく是谦让语，不能命令对方用）
○ どうぞ、召し上がってください

× お客様がお見えになられました（二重敬語）
○ お客様がお見えになりました
```

## 17.8 バイト敬語・マニュアル敬語（现代错误敬语）

近年来，尤其在服务业年轻员工中，流行一些不规范的敬语表达，俗称「バイト敬語」（兼职敬语）或「マニュアル敬語」（手册敬语）：

### 常见错误表达

```
【よろしかったでしょうか】
× 「ご注文はよろしかったでしょうか？」（您点好了吗？）
→ 确认现在状态不应用过去式
○ 「ご注文はよろしいでしょうか？」

【～のほうがよろしかったでしょうか】
× 「レシートのほうはよろしかったでしょうか？」
○ 「レシートはご入用ですか？」/ 「レシートはいかがですか？」

【～になります】（物品不会"变成"某物）
× 「こちらが本日のメニューになります。」（这是今天的菜单。）
→ 静态存在说「です」，不应说「になります」
○ 「こちらが本日のメニューでございます。」

【千円からお預かりします】（から是不必要的）
× 「一万円からお預かりします。」（收您一万日元。）
○ 「一万円お預かりします。」

【ご注文のほうは】（のほう表示方向/比较，此处不适当）
× 「ご注文のほうはいかがですか？」
○ 「ご注文はいかがですか？」
```

### 为什么会产生バイト敬語？

1. **模板化教育**：服务手册过于简化，不教原理，只教句型
2. **敬语回避心理**：为了避免失礼，过度使用敬语形式
3. **类推错误**：将正确的过去式确认（「お名前はAでよろしかったでしょうか」在确认已知信息时可用）错误扩展到未确认场景

## 练习

### 练习一：选择正确的敬语形式

1. 社長が（　）のでしょうか？（来る）
   a. 来られます　b. いらっしゃいます　c. 参ります

2. 先ほどの書類を（　）いただけますか？（読む）
   a. 拝読して　b. ご覧になって　c. 読まれて

3. お客様のご住所を（　）してもよろしいでしょうか？（確認する）
   a. 確認　b. ご確認　c. お確認

4. 田中部長が「明日来る」と（　）。（言う）
   a. おっしゃいました　b. 申されました　c. 言われました

### 练习二：改正错误敬语

1. 「先生、ご苦労様でした。」
2. 「社長はいらっしゃいますか？」（対外部への電話で）
3. 「こちらがメニューになります。」
4. 「おっしゃられた通りにいたします。」

### 练习三：翻译以下敬语句

1. 我已经拜读了您发来的资料。
2. 明天我将前往贵公司拜访。
3. 如果不明白的话，请随时告知。
4. 社长现在不在（对外部客户说）。

### 本章小结

| 敬语种类 | 作用 | 核心形式 | 记忆要点 |
|--------|------|---------|--------|
| 尊敬語 | 提升对方 | お/ごになる + 特殊动词 | 对方的行为用尊敬语 |
| 謙譲語Ⅰ | 降低自己（有对象） | お/ごする + 特殊动词 | 自己行为，且有尊敬对象 |
| 謙譲語Ⅱ（丁重語） | 降低自己（无特定对象） | おる/参る/申す | 自己行为，无特定对象 |
| 丁寧語 | 整体礼貌化 | です/ます | 适合所有礼貌场合 |
| 美化語 | 提升语言美感 | お/ご前缀 | 非针对特定人 |

掌握日语敬语的核心在于**理解"谁在对谁做什么"**，而不是机械记忆句型。始终思考：我与对方的关系是什么？我在谈论谁的行为？这是对内（ウチ）还是对外（ソト）的表达？
