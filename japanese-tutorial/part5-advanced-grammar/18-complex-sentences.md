# 第18章：复合句与接续表达

> 「文は人なり、接続は論理なり」——句子展示思想，接续揭示逻辑。掌握日语复合句的接续体系，是从"说日语"迈向"用日语思考"的关键一步。

---

## 18.1 接续助词与接续表达的体系

### 什么是接续（接続）

接续是连接两个或多个分句、表达它们之间逻辑关系的语法手段。日语的接续表达极为丰富，从简单的助词到复杂的接续性名词句型，形成了一个多层次体系。

```
接続の体系
├── 接続助词（接続助詞）：て・が・けれど・ので・から・のに・ても 等
├── 接続性文型（接続的文型）：～ために・～ことから・～上に 等
├── 接続詞（独立接续词）：しかし・だから・それに・ところが 等
└── 複合助词（複合助詞）：～にもかかわらず・～としても 等
```

### 接续表达的分类框架

| 逻辑关系 | 日语分类 | 代表形式 |
|--------|---------|---------|
| 因果（順接） | 原因・理由 | から・ので・ために・おかげで |
| 转折（逆接） | 逆接 | が・のに・にもかかわらず |
| 并列（並列） | 並列・添加 | し・上に・だけでなく |
| 时间（時間） | 時間関係 | てから・うちに・たとたん |
| 条件（条件） | 条件・仮定 | としても・さえ～ば・限り |

## 18.2 顺接（因果关係）

### ので vs から（原因：客观 vs 主观——进阶对比）

这是日语学习者长期困惑的区分，在此做深度分析：

| 项目 | ので | から |
|-----|------|------|
| 语感 | 客观、自然因果 | 主观判断、个人理由 |
| 文体 | 礼貌、书面、说明性 | 口语、随意、主张性 |
| 结句要求 | 不能接命令/请求（较自然） | 可接命令/请求句 |
| 说话人态度 | 中立陈述 | 强调理由、辩解语感 |

```
【客观事实作为理由 → ので】
雨が降っているので、試合は中止になりました。
（因为下雨，比赛取消了。）→ 雨是客观事实，不带主观色彩

【个人主张/请求的理由 → から】
明日試験があるから、今日は早く寝ます。
（因为明天有考试，今天早点睡。）→ 说话人的个人安排

【请求句中只能用から（不用ので）】
○ 荷物が重いから、少し手伝ってください。（请帮我拿一下，因为行李很重。）
△ 荷物が重いので、少し手伝ってください。（语法上可以，但请求语感稍弱）

【正式道歉/说明 → ので更合适】
ただいま混雑しておりますので、少々お待ちくださいませ。
（现在比较拥挤，请稍候。）→ 客观说明，礼貌
```

### ため（に）（正式书面原因/目的）

「ため（に）」具有两种用法，须区分：

```
【原因用法（名詞/動詞た形 + ため）】
嵐のため、電車が止まっています。
（由于暴风雨，电车停运了。）

事故のために、道路が渋滞しています。
（因为事故，道路堵塞。）

【目的用法（動詞辞書形/名詞の + ため）】
日本語を上達させるために、毎日練習しています。
（为了提高日语，每天练习。）

健康のために、野菜をたくさん食べています。
（为了健康，大量食用蔬菜。）
```

> **注意**：「ため」的原因用法多用于客观、正式的书面语，「ために」的目的用法主语须与主句一致（否则意义不通）。

### おかげで vs せいで（积极原因 vs 消极原因）

```
【おかげで——积极原因，结果是正面的】
先生のおかげで、試験に合格することができました。
（托老师的福，我通过了考试。）

天気がよかったおかげで、運動会は大成功でした。
（多亏天气好，运动会大获成功。）

【せいで——消极原因，结果是负面的，带责怪语气】
雨のせいで、試合が中止になった。
（因为下雨，比赛取消了。——暗含对雨的不满）

あなたのせいで、遅刻してしまった。
（都是因为你，我迟到了。——明显指责语气）

【对比记忆】
○ 雨のおかげで、涼しくなった。（多亏下雨变凉快了——正面）
○ 雨のせいで、洗濯物が乾かない。（因为下雨衣服没干——负面）
```

### によって/により（由于、因…而）

```
【原因（多用于书面/正式场合）】
地震によって、多くの建物が倒壊した。
（由于地震，许多建筑物倒塌了。）

病気により、入院しております。
（因病住院。）——より是より的书面形，更正式

【手段/方法（由…来做）】
この問題は話し合いによって解決できる。
（这个问题可以通过协商来解决。）

【施为者（被动句中）】
この小説はシェークスピアによって書かれた。
（这部小说是莎士比亚写的。）
```

### ことから（从…这一事实来看）

「ことから」表示以某事实为判断依据：

```
この地域は雨が少ないことから、農業には向いていない。
（从该地区降雨量少这一事实来看，不适合农业。）

形が三角形に似ていることから、「おにぎり山」と呼ばれている。
（因为形状像三角形，所以被称为"饭团山"。）
```

## 18.3 逆接（転折関係）

### が/けど/けれども（普通转折的程度差异）

```
【正式程度：けれども ＞ けれど ＞ が ＞ けど】
この映画はよかったけれども、少し長すぎた。（书面/正式）
この映画はよかったが、少し長すぎた。（中性）
この映画はよかったけど、少し長すぎた。（口语）

【が在句末表示铺垫（结句省略）】
よろしければ、ご連絡いただけると嬉しいのですが。
（如果方便的话，希望能联系我……）——暗示请求，委婉
```

### のに（期待落空——"明明…却"）

「のに」表示结果违反说话人的期待，含有不满、遗憾、惊讶等情绪：

```
【不满/遗憾】
あんなに練習したのに、試験に失敗した。
（明明练习了那么多，却考试失败了。）——遗憾

こんなに頑張っているのに、誰も認めてくれない。
（明明这么努力，却没人认可。）——不满

【惊讶】
彼はもう80歳なのに、とても若く見える。
（他明明已经80岁了，却看起来很年轻。）——惊讶

【用法限制：结句不能是命令/请求句】
× 疲れているのに、手伝ってください。（逻辑矛盾）
○ 疲れているのに、まだ働いている。
```

### にもかかわらず（尽管…仍然）

```
書面语，强调让步关系，语气比「のに」更强：

悪天候にもかかわらず、大勢の人が集まった。
（尽管天气恶劣，仍聚集了大批人。）

彼は病気にもかかわらず、出席した。
（他尽管生病，仍然出席了。）

反対意見が多いにもかかわらず、計画は実行された。
（尽管有很多反对意见，计划仍被执行了。）
```

### ものの/とはいえ（虽说…但是）

```
【ものの——承认前提，但后文有出乎意料的内容】
日本語を三年勉強したものの、まだ会話が苦手だ。
（虽说学了三年日语，但会话仍然不擅长。）

やる気はあるものの、時間が足りない。
（虽然有干劲，但时间不够。）

【とはいえ——承认前提，提出但书】
便利とはいえ、スマートフォンにも欠点はある。
（虽说方便，但智能手机也有缺点。）

彼はリーダーとはいえ、完璧ではない。
（虽说他是领导，但也不是完美的。）
```

### ところが（然而/不料——意外转折）

```
「ところが」表示结果与预期完全相反，强调意外性：

合格できると思っていた。ところが、不合格だった。
（以为能通过的。然而，结果没通过。）

天気予報では晴れだと言っていた。ところが、大雨が降った。
（天气预报说晴天。不料，下起了大雨。）

対比：
が/けど → 普通转折，说话人已有预期
のに → 结果违背期待，含情绪
ところが → 强烈意外，叙事性，常用于故事转折
```

## 18.4 并列与添加

### し（列举理由，复数原因叠加）

```
构成：文（普通体）＋し、文（普通体）＋し

この店は安いし、おいしいし、また来たい。
（这家店既便宜，又好吃，想再来。）

彼女は頭がいいし、やさしいし、努力家だし、本当に尊敬する。
（她既聪明，又温柔，还努力，真的很尊敬她。）

【结句可以是正面也可以是负面】
遅刻するし、ミスするし、使えない部下だ。
（既迟到，又犯错，是个没用的下属。）——负面
```

### 上（に）（不仅…而且，前后同向叠加）

```
【同向添加，且后项比前项更进一步】
この製品は安い上に、品質もいい。
（这个产品不仅便宜，而且质量也好。）

彼は優秀な上に、人柄もいい。
（他不仅优秀，而且人品也好。）

失業した上に、病気にもなった。
（不仅失业，还生病了。）——负面叠加

対比：
～し～し → 平列式列举，没有前后轻重
～上に → 叠加式，后项往往更重要或更出人意料
```

### だけでなく～も（不仅…而且…）

```
彼は英語だけでなく、フランス語も話せる。
（他不仅会英语，还会法语。）

体の健康だけでなく、心の健康も大切だ。
（不仅身体健康，心理健康也很重要。）

【书面形：のみならず】
経済的な問題のみならず、社会的な問題でもある。
（不仅是经济问题，也是社会问题。）
```

### はもちろん/はもとより（不用说…当然也…）

```
【はもちろん——口语/书面均可】
日本語はもちろん、英語も話せます。
（日语自不用说，英语也会说。）

【はもとより——书面语，更正式】
基礎はもとより、応用力も必要だ。
（基础自不用说，应用能力也是必要的。）
```

### ばかりか/ばかりでなく（不仅如此，甚至还）

```
【ばかりか/ばかりでなく——后项比前项更进一步，常有意外/夸张语气】
彼は謝らなかったばかりか、逆に怒り出した。
（他不仅没道歉，反而发起火来。）

この薬は効かなかったばかりか、副作用まで出た。
（这个药不仅没效，甚至还出现了副作用。）

対比：
だけでなく → 中性并列添加
ばかりか → 后项往往令人意外，常有负面语气
```

## 18.5 时间关係

### ～てから / ～た後で（做完之后）

```
【てから——强调先后顺序，前行为是后行为的前提】
手を洗ってから、食事をする。
（洗手之后再吃饭。）——顺序强调

卒業してから、就職する。
（毕业之后再就业。）

【た後で——单纯时间顺序，前行为完了后的一段时间后】
食事をした後で、少し休んだ。
（吃完饭之后，稍微休息了一下。）

対比：
てから → 前行为是后行为的必要条件，顺序不可颠倒
た後で → 时间上的先后，顺序可颠倒不影响意义
```

### ～うちに（趁着…期间）

```
【表示在某状态持续期间完成某事，含"趁机"的语感】
若いうちに、たくさん経験を積みなさい。
（趁年轻，多积累经验。）——趁着年轻的状态

忘れないうちにメモしておこう。
（趁还没忘，先记下来。）

【注意：结句不能是过去式（应该是将要做的事）】
× 忘れないうちにメモした。（不自然）
○ 忘れないうちにメモしておいた。（尚可）
```

### ～際（に）（在…之际，正式）

```
【书面/正式用语，表示"在某场合、某时机"】
ご不明な点がございましたら、遠慮なくお申し出ください。
（如有不明之处，请随时提出。）

ご来店の際には、ぜひお声がけください。
（光临本店时，请一定打声招呼。）

帰国の際に、お土産を買ってきました。
（回国时，买了伴手礼。）
```

### ～たとたん（に）（刚…就立刻…，意外或自然发生）

```
【刚一发生A，B就紧接着发生——有意外或即时性语感】
ドアを開けたとたん、猫が飛び出してきた。
（刚一开门，猫就跑出来了。）

彼の名前を聞いたとたん、顔が赤くなった。
（刚听到他的名字，脸就红了。）

【注意：前句必须是た形，结句不能是意志/命令】
× 外に出たとたん、雨が降ってください。（不合逻辑）
```

### ～次第（一…就，郑重/计划性）

```
【一完成就立即做下一件事，书面语，常用于商务场合】
準備ができ次第、出発します。
（一准备好就出发。）

資料が届き次第、ご連絡いたします。
（资料一到，立即联系您。）

【接续方式：動詞連用形 + 次第】
確認でき次第 / 到着し次第 / 検討でき次第
```

### ～て以来（自从…以来）

```
【表示从某时间点开始，持续到现在】
日本に来て以来、一度も帰国していない。
（自从来日本以来，一次都没回过国。）

彼と別れて以来、会っていない。
（自从和他分手以来，就没见过了。）

【注意：以来用于强调持续至今的状态，不用于短暂行为】
```

## 18.6 条件・假定的高级表达

### ～としても/～にしても（即使…也）

```
【让步条件——即使A成立，B也不受影响】
たとえ失敗したとしても、諦めない。
（即使失败了，也不放弃。）

仕事が忙しいにしても、連絡くらいはできるはずだ。
（即使工作忙，也至少应该能联系吧。）

対比：
としても → 假设性让步（还未发生的情况）
にしても → 接受现实的让步（已发生或可能性高）
```

### ～ないことには（如果不…的话，就无法…）

```
【强条件——前件是后件的绝对必要条件】
実際にやってみないことには、わからない。
（如果不实际尝试的话，就不会明白。）

資金がないことには、事業を始められない。
（如果没有资金的话，就无法开始事业。）

【接续方式：動詞ない形 + ことには / 名詞・形容詞 + でないことには】
```

### ～さえ～ば（只要…就，充分条件）

```
【提示最小/唯一必要条件】
お金さえあれば、何でもできる。
（只要有钱，什么都能做。）

健康でさえあれば、それで十分だ。
（只要健康，那就足够了。）

君さえいてくれれば、他には何もいらない。
（只要你在，其他什么都不需要。）

【接续方式：
名詞 + さえ + あれば/なければ
动词て形 + さえ + いれば
形容词词根 + で + さえ + あれば】
```

### ～限り（只要…就 / 在…范围内）

```
【只要条件持续，结果就持续】
生きている限り、希望を持ち続ける。
（只要活着，就要持续抱有希望。）

私が知っている限り、彼は悪い人ではない。
（就我所知，他不是坏人。）

【否定形：ない限り = 除非……否则……】
努力しない限り、成功はできない。
（除非努力，否则不会成功。）

諦めない限り、夢は実現できる。
（只要不放弃，梦想就能实现。）
```

## 18.7 N2-N1级别文法句型精选30个

以下按接续方式分类，均为N2-N1考试重点：

### 动词普通形接续

| 文型 | 含义 | 例句 |
|-----|------|-----|
| ～にしたがって | 随着…而… | 時代の変化にしたがって、言葉も変わる |
| ～につれて | 随着…（自然变化） | 年を取るにつれて、体力が落ちる |
| ～に伴って | 伴随着… | 経済成長に伴って、生活が豊かになった |
| ～を通じて/を通して | 通过…（手段/期间） | 経験を通じて学ぶ |
| ～をもとに | 以…为基础 | 実話をもとに作られた映画 |
| ～をきっかけに | 以…为契机 | 病気をきっかけに、生活を見直した |

### 名词接续

| 文型 | 含义 | 例句 |
|-----|------|-----|
| ～に関して/に関する | 关于… | 環境問題に関する研究 |
| ～に対して/に対する | 对于… | 子供に対してやさしく接する |
| ～について/についての | 关于…（话题） | 日本文化について話す |
| ～に基づいて | 基于… | 事実に基づいて判断する |
| ～に応じて | 根据… | 状況に応じて判断する |
| ～に反して | 违反/与…相反 | 予想に反して失敗した |

### 动词て形/た形接续

| 文型 | 含义 | 例句 |
|-----|------|-----|
| ～てはじめて | 做了之后才… | 経験してはじめてわかる |
| ～たところ | 做了之后发现… | 調べたところ、ミスが見つかった |
| ～たあげく | 最终（不好的结果） | 悩んだあげく、断った |
| ～たばかりに | 就因为…（负面） | 一言言ったばかりに、関係が壊れた |
| ～たとたん | 刚…就… | ドアを開けたとたん猫が出た |

### 形容词/名词接续

| 文型 | 含义 | 例句 |
|-----|------|-----|
| ～わりに | 虽然…但（不符合预期） | 努力した割に成果が出ない |
| ～だけあって | 不愧是…果然… | プロだけあって、うまい |
| ～だけに | 正因为…所以… | 期待が大きかっただけに、失望も大きい |
| ～ことから | 从…可以得出 | 色が白いことから「雪山」と呼ばれる |
| ～にしては | 就…来说（意外） | 子供にしては、よくできた |

### 意志形接续

| 文型 | 含义 | 例句 |
|-----|------|-----|
| ～（よ）うとする | 正要…/试图… | 出かけようとしたとき、電話が鳴った |
| ～（よ）うとしない | 不肯…/不愿… | 彼は話し合おうとしない |
| ～（よ）うとも | 即使…也 | 何があろうとも、諦めない |

## 练习

### 练习一：选择正确的接续表达

1. あんなに頑張った（　）、失敗してしまった。
   a. から　b. ので　c. のに　d. ために

2. 雨（　）、試合は中止になった。
   a. おかげで　b. せいで　c. ために　d. によって

3. 日本語が上手になる（　）、毎日練習することが大切だ。
   a. ためには　b. うちには　c. のには　d. ことには

4. 準備が整い（　）、出発します。
   a. てから　b. 次第　c. たとたん　d. ながら

### 练习二：改写句子（用指定文型）

1. 「彼は病気だった。でも、来た。」（にもかかわらず を使って）
2. 「田中さんは英語ができる。それから中国語もできる。」（だけでなく を使って）
3. 「もし努力しなければ、成功しない。」（ない限り を使って）

### 练习三：翻译以下句子

1. 随着年龄增长，新技术越来越难以跟上。
2. 就我所知，他没有做过那样的事。
3. 不仅天气好，而且景色也很美，真是完美的一天。
4. 刚踏出家门，就开始下雨了。

### 本章小结

| 逻辑关系 | 核心文型 | 使用要点 |
|--------|---------|--------|
| 客观原因 | ので・ために | 礼貌说明，书面可用 |
| 主观原因 | から | 口语，可接请求/命令 |
| 情绪化转折 | のに | 含不满/遗憾，不用于命令句 |
| 意外转折 | ところが | 叙事性，强调出乎意料 |
| 让步转折 | にもかかわらず | 书面，正式强让步 |
| 叠加并列 | 上に・ばかりか | 后项程度更强 |
| 充分条件 | さえ～ば | 最小/唯一条件 |
| 必要条件 | ないことには | 绝对前提条件 |

接续表达的学习关键不在于死记硬背，而在于**理解其背后的逻辑关系和情感色彩**。建议通过阅读日语新闻社论、小说来感受不同接续表达的实际使用场景，逐渐内化这些语感差异。
