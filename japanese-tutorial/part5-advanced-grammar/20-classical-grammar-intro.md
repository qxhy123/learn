# 第20章：文語（古典语法）入门

> 「古典を学ぶことは、現代語の深みを知ることである。」
> ——学习古典，是为了理解现代语言的深度。

---

## 引言

许多日语学习者在达到中高级水平后，会发现自己在阅读文学作品、歌词、俳句，乃至正式公文时，频繁遭遇一些"看不懂"的表达——它们不像现代日语，却又似乎有规律可循。这就是**文語（ぶんご）**，即古典日语。

古典语法并非博物馆里的陈列品。它活在每一句「言わずもがな」里，活在每一个「～ざるを得ない」里，活在每首演歌和J-pop的歌词里。理解文語，是打开日语深度的一把钥匙。

本章不要求你掌握《源氏物語》的全文翻译能力，而是帮你建立一个"识别框架"——能看懂，能感受，能在现代语境中活用。

## 20.1 为什么现代日语学习者需要了解古典语法

### 20.1.1 日常成语与惯用句中的文語残留

现代日本人每天都在使用大量源自文語的表达，却往往不自知。以下是一些高频例子：

| 现代表达 | 文語来源 | 含义 |
|----------|----------|------|
| さもありなん | 「さもあり」+「なん（なむ）」 | 那也是理所当然的（强调推测） |
| 言わずもがな | 「言わず」+「もがな」 | 不言而喻，不说也罢 |
| よしなに | 「よし（由）」+「なに（古形副詞）」 | 随您处理，多关照（书信结尾） |
| いかんせん | 「いかん（如何）」+「せん（せむ→しよう）」 | 无可奈何，怎奈 |
| さりとて | 「さり（然り）」+「とて」 | 即便如此，话虽如此 |
| さながら | 「さながら（然ながら）」 | 宛如，一如……般 |
| いかんともしがたい | 「いかんとも」+「しがたい」 | 无可奈何，难以改变 |
| もとより | 「元より」（文語的读法） | 本来，当然 |

这些表达在商务日语、新闻报道和正式演讲中极为常见。不懂文語背景，很难准确理解其语感和使用场合。

### 20.1.2 文学与歌词中的文語

日本的演歌（えんか）、J-pop乃至动漫主题曲，都大量借用文語营造古典美感：

```
例：美空ひばり「川の流れのように」
「知らず知らず歩いてきた　細く長いこの道」
→「知らず知らず」源自文語副詞用法，意为"不知不觉"

例：BUMP OF CHICKEN「天体観測」
「まだそこにいるか　声を枯らして呼んだ」
→「枯らして」用了文語性的汉语训读语感

例：古典的な歌詞の例（百人一首より）
「ちはやぶる　神代もきかず　竜田川
　からくれなゐに　水くくるとは」（在原業平）
```

### 20.1.3 正式文体中的文語残留

法律条文、公文书、学术论文中保留了大量文語痕迹：

```
法律文例：
「何人も、法律の定める手続によらなければ、
　その生命若しくは自由を奪はれ、
　又はその他の刑罰を科せられない。」
（日本国憲法第31条）

→「奪はれ」是历史假名遣い（现代写法：奪われ）
→「科せられない」保留了文語被动助动词「る・らる」的痕迹
```

## 20.2 文語与口语的主要差异概览

### 20.2.1 动词活用体系对比

现代日语动词分为**五段/上一段/下一段/カ変/サ変**，而文語动词活用体系更复杂：

| 文語活用种类 | 特征 | 代表动词 | 现代对应 |
|-------------|------|----------|----------|
| 四段活用（よだんかつよう） | ア・イ・ウ・エ（エ）行变化 | 書く、読む | 现代五段 |
| 上二段活用（かみにだん） | イ・ウ段变化 | 起く（起きる） | 现代上一段 |
| 下二段活用（しもにだん） | エ・ウ段变化 | 受く（受ける） | 现代下一段 |
| 上一段活用（かみいちだん） | イ段固定 | 見る、着る | 现代上一段（保留） |
| 下一段活用（しもいちだん） | エ段固定 | 蹴る のみ | 現代下一段 |
| カ行変格活用 | 特殊：来（こ・き・く・くる・くれ・こよ） | 来（く） | 现代カ変 |
| サ行変格活用 | 特殊：す（せ・し・す・する・すれ・せよ） | す | 现代サ変 |
| ナ行変格活用 | な・に・ぬ・ぬる・ぬれ・ね | 死ぬ、往ぬ | 部分消失 |
| ラ行変格活用 | ら・り・り・る・れ・れ | あり、をり、はべり | 消失/統合 |

**六种活用形对照表（以四段動詞「書く」为例）：**

| 活用形 | 文語 | 接续 | 现代对应 |
|--------|------|------|----------|
| 未然形 | 書か | 否定・意志「ず・む」 | 書か（ない） |
| 連用形 | 書き | 助動詞・名詞 | 書き |
| 終止形 | 書く | 文末 | 書く |
| 連体形 | 書く | 体言修飾 | 書く |
| 已然形 | 書け | 「ば・ど・ども」 | 書け（ば）仮定形 |
| 命令形 | 書け | 命令 | 書け |

注意：文語中**終止形≠連体形**（对于某些助动词和形容词），这一点极其重要。

### 20.2.2 文語助动词体系

| 助动词 | 意义 | 接续 | 例句 |
|--------|------|------|------|
| き | 过去（直接体验） | 連用形 | 見き（亲眼看到了） |
| けり | 过去（传闻/感叹） | 連用形 | 見けり（据说看到了/啊，看到了） |
| つ | 完了（意志性） | 連用形 | 書きつ（写完了） |
| ぬ | 完了（自然成立） | 連用形 | 散りぬ（散落了） |
| たり | 存続/完了 | 連用形 | 咲きたり（正在盛开） |
| り | 存続（サ変未然・四段已然） | 特殊 | 咲けり（正在开着） |
| む | 推量/意志 | 未然形 | 行かむ（要去吧） |
| べし | 当然/推量/義務 | 終止形 | 行くべし（应当去） |
| まじ | 否定推量/禁止 | 終止形 | 行くまじ（不该去） |
| ず | 否定 | 未然形 | 行かず（不去） |
| なり | 断定/传闻 | 体言/終止形 | 山なり（是山）/行くなり（听说去） |

### 20.2.3 助词差异

| 文語助词 | 现代对应 | 功能 |
|----------|----------|------|
| の・が（主格） | が（主格） | 主語标记（连体修飾中常用「の」） |
| を（于格） | で | 场所经过（「野を行く」=野原を行く） |
| にて | で | 手段/场所 |
| より | から/より | 起点/比较 |
| もの（の） | こと/の | 形式名词化 |

## 20.3 係り結び（系结构造）

### 20.3.1 什么是係り結び

係り結び（かかりむすび）是文語语法中最独特的规则之一，指句中某些**係り助詞（系助词）**出现时，**文末动词（结び）**必须变为特定活用形，而非通常的終止形。

这是古典文法的核心，理解它能解读大量古文。

### 20.3.2 係り結び规则一览

| 係り助詞 | 结び活用形 | 语感/功能 | 例句 |
|----------|-----------|-----------|------|
| ぞ | 連体形 | 強調（书写强调） | 花ぞ咲く**る**（花啊，盛开着） |
| なむ | 連体形 | 強調（希望/愿望） | 早く来なむ（真希望早点来） |
| や | 連体形 | 疑問（是否……） | 花や咲く**る**（花开了吗？） |
| か | 連体形 | 疑問（什么/哪……） | 誰か来た**る**（是谁来了？） |
| こそ | 已然形 | 強調（最强语气） | 花こそ咲け（正是花盛开） |

```
係り結び例文解析：

原文：「月こそ出でれ」
       ↑係り助詞    ↑已然形（こそ→已然形）
現代語：月が出た（月亮出来了）
語感：正是月亮（出来了）——比单纯的「月が出た」更有强调感

原文：「いかなる人ぞ」
         ↑係り助詞  ↑連体形
現代語：どのような人か（是什么样的人啊）

原文：「世はさだめなきこそいみじけれ」（徒然草）
         ↑こそ                  ↑已然形（いみじ的已然形）
現代語：世の中は無常であることこそが、すばらしいのだ
（正是世事无常，才令人感叹）
```

### 20.3.3 係り結びの流れ（こそ已然形的特殊性）

「こそ」的已然形结び有一个特殊现象——**逆接延续**：当句子以「こそ～已然形」结束后，如果后面还有分句，自动带有逆接语气（虽然……但是）。

```
例：「あはれなることこそ多けれど、かつは忘れもするなり。」
    （感人之事虽多，但也渐渐忘却。）
    ↑こそ～多けれ（已然形）→ 逆接→ かつは忘れもする
```

### 20.3.4 現代日語における係り結びの残留

| 现代表达 | 来源 | 意义 |
|----------|------|------|
| ～こそ（強調） | 係り助詞「こそ」残留 | 正是……（已然形结びが消滅し、こそのみ残留） |
| ～ものか | 「ものぞ」的变形 | 岂会……（反问否定） |
| なんと～な | 「なむ」的感叹用法残留 | 多么……啊 |

```
現代語用例：
「努力こそが成功への鍵だ。」
→「こそ」保留了係り助詞的强调功能，但係り結び规则已消失
（現代語では結びが連体形になるという規則がない）
```

## 20.4 常见文語表达在现代日语中的残留

这是本章最实用的部分。掌握以下表达，你的日语将立刻获得"古雅气质"，在正式场合也能游刃有余。

### 20.4.1 ～ざるを得ない（不得不……）

**文語来源：** 否定助动词「ず」的連体形「ざる」+ 助词「を」+ 動詞「得る（うる）」+ 否定「ない」

| 要素 | 分析 |
|------|------|
| ざる | 文語否定助動詞「ず」の連体形 |
| を | 目的格助詞（文語の「を」） |
| 得ない | 「得る（うる）」+否定 |

```
接続：動詞（否定形の語幹）+ ざるを得ない

例文：
・この状況では、撤退せざるを得ない。（在这种情况下，不得不撤退。）
・真実を話さざるを得なかった。（不得不说出了真相。）
・規則に従わざるを得ない立場だ。（处于不得不遵守规则的立场。）

現代語対応：～しないわけにはいかない / ～するしかない
ニュアンス：比「しかない」更正式，带有"被迫"的无奈感
```

### 20.4.2 ～ごとく / ～ごとき（如……一般）

**文語来源：** 比喩的助動詞「ごとし」的連用形「ごとく」和連体形「ごとき」

```
接続：体言/用言連体形 + ごとく（副詞的）/ ごとき（連体修飾）

例文：
・水が流れるごとく、時は過ぎていった。（时光如流水般逝去。）
・猛虎のごとき眼光で睨みつけた。（以猛虎般的眼神凝视着。）
・彼のごとき才能は稀有だ。（像他那样的才能是稀有的。）

現代語対応：～のように / ～のような
ニュアンス：文語的で格調高く、詩的な表現。日常語より改まった場面で使用。
```

### 20.4.3 ～なり（是……/断定）

**文語来源：** 断定の助動詞「なり」（体言・連体形に接続）

```
主な用法：
① 断定（～である）：「これ真実なり」= これが真実である
② 伝聞（～というよ）：「雨降るなり」= 雨が降るそうだ

現代の残留：
・これが道理というものなり。（这才是道理。）→演説・文学的
・以上が報告の概要なり。（以上是报告概要。）→旧式公文書

慣用的残留：
・なるほど（成る程）→なり的連体形から転化した感嘆詞
・そうなり（そうなのか）→東北方言に残存
```

### 20.4.4 ～べし / ～べき（应当……）

**文語来源：** 推量・当然・義務の助動詞「べし」（終止形接続）

| 用法 | 意义 | 例 |
|------|------|-----|
| 当然 | 应该，理所当然 | 勝つべき試合（应该赢的比赛） |
| 義務 | 必须，应当 | 守るべきルール（必须遵守的规则） |
| 推量 | 大概，应该会 | 来るべき時（迟早会来的时候） |
| 意志 | 要……（决意） | 我行くべし（我定要去） |

```
例文：
・人として守るべき礼儀がある。（作为人，有应该遵守的礼仪。）
・来るべき未来に備える。（为即将到来的未来做准备。）
・これは読むべき名著だ。（这是必读的名著。）
・武士は死すべし。（武士应当赴死。）→文語的断言

現代語対応：～はずだ / ～なければならない / ～にちがいない
```

### 20.4.5 ～まじ（不该……/不可……）

**文語来源：** 否定推量・禁止の助動詞「まじ」（「べし」の否定対応）

```
接続：終止形 + まじ（文語）→ 現代では「まじき」（連体形）が残留

例文：
・あるまじき行為（不该有的行为）
・許すまじき罪（不可饶恕的罪行）
・武士のあるまじき姿（武士不应有的样子）

慣用句：
・なんたることか、あるまじき！（这是何等不该有的行为！）
→ 現代語対応：～であってはならない / ～するはずがない
→ ニュアンス：道徳的な批判や驚愕を伴う
```

### 20.4.6 ～たり（是……；存在着……）

**文語来源：** 断定「たり」（体言に接続、「なり」の別形）または完了存続「たり」

```
現代の残留：
① 断定の「たり」→ 「として」の形で残留
   「教師たるもの（作为教师的人）」
   「社会人たる自覚（身为社会人的自觉）」

② 完了の「たり」→ 「たり～たり（したり～したり）」
   「泣いたり笑ったり（时哭时笑）」
   →文語では「泣きたり笑ひたり」の形

例文（現代語）：
・リーダーたる者、責任を持たねばならない。
  （作为领导者，必须承担责任。）
・今日は晴れたり曇ったりの天気だった。
  （今天时晴时阴。）
```

### 20.4.7 表达对比总结

| 文語表达 | 现代口语対応 | 正式度 | 使用场合 |
|----------|-------------|--------|----------|
| ～ざるを得ない | ～するしかない | ★★★★★ | 商务/论文/新闻 |
| ～ごとく | ～のように | ★★★★☆ | 文学/演讲/歌词 |
| ～べき | ～はずの/～なければならない | ★★★★☆ | 规章/社论/演讲 |
| ～まじき | ～であってはならない | ★★★★★ | 批评/法律/评论 |
| ～たる | ～である（资格を強調） | ★★★★☆ | 正式演讲/文章 |
| ～なり（断定） | ～だ/～である | ★★★☆☆ | 文学/修辞 |

## 20.5 日本古典文学名作导读

### 20.5.1 万葉集（まんようしゅう，8世纪）

**概要：** 日本最古老的和歌集，收录约4500首和歌，跨越约130年（629—759年）。既有天皇贵族之作，也有防人（边境士兵）和农民的歌谣，是了解上代日语的第一手资料。

**代表名句赏析：**

```
原文（万葉仮名を平仮名に直したもの）：
「天地の　分かれし時ゆ　神さびて
　高く貴き　駿河の山」（山部赤人）

現代語訳：
天地分かれた太古の昔から、神々しく高くそびえる駿河の山よ。

中文翻译：
自天地开辟之时起，神圣高耸的骏河之山啊。

文語ポイント：
「ゆ」= 奈良时代の格助詞，表示起点（～から）
「さびて」= 神々しい様子を表す動詞「さぶ」の連用形
```

```
原文：「君待つと　我が恋ひをれば　我が屋戸の
　すだれ動かし　秋の風吹く」（額田王）

現代語訳：
あなたを待って私が恋い慕っていると、私の家のすだれを揺らして秋風が吹いてくる。

中文翻译：
我在等待你，沉浸于思念之中，此时秋风吹来，摇动了我家的竹帘。

文語ポイント：
「をれば」= 「をり（居り）」+「ば」、存続+仮定
「吹く」= 終止形で文末（現代語と同じ）
```

### 20.5.2 源氏物語（げんじものがたり，11世纪初）

**概要：** 紫式部著，世界最古の長編小説之一，共54帖。以光源氏的恋爱与政治生涯为主线，是平安文学的巅峰。

**代表名句赏析：**

```
原文：「いづれの御時にか、女御、更衣あまたさぶらひたまひけるなかに、
　いとやむごとなき際にはあらぬが、すぐれて時めきたまふありけり。」

現代語訳：
どの帝の御代であったか、女御や更衣がたくさん仕えていらっしゃったなかで、
さして高い身分でもない方で、際立ってご寵愛を受けていらっしゃる方がいらっしゃった。

中文翻译：
在某位天皇的时代，众多女御和更衣侍奉宫中，其中有一位并非出身高贵，
却格外受到宠爱的女子。

文語ポイント：
「いづれの御時にか」→ 係り結び（か→連体形が省略：あった「ける」→「けるか」→省略）
「さぶらひたまひけるなかに」→ 「さぶらふ」(侍奉)の四段活用+敬語+過去
「ありけり」→「あり」（ラ変）+「けり」（過去伝聞）
```

### 20.5.3 枕草子（まくらのそうし，10世纪末）

**概要：** 清少納言著，随笔文学的开山之作，以「～は……」的列举形式著名。

**代表名句赏析：**

```
原文：「春はあけぼの。やうやうしろくなりゆく山ぎは、
　すこしあかりて、紫だちたる雲のほそくたなびきたる。」

現代語訳：
春は夜明けがよい。だんだんと白くなっていく山の稜線、
少し明るくなって、紫がかった雲が細くたなびいているのがよい。

中文翻译：
春天，黎明时分最美。山际渐渐泛白，微微透出光亮，
淡紫色的薄云横卧天际，令人心旷神怡。

文語ポイント：
「あけぼの」= 黎明（现代では「夜明け」）
「やうやう」= 「ようよう」、だんだん（副詞）
「しろくなりゆく」= 「なりゆく」は「なり行く」、変化の進行
「たなびきたる」= 「たなびく」+完了存続「たり」の連体形
```

### 20.5.4 徒然草（つれづれぐさ，14世纪初）

**概要：** 吉田兼好著，244段の随笔集，蕴含深刻的人生哲理与无常観。

**代表名句赏析：**

```
原文：「つれづれなるまゝに、日ぐらし硯にむかひて、
　心にうつりゆくよしなし事を、そこはかとなく書きつくれば、
　あやしうこそものぐるほしけれ。」（第一段）

現代語訳：
所在なく、一日中硯に向かって、心に浮かんでは消えていくとりとめのないことを、
なんとなく書き続けると、なんとも奇妙に気が狂ったようになる。

中文翻译：
百无聊赖中，整日对着砚台，心中浮现又消散的漫无边际之事，
随手写来，竟莫名感到一种癫狂之感。

文語ポイント：
「つれづれなるまゝに」= 無聊な（状態）のままに
「日ぐらし」= 一日中（古語）
「心にうつりゆく」= 心に浮かんでは消えていく
「よしなし事」= 意味のないこと、とりとめのないこと
「そこはかとなく」= なんとなく（副詞）
「あやしうこそものぐるほしけれ」→ 係り結び（こそ→已然形「けれ」）
```

## 練習問題

### 問題1：係り結び識別

以下の文から係り助詞を見つけ、結びの活用形を答えなさい。

1. 「花ぞ美しき」
2. 「いかにぞ言ふ」
3. 「風こそ吹け」
4. 「誰かありける」

**答え：**
1. ぞ→連体形（美しき）
2. ぞ→連体形（言ふ）
3. こそ→已然形（吹け）
4. か→連体形（ありける）

### 問題2：文語→現代語変換

以下の文語表現を現代日本語に訳しなさい。

1. 「言わずもがな」→ ＿＿＿
2. 「守るべき約束」→ ＿＿＿
3. 「あるまじき行為」→ ＿＿＿
4. 「猛虎のごとき勢い」→ ＿＿＿

**答え：**
1. 言うまでもないこと / 不言而喻的事
2. 守らなければならない约束 / 应当遵守的约定
3. あってはならない行為 / 不应存在的行为
4. 猛虎のような勢い / 猛虎般的气势

### 問題3：文語表現の使い分け

括弧内の文語表現を現代の文脈で使った例文を作りなさい。

1. ～ざるを得ない
2. ～べき
3. ～たる（資格強調）

## 本章小结

| 学习要点 | 核心内容 |
|----------|----------|
| 文語の重要性 | 成语・歌词・公文に文語は現役で生きている |
| 動詞活用 | 四段・上二段・下二段など文語特有の活用体系 |
| 係り結び | ぞ/なむ/や/か→連体形、こそ→已然形 |
| 現代残留表現 | ～ざるを得ない/～べき/～まじき/～ごとく/～たる |
| 古典文学 | 万葉集・源氏物語・枕草子・徒然草の特徴と名句 |

学习文語不是目标，而是工具。每当你读到「さもありなん」时会心一笑，每当你写出「～ざるを得ない」时语气精准，这就是古典语法带给现代日语学习者的礼物。

> 「古きを温ねて新しきを知る。」——《论语》
> 学习古典，正是为了更深刻地理解当下。
