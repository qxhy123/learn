# 第2章：平假名完全掌握

> 「ひらがなは日本語の母である」——平假名是日语之母。在日本，孩子学会的第一套文字就是平假名；对外国学习者来说，掌握平假名也是踏入日语世界的第一道门。看似简单的46个字符背后，隐藏着千年的文化演变史。

---

## 2.1 平假名的起源

### 万葉仮名（まんようがな）：最初的借音

公元7-8世纪，日本人面临一个根本性问题：日语是有声调的口语，却没有自己的文字系统。汉字的传入解决了这个困境，但方式十分迂回——日本人直接**借用汉字的音来记录日语发音**，而不管汉字本身的意义。

这套系统被称为**万葉仮名**（まんようがな），因用于记录《万葉集》（日本最早的诗歌总集，759年编成）而得名：

```
万葉仮名示例（《万葉集》卷一）：
原文：夜麻等播 久邇能麻保呂婆 多多儺波努 阿袁迦岐余斯 夜麻等斯宇流波斯

读音：やまとは くにのまほろば たたなはぬ あをかきよし やまとしうるはし
现代汉字：大和は 国のまほろば たたなはぬ 青垣 山籠れる 大和し美し
含义：大和（奈良）啊，是最美好的国度，青山环绕，大和真是美丽。
```

每个汉字仅用其音，不用其义，相当于今天的音标系统。但这种方法书写繁琐——一个音需要一个完整汉字。

### 草書体（そうしょたい）的简化

平安时代（794-1185年），贵族社会需要大量书写和歌、日记、书信，万葉仮名的繁复严重阻碍效率。书法家们开始将汉字快速连写，形成**草书体**，进而进一步简化，最终演变为平假名。

主要演变路径：

```
安 → あ     (安的草书简化)
以 → い     (以的草书简化)
宇 → う     (宇的草书简化)
衣 → え     (衣的草书简化)
於 → お     (於的草书简化)

加 → か     (加的草书)
幾 → き     (幾的草书)
久 → く     (久的草书)
計 → け     (計的草书)
己 → こ     (己的草书)
```

### 女手（おんなで）：女性书写文化的产物

平假名最初被称为**女手**（おんなで），因为它主要由女性贵族使用，而男性贵族以使用正式汉文（漢文）为荣。这一历史背景产生了深远影响：

| 书写系统 | 使用者 | 代表作品 | 年代 |
|---------|--------|---------|------|
| 漢文（男手）| 男性贵族、僧侣 | 《日本書紀》| 720年 |
| 平假名（女手）| 女性贵族 | 《源氏物語》| 约1000年 |
| 平假名（男性）| 贵族男性渐接受 | 《土佐日記》（紀貫之假托女性）| 935年 |

**紫式部**（むらさきしきぶ）用平假名写成的《源氏物語》（约54万字）是世界最早的长篇小说之一，也是平假名文学成熟的巅峰标志。

## 2.2 五十音图系统讲解

### 五十音图的逻辑

五十音图（五十音図／ごじゅうおんず）将日语音节按**元音列**和**辅音行**排列，构成一个10行×5列的矩阵。实际有音的格子是46个（并非50个，因为yi、wu等音不存在）。

### 母音行（あ行）：发音要点

五个元音是整个五十音图的骨架，所有其他音节都在这五个元音的基础上添加辅音：

```
あ行（元音行）：
┌────┬────┬────┬────┬────┐
│ あ │ い │ う │ え │ お │
│ [a]│ [i]│ [ɯ]│ [e]│ [o]│
└────┴────┴────┴────┴────┘
```

**各元音发音要点：**

| 假名 | 罗马字 | 发音说明 | 与中文近似音 | 注意事项 |
|------|--------|---------|------------|---------|
| あ | a | 开口元音，嘴张大 | 类似"啊" | 发音干净，不带鼻音 |
| い | i | 前高元音，嘴角横向拉开 | 类似"衣" | 比中文"i"更紧张 |
| う | u | 后高元音，**嘴唇不圆**！ | 接近"乌"但不圆唇 | 最难点：与中文"u"的圆唇发音不同 |
| え | e | 中前元音，嘴半开 | 类似"诶" | 比中文"e"更前 |
| お | o | 后中元音，嘴唇略圆 | 类似"哦" | 与中文"o"相近 |

**「う」的特殊性：**

```
错误发音：う → [u] （像中文"乌"，嘴唇圆）
正确发音：う → [ɯ] （嘴唇不圆，稍微向两侧）

对比：
中文"书"→ shū，嘴唇向前撅
日语「す」→ su，嘴唇扁平，几乎不动
```

### か行～わ行：完整五十音图

**完整清音五十音图（46字）：**

```
     あ段   い段   う段   え段   お段
     [a]    [i]    [ɯ]   [e]    [o]
あ行  あ     い     う     え     お
か行  か     き     く     け     こ
さ行  さ     し     す     せ     そ
た行  た     ち     つ     て     と
な行  な     に     ぬ     ね     の
は行  は     ひ     ふ     へ     ほ
ま行  ま     み     む     め     も
や行  や     ―     ゆ     ―     よ
ら行  ら     り     る     れ     ろ
わ行  わ     ゐ*    ―     ゑ*    を
ん    ん（鼻音，独立）
```
*ゐ(wi)和ゑ(we)是历史假名，现代日语已不使用

**各行发音与记忆技巧：**

| 行 | 辅音 | 特殊注意 | 记忆技巧 |
|----|------|---------|---------|
| か行 | k | 清音，不送气（类似普通话的"嘎卡咕格" 但更轻）| か=卡 |
| さ行 | s/sh | **し** 发音是 [ɕi]（接近"西"），不是 si | さ=撒，し=西 |
| た行 | t/ch/ts | **ち**=[tɕi]（接近"七"），**つ**=[tsɯ]（近"次"）| た=他，ち=七，つ=次 |
| な行 | n | 发音与中文"n"相近 | な=那，に=你，の=喝（の形似茶壶）|
| は行 | h/f | **ふ**=[ɸɯ]（类似轻声f+u），不送气 | は=哈，ふ=夫 |
| ま行 | m | 与中文"m"完全相同 | ま=马，み=蜜 |
| や行 | y | 半元音 | や=呀，ゆ=由，よ=哟 |
| ら行 | r | **日语r是弹舌音[ɾ]**，介于英语r和l之间 | ら=啦（弹舌）|
| わ行 | w | を仅用于宾格助词，发音=[o] | わ=哇 |

**重点辨析——さ行的「し」：**
```
错误：し → si（像中文"私"的发音）
正确：し → [ɕi]（接近"西"，舌面音）

整个さ行：
さ[sa] し[ɕi] す[sɯ] せ[se] そ[so]
注意：只有し是例外，其余保持sa/su/se/so
```

**重点辨析——た行的「ち」和「つ」：**
```
た[ta] ち[tɕi] つ[tsɯ] て[te] と[to]

ち → [tɕi]：接近中文"七"，不是"ti"
つ → [tsɯ]：接近中文"次"，不是"tu"
```

**重点辨析——ら行的弹舌音：**
```
ら[ɾa] り[ɾi] る[ɾɯ] れ[ɾe] ろ[ɾo]

[ɾ] = 弹舌音，舌尖快速弹触上颚一次
- 不同于英语r（卷舌）
- 不同于汉语r（zh/摩擦音）
- 类似西班牙语r（单弹舌）
- 也类似英语butter中间的d/t音（美式）
```

## 2.3 浊音与半浊音

### 浊音（濁音／だくおん）

在清音右上角加两点（**濁点／だくてん**，又称"てんてん"），使辅音由清辅音变为浊辅音：

```
     あ段   い段   う段   え段   お段
が行  が     ぎ     ぐ     げ     ご   [g]
ざ行  ざ     じ     ず     ぜ     ぞ   [z/ʑ]
だ行  だ     ぢ     づ     で     ど   [d/ʑ/z]
ば行  ば     び     ぶ     べ     ぼ   [b]
```

**特殊浊音辨析：**

| 假名 | 发音 | 说明 |
|------|------|------|
| じ | [ʑi] | ざ行的い段浊音 |
| ぢ | [ʑi] | だ行的い段浊音，**与じ发音相同** |
| ず | [zɯ] | ざ行的う段浊音 |
| づ | [zɯ] | だ行的う段浊音，**与ず发音相同** |

现代标准日语中，ぢ=じ，づ=ず，均发相同的音。ぢ和づ只在特定词汇中保留（如：鼻血＝はなぢ，続く＝つづく）。

### 半浊音（半濁音／はんだくおん）

在は行右上角加圆圈（**半濁点／はんだくてん**，"まるぽ"），は行变为ぱ行：

```
     あ段   い段   う段   え段   お段
ぱ行  ぱ     ぴ     ぷ     ぺ     ぽ   [p]
```

**は行的清浊半浊对比：**
```
は[ha] → ば[ba] → ぱ[pa]
ひ[hi] → び[bi] → ぴ[pi]
ふ[ɸɯ]→ ぶ[bɯ]→ ぷ[pɯ]
へ[he] → べ[be] → ぺ[pe]
ほ[ho] → ぼ[bo] → ぽ[po]
```

**历史背景：** 上古日本语中，は行辅音实为 [p]，随时间演变为 [ɸ]，再到 [h]。所以反而是ぱ行保留了最古老的发音。英语借词如 Paris→パリ（pa-ri）就是用ぱ行。

## 2.4 拗音（ようおん）

### 拗音的构成原理

拗音是**辅音+半元音[j]**的复合音节，由**い段假名**+小写的**や/ゆ/よ**组成：

规则：`き（ki）+ や（ya）= きゃ（kya）`  
注意：**や/ゆ/よ写成小写**，这是区分组合读音还是两个独立音节的关键。

```
きや（ki-ya，两个音节）≠ きゃ（kya，一个音节）
```

### 完整拗音表

```
清音拗音：
き行：きゃ[kja] きゅ[kjɯ] きょ[kjo]
し行：しゃ[ɕa]  しゅ[ɕɯ]  しょ[ɕo]   ← 注意：不是sya/syu/syo
ち行：ちゃ[tɕa] ちゅ[tɕɯ] ちょ[tɕo]  ← 注意：不是tya/tyu/tyo
に行：にゃ[nja] にゅ[njɯ] にょ[njo]
ひ行：ひゃ[hja] ひゅ[hjɯ] ひょ[hjo]
み行：みゃ[mja] みゅ[mjɯ] みょ[mjo]
り行：りゃ[ɾja] りゅ[ɾjɯ] りょ[ɾjo]

浊音拗音：
ぎ行：ぎゃ[gja] ぎゅ[gjɯ] ぎょ[gjo]
じ行：じゃ[ʑa]  じゅ[ʑɯ]  じょ[ʑo]
ぢ行：ぢゃ[ʑa]  ぢゅ[ʑɯ]  ぢょ[ʑo]  ← 与じ行相同音
び行：びゃ[bja] びゅ[bjɯ] びょ[bjo]

半浊音拗音：
ぴ行：ぴゃ[pja] ぴゅ[pjɯ] ぴょ[pjo]
```

### 常见拗音词汇示例

```
しゃ：写真（しゃしん）= 照片
しゅ：趣味（しゅみ）= 爱好
しょ：書類（しょるい）= 文件
ちゃ：お茶（おちゃ）= 茶
ちゅ：注意（ちゅうい）= 注意
ちょ：ちょっと = 稍微，一点
にゃ：にゃんこ = 猫咪（俗称）
りゅ：龍（りゅう）= 龙
りょ：旅行（りょこう）= 旅行
```

## 2.5 特殊发音规则

### 长音（長音／ちょうおん）

日语中，**元音延长一拍**即为长音，改变词义：

```
おばさん（伯母さん）= 阿姨（短音）
おばあさん（お婆さん）= 奶奶（长音，第2个"a"延长）

おじさん（叔父さん）= 叔叔（短音）
おじいさん（お爺さん）= 爷爷（长音，第2个"i"延长）

ゆき（雪）= 雪（短音）
ゆうき（勇気）= 勇气（长音，"u"延长）
```

**平假名中长音的书写规则：**

| 元音 | 长音写法 | 示例 | 含义 |
|------|---------|------|------|
| a 段长音 | +あ | おかあさん | 妈妈 |
| i 段长音 | +い | おにいさん | 哥哥 |
| u 段长音 | +う | くうき（空気）| 空气 |
| e 段长音 | +え（少见）或+い | おねえさん / えいご | 姐姐 / 英语 |
| o 段长音 | +う（常见）或+お | がっこう / おおきい | 学校 / 大 |

**注意：** o段长音通常写作 +う（学校=がっこう），但也有少数写作 +お（大きい=おおきい，多い=おおい）。

### 促音（促音／そくおん）——「っ」

促音（小写的「っ」）表示辅音的短暂**停顿**（closure），相当于双写后面的辅音：

```
きて（来て）= 请来（2拍）
きって（切手）= 邮票（3拍，中间有停顿）

はく（履く）= 穿鞋（2拍）
はっく = （无意义，展示促音，3拍）

がこう = （无意义）
がっこう（学校）= 学校（4拍）
```

**促音的发音方法：**
```
きって：ki - [pause] - te
実際（じっさい）：ji - [pause] - sa - i
切符（きっぷ）：ki - [pause] - pu
```

停顿时间约等于一个普通音节的时长（即一拍，モーラ）。

**促音只出现在以下辅音之前：**
- か行（k）：がっか（画家）
- さ行（s）：まっさき（真っ先）
- た行（t）：きって（切手）
- ぱ行（p）：きっぱり（干脆）
- （口语中偶见在か行、さ行外的情况）

### 拨音（撥音／はつおん）——「ん」

「ん」是唯一不以元音结尾的假名，单独占一拍，其实际发音**随后面的音节而变化**：

```
条件1：在ば/ぱ/ま行之前 → [m]（双唇鼻音）
  ─ 新聞（しんぶん）：shin-BUN，n→m，因为b是双唇音
  ─ 三枚（さんまい）：sa-MMAI，n→m

条件2：在た/だ/な/ら行之前 → [n]（齿龈鼻音，标准n音）
  ─ 温度（おんど）：on-do
  ─ 先日（せんじつ）：sen-jitsu

条件3：在か/が行之前 → [ŋ]（软腭鼻音，汉语"昂/英"末尾音）
  ─ 銀行（ぎんこう）：gi-ŊKOU
  ─ 暗記（あんき）：a-ŊKI

条件4：在母音/や行/わ行之前，或句尾 → [ɴ]（悬浮鼻音）
  ─ 本（ほん）：ho-N（句尾）
  ─ 恋愛（れんあい）：re-N-ai（鼻化元音）
```

**中文学习者常见错误：** 将「ん」在任何位置都读成标准的 [n]，导致如「銀行（ぎんこう）」听起来发音不自然。正确应在か行前发 [ŋ]。

## 2.6 平假名的书写规范

### 易混淆假名对比

以下是中国学习者最容易混淆的平假名组合：

**组合1：あ 与 お**
```
あ (a)：有一个向右的横折，下方开口较大
お (o)：类似"お"，中间有竖挑，整体更紧凑

记忆法：あ→"安"字草书，口更大；お→"於"字草书，口更小
```

**组合2：は 与 ほ**
```
は (ha)：右侧两个弯，下方第二个弯往左延伸
ほ (ho)：右侧两个弯，但下方有个回钩往右
```

**组合3：ぬ 与 め**
```
ぬ (nu)：左侧竖撇，右侧有个"8"字形绕圈，末端不收
め (me)：左侧是个"口"形，右侧绕圈，末端向右甩出

记忆法：ぬ→"奴"草书，两个人；め→"女"草书，看脸
```

**组合4：る 与 ろ**
```
る (ru)：有个向右的环形回转，末端有钩
ろ (ro)：与る相似但**没有末端环形**，更简单
```

**组合5：わ 与 れ**
```
わ (wa)：中间竖撇向右，没有右侧的横
れ (re)：中间竖撇，右侧有向右弯出的横
```

**组合6：さ 与 ち**
```
さ (sa)：两横，下方弯钩朝左
ち (chi)：一横，然后是朝右的大弯
```

**完整易混淆假名速查：**

| 易混对 | 区分要点 |
|--------|---------|
| あ/お | あ右边横折开口大；お中间多一竖 |
| は/ほ | は右下往左；ほ右下往右回钩 |
| ぬ/め | ぬ末端无外甩；め末端向右甩 |
| る/ろ | る有末端小环；ろ无小环更简 |
| ね/れ | ね有上方横连接；れ无上横 |
| わ/ね | わ右侧简单竖弯；ね右侧更复杂 |
| き/さ | き三横；さ两横 |
| い/り | い左撇无勾；り左撇有勾朝右 |
| ゐ/ゑ | 历史假名，现代不用 |

### 笔顺要点

平假名笔顺遵循汉字"从上到下、从左到右"的基本原则，以下是几个关键假名的笔顺：

```
あ：①横（上）→ ②撇（中左）→ ③大弧（右转一圈）
き：①横→ ②横→ ③竖撇→ ④弯钩
す：①横→ ②竖弯钩（带回圆）
な：①横→ ②撇→ ③竖折→ ④大圆弧
の：①一笔，左上起，顺时针大圆（不收尾，开口朝右）
も：①横→ ②横→ ③竖勾→ ④横弯钩
```

## 2.7 假名输入法

### 罗马字输入（ローマ字入力）

现代电脑和手机上，日语最常用的输入方式是**罗马字输入**，即键入对应罗马字字母组合，候选词自动转换为假名/汉字。

**基础对应表（特殊音节）：**

```
通常情况：
か → ka     き → ki     く → ku     け → ke     こ → ko
さ → sa     し → si/shi す → su     せ → se     そ → so
た → ta     ち → ti/chi つ → tu/tsu て → te     と → to

特殊组合：
しゃ → sha/sya     しゅ → shu/syu     しょ → sho/syo
ちゃ → cha/tya     ちゅ → chu/tyu     ちょ → cho/tyo
じゃ → ja/jya      じゅ → ju/jyu      じょ → jo/jyo
つ  → tsu          ふ  → fu

促音：重复辅音字母
きって → kitte（重复t）
がっこう → gakkou/gakko（重复k）
っ单独输入 → xtu 或 ltu

长音（片假名中）：
- → （横线）

拨音：
ん → n（后面跟非n开头的辅音时）
ん → nn（后面跟n/元音时，防止被解读为な行）
如：あんない → an'nai 或 annai
```

### 手机九键输入（フリック入力）

智能手机上，日本用户多用**フリック入力**（滑动输入）：

```
手机布局（九宫格）：
┌───┬───┬───┐
│ あ │ か │ さ │
├───┼───┼───┤
│ た │ な │ は │
├───┼───┼───┤
│ ま │ や │ ら │
├───┼───┼───┤
│ わ │ 小 │ 、。│
└───┴───┴───┘

每个键位：
- 点击一次 → あ段（あかさたなはまやらわ）
- 向上滑 → い段（いきしちにひみ―り―）
- 向右滑 → う段（うくすつぬふむゆるー）
- 向下滑 → え段（えけせてねへめーれー）
- 向左滑 → お段（おこそとのほもよろをん）
```

**中文学习者的输入优势：** 中文使用者因为熟悉拼音，罗马字输入法上手很快，比英语母语者更容易适应日语输入系统。

## 本章小结

| 知识点 | 核心内容 |
|--------|---------|
| 起源 | 万葉仮名→草書体简化→平假名（女手文化）|
| 五十音 | 10行×5列，实际46字清音 |
| 特殊音节 | さ行し、た行ちつ、ら行弹舌音是发音难点 |
| 浊音/半浊音 | か/さ/た/は→が/ざ/だ/ば；は→ぱ |
| 拗音 | い段+小字や/ゆ/よ，一个音节而非两个 |
| 长音 | a/i/u+あいう，e段+い，o段+う（或お）|
| 促音 | 小っ=辅音停顿一拍，只在k/s/t/p前 |
| 拨音 | ん的发音随后续音变化（m/n/ŋ/ɴ）|
| 书写 | 易混淆对：あ/お、は/ほ、ぬ/め、る/ろ等 |
| 输入法 | 罗马字输入为主，注意し=si/shi、ち=ti/chi |
