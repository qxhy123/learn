# 第4章：汉字入门与音读训读

> 「漢字は日本語の魂である」——汉字是日语的灵魂。对于中文母语者来说，这一章是最复杂的惊喜：你以为认识的字，未必读得出；你以为相同的词，未必同义。汉字在日语中经历了千年的本土化演变，形成了一套独特而精妙的读音与意义体系。

---

## 4.1 日语汉字的历史

### 汉字传入日本的三个时期

#### 第一期：应神天皇时代（约4-5世纪）——传说与史实的交界

传统记载，应神天皇15年（约285年），百济（朝鲜半岛西南部的古国）的**王仁博士**（わにはかせ）携带《论语》十卷和《千字文》一卷渡海来日，这被视为汉字正式传入日本的起点。

历史背景：
- 日本当时尚无文字系统
- 朝鲜半岛充当中日文化传播的中介
- 汉字最初只有极少数贵族、僧侣掌握

```
传播路径：
中国（汉字原产地）→ 朝鲜半岛（加工/中转）→ 日本（接受/本土化）
```

#### 第二期：遣隋使・遣唐使时代（7-9世纪）——全面汉化

这是日本主动吸收中国文化最积极的时期：

| 时期 | 使节 | 次数 | 重要意义 |
|------|------|------|---------|
| 遣隋使（593-618）| 小野妹子等 | 3-4次 | 正式建立外交关系 |
| 遣唐使（630-894）| 阿倍仲麻呂等 | 约20次 | 全面引入唐朝制度文化 |

**遣唐使的文化贡献：**
- 大批留学生和学问僧赴唐朝长安、洛阳
- 携回大量汉籍：儒家经典、佛教典籍、医书、历法、艺术
- 系统引入**漢音**（汉音，反映盛唐长安音）
- 建立律令国家制度（大化改新645年）

著名遣唐使人物：
```
阿倍仲麻呂（あべのなかまろ，698-770）
  - 赴唐后改名晁衡，任职唐朝官员
  - 与诗人李白、王维交好
  - 李白曾误以为他在回国途中遇难，写下悼诗
  - 实际终老于唐朝，未能归国

吉備真備（きびのまきび，695-775）
  - 赴唐留学18年
  - 带回大量典籍，传入围棋、音乐、天文
  - 相传参与创制片假名
```

#### 第三期：鎌倉・室町时代（12-16世纪）——禅宗文化的汉字

禅宗（ぜんしゅう）从宋、元、明朝传入，带来了：
- **唐音/宋音**（从宋元明时代的南方官话/方言音）
- 茶道、水墨画、枯山水等文化形式
- 大量俚语、俗语词汇

```
禅宗带来的唐音词汇示例：
椅子（いす）→ 唐音：isu（来自宋代"椅子"的读音）
普通（ふつう）→ 唐音传入的读法
行灯（あんどん）→ 灯笼类
```

## 4.2 音読み（音读）与訓読み（训读）

### 音読み（おんよみ）——模仿汉语发音

**音读**是模仿汉字传入时代的汉语发音。由于汉字在不同时期传入，同一汉字往往有多个音读，反映不同历史阶段的汉语语音：

#### 音读的四大分类

**1. 呉音（ごおん）——最古老的音读**
- 传入时期：5-6世纪，经朝鲜半岛或直接从南朝（建康/今南京）传入
- 反映：南朝汉语（江南方言）的发音
- 主要用于：佛教用语

```
呉音示例：
人（にん）：仏教 → 人間（にんげん）
行（ぎょう）：仏教 → 修行（しゅぎょう）
六（ろく）：仏教 → 六道（ろくどう）
明（みょう）：仏教 → 光明（こうみょう）
```

**2. 漢音（かんおん）——最主流的音读**
- 传入时期：7-9世纪，遣唐使携回
- 反映：盛唐长安官话
- 主要用于：一般词汇（现代日语最常用音读）

```
漢音示例：
人（じん）：一般 → 人口（じんこう）
行（こう）：一般 → 行動（こうどう）
六（りく）：一般 → 陸（りく，陸地）
明（めい）：一般 → 明暗（めいあん）
```

**3. 唐音/宋音（とうおん/そうおん）——禅宗传入的音读**
- 传入时期：12-16世纪，禅僧往来
- 反映：宋元明时代（可能是福建/浙江方言）
- 主要用于：禅宗术语、茶道、特定文化词汇

```
唐音/宋音示例：
行（あん）：禅 → 行灯（あんどん）
椅（い）：家具 → 椅子（いす）
路（ろ）：一般 → 路地（ろじ、小路）
```

**4. 慣用音（かんようおん）——约定俗成的误读**
- 由于传抄错误或日本人的自行演变，形成与正规汉音/呉音不同的读法
- 长期使用后被接受为正确读音

```
慣用音示例：
輸（ゆ）：正规应读しゅ，但习惯读ゆ → 輸送（ゆそう）
攻（こう）：正规应读きょう，习惯读こう
情（じょう）：情報（じょうほう）
```

#### 同一汉字的多种音读对比

```
人：
  呉音：にん（仏教：人間にんげん、人形にんぎょう）
  漢音：じん（一般：人口じんこう、外人がいじん）
  慣用：ひと（训读，日语固有词）

明：
  呉音：みょう（仏教：光明こうみょう、冥界めいかい）
  漢音：めい（一般：明暗めいあん、明確めいかく）
  訓読：あか/あき/あかるい（固有词）
```

### 訓読み（くんよみ）——日语固有词的汉字标记

**训读**是用汉字的意义，对应日语中**本来就有的固有词**（大和言葉/やまとことば）。换言之，先有日语固有词，后来找汉字来"配对"。

```
日语固有词 ← 汉字配对
yama（山）← 山（读作やま）
kawa（川）← 川（读作かわ）
hana（花）← 花（读作はな）
mizu（水）← 水（読作みず）
hi（火）  ← 火（读作ひ）
te（手）  ← 手（读作て）
```

这就是为什么日语汉字往往既有音读（发音近似古汉语）又有训读（发音是纯日语）：

| 汉字 | 音读 | 训读 | 
|------|------|------|
| 山 | さん (san) | やま (yama) |
| 川 | せん (sen) | かわ (kawa) |
| 花 | か (ka) | はな (hana) |
| 水 | すい (sui) | みず (mizu) |
| 火 | か (ka) | ひ (hi) |
| 木 | もく/ぼく | き (ki) |
| 日 | にち/じつ | ひ/か |
| 月 | げつ/がつ | つき |

### 「生」——极端的多读字

汉字「生」是日语中读法最多的汉字之一，堪称日语汉字复杂性的极致体现：

```
生（音読み）：
  せい → 先生（せんせい）、生活（せいかつ）、生命（せいめい）
  しょう → 一生（いっしょう）、誕生（たんじょう）

生（訓読み）：
  い（きる）→ 生きる（活着）
  い（かす）→ 生かす（活用/使活）
  い（ける）→ 生ける（插花/使存活）
  う（まれる）→ 生まれる（出生，被动）
  う（む）→ 生む（生育）
  お（う）→ 生う（长出，书面语）
  は（える）→ 生える（生长）
  き → 生（き）（な）形容词：生の魚（生鱼）
  なま → 生（なま）（啤酒生的：生ビール）
  さ（ける）→ （在某些方言/古语）

合计：超过10种读法，为常用汉字中读法最多之一
```

### 熟語中的音训混读规则

两个汉字组成的**熟語**（じゅくご），读法有四种组合：

```
1. 音読み + 音読み（最常见）：
   学校（がっこう）gakkou：学[がく]+校[こう]
   電話（でんわ）  denwa ：電[でん]+話[わ]
   
2. 訓読み + 訓読み（次常见）：
   山道（やまみち）：山[やま]+道[みち]
   手紙（てがみ）  ：手[て]+紙[がみ]
   
3. 訓読み + 音読み = 湯桶読み（ゆとうよみ）：
   「湯桶」(ゆおけ=木桶) 这个词本身就是湯[ゆ,训]+桶[おけ,训]
   ─ 但"湯桶"作为名称，其读み方式 = 訓+音
   例：
   手本（てほん）：手[て,训]+本[ほん,音]
   本屋（ほんや）：本[ほん,音]+屋[や,训]
   台所（だいどころ）：台[だい,音]+所[どころ,训]

4. 音読み + 訓読み = 重箱読み（じゅうばこよみ）：
   「重箱」(じゅうばこ=多层漆盒) = 重[じゅう,音]+箱[はこ,训]
   例：
   団子（だんご）：団[だん,音]+子[ご,训]
   番組（ばんぐみ）：番[ばん,音]+組[ぐみ,训]
   合図（あいず）：合[あい,训]+図[ず,音]
```

## 4.3 常用汉字与教育汉字

### 常用漢字（じょうようかんじ）2136字

**常用漢字表**由日本文化厅（文化庁）制定，是现代日语书写中"常规使用"汉字的官方范围：

| 版本 | 年份 | 字数 | 背景 |
|------|------|------|------|
| 当用漢字 | 1946年 | 1850字 | 战后简化汉字使用 |
| 常用漢字（初版）| 1981年 | 1945字 | 扩充当用漢字 |
| 常用漢字（現行）| 2010年 | 2136字 | 增加196字，减少5字 |

常用漢字的功能定位：
- **非排他性**：日语中可以使用非常用漢字，但超出范围时通常加注假名（振り仮名/ふりがな）
- **教育基准**：学校教育的主要覆盖范围
- **媒体标准**：新闻、广播、政府文件的参考标准

## 4.4 中日汉字差异

### 字形差异：简体/繁体/新字体三足鼎立

日本在1946年战后推行**新字体**（しんじたい）改革，简化了部分繁体字，但并非完全照搬中国简体字，形成了三套并行的汉字体系：

**系统对比表（精选重要差异）：**

| 概念 | 中国简体 | 日本新字体 | 中国繁体/传统 | 说明 |
|------|---------|-----------|------------|------|
| 学习 | 学习 | 学習 | 學習 | 日文"学"同中简，"習"不同 |
| 国家 | 国家 | 国家 | 國家 | 日文"国"同中简 |
| 图书馆 | 图书馆 | 図書館 | 圖書館 | 日文"図"与中简"图"不同 |
| 经济 | 经济 | 経済 | 經濟 | 日文"経"与中简"经"略不同 |
| 传统 | 传统 | 伝統 | 傳統 | 日文"伝"与中简"传"略不同 |
| 医院 | 医院 | 医院 | 醫院 | 日文"医"同中简 |
| 歴史 | 历史 | 歴史 | 歷史 | 日文"歴"与中简"历"略不同 |
| 画 | 画 | 画 | 畫 | 三者一致（不常见）|
| 发展 | 发展 | 発展 | 發展 | 日文"発"与中简"发"不同 |
| 丰富 | 丰富 | 豊富 | 豐富 | 日文"豊"与中简"丰"不同 |
| 继续 | 继续 | 継続 | 繼續 | 日文"継"与中简"继"略不同 |
| 铁路 | 铁路 | 鉄道 | 鐵道 | 日文"鉄"与中简"铁"不同 |

**特殊情况：日本新字体与中国简体相同但繁体不同：**
```
国（くに）：中简=国，日新=国，繁=國
学（まなぶ）：中简=学，日新=学，繁=學
```

**特殊情况：日本新字体与繁体相同但中国简体不同：**
```
強（つよい）：中简=强，日新=強，繁=強（日本与繁体更接近）
齢（れい）：中简=龄，日新=齢，繁=齡
```

```

### 含义相同但使用方式不同

```
例1：「一緒に行きましょう」vs「一起去吧」
日语"一緒"(いっしょ)=一起，与中文基本对应，但用法有差异：
  日语：一緒に（+に助词）+动词
  中文：一起+动词

例2：「よろしく」vs「请多关照」
日语よろしくお願いします是初次见面时的万能客套语，
中文没有完全对等的单一表达，需根据语境翻译。

例3：数量词差异：
日语：一台の車/一枚の紙/一冊の本（量词极多且不同）
中文：一辆车/一张纸/一本书（量词也多，但体系不同）
```

## 4.5 汉字学习策略

### 部首记忆法（部首分類）

汉字按**部首**（ぶしゅ）分类，掌握常用部首能大幅提升汉字认知能力。以下是日语学习中最重要的40个部首：

**人相关部首：**
```

```

```

火/灬（ひへん）= 与火有关：炎（ほのお=火焰）、燃（もえる=燃烧）、熱（あつい=热）
山（やまへん）= 与山有关：岩（いわ=岩石）、峰（みね=山峰）

```

**动作相关部首：**
```

足/⻊（あしへん）= 与脚/行走有关：跳（とぶ=跳）、踊（おどる=跳舞）
```

**建筑/场所相关部首：**
```
門（もんがまえ）= 与门有关：開（ひらく=开）、閉（しめる=关）、間（あいだ=中间）

口（くにがまえ）= 围合：国（くに=国家）、図（ず=图）、囲（かこむ=围）

### 汉字造字法：与中文六书的对应

日本汉字学借用了中国传统的**六书**（りくしょ）理论，但有所简化：

**1. 象形文字（しょうけい）——图画变文字**
```
山 → （三座山峰）
川 → （流水纹）
日 → （圆圈+中间一点）
月 → （月牙形）
木 → （树木形）
口 → （嘴巴形）
```

**2. 指事文字（しじ）——用符号指示**
```
上 → （线上方有点，表示上面）
下 → （线下方有点，表示下面）
一二三 → 直接用横线数量
末 → 木字上方加横，指树梢
本 → 木字下方加横，指树根（本=根本）
```

**3. 会意文字（かいい）——多个象形合并**
```
森 = 木+木+木（很多树=森林）
休 = 亻+木（人靠在树上=休息）
明 = 日+月（日月同辉=明亮）
鳴 = 口+鳥（鸟鸣叫=鸣叫）
愛 = 心+形（有心地爱）
```

**4. 形声文字（けいせい）——最多数，占汉字80%以上**
```
形旁（意义线索）+声旁（发音线索）

例：
清（きよい/せい）= 氵（形/水）+青（声/せい）
悩（なやむ/のう）= 心（形/心情）+呶（声）
語（かたる/ご）  = 言（形/言语）+吾（声/ご）
銀（ぎん）      = 金（形/金属）+艮（声/ぎん）
```

**中文学习者的优势：** 熟悉六书理论的中文学习者在看到不认识的日文汉字时，可以通过：
1. 识别形旁推断意义领域
2. 识别声旁猜测大致发音
3. 联系已知中文含义推断日语含义

### 中国学习者的汉字优势与陷阱总结

```
优势（✓）：
✓ 认字快：能直接读懂大量汉字词义
✓ 记忆快：有汉字知识基础，新字认知快
✓ 词汇量跳跃：许多汉字词汇含义相近或相同
✓ 阅读速度：比英语母语者阅读理解快很多
✓ 文化理解：儒家文化、历史典故有共同基础

陷阱（✗）：
✗ 同形异义词：最大陷阱（手紙/老婆/愛人等）
✗ 过度自信：以为认识汉字就懂意思，忽略训读和特殊义
✗ 字形混淆：日本新字体与简体/繁体都有差异
✗ 发音误导：汉字认识了但音读可能与普通话差距大
✗ 忽视假名：只关注汉字，忽略语法功能的平假名部分
```

## 4.6 数字、日期、时间的汉字表达

### 数字系统

日语有两套数字系统：**汉字数字**（音读）和**日语固有数字**（训读），使用场景不同：

**汉字数字（音読み）——最常用：**
```
一（いち）二（に）三（さん）四（し/よん）五（ご）
六（ろく）七（しち/なな）八（はち）九（く/きゅう）十（じゅう）
百（ひゃく）千（せん）万（まん）億（おく）兆（ちょう）

注意：4和7有两种读法：
4 → し（纯音读）或よん（来自训读よっつ的变形），忌讳し（与死同音，凶数）
7 → しち（音读）或なな（训读），なな更常用（避免与一しちい混淆）
```

**日语固有数字（訓読み）——1-10的物品计数：**
```
一つ（ひとつ）二つ（ふたつ）三つ（みっつ）四つ（よっつ）五つ（いつつ）
六つ（むっつ）七つ（ななつ）八つ（やっつ）九つ（ここのつ）十（とお）
```

**大数字的表达：**
```
日语以「万」（10,000）为进位单位，与汉语相同（但与英语billion等不同）：

100       → 百（ひゃく）
1,000     → 千（せん）
10,000    → 一万（いちまん）
100,000   → 十万（じゅうまん）
1,000,000 → 百万（ひゃくまん）
10,000,000 → 千万（せんまん）
100,000,000 → 一億（いちおく）

年份：
1990年 → 千九百九十年（せんきゅうひゃくきゅうじゅうねん）
2024年 → 二千二十四年（にせんにじゅうよねん）

日本纪元（元号）：
2024年 = 令和6年（れいわろくねん）
```

### 日期表达

**月份（月/つき）：**
```
1月（いちがつ）  7月（しちがつ）
2月（にがつ）    8月（はちがつ）
3月（さんがつ）  9月（くがつ）
4月（しがつ）    10月（じゅうがつ）
5月（ごがつ）    11月（じゅういちがつ）
6月（ろくがつ）  12月（じゅうにがつ）
```

**日期（日/か・にち）——不规则读法：**
```
1日  ついたち（例外！不是いちにち）
2日  ふつか
3日  みっか
4日  よっか
5日  いつか
6日  むいか
7日  なのか
8日  ようか
9日  ここのか
10日 とおか
11日以后 じゅういちにち、じゅうににち...（基本规则）
20日 はつか（例外！不是にじゅうにち）
24日 にじゅうよっか（例外！）
```

**星期（曜日/ようび）：**
```
月曜日（げつようび）= 星期一（月）
火曜日（かようび）  = 星期二（火）
水曜日（すいようび）= 星期三（水）
木曜日（もくようび）= 星期四（木）
金曜日（きんようび）= 星期五（金）
土曜日（どようび）  = 星期六（土）
日曜日（にちようび）= 星期日（日）

记忆法：日月火水木金土 = 七曜（来自中国传统七曜）
```

### 时间表达

**时（時/じ）和分（分/ふん・ぷん）：**
```
时：
1時（いちじ）2時（にじ）3時（さんじ）4時（よじ）
5時（ごじ）6時（ろくじ）7時（しちじ）8時（はちじ）
9時（くじ）10時（じゅうじ）11時（じゅういちじ）12時（じゅうにじ）

分（注意浊音/半浊音变化）：
1分（いっぷん）  6分（ろっぷん）
2分（にふん）    7分（ななふん）
3分（さんぷん）  8分（はっぷん）
4分（よんぷん）  9分（きゅうふん）
5分（ごふん）    10分（じゅっぷん）

完整时间表达：
午前9時30分（ごぜんくじさんじゅっぷん）= 上午9点30分
午後3時15分（ごごさんじじゅうごふん）= 下午3点15分
```
