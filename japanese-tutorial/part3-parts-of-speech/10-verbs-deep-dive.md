# 第10章：动词深度解析

> 「動詞は言語の心臓である。それが動けば、文が生きる。」
> 动词是语言的心脏。它一旦跳动，句子便有了生命。

日语动词体系以其精密的活用规则著称。五段、一段、不规则三大类动词，六种基本活用形，再加上可能形、意志形、命令形等派生形态，构成了一套逻辑严密、规律性极强的系统。本章将从底层规律出发，让你彻底驯服日语动词。

---

## 10.1 动词活用形总览

### 学校文法 vs 教育文法

日本国内教学使用"学校文法"（六活用形），对外日语教学常用"教育文法"（て形/た形/ない形等实用形）。两套体系各有侧重：

| 学校文法（六活用形） | 教育文法（实用形） | 主要功能 |
|---------------------|-------------------|---------|
| 未然形（みぜんけい） | ない形/よう形 | 接否定、接意志 |
| 連用形（れんようけい） | ます形/て形/た形 | 接ます、接続 |
| 終止形（しゅうしけい） | 辞書形（原形） | 句末、接续名词从句 |
| 連体形（れんたいけい） | 辞書形（同上） | 修饰名词（现代日语同终止形） |
| 假定形（かていけい） | ば形 | 条件句 |
| 命令形（めいれいけい） | 命令形 | 命令 |

**现代日语中，终止形=连体形**（古典日语有区别，如「あり」vs「ある」）。

### 六活用形功能总结表

| 活用形 | 接续典型例 | 功能说明 |
|--------|-----------|---------|
| 未然形 | ～ない/～なかった/～(よ)う | 否定/意志/推量 |
| 連用形 | ～ます/～て/～た/～ながら | 礼貌体/接続/过去/同时 |
| 終止形 | 句末/～と思う/～から | 陈述完结/引用/原因 |
| 連体形 | ～名詞/～こと/～ので | 修饰名词/名词化/原因 |
| 假定形 | ～ば | 条件 |
| 命令形 | 单独使用/～よ | 命令/指示 |

### 音便现象详解

音便（おんびん）是语音自然流畅化的结果，发生在て形/た形的连接处：

#### イ音便（い音便）

适用词尾：**く/ぐ**

```
書く → 書いて/書いた   （く→い）
泳ぐ → 泳いで/泳いだ   （ぐ→い+浊化）
聞く → 聞いて/聞いた
漕ぐ → 漕いで/漕いだ

发生原因：き/ぎ+て/た → いて/いだ （更易发音）
例外！行く → 行って （不发生イ音便，历史音变原因）
```

#### 促音便（そくおんびん）

适用词尾：**う/つ/る（五段）**

```
買う → 買って/買った    （う→っ）
待つ → 待って/待った    （つ→っ）
帰る → 帰って/帰った    （る→っ）
切る → 切って/切った    （五段る→っ）

发生原因：母音+て/た → 促音+て/た （减少连读困难）
```

#### 撥音便（はつおんびん）

适用词尾：**ぬ/ぶ/む**

```
死ぬ → 死んで/死んだ    （ぬ→ん）
遊ぶ → 遊んで/遊んだ    （ぶ→ん）
飲む → 飲んで/飲んだ    （む→ん）
呼ぶ → 呼んで/呼んだ

发生原因：ぬ/ぶ/む → ん （鼻音化，更流畅）
浊化：ん+で/だ（だ行濁音）
```

## 10.3 一段動詞（二类动词）的完整活用

一段动词词尾均为**る**，词干最后一个假名为い段（上一段）或え段（下一段）。

### 上一段動詞：起きる（おきる）

词干固定为「おき」，活用只改词尾る部分：

| 活用形 | 形式 | 备注 |
|--------|------|------|
| 未然形 | おき（ない） | 去る |
| 連用形 | おき（ます/て/た） | 去る |
| 終止形 | おき**る** | |
| 假定形 | おき**れ**（ば） | |
| 命令形 | おき**ろ/よ** | ろ或よ |
| 意志形 | おき**よう** | |
| て形 | おき**て** | 无音便 |
| た形 | おき**た** | |
| 可能形 | おき**られる** | |

### 下一段動詞：食べる（たべる）

| 活用形 | 形式 |
|--------|------|
| 未然形 | たべ（ない） |
| 連用形 | たべ（ます/て/た） |
| 終止形 | たべ**る** |
| 假定形 | たべ**れ**（ば） |
| 命令形 | たべ**ろ/よ** |
| 意志形 | たべ**よう** |
| て形 | たべ**て** |
| 可能形 | たべ**られる** |

**一段动词规律：** 去掉词尾る，接各种活用即可。**绝无音便**。

### 看似五段实为一段的陷阱词

这是日语学习最常见的错误来源：

| 动词 | 貌似 | 实际 | 判断依据 |
|------|------|------|---------|
| 帰る（かえる） | 一段？ | **五段** | 词干+る，而非词干末え段 |
| 切る（きる） | 一段？ | **五段** | 词干+る |
| 走る（はしる） | 一段？ | **五段** | 词干+る |
| 入る（はいる） | 一段？ | **五段** | 词干+る |
| 知る（しる） | 一段？ | **五段** | 词干+る |
| 蹴る（ける） | 一段？ | **五段** | 词干+る（历史原因） |
| 起きる（おきる） | — | **一段** | 词干おき（い段） |
| 着る（きる） | 难辨 | **一段** | 词干き（い段）|
| 見る（みる） | — | **一段** | 词干み（い段） |

**判断技巧：** 将る前一个假名转换为あ段，能接ない则是五段；直接接ない则是一段：

```
帰る → 帰ら+ない = 帰らない ✓ → 五段
起きる → 起き+ない = 起きない ✓（不加ら） → 一段
走る → 走ら+ない = 走らない ✓ → 五段
着る → 着+ない = 着ない ✓（不加ら） → 一段
```

## 10.4 不规则动词する/来る的完整活用表

### する的活用

する是日语中使用频率极高的万能动词，可将名词转化为动词：

| 活用形 | 形式 |
|--------|------|
| 未然形 | **し/さ/せ** |
| 連用形 | **し** |
| 終止形 | **する** |
| 連体形 | **する** |
| 假定形 | **すれ** |
| 命令形 | **しろ/せよ** |
| て形 | **して** |
| た形 | **した** |
| 意志形 | **しよう** |
| 可能形 | **できる** （特殊！不是「される」） |
| 使役形 | **させる** |
| 被动形 | **される** |
| 否定形 | **しない** |

```
名詞+する の例：

運動する → 運動して/運動した
料理する → 料理して/料理した
電話する → 電話して/電話した
```

## 10.5 动词的各种形态详解

### 可能形

表示"能够做……"的能力或可能性：

#### 构成规则

```
五段动词：词尾う段 → え段+る
書く → 書ける
読む → 読める
話す → 話せる
買う → 買える
帰る → 帰れる

一段动词：词干+られる（或ら抜き：+れる）
食べる → 食べられる（食べれる）
起きる → 起きられる（起きれる）
見る → 見られる（見れる）

する → できる（特殊）
来る → 来られる（来れる）
```

#### ら抜き言葉（ら脱落现象）

```
规范形：食べられる / 見られる / 来られる
ら抜き：食べれる / 見れる / 来れる

现状：
- 年轻人日常会话广泛使用ら抜き
- 正式场合/书面语应使用标准形
- 消除与被动形的歧义（食べられる既是可能也是被动）
  ら抜きの食べれる → 明确仅为可能形

例：
昨日は忙しくて、ご飯も食べれなかった。（ら抜き，口语）
この映画は子供でも見られます。（标准，正式）
```

### 意志形/意向形（～おう/～よう）

表示"打算/决意/邀请"：

```
五段动词：词尾う段 → お段+う（长音）
書く → 書こう
読む → 読もう
話す → 話そう
帰る → 帰ろう

一段动词：词干+よう
食べる → 食べよう
起きる → 起きよう

する → しよう
来る → 来よう（こよう）

使用场景：
1. 自己的决意：
   明日から運動しよう。 ← 我打算从明天开始运动

2. 邀请/提议（对方参与）：
   一緒に行こう！ ← 一起去吧！
   映画でも見ようか。← 要不要去看电影？

3. ～うとする/～うとしていた（尝试/正要）：
   出かけようとしたとき、電話が来た。
   正要出门时，电话来了。
```

### 命令形与禁止形

命令形语气强烈，在日常会话中要慎用：

```
五段动词：词尾う段 → え段
書く → 書け
読む → 読め
話す → 話せ
帰る → 帰れ

一段动词：词干+ろ（或よ）
食べる → 食べろ（食べよ，书面/古风）
起きる → 起きろ

する → しろ（せよ，书面）
来る → 来い（こい）

使用场景（命令形非常粗鲁，注意场合）：
- 军事/体育指导：走れ！前へ進め！
- 紧急情况：逃げろ！助けろ！
- 叙事/内心独白（小说中）
- ～なさい（より礼貌的命令）：
  早く起きなさい。快点起床。
  宿題をしなさい。去做作业。

禁止形：動詞原形+な
食べるな！ ← 不许吃！
入るな！   ← 不许进！（门牌/警告用）
触るな！   ← 不许碰！
```

### 推量形

```
動詞+だろう/でしょう
行くだろう → 可能会去
来るでしょう → 应该会来（较礼貌）
雨が降るでしょう → 应该会下雨

確信度：
～にちがいない > ～はずだ > ～だろう > ～かもしれない
（一定）      （应该）   （大概）    （也许）
```

## 10.6 自動詞と他動詞（自动词与他动词）

自他动词是日语语法的核心难点，也是体现日语"以物为主体"思维方式的关键。

### 基本区别

```
自動詞（じどうし）：动作自然发生，不需要宾语
  ドアが開いた。 ← 门（自然）开了

他動詞（たどうし）：动作需要施事者对宾语施加
  私がドアを開けた。← 我（主动）把门打开了
```

### 成对的自他动词（30对）

| 意义 | 自動詞 | 他動詞 |
|------|--------|--------|
| 开 | 開く（あく） | 開ける（あける） |
| 关 | 閉まる（しまる） | 閉める（しめる） |
| 开始 | 始まる（はじまる） | 始める（はじめる） |
| 结束 | 終わる（おわる） | 終える（おえる） |
| 出 | 出る（でる） | 出す（だす） |
| 入 | 入る（はいる） | 入れる（いれる） |
| 上 | 上がる（あがる） | 上げる（あげる） |
| 下 | 下がる（さがる） | 下げる（さげる） |
| 转 | 回る（まわる） | 回す（まわす） |
| 挂 | かかる | かける |
| 掉落 | 落ちる（おちる） | 落とす（おとす） |
| 损坏 | 壊れる（こわれる） | 壊す（こわす） |
| 烧 | 燃える（もえる） | 燃やす（もやす） |
| 溶 | 溶ける（とける） | 溶かす（とかす） |
| 沸腾 | 沸く（わく） | 沸かす（わかす） |
| 弯 | 曲がる（まがる） | 曲げる（まげる） |
| 混 | 混ざる（まざる） | 混ぜる（まぜる） |
| 续 | 続く（つづく） | 続ける（つづける） |
| 聚 | 集まる（あつまる） | 集める（あつめる） |
| 决 | 決まる（きまる） | 決める（きめる） |
| 分 | 分かれる（わかれる） | 分ける（わける） |
| 连 | つながる | つなぐ |
| 止 | 止まる（とまる） | 止める（とめる） |
| 立 | 立つ（たつ） | 立てる（たてる） |
| 完成 | できる | — |
| 惊 | 驚く（おどろく） | 驚かす（おどろかす） |

| 暖 | 温まる（あたたまる） | 温める（あたためる） |
| 散 | 散る（ちる） | 散らす（ちらす） |

### 自他动词的语义差异与选择

```
「が～ている」（自動詞+ている）→ 结果状态（无责任归属）
「を～ている」（他動詞+ている）→ 动作进行（施事者明确）

電気がついている。  ← 灯亮着（状态，不关注是谁开的）
電気をつけている。 ← 正在开灯/让灯一直开着（施事者）

窓が開いている。   ← 窗户开着（状态）
窓を開けている。   ← （某人）正在开窗
ドアが壊れている。 ← 门坏了（状态，不含责任）
彼がドアを壊した。 ← 他把门弄坏了（明确责任）
```

**日语的"无责任主语"倾向：**

```
日本人在描述错误或事故时，倾向于用自动词：
書類が紛失しました。  ← 文件丢失了（自动词，无人为责任）
書類を紛失しました。  ← 我把文件弄丢了（他动词，责任明确）

コップが割れました。  ← 杯子碎了（自动词，自然发生感）
コップを割りました。  ← 我打碎了杯子（他动词，负责）
```

## 10.7 複合動詞（复合动词）

复合动词是两个动词合并，第一个动词以**連用形**接续第二个动词，形成新的语义：

### 核心复合动词详解

| 后置动词 | 意义 | 例子 |
|---------|------|------|
| ～始める（はじめる） | 开始～ | 食べ始める/降り始める |
| ～続ける（つづける） | 继续～ | 働き続ける/待ち続ける |
| ～終わる（おわる） | 完成～ | 食べ終わる/書き終わる |
| ～出す（だす） | 突然开始/取出 | 走り出す/泣き出す |
| ～込む（こむ） | 进入/深入 | 飛び込む/考え込む |
| ～上げる（あげる） | 完成/向上 | 書き上げる/仕上げる |
| ～合う（あう） | 互相～ | 助け合う/話し合う |
| ～直す（なおす） | 重新～ | 書き直す/やり直す |
| ～忘れる（わすれる） | 忘记～ | 持ち忘れる/言い忘れる |
| ～過ぎる（すぎる） | 过度～ | 食べ過ぎる/働き過ぎる |
| ～返す（かえす） | 归还/反复 | 繰り返す/言い返す |
| ～切る（きる） | 彻底/完全 | 言い切る/走り切る |
| ～つける（つける） | 习惯/固定 | 言いつける/着つける |
| ～かける（かける） | 做到一半 | 食べかける/言いかける |

```
複合動詞の例文：
1. 子供が突然泣き出した。孩子突然哭起来了。
2. 彼は走り続けた。他一直跑着。
3. レポートを書き上げた。论文写完了。
4. お互いに助け合いましょう。让我们互相帮助吧。
5. もう一度やり直そう。重新来过吧。
6. 食べ過ぎて気分が悪い。吃太多了，感觉不舒服。
7. その問題について話し合った。就那个问题进行了讨论。
8. 言いかけて止まった。说到一半停下来了。
```

## 10.8 练习题

### 练习一：活用形判断

写出以下动词的て形和た形：

| 动词 | 原形 | て形 | た形 |
|------|------|------|------|
| 1 | 飲む | | |

| 3 | 来る | | |
| 4 | 泳ぐ | | |
| 5 | 起きる | | |
| 6 | する | | |
| 7 | 話す | | |
| 8 | 帰る | | |

### 练习二：自他动词辨析

选择正确的动词：

1. 会議が（始まった/始めた）。
2. 先生が授業を（始まった/始めた）。
3. 窓が（閉まっている/閉めている）。
4. 誰かが窓を（閉まった/閉めた）。
5. コップが（落ちた/落とした）。

### 练习三：可能形转换

将以下动词变为可能形（标准形）：

1. 食べる → ___（能吃）
2. 書く → ___（能写）
3. 来る → ___（能来）
4. する → ___（能做）
5. 話す → ___（能说）

### 答案

练习一：
1. 飲んで/飲んだ  2. 食べて/食べた  3. 来て/来た
4. 泳いで/泳いだ  5. 起きて/起きた  6. して/した
7. 話して/話した  8. 帰って/帰った

练习二：
1. 始まった（自動詞）  2. 始めた（他動詞）
3. 閉まっている（自動詞，状态）  4. 閉めた（他動詞）
5. 落ちた（自動詞，自然落下）

练习三：
1. 食べられる  2. 書ける  3. 来られる  4. できる  5. 話せる

## 本章小结

| 知识点 | 核心规律 |
|--------|---------|
| 动词分类 | 五段/一段/不规则，以ない形快速判断 |
| 音便 | イ音便（く/ぐ）、促音便（う/つ/る）、撥音便（ぬ/ぶ/む） |
| 一段陷阱 | 帰る/切る/走る等は五段，用ない形验证 |
| 自他动词 | が+自動詞+ている=状态，を+他動詞=主动行为 |
| 可能形 | 五段→え段+る，一段→られる，ら抜き是口语现象 |
| 复合动词 | 連用形+後続動詞，熟记常用组合 |

> 动词活用不是死记硬背的规则表，而是语言演化的音韵逻辑。当你理解了为什么発音便（音便）会发生，活用表就成了自然而然的结果。
