# 第9章：名词与代名词

> 「言葉は単なる記号ではなく、文化そのものである。」
> 语言不仅仅是符号，它本身就是文化。

日语名词体系看似简单——没有格变化、没有单复数屈折、没有冠词——但这种表面的简洁之下，隐藏着极为精密的分类逻辑、丰富的形式名词体系，以及深刻体现日本社会结构的代词系统。本章将从语言学角度深入剖析这套体系。

---

## 9.1 日语名词的根本特点

### 三大"缺失"背后的逻辑

初学者往往从"日语名词没有XXX"来认识名词，但更准确的理解是：日语用**其他手段**来表达这些概念。

| 特征 | 日语处理方式 | 例句 |
|------|------------|------|
| 无单复数变化 | 靠语境/量词/复数词缀 | 本（ほん）可以是一本书，也可以是很多书 |
| 无性别标记 | 靠语境/称呼方式 | 先生（せんせい）可以是男老师或女老师 |
| 无冠词 | 靠助词は/が+语境 | 猫（ねこ）可表示"一只猫"或"猫这种动物" |

**复数表达的几种方式：**

```
人（ひと）→ 人々（ひとびと）  [叠字式复数]
山（やま）→ 山々（やまやま）
国（くに）→ 国々（くにぐに）

私（わたし）→ 私たち（わたしたち）  [たち/ら+接尾词]
彼（かれ）  → 彼ら（かれら）
学生（がくせい）→ 学生たち（がくせいたち）

本（ほん）が3冊（さつ）あります。  [量词明示数量]
たくさんの本（ほん）があります。   [数量副词]
```

### 日语名词的形态特点

日语名词本身**不发生形态变化**，所有语法信息由助词承担：

```
猫が → 猫（作为主语）
猫を → 猫（作为宾语）
猫に → 猫（方向/给予对象）
猫の → 猫（所有/修饰）
```

这与印欧语系名词格变化形成鲜明对比。

## 9.2 名词的分类

### 9.2.1 普通名詞 vs 固有名詞

| 类别 | 说明 | 例子 |
|------|------|------|
| 普通名詞（ふつうめいし） | 一类事物的通称 | 山、川、犬、学校、愛 |
| 固有名詞（こゆうめいし） | 特定事物的专名 | 富士山、隅田川、東京大学 |

固有名词在日语中有一个特殊现象：**汉字读法多样**，同一汉字组合可能有多种读法，需要单独记忆：

```
東京（とうきょう）  ← 音读
大阪（おおさか）   ← 混合
神奈川（かながわ） ← 训读
```

### 9.2.2 時間名詞（时间名词）

日语时间体系以"今（いま）"为核心，构成完整的时间坐标系：

**日单位：**

| 日语 | 读音 | 含义 |
|------|------|------|
| 一昨日 | おととい | 前天 |
| 昨日 | きのう | 昨天 |
| 今日 | きょう | 今天 |
| 明日/明日 | あした/あす | 明天 |
| 明後日 | あさって | 后天 |

**周单位：**

| 日语 | 读音 | 含义 |
|------|------|------|
| 先々週 | せんせんしゅう | 上上周 |
| 先週 | せんしゅう | 上周 |
| 今週 | こんしゅう | 本周 |
| 来週 | らいしゅう | 下周 |
| 再来週 | さらいしゅう | 下下周 |

**月/年单位：**

| 日语 | 读音 | 含义 |
|------|------|------|
| 先月 | せんげつ | 上个月 |
| 今月 | こんげつ | 本月 |
| 来月 | らいげつ | 下个月 |
| 去年/昨年 | きょねん/さくねん | 去年 |
| 今年 | ことし | 今年 |
| 来年 | らいねん | 明年 |

**注意：** 时间名词不需要助词に也可作为时间状语（相对时间词如今日、明日不加に，绝对时间如3時、月曜日需要に）：

```
✓ 今日、学校に行きます。（今日不加に）
✓ 月曜日に、学校に行きます。（曜日加に）
✓ 3時に来てください。（时刻加に）
✗ 今日に、学校に行きます。（错误）
```

### 9.2.3 数量名詞与助数詞（量词）体系

日语量词（助数詞，じょすうし）是日语学习的重难点，量词的选择取决于**物体的形状、种类、大小**。

#### 量词音变规则（连浊/半浊）

量词与数字组合时发生音变，规律如下：

```
～本（ほん）的变化：
1本 → いっぽん（促音便+半浊）
2本 → にほん
3本 → さんぼん（鼻音+浊音）
4本 → よんほん
6本 → ろっぽん
8本 → はっぽん
10本 → じゅっぽん/じっぽん
```

```
～匹（ひき）的变化：
1匹 → いっぴき
2匹 → にひき
3匹 → さんびき
4匹 → よんひき
6匹 → ろっぴき
8匹 → はっぴき
10匹 → じゅっぴき
```

#### 固有数词系统（ひとつ/ふたつ系）

日语存在源自古日语的固有数词系统，主要用于1-10的不限定物品计数：

| 数字 | 固有数词 | 汉字 | 使用场景 |
|------|---------|------|---------|
| 1 | ひとつ | 一つ | 一个（通用） |
| 2 | ふたつ | 二つ | 两个 |
| 3 | みっつ | 三つ | 三个 |
| 4 | よっつ | 四つ | 四个 |
| 5 | いつつ | 五つ | 五个 |
| 6 | むっつ | 六つ | 六个 |
| 7 | ななつ | 七つ | 七个 |
| 8 | やっつ | 八つ | 八个 |
| 9 | ここのつ | 九つ | 九个 |
| 10 | とお | 十 | 十个 |
| 20 | はたち | 二十 | 仅用于年龄 |

```
ひとつ系的使用场景：
- 量词不明时：何かひとつ取ってください。
- 年龄（小孩）：子供はみっつです。
- 理由列举：ひとつは時間がないから。
```

## 9.3 代名詞（代词）体系

日语代词体系是整个语言中最能体现**日本社会结构与人际关系**的部分。理解代词，就是理解日本人的社会意识。

### 9.3.1 第一人称代词对照表

| 代词 | 读音 | 使用者 | 场合 | 语气 |
|------|------|--------|------|------|
| 私 | わたし | 男女通用 | 正式/日常 | 中性、礼貌 |
| 私 | わたくし | 男女通用 | 极正式 | 非常礼貌、郑重 |
| 僕 | ぼく | 主要男性 | 日常/轻松 | 温和、少年感 |
| 俺 | おれ | 男性 | 朋友间 | 强烈、粗犷 |
| あたし | — | 主要女性 | 日常 | 柔和、女性化 |
| うち | — | 关西女性 | 日常 | 亲密、关西腔 |
| 自分 | じぶん | 男性（军/运动） | 自指 | 严格、集体感 |
| 吾輩/余 | わがはい/よ | 文学/老人 | 文语 | 高傲、古典 |

**使用场景示例：**

```
就职面接（求职面试）：
私（わたし）は山田と申します。  ← 必须用わたし

男性朋友聊天：
俺（おれ）、今日バイトだわ。  ← 自然

教室（学生发言）：
僕（ぼく）は反対です。  ← 较礼貌但轻松

女性日常：
あたし、それ知らなかった！ ← 女性日常用语
```

### 9.3.2 第二人称代词——日语为何"回避你"

这是日语学习最反直觉的点之一。日语中虽有第二人称词，但**在日常会话中极少直接使用**：

| 代词 | 读音 | 使用限制 | 语气风险 |
|------|------|---------|---------|
| あなた | — | 书面语/歌词/夫妻间 | 对上司用会显得失礼 |
| きみ | — | 长辈对晚辈/恋人间 | 有居高临下感 |
| お前/おまえ | — | 男性对亲密者 | 对陌生人极失礼 |
| あんた | — | 关西/愤怒时 | 带攻击性 |
| てめえ/貴様 | — | 骂人/激烈冲突 | 严重失礼 |

**正确替代方式：**

```
❌ あなたは学生ですか？（对初识者显得生硬）
✓ 学生ですか？           （直接省略主语）
✓ 田中さんは学生ですか？（用姓名替代）

❌ あなたはどこから来ましたか？
✓ どちらからいらっしゃいましたか？（用敬语回避主语）

上司に対して（对上司）：
✓ 部長はもうご覧になりましたか？（用职位替代）
✓ ご確認いただけましたか？       （被动/敬语回避）
```

### 9.3.3 第三人称代词

日语第三人称代词同样有文化限制：

| 代词 | 读音 | 特点 |
|------|------|------|
| 彼 | かれ | 原指"那人"，现代用于男性/男友 |
| 彼女 | かのじょ | 原指"那个女人"，现代用于女性/女友 |
| あの人 | あのひと | 中性礼貌，指第三者 |
| あの方 | あのかた | 礼貌/尊重 |
| 彼ら | かれら | 他们（男性为主的群体） |
| 彼女たち | かのじょたち | 她们 |

```
文化注意：
在传统语境中，「彼」和「彼女」有"男/女朋友"的含义，
在职场谈论第三者时，更常用「あの方」「山田さん」等。

彼女はどこですか？ ← 可能被理解为"你女朋友在哪里？"
田中さんはどこですか？ ← 明确指田中这个人
```

### 9.3.4 用名字/职位代替代词的文化习惯

日本社会的集体主义和等级意识使得**称呼方式本身承载了大量社交信息**：

```
家族内（家庭内部）：
父→子：名前で呼ぶ（直呼名字）
子→父：お父さん/パパ（用称呼语）
夫→妻：あなた/名前
妻→夫：あなた/名前/パパ（有子女后）

職場（职场）：
部下→上司：部長、課長（用职位）
上司→部下：田中くん/田中さん（姓+称呼）
同僚：苗字+さん/名前+ちゃん（亲疏程度）

学校：
先生→学生：姓名/苗字（课堂点名）
学生→先生：先生（职业称呼）
学生同士：ニックネーム/名前
```

## 9.4 形式名詞（形式名词）

形式名詞（けいしきめいし）是日语语法的核心机制之一。它们是**失去实质语义、转而承担语法功能**的名词。

### こと（事）

核心语义：**事情、事实的抽象化**

| 用法 | 结构 | 例句 | 含义 |
|------|------|------|------|
| 名词化 | 動詞+こと | 泳ぐことが好きです | 喜欢游泳这件事 |
| 义务 | ～ことがある | 会議に遅れることがある | 有时会迟到 |
| 经验 | ～たことがある | 富士山に登ったことがある | 曾经爬过富士山 |
| 必须 | ～ことになっている | 8時に来ることになっている | 规定8点来 |
| 禁止 | ～ことはできない | 入ることはできない | 不能进入 |

```
例句集：
1. 毎日運動することは大切です。每天运动是很重要的。
2. 彼女に会ったことがありますか？你见过她吗？
3. ここで写真を撮ることはできません。这里不能拍照。
4. 報告書を提出することになっています。需要提交报告书。
5. 諦めないことが大事です。不放弃是很重要的。
```

### もの（物）

核心语义：**物品的具体化、理所当然的事**

| 用法 | 结构 | 例句 | 含义 |
|------|------|------|------|
| 物品化 | 動詞+もの | 食べるものがない | 没有吃的东西 |
| 理所当然 | ～ものだ | 人は死ぬものだ | 人终有一死（自然规律） |
| 怀念/感叹 | ～ものだ | 昔はよく遊んだものだ | 以前常常玩啊（回忆） |
| 欲望 | ～ものだ（想） | 一度行ってみたいものだ | 真想去一次啊 |
| 理由/解释 | ～ものだから | 忙しいものだから | 因为很忙（理由/辩解） |

```
例句集：
1. 買うものがたくさんある。有很多要买的东西。
2. 子供は元気なものだ。孩子们总是精力充沛的（理所当然）。
3. 学生の頃はよく徹夜したものだ。学生时代常常熬夜啊（感叹）。
4. 一人でやり遂げたいものです。真想一个人完成啊（愿望）。
5. 電車が遅れたものですから。因为电车晚点了（辩解/理由）。
```

### こと vs もの 核心辨析

```
のことが好きです → 喜欢某件事（抽象行为）
のものが好きです → 喜欢某样东西（具体物品）

甘いものが好きです。  ← 喜欢甜的东西（具体食品）
甘えることが好きです。← 喜欢撒娇这件事（行为）

約束は守るものだ。  ← 约定是应该遵守的（规范/理所当然）
約束を守ることが大切だ。← 遵守约定是重要的（陈述事实）
```

### ところ（所）

核心语义：**场所、时间点、局面状态**

```
例句集：
1. ちょうど出かけるところです。（时间点：正要出门）
2. 今、食事をしているところです。（进行中：正在吃饭）
3. 映画を見たところです。（刚完成：刚看完电影）
4. いいところに来ましたね。（恰好：来得正好）
5. そこがいいところです。（场所/要点：那就是好的地方）
6. 見せてもらえるところまで来た。（到了能看的地步）
```

### わけ（訳）

核心语义：**理由、道理、结论逻辑**

```
例句集：
1. そういうわけで、遅れました。（就这样，我迟到了——理由总结）
2. 彼が怒るのもわけだ。（他生气也是有道理的）
3. わけがわからない。（不明白道理/搞不懂）
4. 行けるわけがない。（不可能去——逻辑上不可能）
5. そういうわけではありません。（并不是那个意思——否定理解）
6. 結局、無駄だったわけだ。（结果，可以说是白费了——逻辑结论）
```

### はず（筈）

核心语义：**按理应当、预期推断**

```
例句集：
1. 彼はもう来ているはずです。（按理应该到了）
2. そんなはずはない。（不可能/不应该是那样）
3. 明日は晴れるはずだ。（按天气预报应该晴天）
4. 彼女は知っているはずがない。（她不可能知道）
5. こんなに疲れるはずだ。（怪不得这么累——恍然大悟）
```

### つもり（積もり）

核心语义：**意图、预定、（主观上的）认为**

```
例句集：
1. 来週、旅行するつもりです。（打算下周旅行）
2. 本当のことを言ったつもりだ。（我以为自己说了实话）
3. 冗談のつもりだった。（本来是开玩笑的意思）
4. 行かないつもりだ。（打算不去）
5. 努力しているつもりなのに。（明明自认为在努力，却…）
```

## 9.5 名词修饰结构（連体修飾）

日语的修饰语一律**前置**，这与汉语相同，却与英语不同。

### 三种基本修饰结构

**1. 名詞の+名詞**

```
日本語の教科書  ← 日语教科书
東京の空       ← 东京的天空
木で作った机    ← 用木头做的桌子（で+動詞た+名詞）
```

**2. 形容詞+名詞**

```
い形容詞直接连接：
大きい犬   ← 大狗
おいしいケーキ ← 好吃的蛋糕

な形容詞+な+名詞：
きれいな花  ← 漂亮的花
静かな部屋  ← 安静的房间
```

**3. 動詞（普通形）+名詞**

这是日语最灵活的修饰结构：

```
食べる人      ← 吃饭的人（现在/将来）
食べた料理    ← 吃过的料理（过去）
食べている人  ← 正在吃饭的人（进行）
食べない子供  ← 不吃饭的孩子（否定）

複雑な例：
昨日公園で出会ったあの女性は誰ですか？
← 昨天在公园遇到的那位女性是谁？
```

### の的特殊修饰功能

```
の可以代替主格が进入修饰节：
私が作った料理 ≈ 私の作った料理  ← 我做的料理
彼女が書いた手紙 ≈ 彼女の書いた手紙 ← 她写的信

注意：この限定の使い方は連体修飾の中でのみ可能
```

## 9.6 名词化——用の和こと将句子变为名词

日语中，将动词短语变为名词成分（即"名词化"）是句子嵌套的核心机制。

### の vs こと 的系统对比

| 特征 | の | こと |
|------|-----|------|
| 语体 | 口语化 | 较书面/正式 |
| 感知动词（见/听/感觉） | 用の | 不自然 |
| 抽象动词（思考/相信） | 两者皆可 | 更常见 |
| する的名词化 | 少见 | 常见 |
| ～が好き/嫌い | 两者皆可 | 两者皆可 |

```
感知动词必须用の：
✓ 彼が走るのを見た。  ← 看到他跑步
✗ 彼が走ることを見た。

思考动词用こと更自然：
✓ 彼が来ることを知っていた。 ← 知道他会来
△ 彼が来るのを知っていた。

好き/嫌い两者均可：
✓ 歌うのが好きです。
✓ 歌うことが好きです。

正式书面语：
✓ 本研究の目的は～を明らかにすることである。
```

## 9.7 练习题

### 练习一：量词填空

从括号中选择正确的量词：

1. 猫が（　）います。（2：二匹/二台/二枚）
2. 本棚に本が（　）あります。（5：五冊/五本/五枚）
3. 鳥が空を（　）飛んでいます。（3：三羽/三匹/三頭）
4. 車が（　）止まっています。（4：四台/四枚/四本）
5. 昨日、映画を（　）見ました。（1：一本/一冊/一枚）

### 练习二：代词使用判断

判断以下代词使用是否适切，并说明理由：

1. 部長、**あなた**はもう読みましたか？（对部长说话）
2. 田中さん、**田中さん**のご意見はどうですか？（询问意见）
3. 子供たちに：**お前ら**、静かにしろ！（老师对学生）
4. 夫婦間で：**あなた**、これ見て！（妻子对丈夫）

### 练习三：形式名词辨析

用こと/もの/わけ/はず/つもり填空：

1. 彼はもう来ている（　）だ。（按理应该来了）
2. 甘い（　）が食べたい。（想吃甜的东西）
3. 彼女に話した（　）なのに、伝わっていなかった。（以为说了）
4. 約束を守る（　）は大切だ。（遵守约定这件事）
5. だから失敗した（　）だ。（所以才失败了——逻辑结论）

### 答案参考
1. 二匹 / 2. 五冊 / 3. 三羽 / 4. 四台 / 5. 一本
1. 不适切，应用「部長はもうお読みになりましたか？」
2. 不自然，通常直接问「ご意見はどうですか？」
3. 不适切，対学生用「おまえら」失礼，应用「みんな」
4. 适切，夫妻间用あなた自然
形式名词：はず / もの / つもり / こと / わけ

## 本章小结

| 知识点 | 核心要点 |
|--------|---------|
| 名词特点 | 无单复数/性别/冠词变化，靠助词和量词表达语法关系 |
| 时间名词 | 以今为中心的完整坐标体系，相对时间不加に |
| 量词系统 | 按形状/种类选择，数字+量词发生音变 |
| 第一人称 | 多种选择，按性别/场合/亲疏度选用 |
| 第二人称 | 尽量回避，用姓名/职位/省略替代 |
| 形式名词 | こと/もの/ところ/わけ/はず/つもり各有核心语义 |
| 名词化 | の口语化（感知动词必用），こと书面化（抽象动词常用） |

> 名词是语言的砖石，代词是社会关系的镜子。掌握日语的名词体系，就是在构建理解日本文化的基础框架。
