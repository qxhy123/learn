# 第11章：形容词与副词进阶

> 「色彩のない世界に言葉が色を与える。形容詞はその筆である。」
> 语言为无色的世界赋予色彩。形容词，就是那支画笔。

形容词让语言从黑白变为彩色，副词则调节着色彩的浓淡与节奏。日语的形容词体系有其独特的感情限制逻辑，而副词中的拟声拟态词（オノマトペ）更是日语最具魅力的特征之一。本章将带你深入这个充满感性与精度的词语世界。

---

## 11.1 形容词的语感差异

### 客观描述 vs 主观感受

日语形容词在使用时有一个极重要却常被初学者忽视的区别：**い形容词与な形容词在感情描述上有本质不同**，而同一意义的不同词形也可能传达微妙的语感差异。

#### おおきい vs おおきな——连体用法的微妙差异

| 形式 | 用法 | 语感 | 例句 |
|------|------|------|------|
| 大きい犬 | 谓语/连体 | 客观陈述大小 | その犬は大きい |
| 大きな犬 | 仅连体 | 印象性、感性 | 大きな夢を持て |

```
大きな夢（おおきなゆめ）← 感性，强调"宏大的"印象
大きい夢 ← 较少见，不自然

大きな問題（おおきなもんだい）← 严重的、重大的（质的）
大きい問題 ← 偏向字面的"尺寸大的"问题

おおきな比較：
小さな子供（こさな）← 可爱、柔软的感性印象
小さい子供 ← 体型描述
```

**同类对比（い形/な形并存）：**

| い形 | な形 | 区别 |
|------|------|------|
| 大きい | 大きな | な形更有感性印象 |
| 小さい | 小さな | な形有亲昵感 |
| おかしい | おかしな | な形强调"奇怪的性质" |
| — | いろんな | 口语化（いろいろな的缩略） |

### 形容词的感情限制——只能描述自己

这是日语语法最独特的规则之一：**表达感情的形容词（感情形容詞）只能直接描述说话者自己的感情**。描述第三者时必须使用特定语法结构。

**感情形容詞列表：**
うれしい（高兴）、かなしい（悲伤）、さびしい（寂寞）、こわい（害怕）、はずかしい（羞耻）、くやしい（懊悔）、うらやましい（羡慕）、つらい（痛苦）、たのしい（快乐）

```
说话者自身感情（直接使用）：
✓ 私はうれしい。我很高兴。
✓ こわいなあ。真害怕啊。

第三者感情（必须转换）：
✗ 彼はうれしい。（不自然/不符合文法）
✓ 彼はうれしそうだ。（看起来高兴——外部观察）
✓ 彼はうれしがっている。（表现出高兴——外部观察）
✓ 彼はうれしいと言っている。（说他高兴——引用）
✓ 彼はうれしいようだ。（似乎高兴——推测）

疑问句中询问对方（对方暂时等同说话者视角）：
✓ うれしいですか？你高兴吗？（被询问者自我报告）
```

## 11.2 形容词的高级用法

### ～すぎる（过度表达）

```
い形容詞：語幹+すぎる
大きすぎる → 太大了
難しすぎる → 太难了
高すぎる   → 太贵了

な形容詞：語幹+すぎる（去な）
静かすぎる → 太安静了
便利すぎる → 太方便了

動詞：連用形+すぎる
食べすぎる → 吃太多了
働きすぎる → 工作太卖力了

例文：
このコーヒーは苦すぎて飲めない。这咖啡太苦了喝不下去。
昨日飲みすぎて頭が痛い。昨天喝多了头很疼。
```

### ～さ/～み（名词化：抽象度的区别）

| 接尾词 | 名词化类型 | 语义特点 | 例子 |
|--------|----------|---------|------|
| ～さ | 客观度量化 | 可量化的程度 | 高さ/深さ/速さ |
| ～み | 主观感受化 | 内在的质感 | 深み/重み/甘み |

```
高さ vs 高み：
山の高さは3776メートルだ。山的高度是3776米。（客观数值）
人生には深みがある。人生有其深度/厚度。（质感，无法量化）

甘さ vs 甘み：
このケーキは甘さが足りない。这蛋糕甜度不够。（可调节的量）
この料理には自然な甘みがある。这道菜有自然的甜味。（风味质感）

其他对比：
暖かさ（暖和的程度）vs 暖かみ（温暖的感觉/人情味）
白さ（白的程度）vs 白み（发白/微白——白みがかった空）
```

**～み只能接部分形容词（需记忆）：**
深み、重み、甘み、辛み、渋み、弱み、強み、悲しみ、楽しみ、高み

### ～そう（推测/外观）

```
い形容詞：語幹+そう（去い）
おいしそう → 看起来好吃
難しそう   → 看起来难
楽しそう   → 看起来快乐

な形容詞：語幹+そう
元気そう   → 看起来有精神
大変そう   → 看起来很辛苦

動詞：連用形+そう
雨が降りそう → 看起来要下雨
泣きそう    → 快要哭了

特殊：
よさそう（良さそう） ← よい→よさ（不是よいそう）
なさそう（無さそう） ← ない→なさ

例文：
このりんご、おいしそうですね。这苹果看起来很好吃呢。
彼は疲れていそうだ。他看起来很累。（動詞て形+いそう）
今にも雨が降りそうな空だ。感觉随时都要下雨的天空。
```

### ～がる（第三者感情外化）

```
感情形容詞+がる → 表现出某种感情（仅限第三者）

こわい → こわがる（表现出害怕）
うれしい → うれしがる（表现出高兴）
さびしい → さびしがる（表现出寂寞）
恥ずかしい → 恥ずかしがる（表现出羞耻）
欲しい（ほしい）→ 欲しがる（表现出想要）
～たい → ～たがる（表现出想做）

例文：
子供が暗がりをこわがっている。孩子表现出对黑暗的恐惧。
彼女は新しいスマホを欲しがっている。她表现出想要新手机。
彼は何でも知りたがる。他什么都想知道（表现出强烈的求知欲）。
```

### ～くする/～にする（使役变化）

```
い形容詞：～く+する
部屋を暖かくする。 把房间弄暖和。
声を大きくしてください。请把声音调大。
生活を楽しくしたい。想让生活变得有趣。

な形容詞/名詞：～に+する
部屋をきれいにする。把房间弄干净。
子供を医者にする。让孩子成为医生。
これを参考にする。以此为参考。
```

### ～くなる/～になる（自然变化）

```
い形容詞：～く+なる
天気が良くなった。天气变好了。
日が長くなってきた。白天变长了。

な形容詞/名詞：～に+なる
上手になった。变得擅长了。
医者になりたい。想成为医生。
有名になった。出名了。

くする vs くなる的对比：
部屋を暖かくする（人为使其变暖）
部屋が暖かくなる（自然变暖）
```

## 11.3 副詞（副词）的分类与功能

### 11.3.1 状態副詞（状态副词）

描述动作或状态的方式，多为拟声拟态词（详见11.5）：

| 副词 | 读音 | 含义 | 例句 |
|------|------|------|------|
| ゆっくり | — | 慢慢地、放松地 | ゆっくり休んでください |
| はっきり | — | 清楚地、明确地 | はっきり言ってください |
| しっかり | — | 好好地、扎实地 | しっかり勉強する |
| ちゃんと | — | 认真地、规矩地 | ちゃんとやってね |
| きちんと | — | 整齐地、规范地 | きちんと並べる |
| すっかり | — | 完全地 | すっかり忘れた |
| やっと | — | 终于、好不容易 | やっと終わった |
| ようやく | — | 好不容易、终于（较书面） | ようやく着いた |
| わざわざ | — | 特意地、不辞辛劳地 | わざわざ来てくれた |
| いきなり | — | 突然地 | いきなり泣き出した |

### 11.3.2 程度副词（程度梯度表）

```
肯定程度（从强到弱）：
とても/非常に（非常）> かなり（相当）> 結構（挺、蛮）
> だいぶ（相当、不少）> まあまあ（还算）
> 少し/ちょっと（有点）> あまり（不太——接否定）
> ほとんど（几乎不——接否定）> 全然/全く（完全不——接否定）

使用示例：
とても難しい。非常难。
かなり疲れた。相当累了。
だいぶ良くなった。好多了。
ちょっと待ってください。稍等一下。
あまり好きじゃない。不太喜欢。
全然わからない。完全不懂。
```

**程度副词与否定的关系：**

| 副词 | 与肯定 | 与否定 |
|------|--------|--------|
| とても | ✓ とても良い | ✗ とても良くない（不自然） |
| あまり | △ 少见 | ✓ あまり良くない |
| 全然 | △ 近年口语有"全然OK" | ✓ 全然良くない（标准） |
| ほとんど | ✓ ほとんど終わった | ✓ ほとんど来ない |

### 11.3.3 時間副詞（时间频率梯度表）

**频率副词（从高到低）：**

| 副词 | 读音 | 频率 | 例句 |
|------|------|------|------|
| いつも | — | 总是（90%+） | いつも遅刻する |
| よく | — | 经常（70%） | よく図書館で勉強する |
| たまに | — | 偶尔（30%） | たまに映画を見る |
| めったに | — | 很少（接否定） | めったに来ない |
| ほとんど | — | 几乎不（接否定） | ほとんど食べない |
| 全然 | ぜんぜん | 完全不（接否定） | 全然来ない |

**时态副词（时间关系）：**

| 副词 | 含义 | 与否定/肯定 |
|------|------|-----------|
| もう | 已经/还（否定=不再） | もう食べた/もう食べない |
| まだ | 还（仍）/还没 | まだ食べている/まだ食べていない |
| すでに | 已经（书面） | すでに決まっている |
| もうすぐ | 马上/快要 | もうすぐ着く |

```
もう vs まだ 对比：
もう食べましたか？ → 你吃了吗？（期待"已经"）
  はい、もう食べました。是的，已经吃了。
  いいえ、まだです。不，还没有。

まだ食べていますか？ → 还在吃吗？
  はい、まだ食べています。是的，还在吃。
  いいえ、もう食べ終わりました。不，已经吃完了。
```

### 11.3.4 陳述副詞（确信度梯度表）

陳述副词（ちんじゅつふくし）又称呼応の副词，与句末特定表达形成呼应：

**确信度从高到低：**

| 副词 | 读音 | 确信度 | 搭配句末 |
|------|------|--------|---------|
| きっと | — | 90%+ | きっと～だろう/に違いない |
| 必ず | かならず | 必然/承诺 | 必ず～します（承诺） |
| たぶん | — | 70-80% | たぶん～だろう |
| おそらく | — | 60-70%（书面） | おそらく～でしょう |
| もしかしたら | — | 30-40% | もしかしたら～かもしれない |
| まさか | — | 否定期待 | まさか～ないだろう |

```
例文：
きっと彼は来るに違いない。他肯定会来的。
必ず連絡します。我一定会联系你。
たぶん明日は雨だろう。明天大概会下雨。
おそらく彼が犯人でしょう。他大概就是犯人。
もしかしたら間違いかもしれない。或许是搞错了。
まさかそんなことはないだろう。不可能会那样的。
```

### 11.3.5 呼応の副詞（与特定句末呼应）

| 副词 | 必须搭配的句末 | 例句 |
|------|-------------|------|
| 決して | ～ない | 決してあきらめない |
| 必ずしも | ～ない | 必ずしも正しくない |
| まるで | ～ようだ/～みたいだ | まるで夢のようだ |
| もし | ～たら/～ば/～なら | もし雨が降ったら |
| たとえ | ～ても | たとえ失敗しても |
| せっかく | ～のに/のだから | せっかく来たのに |
| どうせ | ～から/～のだ | どうせ無理だから |
| いくら | ～ても | いくら食べても太らない |
| なぜなら | ～からだ | なぜなら～からだ |

```
呼応の例文：
決して嘘をつかない。绝对不说谎。
必ずしも金持ちが幸せとは限らない。有钱不一定幸福。
まるで映画のような話だ。像电影一样的故事。
もし時間があれば、ぜひ来てください。如果有时间，请务必来。
たとえ失敗しても、また挑戦する。即便失败了，也会再次挑战。
せっかく作ったのに、食べてくれなかった。特意做了，却不吃。
いくら説明しても理解してくれない。无论怎么解释都不理解。
```

## 11.4 副词与形容词的转换关系

形容词可以通过变形转化为副词，在句中修饰动词：

```
い形容詞：～い → ～く（副詞用法）
速い → 速く走る（快速地跑）
高い → 高く跳ぶ（高高地跳）
早い → 早く来てください（请早点来）
強く握る（紧紧地握）

な形容詞：～な → ～に（副詞用法）
静かな → 静かに話す（静静地说）
丁寧な → 丁寧に書く（认真地写）
上手な → 上手に歌う（唱得好）
自由に → 自由に選ぶ（自由地选择）

副詞化の特殊例：
良い → 良く（よく）※不是良いく
いい → よく（発話ではよく使う）
```

**形容词同根副词词汇积累：**

| 形容詞 | 副詞形 | 意义转变 |
|--------|--------|---------|
| 正しい | 正しく | 正确地 |
| 優しい | 優しく | 温柔地 |
| 激しい | 激しく | 激烈地 |
| 新しい | 新しく | 重新地 |
| 親切な | 親切に | 亲切地 |
| 急に | — | 突然地（固定副词） |
| 特に | — | 特别地（固定副词） |

## 11.5 擬声語・擬態語（拟声拟态词/オノマトペ）

### 日语オノマトペ的独特性

オノマトペ是日语词汇中最具特色的部分，数量多达数千个，可以进入几乎所有词类位置，且承载着普通词汇难以替代的感官信息。

**基本结构类型：**

```
AB型：ドキドキ（心跳）、キラキラ（闪烁）
ABと型：そっとする（轻轻地做）
ABした：ドキドキした（心跳加速了）
ABする：ウロウロする（徘徊）
ABと+動詞：ぐっすり眠る（熟睡）
```

**四类基本词：**

| 类型 | 定义 | 例子 |
|------|------|------|
| 擬音語（ぎおんご） | 模拟自然/动物声音 | ワンワン（狗叫）、ザーザー（大雨声） |
| 擬声語（ぎせいご） | 模拟人的声音 | ペラペラ（流利地说）、ブツブツ（嘟囔） |
| 擬態語（ぎたいご） | 模拟状态/样子 | キラキラ（闪亮）、ふわふわ（蓬松） |
| 擬情語（ぎじょうご） | 模拟心理感受 | ドキドキ（心跳/紧张）、ワクワク（兴奋期待） |

### 常用オノマトペ50个分类详解

#### 感情/心理类

| オノマトペ | 汉字/含义 | 例句 |
|-----------|----------|------|
| ドキドキ | 心跳/紧张/期待 | 発表の前でドキドキしている |
| ワクワク | 兴奋/期待 | 旅行が楽しみでワクワクする |
| ニコニコ | 微笑/表情愉快 | いつもニコニコしている |
| イライラ | 烦躁/焦躁 | 渋滞でイライラする |
| ビクビク | 惶恐/怕怕的 | 怒られるかとビクビクする |
| ウキウキ | 雀跃/心情好 | 春になるとウキウキする |
| ガッカリ | 失望/灰心 | 結果を聞いてガッカリした |
| うっとり | 陶醉/迷离 | 音楽にうっとりと聴き入る |

#### 天气/自然类

| オノマトペ | 含义 | 例句 |
|-----------|------|------|
| ザーザー | 大雨声 | 雨がザーザー降っている |
| ポツポツ | 小雨开始落 | 雨がポツポツ降り始めた |
| シトシト | 细雨绵绵 | 雨がシトシト降る |
| ヒュービュー | 风声呼啸 | 風がヒューヒュー吹く |
| ピカピカ | 闪闪发光（干净/雷） | 床をピカピカに磨く |
| キラキラ | 闪烁/星光 | 星がキラキラ輝く |

#### 动作/行为类

| オノマトペ | 含义 | 例句 |
|-----------|------|------|
| ゆっくり | 慢慢地 | ゆっくり歩く |
| そっと | 轻轻地/悄悄地 | そっとドアを開ける |
| ぐっと | 用力地/深深地 | ぐっと我慢する |
| ぱっと | 突然/迅速 | ぱっと目が覚めた |
| ふらふら | 摇摇晃晃/漫无目的 | 疲れてふらふらする |
| ウロウロ | 徘徊/转来转去 | 迷子になってウロウロした |
| バタバタ | 忙乱/噼里啪啦 | 朝はいつもバタバタしている |
| ぐるぐる | 转圈/晕头转向 | 目がぐるぐる回る |

#### 食感/触感类

| オノマトペ | 含义 | 例句 |
|-----------|------|------|
| サクサク | 酥脆（咬合声） | クッキーがサクサクしている |
| ふわふわ | 蓬松/轻柔 | このパンはふわふわだ |
| もちもち | Q弹/有嚼劲 | このお餅はもちもちしている |
| とろとろ | 浓稠/入口即化 | チーズがとろとろに溶けた |
| パリパリ | 清脆/爽脆 | 海苔がパリパリだ |
| ぬるぬる | 黏滑/湿滑 | 魚がぬるぬるしている |
| ザラザラ | 粗糙 | 砂がザラザラしている |
| スベスベ | 光滑/细腻 | 肌がスベスベだ |
| ふかふか | 松软/柔软（床/垫） | このソファはふかふかだ |
| ねばねば | 黏稠 | 納豆はねばねばしている |

#### 声音/状态类

| オノマトペ | 含义 | 例句 |
|-----------|------|------|

## 11.6 练习题

### 练习一：形容词感情限制

改正以下不自然的句子：

1. 彼はかなしい。（他很悲伤）
2. 犬はこわい。（狗很害怕——指狗感到害怕）
3. 妹はうらやましい。（妹妹很羡慕）

### 练习二：高级用法填空

用适当的形式填空（すぎる/さ/み/そう/がる）：

1. このスープは塩辛（　）て食べられない。（太咸了）
2. 部屋の暗（　）を調整する。（调整暗度）
3. このワインには深（　）がある。（有深度）
4. 子供がお菓子を欲し（　）ている。（表现出想要）
5. 空が暗くなってきた。雨が降り（　）だ。（看起来要下）

### 练习三：副词搭配

选择正确搭配：

1. 決して（a.行った / b.行かない）。
2. まるで（a.夢のようだ / b.夢だった）。
3. もし明日暇（a.なら / b.だ）、来てください。
4. たとえ失敗し（a.ても / b.たら）、諦めない。
5. いくら練習し（a.ても / b.たら）、上手にならない。

### 练习四：オノマトペ匹配

将オノマトペ与含义连线：

1. ドキドキ　　 a. 发呆
2. サクサク　　 b. 心跳/紧张
3. ぼんやり　　 c. 酥脆
4. ぐっすり　　 d. 熟睡
5. イライラ　　 e. 烦躁

### 答案参考

练习一：
1. 彼はかなしそうだ/かなしんでいる
2. 犬はこわがっている
3. 妹はうらやましがっている

练习二：
1. 塩辛**すぎ**て  2. 暗**さ**  3. 深**み**  4. 欲し**が**っている  5. 降り**そう**

练习三：
1. b  2. a  3. a  4. a  5. a

练习四：
1-b / 2-c / 3-a / 4-d / 5-e

## 本章小结

| 知识点 | 核心要点 |
|--------|---------|
| おおきい vs おおきな | な形有感性印象，仅用于连体位置 |
| 感情形容詞限制 | 只能直接描述自己，第三者用～そう/～がる |
| ～さ vs ～み | さ=客观量化，み=主观质感 |
| 程度副词 | あまり/全然/ほとんど接否定，とても接肯定 |
| 呼応の副词 | 決して～ない/まるで～よう/もし～たら 等固定搭配 |
| オノマトペ | 日语独有，传达感官质感，大量存在于日常会话中 |

> 形容词和副词是语言的表情与韵律。日语オノマトペ所承载的感官精度，是任何翻译都难以完全还原的——它们只能被感受，被体验。
