# 第36章：考试策略（JLPT / J.TEST / EJU）

> 「試験は目的ではなく、通過点に過ぎない。」（考试不是目的，只是途中的一个节点。）——考试证书固然重要，但它的价值在于为你打开一扇门，而不是终点本身。理解这一点，才能以正确的心态制定备考策略，在考过之后继续前行。

## 36.1 日语主要考试体系概览

### 主要考试对比总表

| 考试名称 | 主办方 | 级别 | 题型 | 考试形式 | 费用（约） | 考试时间 | 主要用途 |
|---------|-------|------|------|---------|----------|---------|---------|
| **JLPT**（日本語能力試験） | 国际交流基金+日本国际教育支援协会 | N5~N1 | 选择题（无写作口语） | 笔试（部分地区CBT） | ¥500~¥700 HKD / ¥250~¥350 人民币 | 升学、就业、签证、移民 |

| **NAT-TEST**（日本語NAT-TEST） | NAT-TEST事務局 | 5~1级 | 选择题 | 笔试 | 约¥4000~¥5000日元 | JLPT替代，海外常用 |

| **BJT**（ビジネス日本語能力テスト） | 日商岩井等 | J1~J5 | 选择题（听力为主） | CBT | 约¥5000日元 | 商务就业 |

---

### 各考试特点解析

**JLPT（日本語能力試験）**

- 全球认知度最高，是大多数中国学习者的首选目标
- 每年7月和12月举行两次
- 只有选择题，没有口语和写作测试（这是主要争议点）
- N2是就业的主要门槛，N1是高端职位的准入标准
- 有效期：官方无明确有效期，但多数用人单位认可3~5年内的成绩

**J.TEST**

- 每年举行多次（约6次），更灵活
- 包含写作题，比JLPT更全面
- 在日资企业中认可度较高
- A~C级相当于JLPT N1，D~E级相当于N2~N3

**EJU（日本留学試験）**

- 专门用于申请日本大学（学部）和研究生院
- 日本語科目分四部分：読解、聴解、聴読解、記述（作文）
- 记述（作文）450~500字，是EJU的核心难点
- 理工科学生还需考数学和理科

## 36.2 JLPT各级别详细备考策略

### N5：入门级

**考试规格：**

| 科目 | 时间 | 满分 | 合格线 |
|------|------|------|-------|
| 言語知識（文字・語彙） | 25分 | 120 | 80 |
| 言語知識（文法）・読解 | 50分 | 120 | 80 |
| 聴解 | 30分 | 60 | 38 |
| 总计 | - | 180 | 80（总分） |

**词汇与汉字目标：**
- 词汇：约800词（含基本生活词汇）
- 汉字：约100个（数字、日常常见汉字）
- 语法：约100条（ます形、て形、基本助词）

**备考周期：** 零基础3~6个月（每天1~2小时）

**核心学习内容：**

```
必须掌握的语法：
✓ 基本助词：は、が、を、に、で、へ、と、から、まで
✓ ます形/ました/ません/ませんでした
✓ て形连接：〜て、〜てください、〜てもいいですか
✓ たい形：〜したい（想做）
✓ 疑问词：いつ、どこ、だれ、なに、なぜ、どう、どの
✓ 存在表达：あります/います
✓ 方向：〜に行く/来る/帰る

高频场景词汇：
时间（今日/明日/昨日/来週/先月）
地点（駅/学校/病院/スーパー）
家庭（父/母/兄/姉/弟/妹）
数字与金额
```

**备考建议：**

- 以《みんなの日本語 初級I》或《标准日本语》初级上册为主教材
- 每天背20个单词，配合例句记忆
- 利用Anki或多邻国建立单词记忆系统
- 多听NHK Easy News培养语感

### N3：中级门槛

**词汇与汉字目标：**
- 词汇：约3500词
- 汉字：约600个
- 语法：约350条

**备考周期：** N4基础上4~6个月

**从N4到N3的跨越难点：**

| 难点 | 具体问题 | 解决方法 |
|------|---------|---------|
| 读解速度 | 文章变长，时间紧张 | 每天精读1篇N3文章，计时练习 |
| 词汇量跳跃 | N3词汇是N4的2倍以上 | 系统使用词汇书，每天50词 |
| 语法细节 | 近义语法辨析增多 | 制作对比卡片，记录差异 |
| 口语式文法 | 〜んじゃないか/〜わけだ等 | 多听日常对话素材 |

**N3特色语法：**

```

〜ことにする（决定……）
〜ようにする（尽量……）
〜ばかりだ（只是……/净是……）
〜だけでなく（不仅……）
〜に対して（针对……/相对于……）
〜によって（根据……/因……而异）
〜上で（在……方面/……之后）
〜つもりだ（打算……）
〜ところだ（正在……/刚要……/刚……完）
〜始める/〜続ける/〜終わる（开始……/继续……/结束……）
```

### N2：就业升学分水岭

N2是日本就业市场和升学申请中最重要的门槛。持N2证书可以：
- 申请绝大多数日资企业的日语相关职位
- 在日本研究生院入学申请中满足语言要求
- 申请日本国际学生交换项目

**词汇与汉字目标：**
- 词汇：约6000词
- 汉字：约1000个
- 语法：约500条（含约200个高频句型）

**备考周期：** N3基础上6~10个月

**文法200句型——核心200选40：**

```
必须掌握的N2文法精选：

表原因/根据：
〜に基づいて（基于……）
〜に伴って（伴随……）
〜につれて（随着……）
〜をきっかけに（以……为契机）
〜をもとに（以……为基础）

表逆接/让步：
〜にもかかわらず（尽管……，却……）
〜とはいえ（虽说……）
〜ながらも（虽然……但……）
〜ものの（虽然……但……）

表条件/限定：
〜限り（只要……就……/在……范围内）
〜からには（既然……就……）
〜以上（は）（既然……就……）
〜をはじめ（以……为首）

表评价/判断：
〜に違いない（一定是……）
〜に決まっている（肯定是……）
〜はずがない（不可能……）
〜っこない（根本不可能……）

形式名词：
〜こと（将事情名词化）
〜もの（将事情名词化，表情感）
〜わけ（理由/当然的结果）
〜はず（应该/理应）
```

**读解速度提升策略：**

```
N2读解时间管理：
总时间：105分钟（语法+读解合计）
建议分配：
- 语法题（12题）：约20分钟
- 内容理解·短文（4篇）：约25分钟
- 内容理解·中文（3篇）：约25分钟
- 综合理解（1篇）：约15分钟
- 信息检索（1篇）：约10分钟
- 剩余检查时间：10分钟

速读技巧：
1. 先看问题，再读文章（目的性阅读）
2. 找文章的「ポイント文」（要点句）
3. 注意接续词：しかし/でも → 逆接，一般后面是重点
4. 段落首尾句通常含主旨
```

### N1：最高级别

N1代表日语使用者的最高认证层次，通过者能够理解：
- 各类场合的日语（包括新闻、学术论文、文学作品）
- 语言细节与暧昧表达
- 高度语境依赖的表达方式

**词汇与汉字目标：**
- 词汇：约10000词以上
- 汉字：约2000个（含常用汉字全部2136字）
- 语法：700条以上（含高度语境化表达）

**备考周期：** N2通过后6~18个月（个体差异大）

**N1核心难点：**

| 难点类型 | 描述 | 备考对策 |
|---------|------|---------|
| 近义语法辨析 | 语义极为相近的语法，需结合语境判断 | 精读《新完全マスター》N1文法 |
| 超长文读解 | 单篇文章达800~1000字，时间极紧 | 每天精读练习，计时模拟 |
| 高频汉字词 | 生僻汉字读音 | 汉字专项练习 |
| 语言风格感知 | 文章的文体（文学/论说/新闻）判断 | 多读日文原著与报纸 |
| 信息整合 | 对比两篇文章观点的题型 | 专项练习信息检索题 |

**N1特色文法（超难型）：**

```
表书面语/文学语感：
〜ごとき/〜ごとく（如……一般，文语）
〜をもって（以……/凭借……）
〜をもとより（不用说……）
〜いかんによって（取决于……）
〜いかんにかかわらず（无论……如何）

表强调/感情色彩：
〜てやまない（不停地……/衷心地……）
〜に堪えない（不胜……/令人……）
〜を禁じ得ない（忍不住……/不禁……）
〜ずにはおかない（不得不……/必定会……）

近义辨析（N1最难点）：
〜たところ vs 〜たとたん vs 〜やいなや
→ 三者都表"刚……就……"，但语气强度和接续不同

〜だけあって vs 〜だけに vs 〜にしては
→ 三者都含"毕竟/就……而言"的语感，需精细辨别
```

**超长文策略：**

```
N1长文处理五步法：
1. 快速扫描标题/小标题（10秒）
2. 阅读首段+末段，把握论点（1分钟）
3. 带着问题阅读全文（标注关键句）
4. 回答问题时定位原文，不凭记忆
5. 对比题：同时看两篇文章对应段落

时间控制：
读解总时间约55分钟（N1语法+读解105分钟中分配）
每题作答不超过2分钟，如超时先跳过
```

## 36.3 EJU备考策略

EJU（日本留学試験）是赴日攻读本科学位的必考项目，日本語科目是其中的核心。

### EJU日本語科目构成

| 部分 | 内容 | 时间 | 满分 |
|------|------|------|------|
| 読解（どっかい） | 阅读理解（选择题） | 约40分钟 | 200分 |
| 聴解（ちょうかい） | 听力理解 | 约55分钟 | 200分 |
| 聴読解（ちょうどっかい） | 听读结合理解 | 约30分钟（含在聴解中） | - |
| 記述（きじゅつ） | 作文（450~500字） | 20分钟 | 50分 |

**总分：400~450分（含记述）**

---

### 記述（作文）评分标准与模板

EJU記述考试要求考生在20分钟内用日语写出450~500字的说明文/论述文。

**评分维度：**

| 评分项目 | 权重 | 具体要求 |
|---------|------|---------|
| 課題への対応 | 最重要 | 是否紧扣题目，有明确观点 |
| 文章の構成 | 重要 | 结构是否清晰，段落是否有逻辑 |
| 論理の展開 | 重要 | 论点是否有支撑，例证是否恰当 |
| 言語の正確さ | 重要 | 语法词汇是否准确 |

**记述作文基本模板：**

```
【段落结构】
序论（约80字）：提出话题背景，明确自己的立场
本论（约300字）：论点1（约100字）+ 论点2（约100字）+ 反驳意见处理（约100字）
結論（约100字）：总结观点，呼应开头

【实用表现】

开篇：
「近年、〜という問題が注目されている。この問題について、私は〜と考える。」
（近年来，……这一问题备受关注。关于此问题，我认为……。）

提出论点：
「まず第一に、〜が挙げられる。」（首先第一点，可以举出……。）
「次に、〜という点も重要だ。」（其次，……这一点也很重要。）

举例论证：
「例えば、〜という事例がある。」（例如，有……这样的案例。）
「実際に、〜というデータがある。」（实际上，有……这样的数据。）

处理反对意见：
「確かに、〜という意見もある。しかし、〜だと私は考える。」
（确实，也有……这样的意见。但是，我认为……。）

结尾：
「以上の点から、私は〜と結論付ける。」
（综合以上各点，我得出的结论是……。）
「これからも、〜ことが重要だと思う。」
（今后，我认为……仍然很重要。）
```

**20分钟时间分配：**

```
3分钟：审题，确定立场，草拟大纲
15分钟：写作（按模板填充）
2分钟：检查（字数、语法、汉字）
```

## 36.4 各考试通用技巧

### 时间管理矩阵

在任何日语考试中，有效的时间管理都是得分关键。

```
【通用时间策略】

┌──────────────────────────────────────┐
│ 题目分类           │ 时间策略           │
├──────────────────────────────────────┤
│ 必拿分（简单确定）  │ 快速作答，不纠结   │
│ 应拿分（有点难）    │ 正常用时，认真分析  │
│ 争取分（较难）      │ 先跳过，最后回来   │
│ 放弃分（极难）      │ 随机猜一个，继续   │
└──────────────────────────────────────┘

时间预警：
- 每题平均用时超过90秒 → 需加快节奏
- 离结束还有5分钟 → 必须检查未答题
- 答案不确定 → 先圈出，留时间回头看
```

---

### 选项排除法

JLPT全部为四选一选择题，排除法是核心技巧。

```
四步排除法：

Step 1：消去明显错误选项
→ 语法形式明显不对的（接续错误/活用错误）

Step 2：消去语义不符选项
→ 词义与上下文明显矛盾的

Step 3：在剩余2个中找细节差异
→ 通常一个在语感/语义细节上更贴切

Step 4：如仍无法确定
→ 选语法最"规整"的那个

典型案例：
问：「彼は毎日____ように努力している。」
选项：A 上手に話す  B 上手に話せる  C 話し上手の  D 上手く話せる

分析：ように接动词基本形 → A和B格式正确
      语义：努力"能说好"→ 未达成状态，用可能形更准确
      答案：B（上手に話せる）
```

---

### 接续判断法（文法题专用）

```
接续判断的核心规则：

通过观察空格前后的词形，快速定位合适语法：

空格前是：
- 名词（N）→ 选以「の/に/が/を/で」等开头的语法
- 形容词て形 → 选表"因为"或"虽然"的语法
- 动词た形 → 选「〜たとたん/〜たところ/〜たら」等
- 动词辞書形 → 选「〜ように/〜ために/〜前に」等

空格后是：
- 名词 → 空格为连体修饰形或の
- 助词 → 前面通常为名词
- 动词ます形 → 前面通常有て形或接续词

实战技巧：先看前后词形，再看语义。
90%的语法题可以通过接续判断排除2~3个选项。
```

---

### 主旨定位法（读解题专用）

```
日语论说文的结构规律：

日本语文章的论点通常位于：
1. 第一段末尾（提出论点）
2. 段落首句（展开论证）
3. 最后一段（总结论点）

关键信号词：
结论信号词：つまり、要するに、したがって、このように
逆接信号词：しかし、でも、ところが、一方で
举例信号词：たとえば、具体的には、〜をはじめ
强调信号词：特に、とりわけ、なかでも

「主旨题」解题步骤：
1. 找文章最后一个「つまり/このように」句
2. 该句通常就是主旨
3. 选项中与该句语义最接近的即为答案
```

---

### 笔记法（听力题专用）

```
听力答题笔记技巧：

在题目纸上快速记录：
- 数字（价格/时间/日期/数量）
- 人名/地名
- 关键动词（做什么）
- 否定词（〜ない/〜ません要特别标记）

符号系统（自定义，快速）：
↑ = 涨价/增加/上升
↓ = 降价/减少/下降
→ = 因此/结果
× = 不行/不对/取消
? = 不确定的信息
! = 重要，特别注意

N1听力特别技巧：
- 选项"换个说法"的题：训练同义改写理解
- 推断态度题：注意说话人的语气和情感词
- 图表题：先快速看图，理解纵横轴含义
```

## 36.6 从"应试日语"到"真实日语"

### 考过N1之后的真实状况

通过N1并不意味着日语能力已经完整，而是刚刚站在了真正使用日语的起跑线上。

```
N1合格者的常见困境：

1. 听力断层：考试的听力和真实对话速度完全不同
   解决：大量收看日本综艺（字幕→无字幕渐进）

2. 口语哑巴：JLPT没有口语测试，很多N1持有者开不了口
   解决：语言交换+日本留学/工作经历

3. 文化理解不足：能看懂文字但不懂字背后的含义
   解决：系统了解日本文化（参考第34章）

4. 书写能力薄弱：JLPT不考写作，写作能力零积累
   解决：坚持日记写作，参加EJU記述练习
```

---

### 工作语言进阶路径

```
阶段1：N2持有（就业准入）
→ 能够完成基本的日语工作任务（邮件、会议理解）
→ 重点：扩展行业专业词汇

阶段2：N1持有（高级日语使用者）
→ 能够独立完成日语工作
→ 重点：提升口语流利度，学习行业术语

阶段3：商务日语熟练者
→ 能够用日语谈判、演讲、写报告
→ 可考BJT（ビジネス日本語能力テスト）

阶段4：高级/专业日语使用者
→ 翻译/口译方向
→ 日本研究生院/大学任职方向
→ 日本企业管理层方向
```

---

### 翻译口译方向

```
笔译（翻訳、ほんやく）路径：
基础：N1 + 母语优秀写作能力
进阶：选择专业方向（法律/医学/技术/文学）
认证：一般社団法人日本翻訳連盟资质
实践：从字幕翻译/技术文档起步

口译（通訳、つうやく）路径：
基础：N1 + 超强的双语口语能力
训练方法：
→ 影子练习（シャドーイング）——必须每日坚持
→ 逐步口译练习（ノートテイキング）
→ 同声传译（初级）
认证：日本通訳技能向上センター等机构
```

---

### 终身学习的视角

```
日语学习的终点不是N1，而是：

● 能够用日语深度思考
  → 阅读哲学/历史/文学原著
  → 理解隐喻、幽默、讽刺

● 能够用日语创作
  → 写出有文采的日语散文
  → 翻译中文作品的细腻之处

● 成为真正的中日文化桥梁
  → 在两种文化之间自如切换
  → 帮助他人理解两种文化的异同

推荐的终身学习路径：
1. 每周读一篇日文原版新闻分析（経済・社会）
2. 每月读完一本日文原版书籍
3. 保持与日本人的定期交流
4. 关注日语的动态变化（新词/网络语/方言）
5. 将日语学习融入生活，而不是单独的"学习时间"
```
