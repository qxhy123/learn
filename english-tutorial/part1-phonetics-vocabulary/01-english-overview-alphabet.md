## 1.1 英语的历史与语系

### 英语从哪里来？

英语（English）属于**印欧语系**（Indo-European Language Family）→ **日耳曼语族**（Germanic Languages）→ **西日耳曼语支**（West Germanic Branch）。它的"兄弟语言"是德语（German）和荷兰语（Dutch），"堂兄弟"是斯堪的纳维亚语言（北日耳曼语支）。

```
印欧语系
├── 日耳曼语族
│   ├── 西日耳曼语支
│   │   ├── 英语 (English)       ← 我们的主角
│   │   ├── 德语 (German)
│   │   └── 荷兰语 (Dutch)
│   └── 北日耳曼语支
│       ├── 瑞典语 (Swedish)
│       └── 挪威语 (Norwegian)
├── 罗曼语族（拉丁语后裔）
│   ├── 法语 (French)            ← 对英语影响最深
│   ├── 西班牙语 (Spanish)
│   └── 意大利语 (Italian)
└── 其他语支（希腊语、梵语等）
```

理解这个谱系至关重要：英语的词汇有两大来源——**日耳曼语底层**（基础词汇）和**拉丁/法语上层**（书面词汇），这直接决定了英语学习的策略。

---

### 三次历史性冲击

英语的形成经历了三次改变其面貌的历史事件：

#### 第一次冲击：盎格鲁-撒克逊定居（5-11世纪）— 古英语时期

公元449年，日耳曼部落（盎格鲁人、撒克逊人、朱特人）从今天的德国/丹麦地区入侵不列颠岛，带来了**古英语**（Old English / Anglo-Saxon）。

古英语与现代英语几乎无法互通，但其遗留的词汇是最核心的基础词：

| 古英语词根 | 现代英语 | 中文含义 |
|-----------|---------|---------|
| mann | man | 人 |
| wīf | wife | 妻子 |
| cild | child | 孩子 |
| hūs | house | 房子 |
| etan | eat | 吃 |
| drincan | drink | 喝 |
| god | good | 好的 |
| heorte | heart | 心脏 |

**规律**：凡是描述日常生活、家庭、身体、自然的最基本词汇，大多来自日耳曼语底层。这些词短促、有力、常见。

#### 第二次冲击：维京人入侵（8-11世纪）— 古诺尔斯语影响

斯堪的纳维亚维京人的入侵带来了古诺尔斯语（Old Norse），这一影响往往被低估。很多日常高频词来自这一时期：

| 来自古诺尔斯语 | 中文含义 |
|-------------|---------|
| they / their / them | 他们（英语原本的第三人称复数是"hie"！） |
| sky | 天空 |
| window | 窗户（wind + eye，"风眼"） |
| knife | 刀 |
| egg | 蛋 |
| get | 获得 |
| call | 呼叫 |
| die | 死亡 |

#### 第三次冲击：诺曼征服（1066年）— 法语/拉丁语洪流

**1066年**是英语史上最重要的年份。法国诺曼底公爵威廉一世（William the Conqueror）征服英格兰，法语成为贵族、法律、教会、文学的语言，持续约300年。

这次冲击造就了英语独特的**"双层词汇系统"**：

| 日耳曼语（日常/口语） | 法语/拉丁语（正式/书面） | 中文含义 |
|-------------------|-------------------|---------|
| ask | inquire | 询问 |
| begin | commence | 开始 |
| end | terminate | 终止 |
| buy | purchase | 购买 |
| help | assist | 帮助 |
| holy | sacred | 神圣的 |
| kingly | royal | 皇家的 |
| pig（猪圈里的猪） | pork（餐桌上的猪肉） | 猪 |
| cow（农民养的牛） | beef（贵族吃的牛肉） | 牛 |
| sheep（农民的羊） | mutton（贵族的羊肉） | 羊 |

> **深层洞察**：农民养动物（日耳曼词），贵族吃肉（法语词）。这个差异一千年后仍清晰可见。

#### 文艺复兴与科学时代：拉丁语和希腊语词汇大量涌入

15-17世纪，大量科学、哲学、医学、法律词汇从拉丁语和希腊语直接借入：

- 来自希腊语：**philosophy**（哲学）、**democracy**（民主）、**biology**（生物学）、**telephone**（电话）
- 来自拉丁语：**education**（教育）、**literature**（文学）、**science**（科学）、**gravity**（重力）

---

### 英语词汇的来源比例

```
现代英语词汇来源分布（约估）：
┌─────────────────────────────────────────┐
│  拉丁语（直接借入）        约 28%         │
│  法语（含古法语）          约 28%         │
│  日耳曼语底层              约 25%         │
│  希腊语                    约 6%          │
│  其他语言                  约 13%         │
└─────────────────────────────────────────┘
```

但按**使用频率**排列，日耳曼语底层词汇占了最常用词的大多数——因为 the、is、you、I、and、have 这些高频词都是日耳曼起源的。

**实践意义**：学英语应该先掌握日耳曼底层词（最高频、最实用），再扩展拉丁/法语词汇（提升书面表达力）。

## 1.2 26个字母的前世今生

### 字母的演变之路

英语字母表来自**罗马字母**（Latin/Roman Alphabet），而罗马字母来自**希腊字母**，希腊字母又来自**腓尼基字母**（Phoenician），腓尼基字母则源于古埃及的象形文字。

```
演变链条：
古埃及象形文字 → 原始闪米特字母 → 腓尼基字母 → 希腊字母 → 拉丁字母 → 现代英语字母
（约公元前3000年）                  （约前1050年）  （约前800年）  （约前600年）   （现代）
```

以字母 **A** 为例追溯其起源：

```
𓃾 (古埃及牛头象形文字)
  ↓
𐤀 (腓尼基字母 "aleph"，意为"牛")
  ↓
Α α (希腊字母 "alpha")
  ↓
A a (拉丁字母，最终进入英语)
```

字母 A 最初是倒过来的牛头（∧），旋转后成为今天的 A 形——字母表第一个字母承载着五千年文明史。

### 大小写使用规则

英语大写规则远比中文复杂，以下是核心规则：

**必须大写的情况：**

```
1. 句子首字母
   - The cat sat on the mat.

2. 专有名词（人名、地名、机构名）
   - John Smith, New York, Harvard University

3. 月份和星期
   - January, February / Monday, Tuesday

4. 国籍、语言、宗教
   - Chinese, English, Buddhism

5. 书名、电影名（实词首字母）
   - The Lord of the Rings
   - Harry Potter and the Sorcerer's Stone

6. 代词 "I"（第一人称单数永远大写）
   - She and I went to the store.

7. 头衔（在名字前）
   - President Biden, Dr. Smith, Professor Wang
```

**常见错误：**
```
错误：I am a chinese student studying english.
正确：I am a Chinese student studying English.

错误：I went to the park on monday with my friend john.
正确：I went to the park on Monday with my friend John.
```

## 1.3 英语的全球变体

### 世界英语的格局

英语是世界上使用人口最多的语言之一，但"英语"并非铁板一块。全球约15亿英语使用者分布在不同地区，形成了显著的变体差异。

```
主要英语变体：
├── 英式英语 (British English, BrE)    — 约6500万母语者
├── 美式英语 (American English, AmE)   — 约2.5亿母语者（最广泛使用）
├── 澳式英语 (Australian English, AuE) — 约2500万母语者
├── 加拿大英语 (Canadian English)
├── 新西兰英语 (New Zealand English)
├── 南非英语 (South African English)
└── 印度英语 (Indian English)
```

### 美式 vs 英式：核心差异对比

#### 拼写差异

| 美式英语 (AmE) | 英式英语 (BrE) | 中文含义 |
|-------------|-------------|---------|
| color | colour | 颜色 |
| favorite | favourite | 最喜欢的 |
| honor | honour | 荣誉 |
| organize | organise | 组织 |
| center | centre | 中心 |
| theater | theatre | 剧院 |
| traveling | travelling | 旅行（进行时） |

| check | cheque | 支票 |

### 澳式英语的特点

澳大利亚英语（Australian English）是英式英语在澳大利亚200年演化的结果，有自己鲜明特色：

**元音特征**：所谓的"澳大利亚腔"主要源于以下几个元音变化：
- /eɪ/ 变为 /æɪ/（day 听起来像"die"）
- /aɪ/ 变为 /ɔɪ/（time 听起来接近"toim"）

**特有词汇**：
```
arvo    = afternoon（下午）
brekkie = breakfast（早餐）
footy   = football（足球）
servo   = service station（加油站）
ute     = utility vehicle（皮卡）
G'day   = Good day（打招呼用语）
```

### 中国学生应学哪种英语？

| 场景 | 推荐变体 | 原因 |
|-----|---------|-----|
| 雅思考试 | 英式为主 | 考试基于英式标准 |
| 托福/GRE | 美式为主 | 美国教育体系 |
| 工作/商务 | 美式为主 | 全球商业更通用 |
| 学术写作 | 选定一种，保持一致 | 避免混用 |
| 日常交流 | 两者均可 | 培养双向理解能力 |

> **建议**：选择一种变体作为主要输入/输出标准，同时培养理解另一种变体的能力。混用拼写（colour + traveling）是大忌。

## 1.4 英语学习的认知科学基础

### 记忆曲线与遗忘规律

1885年，德国心理学家赫尔曼·艾宾浩斯（Hermann Ebbinghaus）发现了**遗忘曲线**（Forgetting Curve）：

```
记忆保留率（%）
100 |*
 80 |  *
 60 |      *
 40 |           *
 20 |                  *  *    *
  0 +--+--+--+--+--+--+--+--+--→ 时间
    0  1  2  4  8  16 1d 2d 7d 30d
    (小时)                (天)
```

**核心发现**：
- 学习后 1 小时遗忘约 50%
- 24 小时后遗忘约 70%
- 一周后遗忘约 90%
- **但**：每次复习都会重置曲线，且每次遗忘速度变慢

**间隔重复**（Spaced Repetition）正是基于这一发现：在即将遗忘前复习，效率最高。

```
最优复习时间间隔（参考）：
第1次学习 → 1天后复习 → 3天后复习 → 7天后复习 → 15天后复习 → 30天后复习
```

### Krashen 的输入假说

Stephen Krashen 提出的**可理解输入假说**（Comprehensible Input Hypothesis）是现代语言习得理论的基石：

**核心公式：i + 1**
- `i` = 当前语言水平
- `i + 1` = 稍高于当前水平的输入
- 只有 `i + 1` 级别的输入才能推动语言习得

```
输入难度 vs 学习效果：

太难（i + 10）: ████████████ 无法理解 → 没有习得
适当（i + 1）:  ████████░░░░ 可理解 → 最佳习得区间
太简单（i - 1）: ████░░░░░░░░ 完全理解 → 没有新习得
```

**实践含义**：
- 不要只读"懂100%"的材料（没有挑战）
- 不要读"懂20%以下"的材料（太受挫）
- **最佳区间**：理解率约80-95%，有生词但能从语境猜测

### 语言习得的关键时期

| 年龄段 | 习得特点 | 对成人学习者的启示 |
|-------|---------|----------------|
| 0-7岁 | 无意识习得，发音近乎母语 | 成人很难达到母语发音 |
| 7-12岁 | 依然高效，发音接近母语 | 青少年有优势 |
| 12岁后 | 意识性学习增强，发音有口音 | 成人学习者需更多刻意练习 |
| 成人 | 词汇/语法可超越儿童，发音有口音 | 利用逻辑分析弥补"关键期"劣势 |

**好消息**：成年人的**词汇习得速度**和**语法推理能力**实际上比儿童更强。儿童的优势主要在发音，不在其他方面。

### 五大语言习得策略

**1. 大量可理解输入（Massive Comprehensible Input）**
- 每天1-2小时接触真实英语材料
- 材料难度略高于当前水平
- 涵盖听读两个渠道

**2. 注意力机制（Noticing Hypothesis）**
- 单纯"暴露"不够，需要主动注意语言现象
- 例：看到新词，主动分析其词根、词性、用法

**3. 有意义的输出（Meaningful Output）**
- 口语/写作不仅是"展示"，也是促进习得的过程
- Swain 的输出假说：输出迫使你注意语言的精确性

**4. 情感过滤（Affective Filter）**
- 焦虑感高时，大脑对语言输入的"接收度"降低
- 轻松、有趣的学习环境比高压环境更利于习得

**5. 隐性知识 vs 显性知识**
```
显性知识（Explicit Knowledge）：
"know that" — 知道规则，如"第三人称单数加 -s"
↓ 大量练习、内化
隐性知识（Implicit Knowledge）：
"know how" — 自动使用，无需思考规则
```

目标：将显性语法知识转化为隐性使用能力。

## 1.5 学习路线规划与目标设定

### 英语能力分级框架

国际通用的**欧洲语言共同参考框架**（CEFR, Common European Framework of Reference）将语言能力分为六级：

| CEFR等级 | 描述 | 词汇量（约） | 对应考试 | 学习时长（成人，零基础） |
|---------|-----|-----------|---------|-------------------|
| A1 入门 | 理解简单日常用语 | 500-1000 | — | 约80小时 |
| A2 基础 | 理解简单句子和常见表达 | 1000-2000 | — | 约160小时 |
| B1 中级 | 能处理大多数旅游/工作情境 | 2500-3500 | 雅思4.5-5.5 / 托福57-86 | 约350小时 |
| B2 中高级 | 理解复杂文本，流利交流 | 4000-6000 | 雅思6.0-7.0 / 托福87-109 | 约600小时 |
| C1 高级 | 自由流利使用语言 | 8000-12000 | 雅思7.0-8.0 / 托福110+ | 约900小时 |
| C2 精通 | 接近母语水平 | 16000+ | 雅思8.5+ | 约1200小时 |

### 从零到实用的路线图

```
阶段一：语音基础（第1-3个月）
├── 目标：掌握所有音标，能拼读任何单词
├── 每日：30分钟音标练习 + 跟读训练
└── 里程碑：遇到新词能通过词典音标自主学习发音

阶段二：基础词汇与语法（第4-9个月）
├── 目标：掌握3000高频词，理解基本句型
├── 每日：20个新词（间隔复习）+ 基础语法学习
└── 里程碑：能读懂A2级别文章，能进行基础对话

阶段三：大量输入积累（第10-24个月）
├── 目标：词汇量突破6000，能流畅阅读
├── 每日：1小时英语阅读/听力输入
└── 里程碑：能理解英语新闻，能写基础英语邮件

阶段四：精细化与流利度（第25个月起）
├── 目标：词汇量10000+，输出流利自然
├── 每日：大量真实材料输入 + 持续输出练习
└── 里程碑：能参加英语会议，能写学术/商务文章
```

### 设定 SMART 目标

```
S (Specific  具体的)：不是"提高英语"，而是"三个月内通过雅思5.5"
M (Measurable 可衡量的)：词汇量从1000到3000（可用词汇测试验证）
A (Achievable 可实现的)：每天1小时学习，三个月内完成A2→B1
R (Relevant 相关的)：学习材料与目标场景相关（如商务英语 vs 学术英语）
T (Time-bound 有期限的)：六个月内完成，设置月度检查点
```

### 中国学习者的常见误区

| 误区 | 现实 | 正确做法 |
|-----|-----|---------|
| 背完单词书再说话 | 孤立背词效率极低 | 语境中习得词汇 |
| 等语法全学完再开口 | 语法在使用中内化 | 从第一天开始说简单句 |
| 追求发音完美再说话 | 发音在大量使用中改善 | 接受口音，重视沟通 |
| 只看中文解释 | 依赖中文思维 | 尽早用英英词典 |
| 每天背100个单词 | 大量遗忘，效率低 | 每天20词+间隔复习 |
| 只练阅读不练听说 | 哑巴英语，技能不平衡 | 四技能（听说读写）均衡发展 |

## 本章小结

| 知识点 | 核心内容 |
|-------|---------|
| 语系归属 | 英语属于印欧语系→日耳曼语族，兄弟语言是德语、荷兰语 |
| 三次冲击 | 盎格鲁-撒克逊(日耳曼底层) + 维京入侵(古诺尔斯语) + 诺曼征服(法语/拉丁语上层) |

| 字母起源 | 英语字母表源自腓尼基→希腊→拉丁字母，五千年演化 |
| 变体差异 | 美式vs英式：拼写(-or/-our)、词汇(elevator/lift)、发音(car的r音) |
| 认知科学 | 艾宾浩斯遗忘曲线→间隔重复；Krashen的i+1可理解输入 |
