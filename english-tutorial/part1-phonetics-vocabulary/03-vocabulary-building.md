## 3.1 英语词汇量级与学习目标

### 词汇量的临界点

词汇量并非线性积累，而是存在**能力跃迁的临界点**（Vocabulary Threshold）。突破某个词汇量级，语言能力会出现质的飞跃。

```
词汇量 → 语言能力跃迁图：

1,000词  ──┐ 理解最基础日常对话（约占日常用语的85%）
           │ ← 能力跃迁点：可以开始简单交流
2,000词  ──┤ 理解简单文章，进行基础交流
           │
3,000词  ──┤ 覆盖日常口语的约95%（理解报纸文章变得可行）
           │ ← 能力跃迁点：英语开始"能用"
5,000词  ──┤ 覆盖大多数非专业文章的98%+
           │
8,000词  ──┤ 理解大学教材，流畅阅读英语小说
           │ ← 能力跃迁点：英语开始"自由"
10,000词 ──┤ 接近教育程度成人母语者基础词汇量
           │
20,000词 ──┘ 英语母语者平均词汇量（成人，书面+口语）
```

### 各阶段具体目标

| 词汇量 | CEFR 等级 | 能做什么 | 对应考试 |
|-------|---------|---------|---------|
| 1,000 | A1-A2 | 基础购物、问路、自我介绍 | — |
| 2,000 | A2-B1 | 简单书信、基础新闻 | 初中毕业约2500词 |
| 3,500 | B1-B2 | 雅思/托福备考核心词汇 | 高中毕业约3500词 |
| 5,000 | B2 | 流畅阅读一般文章 | 雅思6.0-6.5 |
| 8,000 | C1 | 学术阅读、商务写作 | 雅思7.0+ / 托福100+ |
| 10,000+ | C1-C2 | 接近母语者理解力 | GRE词汇范围 |

> **中国高考英语**：要求约3500词；**大学英语四级**：约4000词；**六级**：约5500词；**考研英语**：约5500词；**雅思/托福高分**：需8000-10000词汇量。

### "懂多少才算懂一个词？"

掌握一个词（Word Knowledge）有深浅之分：

```
词汇知识层次（从浅到深）：

层次1（识别）：见到这个词，知道是个词
层次2（认知）：见到这个词，知道大概意思
层次3（理解）：理解词义、词性、基本用法
层次4（使用）：能在适当语境中正确使用
层次5（精通）：了解搭配、语域、辨义、引申义

例：词汇 "run"
层次3：run = 跑
层次5：run a company（管理公司）
        run out of time（用完时间）
        run a temperature（发烧）
        run in the family（遗传）
        a run of bad luck（一连串厄运）
        in the long run（从长远来看）
```

学习策略：前3000词要达到层次4-5，之后的词先达到层次2-3，边用边深化。

## 3.2 词根词缀法

### 为什么词根词缀如此重要？

英语词汇中约60%来自拉丁语和希腊语，这些词大量使用词根词缀构词。掌握300个高频词根词缀，可以推断数万个单词的含义。

```
示例：词根 "port"（来自拉丁语 portare，意为"携带"）

port + able → portable（可携带的，形容词）
trans + port → transport（运输，跨越+携带）
im + port → import（进口，向内+携带）
ex + port → export（出口，向外+携带）
sup + port → support（支撑，从下+携带）
re + port → report（报告，回来+携带）
de + port → deport（驱逐出境，往下+携带）
port + folio → portfolio（档案袋，携带+叶/页）

一个词根 → 8个以上单词
```


## 3.3 词汇记忆科学

### 间隔重复（Spaced Repetition）

间隔重复是目前**科学证明效率最高**的记忆方法，基于艾宾浩斯遗忘曲线，在即将遗忘前复习。

**工作原理**：

```
SM-2 算法（Anki 使用的间隔重复算法）：

每次复习后，根据记忆质量（0-5分）调整下次复习间隔：
- 质量5（完全记住）：间隔 × 2.5
- 质量4（有点费力）：间隔 × 2.0
- 质量3（勉强记住）：间隔 × 1.5
- 质量0-2（忘记了）：重置到第1天

实际间隔序列（理想情况）：
第1次学：今天
第2次复习：1天后
第3次复习：3天后
第4次复习：7天后
第5次复习：15天后
第6次复习：30天后
第7次复习：60天后...
```

**Anki 的科学原理**：

Anki 使用**主动回忆**（Active Recall）而非被动认知：
- 看到卡片正面（问题），主动想答案
- 翻到背面验证
- 评估记忆质量
- 算法决定下次复习时间

研究表明，主动回忆比反复阅读高效**高达500%**（实验来源：Roediger & Karpicke, 2006）。

### 记忆宫殿（Method of Loci）

记忆宫殿（Memory Palace）是古希腊演说家发明的记忆技术，适合记忆抽象词汇：

**操作步骤**：

```
1. 选择熟悉的空间（家、学校路线）
2. 沿固定路线设置"记忆位置"（loci）
3. 将词汇与夸张生动的画面绑定到位置上
4. 复习时"走"这条路线，触发画面

示例：记忆单词 "magnanimous"（慷慨大度的）
发音：/mæɡˈnænɪməs/
词根：magnus（大）+ animus（心）
画面：在家门口（位置1），想象一个巨大的人（magnus=大），
     胸口有一颗发光的心（animus），把金币撒向所有人
→ 大心脏的人 = 慷慨大度的人
```

**联想记忆法**（Keyword Method）：

```
词汇：precipice（悬崖，/ˈpresɪpɪs/）
联想：press（按压）+ ice（冰）
画面：在悬崖边上，被冰冻住只能按压着冰往下看
→ 危险的悬崖

词汇：gregarious（爱社交的，/ɡrɪˈɡeəriəs/）
词根：greg（群体，如 congregation=会众）
画面：一只快乐的羊群中的羊（greg=羊群），四处社交
→ 爱社交的，合群的
```

### 语义网络与聚类记忆

研究表明，词汇以**语义网络**（Semantic Network）形式存储在大脑中，相关词之间有激活联系。利用这一点，**聚类记忆**比逐词记忆高效：

```
主题聚类示例：情绪词汇网络

核心：emotion（情绪）
        ├── positive（积极）
        │   ├── joy（喜悦）→ joyful, joyous, rejoice
        │   ├── excitement（兴奋）→ exciting, excited, excite
        │   ├── contentment（满足）→ content, contented
        │   └── enthusiasm（热情）→ enthusiastic, enthusiast
        └── negative（消极）
            ├── sadness（悲伤）→ sad, sadly, sadden
            ├── anxiety（焦虑）→ anxious, anxiously
            ├── frustration（沮丧）→ frustrated, frustrating
            └── despair（绝望）→ desperate, despairing
```

## 3.4 语境记忆法

### 孤立背词 vs 语境习得：效率对比

大量研究对比了两种词汇学习方式：

| 维度 | 孤立背词（词汇表） | 语境习得（阅读/听力） |
|-----|----------------|------------------|
| 初次习得速度 | 快（直接记忆） | 慢（需要多次遭遇） |
| 长期保留率 | 低（48小时遗忘率高） | 高（附带情景记忆） |
| 搭配和用法 | 不习得 | 自然习得 |
| 词义广度 | 窄（只知道一个译义） | 宽（了解多种用法） |
| 适用词汇数 | 低频词可用 | 高频词更优 |
| 情感联结 | 弱 | 强 |

**研究结论**（Nation, 2001）：词汇通过语境遇见 **8-12次**后才能真正习得。孤立背词能提供"认识"，不能提供"使用"。

### 大量阅读（Extensive Reading）

**大量阅读**（Extensive Reading, ER）是指：
- 阅读大量适合自己水平的英语材料
- 以理解内容为目的，不逐词查字典
- 培养"词汇附带习得"（Incidental Vocabulary Acquisition）

**材料选择标准**：

```
Goldilocks Zone（最优区间）：
每页未知词 < 5个 → 太简单，学不到新词
每页未知词 5-15个 → 最佳区间，能从语境猜测
每页未知词 > 20个 → 太难，理解受损，无法猜测
```

**推荐分级材料**：

| 级别 | 词汇量要求 | 推荐材料 |
|-----|----------|---------|
| Starter | 300词 | Oxford Bookworms Starter |
| Level 1 | 400词 | Penguin Readers Level 1 |
| Level 2 | 700词 | Oxford Bookworms Level 2 |
| Level 3 | 1000词 | Penguin Readers Level 3 |
| Level 4 | 1400词 | Oxford Bookworms Level 4 |
| Level 5 | 1800词 | Original simplified texts |
| Level 6 | 2500词 | Easy original novels |
| 原著 | 5000词+ | Harry Potter, Roald Dahl等 |

### 精读技巧（Intensive Reading）

精读与泛读互补，精读时应：

```
词汇深度学习步骤（以 "conspicuous" 为例）：

第1步：语境猜测
  "Her bright red coat made her conspicuous in the crowd."
  → 判断词性：形容词（修饰她）
  → 猜义：让她在人群中显眼/突出

第2步：词典验证
  conspicuous /kənˈspɪkjuəs/
  adj. 明显的，显眼的，引人注目的

第3步：词根分析
  con-（共同/强调）+ spic（看，同spectacle）+ -uous（形容词）
  → 被众人看见的 → 显眼的

第4步：搭配采集
  conspicuous consumption（炫耀性消费）
  conspicuous absence（明显缺席）
  make oneself conspicuous（引人注意）
  be conspicuous by one's absence（因缺席而引人注意）

第5步：造句输出
  His tattooed arms made him conspicuous in the formal setting.
```

## 3.5 同义词辨析与词汇网络

### 语义场理论（Semantic Field Theory）

**语义场**（Semantic Field）是一组共享核心语义特征的词汇集合。深入理解语义场，能帮助你准确区分近义词，避免用词不当。

### 高频同义词辨析

**"看"的语义场**：

```
look (at)    — 故意看，但不一定有所见（中性，最基本）
see          — 无意看见（强调结果，视觉感知）
watch        — 持续观看（强调过程，如看比赛、看电视）
observe      — 仔细观察（正式，科学语境）
examine      — 详细检查（医学/法律语境）
inspect      — 官方检查（如检查工厂、护照）
glance (at)  — 瞥一眼（短暂、不专注）
glimpse      — 瞥见（短暂，往往意外）
gaze (at)    — 凝视（长时间，充满情感）
stare (at)   — 盯视（长时间，可能失礼）
peep/peek    — 偷看（隐秘地看）
witness      — 目睹（见证，强调重要性）
```

**"说"的语义场**：

```
say     — 说（最通用，引述话语）
tell    — 告诉（传递信息，后接人+内容）
speak   — 说话（语言能力，公开发言）
talk    — 交谈（双向，口语）
state   — 陈述（正式，官方）
mention — 提到（简短，不重点）
claim   — 声称（有争议，未经证实）
argue   — 争辩，论证（有理由支持）
assert  — 断言（自信、强调）
declare — 宣布（正式，公开）
announce — 公告（正式通知）
whisper — 耳语（轻声）
mutter  — 咕哝（不满，不清晰）
shout/yell — 大喊
utter   — 发声（正式，书面）
```

**"重要"的语义场（程度差异）**：

```
程度从高到低：

vital / crucial / critical（至关重要的，缺少将有严重后果）
  → "Water is vital to life."

essential（必不可少的，构成要素）
  → "Teamwork is essential for success."

significant（显著的，值得注意的）
  → "This is a significant development."

important（重要的，最中性，最常用）
  → "It's important to stay healthy."

notable / noteworthy（值得注意的，可能不太常见）
  → "a notable exception"

relevant（相关的，有关的）
  → "Is this relevant to our discussion?"

minor / trivial / negligible（次要的，微不足道的）→ 相反方向
```

### 搭配（Collocation）的重要性

词汇不是孤立存在的，词与词之间存在习惯性搭配（Collocations），错误搭配会让英语听起来不自然：

```
正确搭配 vs 直译错误：

✓ make a decision（做决定）
✗ do a decision（受中文"做"的影响）

✓ take a photo（拍照）
✗ make a photo

✓ heavy rain（大雨）
✗ big rain（中文"大雨"直译）

✓ strong coffee（浓咖啡）
✗ thick coffee / heavy coffee

✓ fast food（快餐）
✗ quick food

✓ break a habit（改掉习惯）
✗ destroy a habit

✓ catch a cold（感冒）
✗ get a cold（也对，但 catch 是习惯搭配）

常见动词搭配（动宾搭配）：
do：homework, damage, harm, research, business
make：progress, money, effort, mistake, decision
have：fun, difficulty, experience, influence
take：care, notes, part, place, time
give：advice, help, support, speech, priority
```

### 词汇学习优先级策略

```

```
