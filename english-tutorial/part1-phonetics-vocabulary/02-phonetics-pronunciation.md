## 2.1 国际音标（IPA）完整系统

### 什么是 IPA？

**国际音标**（International Phonetic Alphabet，IPA）是由国际语音学学会（IPA, International Phonetic Association）于1888年创立的一套标准化音标体系。其核心理念是：**一个符号，只对应一个音；一个音，只对应一个符号**。

这与英语拼写形成鲜明对比：
```
英语拼写的混乱（同一拼写，不同发音）：
read  今天读 /riːd/，昨天读 /red/
lead  金属读 /led/，动词读 /liːd/
tear  眼泪读 /tɪər/，撕裂读 /teər/

（同一发音，不同拼写）：
/iː/ 音：see, sea, me, key, field, machine, people, pizza
```

IPA 彻底解决了这个问题。掌握 IPA，你就能用词典自主学习任何单词的发音。

### 英语 IPA 符号总览

英语共使用以下 IPA 符号（以英式 RP 发音为基准）：

```
元音（Vowels）：20个
单元音（Monophthongs）：12个
  /ɪ/  /e/  /æ/  /ʌ/  /ɒ/  /ʊ/
  /iː/ /ɜː/ /ɑː/ /ɔː/ /uː/ /ə/

双元音（Diphthongs）：8个
  /eɪ/ /aɪ/ /ɔɪ/ /əʊ/ /aʊ/ /ɪə/ /eə/ /ʊə/

辅音（Consonants）：24个
  清音（Voiceless）：/p/ /t/ /k/ /f/ /θ/ /s/ /ʃ/ /tʃ/ /h/
  浊音（Voiced）：  /b/ /d/ /g/ /v/ /ð/ /z/ /ʒ/ /dʒ/
  鼻音（Nasals）：  /m/ /n/ /ŋ/
  流音（Liquids）： /l/ /r/
  半元音（Glides）：/j/ /w/
```

## 2.2 元音系统

### 口腔元音位置图

元音的发音由**舌位高低**（tongue height）、**舌位前后**（tongue backness）和**嘴唇形状**（lip rounding）决定。

```
元音四边形（Vowel Quadrilateral）：

前元音 ←——————————→ 后元音
高  iː              uː
    ɪ               ʊ
中  e        ə      ɔː
    ɜː
    æ        ʌ      ɒ
低              ɑː
```

### 12个单元音详解

| 符号 | 例词 | 中文近似音 | 嘴型描述 | 常见拼写 |
|-----|-----|----------|---------|---------|
| /ɪ/ | **s**i**t**, k**i**d, b**i**g | 短促的"衣"，放松不绷紧 | 嘴微开，舌前高位 | i, y, e(英) |
| /iː/ | s**ee**, m**e**, t**ea** | 拉长的"衣——" | 嘴角展开，舌高前位 | ee, ea, e, ie |
| /e/ | **e**gg, b**e**d, s**ai**d | 类似中文"诶"短促版 | 嘴半开，舌中前位 | e, ea, a(any) |
| /æ/ | c**a**t, b**a**d, m**a**n | 介于"啊"和"诶"之间 | 嘴大开，舌前低位 | a（短a词） |
| /ʌ/ | c**u**p, l**o**ve, b**u**s | 类似中文"啊"但短促靠前 | 嘴半开，舌央位 | u, o, ou |
| /ɑː/ | c**ar**, f**a**ther, b**a**th | 拉长的"啊——" | 嘴大开，舌后低位 | ar, a(英式) |
| /ɒ/ | d**o**g, h**o**t, c**o**d | 类似"哦"，嘴圆开 | 嘴圆开，舌后低位（英式） | o（英式短音） |
| /ɔː/ | d**oor**, **all**, w**a**ter | 拉长的"哦——" | 嘴圆半开，舌后中位 | or, oor, aw, all |
| /ʊ/ | b**oo**k, p**u**t, f**oo**t | 短促的"乌"，放松 | 嘴唇略圆，舌后高位 | oo, u |
| /uː/ | f**oo**d, bl**ue**, t**oo** | 拉长的"乌——" | 嘴唇圆拢，舌后高位 | oo, ue, ew, ou |
| /ɜː/ | b**ir**d, h**er**, t**ur**n | 类似说"额"同时卷舌（美式） | 舌居中，嘴半开 | ir, er, ur, ear |
| /ə/ | **a**bout, sof**a**, b**u**tter | 最弱的"呃"，模糊央元音 | 最放松状态，不强调 | 任何非重读元音 |

> **关键提示**：/ə/（schwa，央元音/弱化元音）是英语中**出现频率最高**的音！所有非重读音节几乎都趋向于 /ə/。掌握 schwa，英语听感会大幅提升。

### 8个双元音详解

双元音（Diphthong）= 两个元音的滑动，从一个位置滑向另一个位置，前面音重后面音轻：

| 符号 | 例词 | 记忆口诀 | 中文近似 | 常见拼写 |
|-----|-----|---------|---------|---------|
| /eɪ/ | d**ay**, **a**te, **a**ge | "诶→衣"滑动 | 类似中文"诶" | a, ay, ai, ey |
| /aɪ/ | **I**, l**igh**t, k**i**te | "啊→衣"滑动 | 类似中文"爱" | i, igh, y, ie |
| /ɔɪ/ | b**oy**, n**oi**se, j**oy** | "哦→衣"滑动 | 类似中文"哦衣" | oi, oy |
| /əʊ/ | g**o**, kn**ow**, b**oa**t | "呃→乌"滑动（英式） | 类似中文"欧" | o, oa, ow, oe |
| /aʊ/ | h**ou**se, n**ow**, **ou**t | "啊→乌"滑动 | 类似中文"啊乌" | ou, ow |
| /ɪə/ | h**ere**, **ear**, d**ear** | "衣→呃"滑动 | 类似中文"衣啊" | ear, eer, ere |
| /eə/ | h**air**, wh**ere**, th**ere** | "诶→呃"滑动 | 类似中文"诶啊" | air, are, ear |
| /ʊə/ | p**ure**, t**our**, c**ure** | "乌→呃"滑动 | 类似中文"乌啊" | ure, oor, our |

### 美式英语元音特点

美式英语与英式英语最大的系统性元音差异：

```
1. r-colored vowels（卷舌元音）
   英式 /ɑː/ → 美式 /ɑːr/（car, far, star）
   英式 /ɜː/ → 美式 /ɜːr/（bird, word, turn）
   英式 /ɔː/ → 美式 /ɔːr/（door, more, store）

2. Bath-Trap Split（英式特有）
   英式：path /pɑːθ/，dance /dɑːns/（长音 /ɑː/）
   美式：path /pæθ/，dance /dæns/（短音 /æ/）

3. Cot-Caught Merger（美式特有）
   美式：cot（小床）和 caught（抓住）发音相同，都是 /kɑːt/
   英式：cot /kɒt/ ≠ caught /kɔːt/
```

## 2.3 辅音系统

### 辅音分类框架

辅音通过三个维度分类：
1. **发音部位**（Place of Articulation）— 气流被阻碍的位置
2. **发音方式**（Manner of Articulation）— 气流被阻碍的方式
3. **清浊**（Voicing）— 声带是否振动

### 清浊对立：英语中的最小音对

清浊对比是英语辅音的核心特征，很多单词仅靠清浊区分：

```
清音 /p/ vs 浊音 /b/：   pin /pɪn/ vs bin /bɪn/
清音 /t/ vs 浊音 /d/：   ten /ten/ vs den /den/
清音 /k/ vs 浊音 /g/：   cap /kæp/ vs gap /gæp/
清音 /f/ vs 浊音 /v/：   fine /faɪn/ vs vine /vaɪn/
清音 /s/ vs 浊音 /z/：   sip /sɪp/ vs zip /zɪp/
清音 /θ/ vs 浊音 /ð/：   thing /θɪŋ/ vs this /ðɪs/
清音 /ʃ/ vs 浊音 /ʒ/：   shoe /ʃuː/ vs (measure /ˈmeʒər/)
清音 /tʃ/ vs 浊音 /dʒ/： chin /tʃɪn/ vs gin /dʒɪn/
```

### 中国学习者的发音难点

| 难点音 | 问题描述 | 错误示例 | 正确练习方法 |
|-------|---------|---------|------------|
| /θ/ /ð/ | 中文无此音，常被替换为 /s/ 或 /t/ | think→"sink", the→"de" | 将舌尖伸至上下牙之间，轻触 |
| /v/ | 常被替换为 /w/ 或 /f/ | very→"wery" | 上齿咬下唇，振动声带 |
| /r/ | 美式卷舌，常被替换为类似 /l/ | right→"light" | 舌尖后卷，不触硬腭 |
| /l/ vs /r/ | 混淆，以 /r/ 替代 /l/ 或反之 | light/right 混淆 | 对比最小音对，感受舌位差异 |
| /æ/ | 发成 /e/ 或 /a/ | cat→/ket/ | 嘴巴大开，像要说"啊"同时嘴角展开 |
| 末尾辅音 | 省略词尾辅音（汉语无闭音节习惯） | good→"gu" | 练习 CVC 结构单词 |
| 辅音丛 | 简化辅音群 | street→"sutrit" | 逐步添加辅音：t→tr→str→street |

## 2.4 自然拼读法（Phonics）核心规则

### 什么是 Phonics？

**自然拼读法**（Phonics）是一套通过字母/字母组合与发音对应关系来学习阅读的方法。掌握 Phonics，可以拼读约90%的常规英语单词，实现"见词能读、听音能写"。

### 核心规则一：元音规则

**短元音**（Short Vowels）— 封闭音节（CVC结构，辅音+元音+辅音）

```
a → /æ/：cat, hat, map, bag, ran
e → /e/：bed, hen, pet, red, ten
i → /ɪ/：sit, big, fin, pin, hit
o → /ɒ/：hot, dog, pot, box, cob
u → /ʌ/：cup, bug, run, sun, mud
```

**长元音**（Long Vowels）— 开放音节（CV结构，或 CVCe 结构）

```
规则1：Magic E（VCe结构 — 元音+辅音+哑音e）
a_e → /eɪ/：cape, lake, name, game, late
i_e → /aɪ/：bike, kite, pine, time, fine
o_e → /əʊ/：note, home, pole, hope, bone
u_e → /juː/：cube, cute, tune, mule, dune

规则2：开放音节（元音在末尾，读长音）
a → /eɪ/：me, he, she, be
e → /iː/：me, he, she, be
i → /aɪ/：hi, fly, try
o → /əʊ/：go, no, so
u → /juː/：flu, clue
```

### 核心规则二：元音字母组合（Vowel Teams）

```
ee/ea → /iː/：see, tree, bean, lead, clean
ai/ay → /eɪ/：rain, mail, day, play, away
oa/ow → /əʊ/：boat, toad, low, snow, grow
oo   → /uː/ 或 /ʊ/：food/foot, moon/book
oi/oy → /ɔɪ/：coin, oil, boy, toy
ou/ow → /aʊ/：out, cloud, cow, how
ew/ue → /juː/：new, few, blue, clue
aw/au → /ɔː/：claw, paw, caught, sauce
```

### 核心规则三：辅音规则

**软硬音规则（C 和 G 的两种发音）**：

```
C 的规则：
  C + a/o/u/辅音 → /k/（硬音）：cat, corn, cup, clap
  C + e/i/y     → /s/（软音）：cell, city, cycle

G 的规则：
  G + a/o/u/辅音 → /g/（硬音）：gas, go, gun, glass
  G + e/i/y     → /dʒ/（软音）：gem, giant, gym（有例外）
  
注意：get, give, girl 中 G 前接 e/i 仍读 /g/，是例外
```

**常见辅音字母组合（Consonant Digraphs）**：

```
ch → /tʃ/：chair, cheese, lunch
sh → /ʃ/：ship, shoe, wash
th → /θ/ 或 /ð/：think(清), this(浊)
wh → /w/（美式）：where, when, white
ph → /f/：phone, photo, graph
ng → /ŋ/：sing, ring, long
ck → /k/：back, duck, clock
```

**辅音丛（Consonant Clusters）**：

```
初始辅音丛（词首）：
bl-, cl-, fl-, gl-, pl-, sl-：black, class, fly, glass, play, slow
br-, cr-, dr-, fr-, gr-, pr-, tr-：brown, cry, dry, from, grow, pray, try
sc-, sk-, sm-, sn-, sp-, st-, sw-：scar, skin, small, snow, spin, stop, swim
str-, spr-, scr-：street, spring, scream

末尾辅音丛（词尾）：
-lk：milk, silk, walk
-nd：end, hand, mind
-ng：ring, song, long
-nk：bank, drink, think
-st：best, last, most
```

## 2.5 音节划分与重音规则

### 什么是音节？

**音节**（Syllable）是语音的基本节奏单位，每个音节有且只有一个**元音音峰**（vowel peak）。

```
音节数量判断：
one     = 1音节（单音节）
water   = wa-ter = 2音节
beautiful = beau-ti-ful = 3音节
information = in-for-ma-tion = 4音节
communication = com-mu-ni-ca-tion = 5音节
```

### 音节划分规则

**规则1：VC/CV 规则**
```
两个辅音之间的元音，辅音分配到两个音节：
VC-CV：pep-per, sin-gle, mar-ket, bas-ket

一个辅音在两个元音之间，通常并入后一音节（开放音节）：
V-CV：na-tion, ti-ger, o-pen, ba-by
```

**规则2：前缀和后缀各成音节**
```
un-hap-py（un- 是前缀）
teach-er（-er 是后缀）
re-mem-ber（re- 是前缀）
```

**规则3：le 结尾音节**
```
辅音 + le = 单独音节：
ta-ble, sim-ple, gen-tle, bot-tle, mid-dle
```

### 单词重音规则

英语重音（Word Stress）是英语节奏的核心，错误重音会导致完全听不懂。

**双音节规则**：

```
规则1：大多数双音节名词/形容词，重音在第一音节：
'ta-ble, 'mu-sic, 'hap-py, 'cit-y, 'doc-tor, 'stu-dent

规则2：大多数双音节动词，重音在第二音节：
be-'gin, for-'get, de-'cide, pre-'fer, re-'peat

重音决定词性（名动转换）：
'record（记录，名词） vs re-'cord（录音，动词）
'permit（许可证，名词）vs per-'mit（允许，动词）
'present（礼物，名词）vs pre-'sent（展示，动词）
'conduct（行为，名词）vs con-'duct（指挥，动词）
'object（物体，名词） vs ob-'ject（反对，动词）
'protest（抗议，名词）vs pro-'test（抗议，动词）
```

**多音节词规则**：

```
后缀决定重音（常见规律）：

-tion, -sion, -cian：重音在其前一音节
  na-'tion, in-for-'ma-tion, mu-si-'cian

-ity, -ety, -gy, -phy, -cy, -ry：重音在其前第二音节
  e-lec-'tri-ci-ty, de-'moc-ra-cy, bi-'ol-o-gy

-ic, -ical：重音在 -ic 前一音节
  e-co-'nom-ic, his-'tor-ic, e-lec-'tric-al
```

### 句子重音（Sentence Stress）

英语是**重音计时语言**（stress-timed language），重读音节之间的时间间隔大致相等。

**内容词**（Content Words）通常重读：名词、动词、形容词、副词、疑问词

**功能词**（Function Words）通常弱读：冠词、介词、代词、连词、助动词

```
例句重音标记（大写=重读，小写=弱读）：
"I WANT to GO to the STORE."
  i  want tə  go  tə ðə  stɔːr

"She's BEEN WORKING all DAY."
  ʃɪz  bɪn  wɜːkɪŋ əl  deɪ
```

**对比重音**（Contrastive Stress）：强调特定信息

```
"I said JOHN did it."（不是别人，是约翰）
"I SAID John did it."（我确实说过）
"I said John DID it."（他做了，不是没做）
"I said John did IT."（做的是这件事，不是别的）
```

## 2.6 连读、弱读、失去爆破等语流规则

### 为什么要学语流规则？

真实英语口语与"逐词朗读"差异巨大。如果不了解语流规则（Connected Speech），即使每个词都认识，也可能听不懂自然语速的英语。

### 连读（Liaison / Linking）

**辅音+元音连读**：前词末辅音 + 后词首元音 → 连读为一个音节

```
turn it off → tur-nit-off → /ˈtɜːr.nɪt.ɒf/
pick it up  → pi-ki-tup   → /ˈpɪ.kɪ.tʌp/
not at all  → no-ta-tall  → /ˌnɒ.tə.ˈtɔːl/
an apple    → a-napple    → /ə.ˈnæ.pəl/
```

**元音+元音连读**（加入过渡音）：

```
前词末为 /iː/ /ɪ/ /eɪ/ /aɪ/ /ɔɪ/ → 加入 /j/：
go away  → go-jaw-way → /gəʊ.dʒə.weɪ/
he asked  → he-jasked → /hɪ.jɑːskt/

前词末为 /uː/ /ʊ/ /əʊ/ /aʊ/ → 加入 /w/：
do it   → do-wit → /duː.wɪt/
who is  → who-wiz → /huː.wɪz/
```

### 弱读（Weak Forms）

功能词在句子中通常发弱读形式（schwa化）：

| 词 | 强读 | 弱读 | 例句中的弱读 |
|---|-----|-----|-----------|
| a | /eɪ/ | /ə/ | a cat → /ə kæt/ |
| the | /ðiː/ | /ðə/ | the book → /ðə bʊk/ |
| and | /ænd/ | /ənd/ /ən/ /n/ | bread and butter → /bred.ən.ˈbʌtər/ |
| to | /tuː/ | /tə/ | I want to go → /aɪ.wɒntə.gəʊ/ |
| of | /ɒv/ | /əv/ /ə/ | cup of tea → /kʌp.ə.ˈtiː/ |
| for | /fɔː/ | /fər/ /fə/ | wait for me → /weɪt.fə.miː/ |
| from | /frɒm/ | /frəm/ | from here → /frəm.hɪər/ |
| can | /kæn/ | /kən/ | I can do it → /aɪ.kən.duːɪt/ |
| have | /hæv/ | /həv/ /əv/ | I have seen → /aɪ.həv.siːn/ |
| was | /wɒz/ | /wəz/ | he was there → /hiː.wəz.ðeər/ |

### 失去爆破（Plosive Deletion / Unreleased Stops）

当两个爆破音相邻，前一个爆破音通常"失去"（不完全爆破，即停顿但不发出音来）：

```
/p/ /b/ /t/ /d/ /k/ /g/ 后接辅音时，前爆破音失去：

sto(p) watch → stop 的 /p/ 不完全爆破
ba(d) dog    → bad 的 /d/ 不完全爆破  
bla(ck) cat  → black 的 /k/ 不完全爆破
goo(d) night → good 的 /d/ 不完全爆破
si(t) down   → sit 的 /t/ 不完全爆破
```

### 同化（Assimilation）

相邻的音互相影响，改变对方的发音：

```
逆行同化（后音影响前音）：
ten pence → /tem pens/（n 受 p 影响变为 m）
good boy  → /gub bɔɪ/（d 受 b 影响变为 b）
in case   → /ɪŋ keɪs/（n 受 k 影响变为 ŋ）

/t/ /d/ + /j/ 的腭化：
did you → /dɪdʒu/（d + y → dʒ）
what you → /wɒtʃu/（t + y → tʃ）
don't you → /dəʊntʃu/
```

### 省略（Elision）

语流中某些音被完全省略：

```
/t/ 在辅音丛中的省略：
next door → /neks dɔːr/（t 省略）
last night → /lɑːs naɪt/（t 省略）
facts     → /fæks/（t 省略）

/h/ 在弱读代词中的省略：
tell him → /tel.ɪm/（h 省略）
give her → /gɪv.ər/（h 省略）
ask him  → /ɑːsk.ɪm/
```

### 英美语音流差异总结

```
美式特有（American English Features）：

1. Flapping（/t/ 弱化）：辅音 /t/ /d/ 在元音之间变为弹舌音 /ɾ/
   water  → /ˈwɔːɾər/（听起来像"wader"）
   butter → /ˈbʌɾər/
   better → /ˈbeɾər/
   city   → /ˈsɪɾi/

2. T-glottalization（喉塞音）：/t/ 在音节末或弱化时变为喉塞音 /ʔ/
   button → /ˈbʌʔn/
   certain → /ˈsɜːʔn/

3. Cot-Caught Merger：/ɒ/ 和 /ɔː/ 合并为 /ɑː/
```

## 练习

### 练习1：音标转录
用 IPA 音标写出以下单词的发音（参考词典）：
1. through（穿过）
2. thought（想到）
3. thorough（彻底的）
4. though（虽然）
5. tough（艰难的）

> 提示：这5个词拼写相似但发音完全不同，展示了英语拼写的不规则性。

### 练习2：重音标注
在以下单词的正确音节上标注重音（用大写表示）：
1. photograph / photographer / photographic
2. economy / economic / economical
3. democracy / democratic
4. present（名词）/ present（动词）

### 练习3：Phonics 解码
根据 Phonics 规则，解释以下单词的发音逻辑：
1. **cake**（Magic E 规则）
2. **street**（元音组合 + 辅音丛）
3. **city**（软 C 规则）
4. **phone**（ph 规则）
5. **bridge**（软 G 规则）

### 练习4：语流听写
听以下句子（或请老师朗读），注意连读和弱读现象，写出你听到的实际发音（用斜体标注弱化的词）：
1. "What do you want to eat?"
2. "I've been waiting for about an hour."
3. "Did you pick it up from the shelf?"

### 练习5：清浊对比练习
朗读以下最小音对，注意清音和浊音的差别：
```
/p/-/b/：pit-bit, cup-cub, rope-robe
/t/-/d/：tie-die, cat-cad, seat-seed
/f/-/v/：fan-van, leaf-leave, safe-save
/s/-/z/：sip-zip, rice-rise, place-plays
/θ/-/ð/：thing-this, breath-breathe, mouth-mouthe
```

## 本章小结

| 知识点 | 核心内容 |
|-------|---------|
| IPA系统 | 英语有20个元音（12单+8双）+ 24个辅音，IPA一符一音 |
| 单元音 | /ə/(schwa)最高频；/iː//uː/ 为长元音；口腔位置决定音质 |
| 双元音 | 8个双元音均为滑动音，从前位置滑向后位置 |
| 辅音 | 清浊+发音部位+发音方式三维分类；/θ//ð//v//r/ 是中国学生难点 |
| Phonics | Magic E规则、元音组合、软硬C/G、辅音丛是核心规则 |
| 重音 | 名词/形容词重前，动词重后；重音可区分词性 |
| 语流 | 连读、弱读、失去爆破、同化、省略是自然口语的必备知识 |
