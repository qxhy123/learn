## 引言

每年全球有数百万人参加英语标准化考试——雅思、托福、GRE、四六级、考研……这些考试是留学申请、职业发展、学历认证的敲门砖。然而，许多人花费大量时间备考，却因为不了解考试机制、题型特点和应试策略而事倍功半。本章将系统拆解各主要英语考试的备考策略，提供可操作的具体方法，并在最后探讨超越考试的终身英语学习路径。

## 32.2 雅思（IELTS）备考策略

### 听力（Listening）：满分40分，折算9分制

雅思听力分4个Section，难度递增，每题只播放一遍。

| Section | 场景类型 | 题型特点 | 难度 |
|---------|----------|----------|------|
| Section 1 | 日常对话（如预订、查询） | 填空为主，信息清晰直接 | ★☆☆☆ |
| Section 2 | 独白（如景点介绍、活动说明） | 填空+选择，需提炼关键信息 | ★★☆☆ |
| Section 3 | 学术讨论（2-4人） | 选择+配对，观点识别复杂 | ★★★☆ |
| Section 4 | 学术讲座（独白） | 填空+选择，术语多，速度快 | ★★★★ |

**核心技巧**：

```
技巧1：预读题目（Prediction）
录音播放前，利用30秒读题，预判答案类型：
- 空格前后是什么词性？→ 预判填什么（数字/名词/形容词）
- 选项的区别在哪里？→ 提炼关键区分词

技巧2：同义替换识别（Paraphrase Recognition）
雅思答案几乎不会原词重复，一定要熟悉同义替换：
题目："What is the main reason?"（什么是主要原因？）
录音："The primary factor is..."（主要因素是……）
→ main → primary；reason → factor

技巧3：答案预判与捕捉
填空题答案通常是1-3个词，注意：
- 数字（日期、价格、数量）
- 专有名词（地点、人名）→ 注意大写和拼写
- 修饰语+名词（the blue folder, Wednesday morning）

技巧4：地图题（Map/Plan）专项
先读题，标注起点（you are here），
听到"opposite/next to/between/turn left/go straight"等方向词立即标记
```

**各题型专项策略**：

```
判断题（Multiple Choice）：
- 听到选项原词并不一定是答案（可能是干扰项）
- 真正的答案往往在说话者"转折"之后出现
  "I thought X, but actually Y." → 答案是Y

配对题（Matching）：
- 同时追踪多个名字/数字+对应特征
- 先处理简单的，再回头填写困难的
```

### 阅读（Reading）：满分40分，60分钟3篇文章

**时间分配建议**：
```
文章1（最简单）：15分钟
文章2（中等）：20分钟
文章3（最难）：25分钟
紧急情况：发现某题组特别难 → 跳过，先做其他题，最后回填
```

**判断题 True/False/Not Given（YES/NO/NG）策略**：

这是中国考生失分最多的题型。关键区别：

```
TRUE (YES)：文章中有明确陈述与题目一致
FALSE (NO)：文章中有明确陈述与题目相反/矛盾
NOT GIVEN (NG)：文章中根本没有提到此信息（无法判断）

❌ 常见错误：
- 用"常识"判断（"这件事常识上是错的，所以是FALSE"）→ 不对！
  只能根据文章内容判断

- 将"文章没提到"判断为FALSE → 不对！
  文章没提到 = NOT GIVEN，不是FALSE

正确方法：
1. 找到题目关键词在文章中的位置
2. 比较题目信息与文章信息
3. 完全一致 → TRUE；明确相反 → FALSE；找不到 → NOT GIVEN
```

**标题匹配（Heading Matching）策略**：

```
步骤1：先读标题列表，提炼每个标题的核心词
步骤2：读段落的前2句和最后1句（主题句通常在开头）
步骤3：匹配核心含义，不要被个别单词迷惑

注意：标题是对段落主旨的概括，不是对细节的描述
一个段落可能有多个标题貌似匹配 → 选最能概括整段主旨的
```

### 写作（Writing）：Task 1（20分钟，150词）+ Task 2（40分钟，250词）

**Task 1——图表描述高分结构**：

```
【折线图/柱状图/饼图通用结构】

段落1：Overview（总览）—— 最重要，直接影响分数
→ 不描述具体数字，描述最显著的1-2个趋势
"Overall, it is clear that..."
"The most striking feature is that..."

段落2：第一组数据分析
→ 选择2-3个最有代表性的数据，支撑Overview观点

段落3：第二组数据对比分析
→ 横向或纵向对比，使用对比连接词

高分句型：
"The figure rose/climbed/surged dramatically from X to Y over the period."
"By contrast, the proportion of X declined sharply, falling from..."
"Despite a slight dip in [year], the overall trend remained upward."
"The two categories followed opposite trends: while X increased, Y..."

❌ 避免：
- 只列数字，不分析趋势
- 照抄题目中的描述文字
- 没有Overview段
```

**Task 2——议论文高分框架**：

```
【通用五段式结构】

段落1：Introduction（引言）
→ 背景句（改写题目信息）+ 立场句（thesis statement）
"In recent years, [topic] has become a subject of considerable debate.
While some argue that..., I believe that..."

段落2：支持论点一
→ 主题句 + 解释 + 例子/数据 + 小结

段落3：支持论点二
→ 同上结构

段落4：反驳/让步段（Concession Paragraph）—— 高分关键
"Admittedly, some might argue that [对立观点].
However, this view overlooks the fact that..."
→ 承认对立观点的合理性，但随后驳斥，展示思维深度

段落5：Conclusion（结论）
→ 重申立场 + 总结论点（改写，不要原句复制）
"In conclusion, despite the arguments suggesting...,
I firmly maintain that..."

高分词汇搭配：
问题类：address/tackle/combat + this issue/challenge/problem
增长类：witness/experience + a dramatic/steady + increase/surge/rise
影响类：have a profound/significant/detrimental impact/effect on
建议类：it is imperative/essential that + subjunctive mood
```

**各分数段特征与突破**：

| 分数 | 典型特征 | 突破重点 |
|------|----------|----------|
| 5.0 | 意思基本能传达，大量基础错误 | 消除语法基础错误（时态、单复数、冠词）|
| 6.0 | 结构基本完整，词汇范围有限，有些不自然表达 | 扩大词汇量，练习同义替换，增强语篇连贯 |
| 6.5 | 内容相关，偶有错误，连接词使用较好 | 提升论证深度，减少重复用词，加入对立观点 |
| 7.0 | 清晰有效，词汇灵活，偶有错误 | 打磨语言精准度，增强高级语法结构（倒装、虚拟语气）|
| 7.5+ | 流利、灵活，几乎无错误，风格成熟 | 培养个人风格，使用学术习语，细节精准 |

### 口语（Speaking）：11-14分钟，3个Part

**评分标准四维度**（各占25%）：
- **Fluency & Coherence（流利度与连贯性）**：能否自然流畅地表达，不频繁停顿
- **Lexical Resource（词汇资源）**：词汇的丰富程度和准确性
- **Grammatical Range & Accuracy（语法范围与准确性）**：语法结构的多样性
- **Pronunciation（发音）**：发音的清晰度和自然度（不要求英式或美式口音）

```
Part 1（4-5分钟）：简短问答（家乡、工作、爱好等）
策略：
- 不要只用一句话回答，扩展2-3句
- 使用自然连接词："Well, actually..." / "To be honest..."
- 避免背诵的痕迹，保持对话感

Part 2（3-4分钟）：话题卡独白，准备1分钟
策略：
- 利用1分钟准备时间打草稿（关键词，不是完整句）
- 严格按照话题卡上的要点组织内容
- 讲一个具体的故事/例子，比泛泛而谈得分更高
- 预留"and the most memorable part was..."拉长时间

话题卡准备方法（STAR方法）：
Situation（情境）：设置背景
Task/Topic（任务/主题）：你要描述的核心
Action（行动/细节）：具体发生了什么
Result/Reflection（结果/感想）：意义或影响

Part 3（4-5分钟）：深入讨论（抽象话题、社会问题）
策略：
- Part 3考查分析能力，不只是描述
- 使用"框架化"开头："There are several factors to consider..."
- 发表个人观点并支撑："Personally, I think... because..."
- 适当提出对立观点然后反驳："While some may argue..., I believe..."
- 不确定时的缓冲表达："That's an interesting question. Let me think..."
```

## 32.3 托福（TOEFL iBT）备考策略

### 阅读（Reading）：54-72分钟，2-3篇文章，各10题

**10种题型分类与策略**：

```
【事实信息题（Factual Information）】
关键词：according to the passage, the author states that...
策略：定位关键词，精读该段，答案在原文中直接找到

【否定事实题（Negative Factual Information）】
关键词：EXCEPT, NOT, LEAST
策略：找到3个正确选项，排除法确定NOT符合原文的选项

【推断题（Inference）】
关键词：implies, suggests, can be inferred...
策略：答案在原文中没有直说，需要逻辑推断；
避免"过度推断"——答案要有文本支持

【修辞目的题（Rhetorical Purpose）】
关键词：Why does the author mention...?
策略：看该信息在段落结构中的作用（举例/对比/支持论点）

【词汇题（Vocabulary）】
关键词：the word X in paragraph Y is closest in meaning to...
策略：回到原文，用上下文线索判断词义，不靠记忆

【参考题（Reference）】
关键词：The word "they/it/these" refers to...
策略：找最近的同类名词，注意单复数匹配

【句子简化题（Sentence Simplification）】
关键词：Which of the following best expresses the essential information...?
策略：找包含"主要信息"的选项，排除遗漏关键信息或增加错误信息的

【句子插入题（Sentence Insertion）】
关键词：Look at the four squares ■
策略：注意前后逻辑衔接（代词指代、连接词、话题延续）

【摘要题（Prose Summary）】
关键词：Introductory sentence provided, choose 3 more sentences
策略：选支持"主旨"的句子，排除细节/错误/文中未提及的

【表格分类题（Fill in a Table）】
策略：先读两类别的定义，再逐条对照原文分类
```

### 听力（Listening）：41-57分钟，2-3 Conversations + 3-4 Lectures

**两种场景的不同笔记策略**：

```
【Conversation 笔记框架】
两人对话，通常有一个问题要解决

Who:  [说话者身份]
What: [讨论什么问题/话题]
Why:  [找教授/学生服务的原因]
Solution/Result: [最终结果或建议]

笔记关键词：
- 态度词（positive/negative/hesitant）
- 转折词（but, however, actually）→ 转折后往往是答案

【Lecture 笔记框架】
学术讲座，有主题和支撑例子

Topic: [讲座主题]
Main Point 1:
  - Example/Detail:
Main Point 2:
  - Example/Detail:
Conclusion/Significance:

笔记技巧：
→ 记符号而非完整词：增加(↑)、减少(↓)、等于(=)、因为(∵)、因此(∴)
→ 缩写：development = dvlp, important = imp, according to = acc to
→ 重点标记教授重复的信息和语气词："This is key...", "Importantly..."
```

### 口语（Speaking）：约17分钟，4个任务

```
【Task 1 - Independent Speaking（独立口语）】
题型：个人观点/偏好（45秒准备，60秒作答）
模板：
"I would [choose/prefer] [option] because [reason 1] and [reason 2].
For instance, [specific example].
Therefore, [restate choice] is better for me."

计时要求：
- 前10秒：明确表态
- 中间40秒：两个理由各15-20秒
- 最后10秒：总结

【Task 2 - Integrated Reading + Listening（综合口语）】
题型：读文章（45秒）→ 听对话→ 说明对话观点如何支持/反对文章
模板：
"According to the reading passage, [主要观点].
The [man/woman] in the conversation [agrees/disagrees] because [理由1].
[He/She] also points out that [理由2]."

【Task 3 - Integrated Lecture（综合讲座）】
题型：听讲座→ 概括教授的两个例子如何说明阅读中的概念
模板：
"The reading introduces the concept of [概念].
The professor illustrates this with two examples.
First, [例子1] demonstrates [如何体现概念].
Second, [例子2] shows [如何体现概念]."

【Task 4 - Integrated Lecture Only（仅讲座）】
题型：听讲座→ 总结主要概念和支撑细节
模板：
"The lecture discusses [topic].
The professor explains that [主要观点].
[He/She] provides two key points: first... second..."
```

### 与雅思的横向对比

| 方面 | IELTS | TOEFL iBT |
|------|-------|-----------|
| **阅读风格** | 更多观点类文章，需要识别作者态度 | 更多学术性文章，侧重信息提取 |
| **听力风格** | 英式/澳式口音更多，生活场景 | 美式口音为主，学术场景 |
| **写作Task 1** | 图表描述（非议论文） | 综合写作（读+听+写） |
| **口语形式** | 与考官面对面 | 对着麦克风录音 |
| **可迁移技能** | 笔记技能、同义替换识别、议论文写作均可互通 |

## 32.4 GRE 备考策略

### Verbal Reasoning（语文推理）：满分170分

**词汇量要求与特点**：

```
GRE词汇级别：高于TOEFL/IELTS，接近英语母语者阅读高级学术文献

核心高频词汇类别：
1. 描述态度/情感的词汇（考试最爱）
   equivocal（模棱两可）、sanguine（乐观的）、
   laconic（言简意赅）、loquacious（话多的）、
   reticent（沉默的）、effusive（感情横溢的）

2. 描述强化/削弱的词汇
   corroborate（证实）、refute（反驳）、
   bolster（支撑）、undermine（削弱）、
   substantiate（证实）、contradict（矛盾）

3. 描述变化的词汇
   burgeon（迅速发展）、wane（减弱）、
   ameliorate（改善）、exacerbate（使恶化）
```

**填空题（Text Completion & Sentence Equivalence）逻辑解题**：

```
GRE填空的核心不是"认识这个词"，而是"理解句子逻辑结构"

步骤1：找逻辑关键词
正向逻辑（同向）：and, also, moreover, similarly, as expected
反向逻辑（反转）：but, however, although, despite, yet, rather

步骤2：根据逻辑预判空格含义（不看选项）

步骤3：从选项中选最接近预判含义的词

例题：
"The professor was known for being ___; she never spoke more
than three words when one would suffice."
逻辑：冒号后的说明是解释（同向），"说话少"→ 找"言简意赅"的词
答案：laconic / terse / taciturn（均为"寡言的"）

双空题：两个空格有逻辑关系，确认一个往往能推断另一个
```

### Analytical Writing（分析写作）：满分6.0分

```
【Issue Task（议题写作）】- 30分钟，550-700词

题型：给出一个有争议的观点，表达并论证你的立场

高分结构：
段落1：引言 + 条件化立场（Qualified Position）
→ 不要简单说"我同意/不同意"
→ 说"在特定条件下，我同意；但当X时，这个观点站不住脚"

段落2-3：支持你立场的论证（各含具体例子）

段落4：最重要的反驳/让步段
→ GRE评分者特别看重你是否能考虑到对立观点的合理性

段落5：结论（重申条件化立场）

关键得分技巧：
- 使用具体例子（历史事件、科学研究、现实案例），不要只有抽象论断
- 识别题目中的假设（unwarranted assumptions）并挑战它
- 展示批判性细微差别（nuance）：不是非黑即白

【Argument Task（论证写作）】- 30分钟，550-700词

题型：给出一段有逻辑缺陷的论证，指出其中的问题

常见逻辑谬误（Logical Fallacies）：
1. 错误的因果关系（Post hoc fallacy）：
   "After we hired the new manager, profits rose." → 相关≠因果
   
2. 样本不充分（Hasty Generalization）：
   "3 out of 10 customers preferred X" → 样本太小
   
3. 类比不当（False Analogy）：
   "This worked in City A, so it will work in City B"
   → 两城市可能有根本差异

4. 错误的二分法（False Dichotomy）：
   "Either we invest more or the company will fail" → 忽略其他可能
   
写法：不是给出正确答案，而是指出论证的漏洞，并说明需要什么额外信息才能支持该结论。
```

### GRE词汇记忆策略

```
词根词缀法（推荐优先）：
前缀：
bene- = 好（beneficial, benevolent, benign）
mal- = 坏（malicious, malevolent, malignant）
pre- = 前（precede, preemptive, precedent）
mis- = 错误（misconception, misnomer, misanthrope）

-cred- = 相信（credulous, incredulous, credibility）

使用Anki设置间隔重复，比死背效率高5-10倍
```

## 32.5 大学英语四六级与考研英语

### 四六级各题型时间分配

| 题型 | 时间 | 分值 | 策略要点 |
|------|------|------|----------|
| 写作（作文） | 30分钟 | 15% | 开头段+3论点+结尾段，模板+灵活调整 |
| 听力（长对话/讲座） | 25分钟 | 35% | 预读题干，抓关键词，注意转折 |
| 阅读（仔细阅读） | 约40分钟 | 35% | 先读题后读文，同义替换敏感 |
| 翻译（汉译英） | 15-20分钟 | 15% | 直译为主，语法正确优先于文采 |

**四六级阅读——"词义猜测"专项**：

```
猜测词义的5种技巧：

1. 利用定义线索：
   "Philatelist, or stamp collectors, are..."
   → 定义从句中包含解释

2. 利用举例线索：
   "Herbivores such as cows, horses, and deer..."
   → such as后的例子揭示上位词含义

3. 利用对比线索：
   "Unlike the extrovert, the introvert..."
   → unlike/but/however提示反义

4. 利用语法结构：
   "The students were ebullient; they were jumping for joy."
   → 分号连接的说明句

5. 利用词根词缀：
   → mis- + anthrope（人）= misanthrope（厌世者）
```

### 考研英语一 vs 英语二

| 特征 | 英语一（学术硕士） | 英语二（专业硕士） |
|------|-----------------|-----------------|
| **难度** | 更难，词汇更深，逻辑更复杂 | 相对较易，贴近实际应用 |
| **阅读** | 长篇学术文章，推断题多 | 略短，信息题多 |
| **写作** | 小作文：图表/书信；大作文：议论文 | 小作文：图表；大作文：应用文 |
| **翻译** | 英译汉（学术长句） | 英译汉（相对简单） |
| **完形填空** | 考逻辑和语境 | 同上但略简单 |
| **报考人群** | 985/211等学术型硕士 | MBA/法学/教育等专业硕士 |

**考研英语翻译专项策略**：

```
长难句拆分步骤：

1. 找主干（主语+谓语+宾语）
2. 识别从句（定语从句/状语从句/名词性从句）
3. 识别分词结构（过去分词/现在分词短语）
4. 按中文习惯调整语序（定语前置、原因在前等）

例题：
"The same dramatic technological changes that have provided
millions with leisure, education and unprecedented control
over their own lives have also given a small group of
irresponsible and often socially inadequate individuals
the means to do great damage."

主干：The same changes have given individuals the means to do damage
前置定语从句：that have provided millions with leisure, education and unprecedented control
插入成分：often socially inadequate
处理：那些为数百万人提供了休闲、教育和前所未有的生活自主权的技术变革，
      同样也给一小撮不负责任、往往缺乏社会适应能力的人提供了造成巨大破坏的手段。
```

**完形填空解题策略**：

```
四六级/考研完形填空的四类考查点：

1. 逻辑关系词（最高频）：
   转折：however, nevertheless, yet, despite, while
   递进：moreover, furthermore, in addition
   因果：therefore, thus, consequently, as a result

2. 固定搭配：
   attach importance to（重视）
   give rise to（引起）
   in terms of（就……而言）

3. 上下文语义：
   选择与上下文语义最协调的词，不是最漂亮的词

4. 语法结构：
   空格处需要什么词性？（动词？名词？形容词？）
   时态是否一致？
```

## 32.6 通用应试技巧

### 时间管理矩阵

```
各题型时间投入产出比（以雅思为例）：

高产出（必须保证时间）：
- 写作Task 2（40分钟）→ 占总分比例最大
- 听力填空题（答案明确，不易争议）
- 阅读判断题（逻辑清晰后效率高）

中产出（适量时间）：
- 写作Task 1（20分钟）
- 阅读信息题

低产出/高耗时（控制时间）：
- 阅读中的难题 → 超过2分钟未解决，标记跳过，最后回填
- 不确定的判断题 → 凭直觉作答，继续前进

黄金规则：
"Never leave a question blank." → 所有考试均无倒扣分
→ 哪怕猜，选最有可能的，空白一定是0分
```

### 猜题与排除法

```
四选一题型的猜题策略：

步骤1：排除明显错误的选项
- 包含绝对化词语的往往是陷阱：always, never, all, none
  （除非原文有对应的绝对化表述）
- 与原文意思相反的选项

步骤2：排除"过度推断"的选项
- 选项信息超出原文给出信息的范围 → 通常不选

步骤3：在剩余选项中选择最接近原文语义的
- 注意"同义替换"而非"原词重复"

步骤4：若仍有两个选项难以区分：
- 选内容更"全面"、更"中性"的选项
- 选"范围更小、更具体"的（而不是"过度泛化"的）
```

### 心理调适与考场策略

```
考前一周：
✓ 停止学习新内容，转为复习已掌握的内容
✓ 模拟考场时间规律（与考试同一时间段练习）
✓ 每天保证7-8小时睡眠（睡眠严重影响语言处理效率）
✓ 避免与其他备考者讨论"我感觉好难"类话题（情绪传染）

考试当天：
✓ 提前45分钟到达考场，熟悉环境
✓ 携带必需证件（多检查一次）
✓ 轻食+适量水（不要空腹，不要过饱）
✓ 深呼吸：4秒吸气，7秒屏息，8秒呼气（4-7-8呼吸法）

考场焦虑处理：
✓ "焦虑是精力的一种形式，我可以把它转化为专注。"
✓ 遇到不懂的题 → 标记，跳过，继续前进
   不要在一道题上消耗情绪，影响后续发挥
✓ 听力/阅读遇到困难段落 → 专注于"我能回答什么"，而非"我不知道什么"
```

### 模考与错题分析方法

```
有效模考的五步法：

步骤1：全真模拟
- 严格计时，与真实考试完全相同的条件
- 不查字典，不暂停，不回头更改（模拟真实心理压力）

步骤2：对答案，不要立即分析
- 记录正确率，先了解宏观表现

步骤3：错题三分类
A类（粗心错误）：知道答案但操作失误 → 记录习惯性失误点
B类（知识漏洞）：不认识词汇/语法规则不熟 → 系统学习
C类（技巧问题）：词汇认识但不知如何解题 → 专项技巧训练

步骤4：建立错题本
每道错题：题目 + 错误选项 + 正确选项 + 错误原因 + 规律总结

步骤5：针对性重做
一周后，对B类和C类错题重新做（不看答案）
如果仍然错 → 这是你的核心突破点
```

### 练习一：考试选择决策

根据以下情境，推荐最合适的考试：

1. 小明，大学本科生，目标是申请英国UCL的硕士项目。
2. 小红，已工作3年，目标是申请澳大利亚技术移民。
3. 小李，大四学生，想申请美国Top 30大学的博士项目。
4. 小华，考研党，报考北京大学学术型硕士。

**参考答案**：
1. IELTS学术类（Academic），目标6.5-7.0；2. IELTS通用培训类或PTE Academic，目标6.0-7.0；3. TOEFL iBT（主要）+ GRE（必需），TOEFL目标100+，GRE目标320+；4. 考研英语一。

### 练习二：雅思Task 2审题练习

分析下题的核心争议点，写出你的立场和两个支撑论点：

题目："Some people believe that university education should be free for all students. Others argue that students should pay for their own tuition. Discuss both views and give your own opinion."

**参考分析框架**：
- 免费大学教育论点：促进教育平等、减少社会流动障碍、教育是公共产品
- 收费大学教育论点：学生更珍惜教育资源、减轻政府财政压力、市场竞争提升教学质量
- 平衡立场：根据国家经济条件，对低收入学生提供助学金/贷款制度，实现机会平等而非绝对免费

### 练习三：GRE填空逻辑分析

分析下题的逻辑关系，选出答案：

"The professor was ___; whenever students came to her for advice, she listened attentively and responded with genuine care."

(A) aloof　(B) approachable　(C) imperious　(D) pedantic　(E) reticent

**分析**：
冒号后的说明"attentively listened, genuine care"是正向描述（亲近、关心学生）。
空格需要一个表示"平易近人、愿意帮助"的词。
答案：(B) approachable（平易近人的）
排除：(A) aloof（冷漠的）、(C) imperious（专横的）、(D) pedantic（迂腐的）、(E) reticent（沉默寡言的）均与后文矛盾。

## 本章小结

| 考试 | 核心难点 | 最关键策略 |
|------|----------|-----------|
| IELTS听力 | 同义替换，一次性信息 | 预读题目，预判答案类型 |

| IELTS口语 | Part 3抽象讨论，流利度 | STAR方法，框架化开头 |

| 通用策略 | 时间管理，心理调适 | 先易后难，不留空白，错题分类 |

**最终寄语**：考试分数是敲门砖，真实的英语能力是终身的财富。愿你在考取满意分数之后，继续探索这门语言无穷无尽的魅力——因为语言本身，远比任何考试更广阔、更深邃、更值得投入一生去追求。
