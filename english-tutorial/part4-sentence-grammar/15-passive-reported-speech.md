# 第15章：被动语态与间接引语

> "The passive voice is to be avoided."
> —— William Strunk Jr.（这句话本身就是被动语态，颇具讽刺意味）

被动语态（Passive Voice）是英语语法中最具标志性的结构之一，它的存在并非单纯的语法把戏，而是承载着深刻的语用功能：控制信息的焦点与流向。间接引语（Reported Speech）则是转述他人话语的系统规则，涉及时态、人称、时间地点词的系统性转变。本章深入剖析这两个核心语法点，帮助你在写作和口语中做出准确而有效的选择。

---

## 15.1 被动语态的构成（be + 过去分词）

被动语态的核心公式：

```
被动语态 = be（助动词，随时态变化）+ 过去分词（Past Participle，不变）

主动：The chef   cooked    the meal.
         主语S    谓语V      宾语O

被动：The meal   was cooked   (by the chef).
         主语S    be+PP       动作发出者（可省略）
```

被动语态的基本构成要素：
1. **be 动词**：承担时态、人称、数的变化
2. **过去分词**：保持不变，表示动作
3. **by 短语**（optional）：表明动作发出者（agent），可省略

```
变换规律：
主动句的宾语 → 被动句的主语
主动句的主语 → by 短语（可省略）
```

## 15.2 各时态的被动语态形式

下表系统整理常用时态的主动与被动对照：

| 时态 | 主动形式 | 被动形式 | 例句（被动） |
|------|---------|---------|------------|
| **一般现在时** | writes | is/are written | The report is written every month. |
| **一般过去时** | wrote | was/were written | The letter was written yesterday. |
| **一般将来时** | will write | will be written | The letter will be written tomorrow. |
| **现在进行时** | is writing | is/are being written | The report is being written now. |
| **过去进行时** | was writing | was/were being written | The letter was being written when I arrived. |
| **现在完成时** | has/have written | has/have been written | The report has been written. |
| **过去完成时** | had written | had been written | The letter had been written before noon. |
| **将来完成时** | will have written | will have been written | It will have been completed by Friday. |
| **情态动词现在** | can write | can be written | This can be done easily. |
| **情态动词过去** | could write | could be written | It could have been avoided. |
| **情态动词+完成** | should have written | should have been written | It should have been reported. |

**关键构成规律：**

```
一般时态：  am/is/are/was/were + PP
进行时态：  am/is/are/was/were + being + PP
完成时态：  have/has/had + been + PP
将来时态：  will + be + PP
情态+现在：  modal + be + PP
情态+完成：  modal + have been + PP

记忆口诀：be动词承接时态，been承接完成，being承接进行
```

### 复杂时态例句对照

```
现在进行被动：
Active:  They are building a new bridge.
Passive: A new bridge is being built.

过去完成被动：
Active:  Someone had stolen her wallet before she noticed.
Passive: Her wallet had been stolen before she noticed.

将来完成被动：
Active:  The team will have completed the project by March.
Passive: The project will have been completed by March.

情态+完成被动：
Active:  Someone should have reported this earlier.
Passive: This should have been reported earlier.

双重情态（可能+完成）：
Active:  Someone might have made a mistake.
Passive: A mistake might have been made.
```

## 15.3 被动语态的使用场景

选择主动还是被动，本质上是**修辞与信息结构**的选择，而非单纯语法问题。

### 15.3.1 动作发出者未知或不重要

```
My car was broken into last night.
（不知道是谁闯入的；who 不重要）

The ancient temple was built around 300 BC.
（建造者不可考；who 不重要）

Three people were injured in the accident.
（强调受害者，而非事故原因）

English is spoken in over 50 countries.
（谁说不重要；事实本身是焦点）
```

### 15.3.2 强调动作承受者（信息焦点控制）

**英语句子末尾位置（end-focus）是信息焦点所在**：

```
主动（焦点在 the solution）：
The research team discovered the solution.

被动（焦点在 the research team）：
The solution was discovered by the research team.
（将 the research team 移至末尾，强调是研究团队的发现）

信息流原则（Given → New）：
Previously unknown, the compound was synthesized by Dr. Chen in 2019.
（compound 是已知信息作主语，Dr. Chen 是新信息放末尾）
```

### 15.3.3 科学/学术写作中的客观表达

学术写作传统上偏爱被动语态，以彰显客观性，避免过多的人称主语：

```
实验描述（客观、可重复）：
The samples were prepared at room temperature.
The data were collected over a period of six months.
The variables were controlled for age and gender.

结论表述：
It was found that the treatment was effective.
It has been demonstrated that...
It is widely accepted that...
It should be noted that...

方法描述：
Participants were randomly assigned to two groups.
The questionnaire was administered online.
Statistical analyses were performed using SPSS.
```

### 15.3.4 新闻报道中的被动

新闻被动（Journalistic Passive）常用于以下场景：

```
引用未经证实的消息（被动规避责任）：
The suspect is reported to have fled the country.
The minister is alleged to have accepted bribes.
Three people are said to have been killed.
The policy is believed to be under review.

报道事件（焦点在事件，不在行为者）：
The building was demolished yesterday.
Fifteen people were arrested during the protest.
The treaty was signed in Geneva on Monday.
```

## 15.4 不能用被动语态的情况

并非所有动词都可以构成被动语态。以下情况不能使用被动：

### 15.4.1 不及物动词（Intransitive Verbs）

不及物动词没有宾语，因此无法变为被动（没有宾语成为被动主语）：

```
不及物动词（无被动）：
arrive, appear, disappear, happen, occur, exist, emerge, rise, fall,
sleep, sit, stand, lie, smile, laugh, cry, die, grow (不及物义)

❌ He was arrived at 8 o'clock.
✓  He arrived at 8 o'clock.

❌ The accident was happened yesterday.
✓  The accident happened yesterday.

❌ She was smiled.
✓  She smiled.

注意：有些动词既可及物又可不及物，及物用法可被动：
grow（种植，及物）：Vegetables are grown here.（可以被动）
grow（生长，不及物）：Plants grow here.（不能被动）
```

### 15.4.2 系动词（Linking Verbs）

系动词后接表语，无宾语，不能被动：

```
be, seem, appear, become, feel, look, sound, smell, taste, remain,
stay, keep（系动词用法）

❌ She was seemed tired.
✓  She seemed tired.

❌ The soup was tasted delicious.
✓  The soup tasted delicious.
```

### 15.4.3 反身动词与对等关系动词

```
❌ The two countries were resembled each other.
✓  The two countries resembled each other.

❌ John was married Mary.（marry 作"娶/嫁"时不用被动，但"结婚"状态可用）
✓  John married Mary.
✓  They were married in Paris.（状态被动，可以）
```

### 15.4.4 某些固定表达中的动词

```
have（拥有）：❌ A car was had by him.
cost（花费）：❌ A lot of money was cost by the project.
lack（缺乏）：❌ Funds were lacked by the project.
fit（适合）：❌ The description was fit by him.
suit（适合）：❌ The role was suited by her.
```

## 15.5 被动语态的特殊结构

### 15.5.1 get + 过去分词（口语被动）

在口语和非正式文体中，get 常代替 be 构成被动，通常暗示**意外、负面、或动态变化**：

```
get + 过去分词：

She got fired last week.（被解雇，带有意外/倒霉含义）
He got arrested for speeding.（被逮捕）
The window got broken.（窗户被打破了，暗示意外）
They got married in June.（结婚了，动态变化）
Don't get caught!（别被抓住）
I got confused by the instructions.（被搞糊涂了）

be + PP vs get + PP 对比：
She was promoted.（被晋升，中性陈述状态）
She got promoted.（被晋升了！带有积极情绪/变化感）

The vase was broken.（花瓶是碎的——描述状态）
The vase got broken.（花瓶被打破了——描述事件/意外）
```

### 15.5.2 have something done（使役被动 / 经历被动）

**have + 宾语 + 过去分词**有两层含义：

**含义一：使役（让别人做某事）**
```
I had my hair cut yesterday.（我让人剪了头发——花钱请人做）
She had her car repaired.（她让人修了车）
They had the house painted.（他们让人粉刷了房子）
I need to have my eyes checked.（我需要去检查眼睛）

比较：
I cut my hair.（我自己剪的）
I had my hair cut.（让别人帮我剪的）
```

**含义二：经历（遭受某事，通常是负面的）**
```
She had her wallet stolen on the bus.（她的钱包在公车上被偷了——遭遇）
He had his leg broken in the accident.（他腿在事故中断了）
They had their house broken into.（他们家被闯入了）
```

**get something done（比 have 更口语化）**
```
I need to get my teeth checked.（我需要检查牙齿）
He got his car stolen.（他的车被偷了）
```

### 15.5.3 双宾语动词的两种被动形式

双宾语动词（give, send, show, tell, offer, teach, lend, pay, award 等）有两种主动结构，对应两种被动结构：

```
主动（以 give 为例）：
They gave him a prize.（SVOO：IO = him, DO = a prize）
They gave a prize to him.（SVO + 介词短语）

被动形式一（间接宾语作主语）：
He was given a prize.（更常见）

被动形式二（直接宾语作主语）：
A prize was given to him.（保留 to）

更多示例：
主动：She showed us the photos.
被动一：We were shown the photos.（更自然）
被动二：The photos were shown to us.

主动：The company offered him a job.
被动一：He was offered a job.（更常用）
被动二：A job was offered to him.
```

## 15.6 主动与被动的选择策略

### 何时用被动

```
✓ 动作发出者未知：A mistake was made.
✓ 动作发出者不重要：The package has been delivered.
✓ 强调受事者：The prime minister was assassinated.
✓ 学术/科学写作中保持客观：The experiment was conducted...
✓ 新闻报道引用未证实信息：The CEO is alleged to have...
✓ 礼貌地表达批评（避免直接指责）：
  Errors have been made in this report.（比 You made errors 更婉转）
✓ 信息流控制：将重要/新信息置于句末
```

### 过度使用被动的问题

```
过度使用被动的文章会显得：
1. 冗长（be + PP 比主动动词多一个词）
2. 逃避责任（谁做的？不清楚）
3. 僵硬、死板（academic hedging 走向极端）
4. 不够直接有力

对比：
被动（弱）：Mistakes were made. Lessons have been learned.
主动（强）：We made mistakes. We learned from them.

被动（繁）：It has been decided by the committee that the proposal 
           will be rejected.
主动（简）：The committee has decided to reject the proposal.
```

## 15.7 直接引语变间接引语（Reported Speech）

间接引语（Reported Speech / Indirect Speech）是转述他人话语的语法结构，区别于直接引用原话的直接引语（Direct Speech）。

```
直接引语：He said, "I am tired."
（原话原貌，用引号）

间接引语：He said (that) he was tired.
（转述，无引号，时态/人称/时间词改变）
```

### 15.7.1 时态后退规则（Backshift of Tenses）

当引述动词（reporting verb）用**过去时**时，被引述的内容中的时态通常需要**后退一个时态**（backshift）：

| 直接引语时态 | 间接引语时态 |
|------------|------------|
| 一般现在时（am/is/are） | 一般过去时（was/were） |
| 现在进行时（is doing） | 过去进行时（was doing） |
| 现在完成时（has done） | 过去完成时（had done） |
| 一般过去时（did） | 过去完成时（had done） |
| 将来时（will do） | 过去将来时（would do） |
| 过去进行时（was doing） | 过去完成进行时（had been doing） |
| 情态 can | could |
| 情态 may | might |
| 情态 shall | should/would |
| 情态 must（表义务） | had to |

```
系统示例：

"I work here."
→ She said (that) she worked there.

"I am working on a project."
→ He said he was working on a project.

"I have finished the report."
→ She said she had finished the report.

"I finished it yesterday."
→ He said he had finished it the day before.

"I will call you tomorrow."
→ She said she would call me the next day.

"You can leave now."
→ He said I could leave then.

"You must submit the form."
→ She said I had to submit the form.

"It may rain later."
→ He said it might rain later.
```

### 15.7.2 人称、时间、地点词的变化

除时态外，以下词语也需要相应调整：

**人称代词变化（根据说话者关系调整）：**

```
"I love you," she told him.
→ She told him (that) she loved him.
（I → she；you → him）

"We are leaving," they announced.
→ They announced (that) they were leaving.
（we → they）
```

**时间词变化：**

| 直接引语 | 间接引语 |
|---------|---------|
| now | then / at that moment |
| today | that day |
| yesterday | the day before / the previous day |
| tomorrow | the next day / the following day |
| last week | the previous week / the week before |
| next year | the following year |
| ago | before |
| this | that |
| these | those |

**地点词变化：**

| 直接引语 | 间接引语 |
|---------|---------|
| here | there |
| this place | that place |
| come | go |
| bring | take |

```
综合示例：
"I'll meet you here tomorrow," she said.
→ She said she would meet me there the next day.

"I bought this car here last week," he told us.
→ He told us he had bought that car there the previous week.
```

### 15.7.3 不同句型的间接引语转换

**陈述句的转述：**

```
引述动词 + that 从句（that 可省略）：
"The meeting is cancelled," he announced.
→ He announced (that) the meeting was cancelled.

"She doesn't know the answer," he said.
→ He said (that) she didn't know the answer.
```

**一般疑问句的转述：**

```
引述动词 + whether/if + 宾语从句（陈述句语序）：
"Are you ready?" she asked.
→ She asked if/whether I was ready.

"Have you seen my keys?" he asked.
→ He asked if/whether I had seen his keys.

"Will you help me?" she asked him.
→ She asked him if/whether he would help her.

注意：疑问句语序 → 陈述句语序（主语 + 动词，不倒装）
❌ She asked if was I ready.
✓  She asked if I was ready.
```

**特殊疑问句的转述：**

```
引述动词 + 疑问词 + 宾语从句（陈述句语序）：
"What time is it?" she asked.
→ She asked what time it was.

"Where do you live?" he asked her.
→ He asked her where she lived.

"Why did you leave so early?" she asked him.
→ She asked him why he had left so early.

"How long have you been waiting?" he asked.
→ He asked how long I had been waiting.

注意语序：
❌ She asked where did I live.
✓  She asked where I lived.（疑问语序→陈述语序）
```

**祈使句的转述：**

```
引述动词（tell/ask/order/warn/advise/beg/remind/urge等）
+ 宾语 + to do / not to do

命令（order/tell + to do）：
"Open the window!" he said to her.
→ He told/ordered her to open the window.

"Don't touch that!" she said to the child.
→ She told/warned the child not to touch that.

请求（ask/beg + to do）：
"Please help me," he said to her.
→ He asked/begged her to help him.

建议（advise + to do）：
"You should see a doctor," she said to him.
→ She advised him to see a doctor.

警告（warn + not to do）：
"Don't go out alone at night," he warned her.
→ He warned her not to go out alone at night.
```

### 15.7.4 不需要时态后退的例外情况

以下情况转述时时态**不需要后退**：

**1. 引述动词用现在时/将来时：**
```
She says (that) she is busy.（现在时引述 → 不后退）
He will say (that) he didn't do it.
```

**2. 所述内容仍为当前真实情况（普遍真理、科学事实）：**
```
He told me that the sun rises in the east.（普遍真理，不后退）
She said that water boils at 100 degrees Celsius.
The teacher explained that light travels faster than sound.
```

**3. 所述内容与现在仍然相关/仍然为真：**
```
She said she lives in Paris.（她现在仍住在巴黎）
He told me that he loves jazz.（他现在仍然喜欢爵士乐）
（若强调过去时刻的陈述：He said he loved jazz.）
```

**4. 假设性/虚拟语气句子（不后退以避免混乱）：**
```
He said, "If I were rich, I would travel."
→ He said that if he were rich, he would travel.（were 和 would 保持不变）

She said, "I wish I had more time."
→ She said she wished she had more time.
```

## 15.8 转述动词的丰富表达

仅用 say/tell 转述显得单调，英语有大量**转述动词（reporting verbs）**可以更精确地表达说话者的态度、意图和说话方式。

### 转述动词语法模式分类

| 语法结构 | 转述动词 | 示例 |
|---------|---------|------|
| V + that 从句 | say, state, claim, argue, insist, admit, deny, suggest, mention, explain, complain, promise, warn, note, observe, point out | He claimed that he was innocent. |
| V + sb + that 从句 | tell, remind, warn, convince, assure, inform, notify | She told him that the flight was delayed. |
| V + to do | offer, promise, threaten, refuse, agree, decide, claim | She promised to return the book. |
| V + sb + to do | tell, ask, order, warn, advise, urge, beg, instruct, encourage, remind, convince, persuade | He advised her to take a break. |
| V + doing | admit, deny, suggest, recommend, regret | He admitted stealing the money. |
| V + (that) + 虚拟 | suggest, recommend, insist, demand, propose, urge, request | She insisted (that) he (should) apologize. |
| V + sb + of sth | accuse, remind, warn, inform, notify | He accused her of lying. |

### 转述动词语义分组

**陈述类（Stating）：**
```
say（中性）→ state（正式）→ declare（郑重宣布）→ announce（公开宣布）
→ maintain（坚持认为）→ mention（提及）→ note（指出）→ observe（观察到）
→ point out（指出）→ remark（评论）→ report（报告）→ indicate（表明）
```

**主张/辩论类（Arguing）：**
```
claim（声称，含质疑）→ argue（论证）→ contend（争辩）→ assert（断言）
→ insist（坚持）→ maintain（坚持认为）→ allege（声称，含怀疑）→ suggest（暗示）
```

**承认/否认类（Admitting/Denying）：**
```
admit（承认）→ confess（承认，含内疚）→ acknowledge（承认）→ concede（承认让步）
→ deny（否认）→ refuse（拒绝）
```

**建议类（Recommending）：**
```
suggest（建议）→ recommend（推荐）→ propose（提议）→ urge（敦促）→ advise（建议）
→ encourage（鼓励）→ persuade（说服）→ convince（使信服）
```

**情感类（Emotive）：**
```
complain（抱怨）→ protest（抗议）→ boast（自吹）→ brag（吹嘘）→ grumble（抱怨）
→ warn（警告）→ threaten（威胁）→ beg（恳求）→ plead（恳求）→ promise（承诺）
→ reassure（安慰）→ comfort（安慰）→ apologize（道歉）→ regret（遗憾）
```

```
语义精度对比示例：

He said the deadline was tomorrow.（中性转述）
He reminded me that the deadline was tomorrow.（提醒）
He warned me that the deadline was tomorrow.（警告，暗示你要赶快）
He insisted that the deadline was tomorrow.（坚持，暗示别人质疑过）
He confirmed that the deadline was tomorrow.（确认，暗示此前有不确定）
He announced that the deadline was tomorrow.（宣布，正式公开）
He complained that the deadline was tomorrow.（抱怨，暗示截止日期太早）
```

## 综合对比：主动 vs 被动 选择流程图

```
需要表达某个动作/事件
        ↓
动作发出者（施事）重要且已知吗？
   ↙            ↘
  是              否
   ↓               ↓
施事是话题焦点吗？    → 考虑被动
   ↙     ↘
  是       否（受事是焦点）
   ↓              ↓
用主动        用被动（by短语可选）

另外考虑：
- 学术/科学写作 → 倾向被动
- 口语/叙事写作 → 倾向主动
- 礼貌批评 → 被动更婉转
- 新闻未证实消息 → 被动规避责任
```

## 练习

**练习一：主动改被动（注意时态）**

1. Someone has stolen my wallet.
2. They are building a new subway line.
3. The company had fired 200 workers before the merger.
4. You must submit the application before Friday.
5. Somebody should have reported this incident.

**练习二：被动改主动（合理添加主语）**

## 本章小结

| 知识点 | 核心要点 |
|--------|---------|
| 被动构成 | be + PP；时态体现在 be 的变化上 |
| 各时态被动 | 进行：be+being+PP；完成：have+been+PP；情态：modal+be+PP |
| 使用场景 | 施事未知/不重要；强调受事；学术客观性；信息焦点控制 |
| 不能被动 | 不及物动词；系动词；lack/cost/have等特定动词 |
| get + PP | 口语被动，暗示意外/动态变化 |
| have sth done | 使役（让别人做）或经历（遭受） |
| 双宾被动 | 间接宾语或直接宾语均可作被动主语 |
| 时态后退 | 引述动词为过去时 → 从句时态后退一级 |
| 不后退情况 | 引述动词现在时；普遍真理；仍为真的事实；虚拟语气 |
| 时间地点词 | now→then; today→that day; here→there; tomorrow→the next day |
| 疑问句转述 | 疑问语序→陈述语序；一般疑问→whether/if |
| 祈使句转述 | tell/ask/order/warn + 宾语 + to do / not to do |
| 转述动词 | 用语义精确的动词代替 said/told，提升表达精度 |

掌握被动语态和间接引语，你的英语表达将更加灵活、客观，尤其在学术写作和新闻转述中具有重要的实用价值。下一章将进入英语语法中最精妙的领域之一——条件句与虚拟语气。
