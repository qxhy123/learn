# 第13章：简单句与并列句

> "A sentence should contain no unnecessary words, a paragraph no unnecessary sentences, for the same reason that a drawing should have no unnecessary lines and a machine no unnecessary parts."
> —— William Strunk Jr., *The Elements of Style*

语言的基本单位是句子（sentence）。在英语中，无论多么复杂的文章，拆解到最底层，都由最基础的句型构成。本章从最简单的简单句（Simple Sentence）出发，深入剖析其内部结构与变体形式，然后过渡到由两个或多个独立分句组成的并列句（Compound Sentence）。掌握这两种句型，是迈向复杂表达的必经之路。

---

## 13.1 简单句的定义与结构

**简单句**（Simple Sentence）的核心定义：**只含一个独立分句（independent clause），即只有一套主语（Subject）+ 谓语（Predicate）结构的句子**。

简单句并不等于"短句"或"简单的句子"。一个简单句可以有多个修饰语、多个并列主语或多个并列谓语，只要它只包含一个主谓核心，就仍是简单句。

### 英语句子的五大基本句型（Five Basic Sentence Patterns）

| 句型编号 | 模式 | 示例 |
|---------|------|------|
| SV | 主语 + 不及物动词 | Birds fly. |
| SVO | 主语 + 及物动词 + 宾语 | She reads books. |
| SVC | 主语 + 系动词 + 表语 | He seems tired. |
| SVOO | 主语 + 动词 + 间接宾语 + 直接宾语 | She gave him a gift. |
| SVOC | 主语 + 动词 + 宾语 + 宾补 | They named the dog Max. |

### 句子成分详解

```
完整句子分析示例：
The young professor / quickly explained / the complex theorem / to her students.
       S(主语)         V+Adv(谓语+副词)      O(宾语)              IO(间接宾语)
```

**主语（Subject）**：句子的话题，通常是名词、代词、动名词短语或不定式短语。
- *The children* are playing outside.（名词短语）
- *Running every day* keeps me healthy.（动名词短语）
- *To forgive* is divine.（不定式）

**谓语（Predicate）**：说明主语做什么或状态如何，核心是动词（verb）。谓语可以包含助动词、情态动词和主动词。

**宾语（Object）**：
- 直接宾语（Direct Object）：动作的直接承受者
- 间接宾语（Indirect Object）：动作的间接受益/受损者，通常置于直接宾语之前

**表语（Subject Complement / Predicative）**：跟在系动词（be, seem, appear, become, feel, look, smell, sound, taste等）之后，说明主语的性质或状态。

**宾补（Object Complement）**：说明宾语的状态或结果。

```
对比：表语 vs 宾补
He is happy.          → happy 是表语（说明主语 He 的状态）
They made him happy.  → happy 是宾补（说明宾语 him 的状态）
```

### 主语与谓语的一致性（Subject-Verb Agreement）

简单句中最常见的错误之一是主谓不一致。

| 规则 | 示例 |
|------|------|
| 单数主语 + 单数动词 | The dog barks. |
| 复数主语 + 复数动词 | The dogs bark. |
| 不可数名词 + 单数动词 | Water is essential. |
| 不定代词（everyone, nobody等）+ 单数动词 | Everyone knows this. |
| 集合名词（team, family等）取决于语境 | The team is ready. / The team are arguing. |

## 13.2 简单句的变体

简单句并非只有一种面貌。根据交际目的，简单句可以呈现出多种变体形式。

### 13.2.1 省略句（Elliptical Sentences）

省略句是在上下文已知的前提下，删去可恢复的成分，使表达更简洁。

```
完整形式 → 省略形式
"Are you going?" "Yes, I am going." → "Yes, I am."
"Who broke the window?" "Tom broke the window." → "Tom did."
"She works harder than he works hard." → "She works harder than he does."
```

### 13.2.2 倒装句（Inverted Sentences）

**完全倒装**（Full Inversion）：谓语动词完全置于主语之前。

```
常规语序：
The train came around the corner.
A large crowd gathered in the square.

完全倒装：
Around the corner came the train.（方位副词短语置首）
In the square gathered a large crowd.
```

**部分倒装**（Partial Inversion）：仅助动词或情态动词置于主语之前。

```
触发部分倒装的结构：
1. 否定副词/副词短语置首：
   Never have I seen such beauty.（正常：I have never seen...）
   Rarely does he make mistakes.
   Not only does she speak French, but she also...

2. only + 副词短语置首：
   Only then did I understand.
   Only by working hard can you succeed.

3. So/Neither/Nor 表示"也..."：
   "I love jazz." "So do I." / "Neither do I."

4. 条件句中省略 if：
   Were I you, I would leave.（= If I were you...）
   Had I known, I would have helped.（= If I had known...）
   Should you need help, call me.（= If you should need help...）
```

### 13.2.3 感叹句（Exclamatory Sentences）

英语感叹句主要有两种结构：

```
What + (a/an) + 形容词 + 名词 + 主语 + 动词！
What a beautiful day (it is)!
What talented musicians they are!
What nonsense (that is)!（不可数名词，无冠词）

How + 形容词/副词 + 主语 + 动词！
How wonderful (this is)!
How quickly time flies!
How kind of you to help!（固定结构）
```

### 13.2.4 祈使句（Imperative Sentences）

祈使句用于表达命令、请求、建议或邀请，主语 you 通常省略。

```
基本形式：动词原形 + ...
Close the door.
Please sit down.（please 使语气更礼貌）
Don't touch that!（否定祈使）
Never give up.

第一/第三人称祈使（通过 let 实现）：
Let me explain.（第一人称单数）
Let's go!（第一人称复数，= Let us go）
Let him speak.（第三人称，表允许）
Don't let them see this.（否定形式）

强调形式（do + 动词原形）：
Do come in!（表诚恳邀请，带强调意味）
Do be careful!
```

## 13.3 there be 句型深度解析

**there be 句型**是英语中表达"存在"的核心结构，是一种**存在句（Existential Sentence）**。注意：句首的 there 是**引导词（expletive/dummy subject）**，不是副词"那里"；真正的逻辑主语是 be 之后的名词短语。

### 基本结构

```
There + be动词 + 主语（真实主语）+ 地点/时间状语

There is a book on the desk.
There are three students in the room.
```

### 主谓一致规则

there be 句型的动词形式由**紧跟 be 之后的名词**（逻辑主语）决定：

```
There is a cat in the garden.（单数名词 → is）
There are two cats in the garden.（复数名词 → are）
There is some water in the glass.（不可数名词 → is）
There are many problems to solve.（复数名词 → are）

当并列名词作主语时，动词与最近的名词一致（就近原则）：
There is a pen and two books on the desk.（pen 最近 → is）
There are two books and a pen on the desk.（books 最近 → are）
```

### 各时态的 there be 形式

| 时态 | 肯定形式 | 示例 |
|------|---------|------|
| 一般现在时 | There is/are | There are many options. |
| 一般过去时 | There was/were | There were no clouds. |
| 一般将来时 | There will be | There will be a meeting. |
| 现在进行时 | There is/are + being | There is being a party held. (少用) |
| 过去进行时 | There was/were + being | There were being changes made. |
| 现在完成时 | There has/have been | There has been an accident. |
| 过去完成时 | There had been | There had been no warning. |
| 将来完成时 | There will have been | There will have been... |
| 情态动词 | There can/must/might be | There must be a mistake. |
| 不定式 | for there to be | I want there to be peace. |

### there be 的扩展形式

```
there be 可以替换为其他存在动词，表示更具体的存在方式：
There stands a tall tower in the center.（矗立）
There lies a small village beyond the hills.（位于）
There remains one problem to discuss.（仍有）
There exist many unanswered questions.（存在）
There followed a long silence.（随之而来）
There comes a time when...（有时候...）
```

### there be 的常见误用

```
错误：There has a book on the desk.（中文"有"的直译）
正确：There is a book on the desk.

错误：There is many students.
正确：There are many students.

错误：There is a lot of problems.
正确：There are a lot of problems.
（a lot of + 复数可数名词 → are）
```

## 13.4 it 的特殊用法

it 在英语中除了作普通的人称代词（指代具体事物）之外，还有三种重要的特殊用法，这三种用法中的 it 都是**形式性的（formal/dummy）**，没有实际指代意义。

### 13.4.1 形式主语（Formal Subject / Anticipatory Subject）

当真正的主语是不定式短语、动名词短语或从句时，通常将真正主语后移，用 it 作形式主语占据句首位置，使句子更加平衡自然。

```
真实主语（后置）                     形式主语（前置）
To learn a language is difficult.  → It is difficult to learn a language.
That he lied shocked us.           → It shocked us that he lied.
Telling lies is wrong.             → It is wrong to tell lies.（动名词少用后置）
```

**常见句型模板：**

```
It is + 形容词 + to do sth.
It is important to exercise regularly.
It is difficult to understand his behavior.

It is + 形容词 + that 从句
It is obvious that he is lying.
It is essential that you (should) be on time.

It takes + 时间/精力 + to do sth.
It takes years to master a language.
It took me three hours to finish the report.

It seems/appears/happens/turns out + that...
It seems that she is not coming.
It turned out that we were both wrong.
```

### 13.4.2 形式宾语（Formal Object）

当宾语是不定式短语或从句，且句中还有宾语补足语时，将真正宾语后置，用 it 作形式宾语。

```
结构：主语 + 动词 + it + 形容词/名词 + 真正宾语（不定式/从句）

I find it difficult to wake up early.
（= I find to wake up early difficult. ← 笨拙）

She made it clear that she was unhappy.
They consider it wrong to cheat.
He thought it strange that no one called.

常见动词：find, make, think, consider, feel, deem, regard, keep
```

### 13.4.3 强调句型（Cleft Sentence / It-cleft）

强调句型是将句子中某个成分（主语、宾语、状语等）突出强调的结构。

**基本模式：It is/was + 被强调成分 + that/who/whom + 句子其余部分**

```
原句：Tom broke the window yesterday.

强调主语（Tom）：
It was Tom who/that broke the window yesterday.

强调宾语（the window）：
It was the window that Tom broke yesterday.

强调时间状语（yesterday）：
It was yesterday that Tom broke the window.

强调地点状语（here）：
It was here that the accident happened.
```

**强调句的判断方法：**

```
测试：去掉 "It is/was...that" 后，剩余部分能否构成完整句子？

It was Tom who broke the window.
去掉强调结构 → "Tom broke the window." ✓ 完整 → 这是强调句

It was a dark night that she dreaded.
去掉 → "She dreaded a dark night." ✓ 这也是强调句

对比—非强调句（it 作形式主语）：
It is true that he resigned.
去掉 → "True he resigned." ✗ 不完整 → 这是形式主语
```

**强调句的否定与疑问：**

```
否定：It was not until midnight that she arrived.
疑问：Was it Tom that broke the window?
Was it yesterday that you met her?
特殊疑问：Who was it that called you?
When was it that you first noticed?
```

## 13.5 并列句（Compound Sentences）的构成

**并列句**由两个或两个以上**独立分句（independent clauses）**组成，各分句在语法上地位平等，通过特定手段连接。并列句有三种主要连接方式。

### 13.5.1 并列连词 FANBOYS 的精确语义差异

FANBOYS 是七个并列连词的首字母缩写，这是英语写作中最基础也最重要的连接工具之一。

| 连词 | 语义功能 | 典型用法 | 误用警示 |
|------|---------|---------|---------|
| **F**or | 表原因（较正式，类似 because） | She was cold, for the window was open. | 不置于句首；书面语 |
| **A**nd | 表添加、递进、结果 | He studied hard, and he passed. | 口语过度使用 |
| **N**or | 表否定添加 | He can't swim, nor can he dance. | 后接倒装句 |
| **B**ut | 表对比、转折 | It rained, but we went out anyway. | 不等于 however |
| **O**r | 表选择、否则 | Hurry, or you'll miss the bus. | 也可表警告 |
| **Y**et | 表让步转折（比 but 语气强） | It was hard, yet she succeeded. | 比 but 更文学化 |
| **S**o | 表结果 | It was late, so we left. | 仅表结果，不表目的 |

**深度辨析：for vs because**

```
for（并列连词，连接两个独立分句，表推断的原因）：
He must be ill, for he looks pale.
（说话者从"他脸色苍白"推断他生病了）

because（从属连词，引导原因状语从句，表直接原因）：
He was absent because he was ill.
（直接原因）

关键区别：
- for 引出的原因是"说话者的推断/判断"，不一定是客观直接原因
- for 引出的分句不能置于主句之前
- because 引出的是直接、明确的原因
```

**nor 的特殊语法：**

```
nor 连接的第二个分句必须倒装：
She didn't cry, nor did she complain.（倒装：did she complain）
He can't speak French, nor can he speak Spanish.
I won't give up, nor will I compromise.

对比 neither...nor（关联连词，不需倒装）：
Neither she nor he was present.（主谓一致遵循就近原则）
```

**or 的多重语义：**

```
表选择（Choice）：
You can take the bus or the subway.

表等同/解释（Equivalence，即"换言之"）：
Biology, or the study of living organisms, fascinates her.

表否则（Consequence）：
Work hard, or you'll fail.（= Work hard; otherwise, you'll fail.）
```

### 13.5.2 分号连接并列句

分号（semicolon ; ）是比逗号强、比句号弱的标点，用于连接**意义紧密相关的两个独立分句**，表明两者之间存在紧密联系（对比、补充、因果等）。

```
基本用法：
Independent clause; independent clause.

The sun was setting; the sky turned crimson.
He was exhausted; he had worked for eighteen hours straight.
She loves classical music; he prefers jazz.（对比关系不言而喻）
```

**分号 vs 逗号 vs 句号：**

```
逗号（弱）：
The weather was nice, and we went for a walk.（需要连词）

分号（中）：
The weather was nice; we went for a walk.（省略连词，意义自明）

句号（强分隔）：
The weather was nice. We went for a walk.（两个完全独立的句子）

原则：分号暗示读者"这两件事高度相关"，比两个句号的分隔感更弱
```

### 13.5.3 连接副词（Conjunctive Adverbs）与标点搭配

连接副词（也称"过渡词"transition words）不是连词，**不能单独用逗号连接两个独立分句**，必须与分号或句号搭配使用。

**规范格式：**
```
独立分句; 连接副词, 独立分句.
独立分句. 连接副词, 独立分句.
```

| 语义关系 | 常用连接副词 | 示例 |
|---------|------------|------|
| 对比/转折 | however, nevertheless, nonetheless, on the other hand, in contrast | He worked hard; however, he failed. |
| 因果/结论 | therefore, thus, consequently, as a result, hence | She studied diligently; therefore, she succeeded. |
| 递进/补充 | moreover, furthermore, in addition, additionally, besides | The plan is costly; furthermore, it is risky. |
| 举例 | for example, for instance, namely, that is | I love sports; for example, I play tennis daily. |
| 时间顺序 | then, subsequently, meanwhile, afterward | She finished the report; then, she left the office. |
| 总结 | in conclusion, in summary, in short, overall | The evidence is mixed; in short, we cannot be sure. |

```
重要警示——逗号拼接错误：
❌ He was tired, however, he continued working.
✓  He was tired; however, he continued working.
✓  He was tired. However, he continued working.

❌ The data was incomplete, therefore the results are unreliable.
✓  The data was incomplete; therefore, the results are unreliable.
```

## 13.6 并列结构的平行原则（Parallelism）

**平行原则（Parallelism / Parallel Structure）**是英语写作中最重要的语法美学原则之一：**并列的成分必须在语法形式上保持一致**。

### 平行原则的适用范围

**并列连词连接的成分必须平行：**

```
❌ She likes hiking, to swim, and dancing.
   （动名词 + 不定式 + 动名词，形式不一致）
✓  She likes hiking, swimming, and dancing.
✓  She likes to hike, to swim, and to dance.

❌ The job requires dedication, you must be creative, and having patience.
✓  The job requires dedication, creativity, and patience.（全为名词）
✓  The job requires being dedicated, creative, and patient.（全为形容词）
```

**关联连词（Correlative Conjunctions）强制平行：**

```
both...and, either...or, neither...nor, not only...but also,
whether...or, as...as, more...than

❌ She is both intelligent and has creativity.
✓  She is both intelligent and creative.（形容词 + 形容词）

❌ He not only speaks English but also French is spoken by him.
✓  He not only speaks English but also speaks French.（动词短语 + 动词短语）

❌ Either you leave now or staying is your option.
✓  Either you leave now or you stay.（主谓结构 + 主谓结构）
```

**列举三项及以上时（the rule of three）：**

```
❌ The politician promised tax cuts, creating more jobs, and that he would
   improve education.
✓  The politician promised tax cuts, job creation, and educational improvement.
   （全为名词短语）
✓  The politician promised to cut taxes, to create more jobs, and to improve
   education.（全为不定式短语）
```

**平行结构中重复 vs 省略的策略：**

```
清晰起见，并列的第一项之后，后续项中重复的成分可以省略：
She can speak French, (can speak) German, and (can speak) Spanish.
→ She can speak French, German, and Spanish.

但当省略会造成歧义时，应保留：
I like him better than her.（歧义：我比她更喜欢他？还是我喜欢他胜过她喜欢他？）
I like him better than I like her.（清晰）
```

## 13.7 逗号拼接错误（Comma Splice）与流水句（Run-on Sentence）

这是英语写作中最常见、也最容易失分的语法错误之一，尤其对于以中文为母语的学习者。

### 13.7.1 逗号拼接错误（Comma Splice）

**定义**：仅用逗号（而不用连词或其他适当标点）连接两个独立分句。

```
错误示例：
❌ I was hungry, I ate a sandwich.
❌ The experiment failed, the scientists were disappointed.
❌ She studied all night, she passed the exam.
❌ It was raining, we decided to stay inside.
```

**修正方法（四种）：**

```
方法一：加入并列连词（FANBOYS）
✓  I was hungry, so I ate a sandwich.
✓  The experiment failed, and the scientists were disappointed.

方法二：改用分号
✓  I was hungry; I ate a sandwich.
✓  The experiment failed; the scientists were disappointed.

方法三：改用分号 + 连接副词
✓  The experiment failed; consequently, the scientists were disappointed.

方法四：改为两个独立句子
✓  I was hungry. I ate a sandwich.
✓  The experiment failed. The scientists were disappointed.

方法五（进阶）：改为复合句，表明逻辑关系
✓  Because I was hungry, I ate a sandwich.
✓  Since the experiment failed, the scientists were disappointed.
```

### 13.7.2 流水句（Run-on Sentence / Fused Sentence）

**定义**：两个独立分句之间没有任何标点或连词，直接连在一起。

```
错误示例：
❌ It was dark outside we turned on the lights.
❌ The meeting ended everyone left the room.
❌ She is a talented artist her paintings have won many awards.
```

修正方法与逗号拼接相同，关键是要**明确分句之间的逻辑关系**，选择最能表达该关系的连接手段。

### 13.7.3 中文母语者的特殊陷阱

中文习惯用逗号分隔大量分句（流水句风格），直接迁移到英文写作中会产生大量逗号拼接错误：

```
中文：我昨天去了图书馆，借了几本书，然后回家看了一下午。
（中文中这是完全合法的流水句）

错误英译：
❌ I went to the library yesterday, I borrowed some books, then I read at
   home all afternoon.

正确选项：
✓  I went to the library yesterday, borrowed some books, and then read at
   home all afternoon.
   （单一主语 I，三个并列谓语动词：went, borrowed, read）

✓  Yesterday I went to the library and borrowed some books; afterward, I
   read at home all afternoon.
   （分号分隔两个关系紧密的分句）
```

## 13.8 综合对比：三种句型的选择策略

| 情况 | 推荐句型 | 示例 |
|------|---------|------|
| 两个想法对等、地位相当 | 并列句（并列连词/分号） | She sings, and he dances. |
| 两个想法关系紧密，一方更重要 | 复合句（下章）| Because she practiced, she improved. |
| 强调某一成分 | 强调句（it-cleft） | It was she who solved the problem. |
| 表示存在 | there be 句型 | There are three options. |
| 想法之间有强逻辑联系需过渡 | 分号 + 连接副词 | Results varied; therefore, more research is needed. |

## 练习

**练习一：识别并改正以下错误（每句指出错误类型并提供修正）**

1. The concert was amazing, I bought a CD afterward.
2. Neither the manager or the staff were informed.
3. She enjoys to run, swimming, and to cycle.
4. There was twenty students in the classroom.
5. It is him who made the decision.
6. He studied hard, however he failed the exam.
7. Do not be fooled, it looks simple but it isn't.

**练习二：用适当的 FANBOYS 或连接副词连接以下句子对，并解释你的选择**

1. The report is due tomorrow. / I haven't started it yet.
2. The new policy saves money. / It reduces employee morale.
3. She speaks three languages. / She has lived in many countries.
4. The project was delayed. / The team worked overtime to meet the deadline.

**练习三：强调句变换**

将以下句子改写为强调括号内成分的强调句：
1. [My sister] won the first prize in the competition.
2. I met him [at the airport] last Friday.
3. She realized her mistake [after the meeting].

**练习四：there be 句型时态变换**

用括号中的时态改写：
1. There is a park near my house.（一般将来时）
2. There were many problems.（现在完成时）
3. There will be a solution.（一定有解决办法——情态动词 must）

## 本章小结

| 知识点 | 核心要点 |
|--------|---------|
| 简单句 | 一套主谓结构；五大句型（SV/SVO/SVC/SVOO/SVOC） |
| 简单句变体 | 倒装（否定副词/only/条件句省略if）；感叹（What/How）；祈使（动词原形/let） |
| there be | 存在句；动词与紧随其后的名词一致；可替换为 stand/lie/remain 等 |
| it 的特殊用法 | 形式主语（It is...to/that）；形式宾语（find it...to）；强调句（It is...that/who） |
| 并列句 | FANBOYS 七大连词各有精确语义；分号连接紧密分句；连接副词须与分号/句号搭配 |
| 平行原则 | 并列成分语法形式须一致；关联连词强制要求平行 |
| 常见错误 | 逗号拼接（comma splice）；流水句（run-on）；中文式流水句迁移 |

掌握本章内容，你将能够准确构建各种简单句和并列句，并在写作中做出正确的标点和连接词选择。下一章将进入更复杂的从句世界，学习如何用从属关系表达更精细的逻辑。
