# 第23章：学术写作与论文

> "Academic writing is not about sounding smart. It is about being precise enough that the reader can verify, challenge, or build upon your thinking."
> — Helen Sword, *Stylish Academic Writing*

---

## 引言

学术写作（Academic Writing）是高等教育中最重要的核心技能之一，也是中国学生赴英美留学后面临的最大挑战。许多学生发现，即便英语成绩优异，真正进入大学后仍会在论文写作上举步维艰——不是因为词汇量不够，而是因为不理解学术写作的**思维方式**：如何构建论证、如何正确引用、如何保持客观而又不失立场、如何让文章既精确又流畅。

本章将系统讲解学术写作的核心原则、语言特点、引用规范、研究论文写作流程，以及摘要和文献综述的专项写法。无论你是准备写大学论文、申请研究生，还是在国际期刊发表，本章都将为你建立坚实的学术写作基础。

## 23.1 学术写作的核心原则

学术写作建立在四大核心原则之上。违反任何一条，论文的可信度和学术价值都会受损。

### 原则一：客观性（Objectivity）

学术写作必须基于证据，而非个人感受或偏见。这不意味着没有立场，而是立场必须由证据支撑。

```
❌ 主观：This is clearly the best approach to solving poverty.
✓ 客观：Among the interventions reviewed, conditional cash transfers
        demonstrated the highest effect size (d = 0.68) in reducing
        child malnutrition across the five studies that met inclusion criteria.
```

### 原则二：精确性（Precision）

每个词都必须准确传达确切含义。模糊的词汇会削弱论证的力度。

```
❌ 模糊：The results were quite good.
✓ 精确：Test scores improved by a statistically significant margin
        (p < 0.05), with an average increase of 12.4 percentage points.
```

### 原则三：正式性（Formality）

学术写作使用正式书面语域，避免口语、缩写、俚语和过于主观的感叹。

```
❌ 非正式：It's kind of obvious that climate change is a huge deal.
✓ 正式：The evidence strongly indicates that anthropogenic climate change
        represents one of the most significant challenges of the 21st century.
```

### 原则四：证据支撑（Evidence-Based Reasoning）

所有论点必须有可追溯的证据支持。个人意见需要与有据可查的知识明确区分。

## 23.2 学术写作的语言特点

### 正式语域选择

学术写作要求使用正式词汇，避免口语化表达。以下是100+常见的正式词汇替换：

| 口语/非正式 | 学术/正式 | 语义细节 |
|---|---|---|
| use | utilize / employ / apply | utilize强调充分利用，employ强调有目的地使用 |
| show | demonstrate / indicate / reveal / suggest | reveal含有揭露含义，suggest语气更谨慎 |
| talk about | discuss / examine / investigate / address | examine含有分析之意 |
| find out | determine / ascertain / establish | ascertain更强调核实 |
| look at | examine / analyze / assess / evaluate | evaluate含有判断价值之意 |
| big / large | substantial / considerable / significant | significant常用于统计学语境 |
| small | minimal / negligible / marginal | negligible指微乎其微 |
| good | effective / beneficial / advantageous / favorable | 依据语境选择 |
| bad | detrimental / adverse / deleterious / problematic | deleterious专指有害健康 |
| very | considerably / substantially / markedly / notably | 避免simply说"very" |
| get | obtain / acquire / achieve / attain | attain多用于目标达成 |
| need | require / necessitate / demand | necessitate语气最强 |
| think | argue / contend / maintain / propose / hypothesize | 依据确定程度选择 |
| say | assert / claim / state / argue / contend | claim有时含有"尚待证实"的意味 |
| change | alter / modify / transform / shift | transform表根本性变化 |
| important | significant / critical / essential / fundamental | fundamental表最基础的重要性 |
| help | facilitate / assist / support / enable | facilitate表使某事变得更容易 |
| end | conclude / terminate / cease | terminate更正式 |
| start | initiate / commence / undertake / embark upon | commence最正式 |
| make | create / produce / generate / establish | generate常用于结果、数据 |
| check | verify / examine / assess / evaluate | verify强调核实真实性 |
| avoid | refrain from / abstain from / prevent | refrain from含有自我克制之意 |
| happen | occur / take place / arise / emerge | emerge含有逐渐出现之意 |
| deal with | address / tackle / handle / manage | tackle含有主动克服之意 |
| give | provide / offer / supply / present | provide最中性 |
| about | regarding / concerning / pertaining to | pertaining to最正式 |

### 第一人称的使用：不同学科的差异

**传统观点**：学术写作应完全避免第一人称（I, we, my, our）。

**现代实践**：这一规则已大幅松动。不同学科和期刊有不同规范：

| 学科领域 | 第一人称使用规范 |
|---|---|
| 理工科（STEM） | 传统上避免，但"we argue" / "we propose"在实验性论文中常见 |
| 社会科学 | 越来越接受，尤其是质性研究中承认研究者立场的必要性 |
| 人文学科 | 普遍接受，"I argue" / "I contend"在文学、历史、哲学中广泛使用 |
| 医学 | 通常回避，倾向于被动语态和"the authors" |
| 商科 | 因期刊而异，管理学期刊近年趋向接受 |

**推荐策略**：查看目标期刊或课程要求中的范文，以其实际用法为准。

### 对冲语言（Hedging）的恰当使用

对冲语言是学术写作的重要特征，用于表示研究结论的确定程度，避免过度概括。

**对冲的必要性**：大多数学术发现都有其适用边界和局限性。适当的对冲体现学术诚实，而非软弱。

| 对冲强度 | 常用表达 |
|---|---|
| 强（确定性较高）| The data clearly demonstrate... / The evidence strongly suggests... |
| 中（有合理依据）| The findings indicate... / It appears that... / The results suggest... |
| 弱（谨慎推测）| It is possible that... / The data may suggest... / This could imply... |
| 限定范围 | In the context of this study... / Among the sample population... |
| 归因来源 | According to X, ... / X argues that... / X maintains that... |

**过度对冲与对冲不足的对比**：

```
对冲不足（过于武断）：
"Social media causes depression in teenagers."

对冲适当：
"The results of this longitudinal study suggest a significant association
between heavy social media use and depressive symptoms in adolescents aged
13-17, though causal direction requires further investigation."

过度对冲（失去立场）：
"It might perhaps be possible that some social media use could
potentially be associated with certain mood-related outcomes in
some young people under some circumstances."
```

### 名词化（Nominalization）

名词化是指将动词或形容词转化为名词，这是学术写作的重要语言特征，使文本更正式、更抽象、信息密度更高。

| 动词/形容词 | 名词化形式 | 例句 |
|---|---|---|
| analyze | analysis | The **analysis** of variance revealed... |
| investigate | investigation | The **investigation** into gene expression... |
| develop | development | The **development** of new methodologies... |
| implement | implementation | The **implementation** of the policy... |
| conclude | conclusion | The **conclusion** drawn from these findings... |
| evaluate | evaluation | A comprehensive **evaluation** of the data... |
| significantly | significance | The **significance** of these results... |
| accurate | accuracy | The **accuracy** of the measurement... |

**注意**：名词化不应过度使用，否则会导致文章晦涩难读。现代学术写作风格越来越提倡在准确的前提下，使用动作性动词而非过度名词化。

### 50个学术写作高频表达汇总

**引出论点**：
```
This paper argues that...
This study examines/investigates/explores...
It is contended that...
The central thesis of this paper is...
This research seeks to demonstrate that...
```

**描述研究目的**：
```
The purpose of this study is to...
This research aims to / is designed to...
The primary objective of this paper is...
This investigation was undertaken in order to...
```

**引用文献**：
```
According to [Author], ...
As [Author] argues/notes/observes/points out...
[Author] contends/maintains/asserts that...
In [Author]'s view, ...
[Author]'s work demonstrates that...
```

**表示对比**：
```
While X argues that..., this paper maintains that...
In contrast to X's position, ...
Unlike the approach adopted by X, the present study...
This finding stands in contrast to those of...
```

**表示因果**：
```
This can be attributed to...
This is largely a consequence of...
The primary driver of this phenomenon appears to be...
These outcomes are consistent with the hypothesis that...
```

**承认局限**：
```
It should be noted that...
One limitation of this study is...
These findings must be interpreted with caution, as...
The present study does not account for...
Further research is needed to determine whether...
```

**总结/结论**：
```
In summary, this paper has argued that...
The evidence presented here suggests...
These findings have significant implications for...
Future research should investigate...
It is hoped that this study will contribute to...
```

## 23.3 引用与参考文献

### 为什么要引用

学术引用建立在三大支柱之上：

1. **学术诚信（Academic Integrity）**：承认他人的知识产权，避免抄袭
2. **论证支撑（Evidence Support）**：让读者能够核实你的证据来源
3. **学术对话（Scholarly Dialogue）**：将你的研究置于更大的学术脉络中

### 三种引用方式的区别

| 引用方式 | 英文 | 定义 | 何时使用 |
|---|---|---|---|
| 直接引用 | Direct Quote | 原文一字不差地引用，加引号 | 原文措辞本身有分析价值；定义、著名表述 |
| 间接引用 | Paraphrase | 用自己的话重述原文内容，保留全部细节 | 整合信息为自己的论证；原文较长时 |
| 总结 | Summary | 用自己的话概括原文的核心要点，更简洁 | 只需要某段文字的大意；综述多篇文献时 |

**直接引用的规范**：

短引用（少于40词，APA；少于4行，MLA）——行内引用，加引号：
```
Foucault (1977) argues that "visibility is a trap" (p. 200), suggesting
that the Panopticon's power lies not in actual observation but in the
prisoner's internalization of the gaze.
```

长引用（40词以上，APA）——块引用（block quote），独立成段，缩进，不加引号：
```
    According to Bourdieu (1990):
        The body believes in what it plays at: it weeps if it mimes
        grief. It does not represent what it performs, it does not
        memorize the past, it enacts the past, bringing it back to life.
        (p. 73)
    This insight helps explain why habitus is reproduced through
    practice rather than explicit instruction.
```

**改写（Paraphrase）的正确方法**：

不能只替换几个词——这仍然构成抄袭！真正的改写要求你：
1. 理解原文的完整意思
2. 放下原文（不看着它写）
3. 用完全不同的句子结构和词汇重新表达
4. 检查与原文的相似度

```
原文：
"The unconscious is structured like a language." (Lacan, 1966, p. 234)

❌ 错误改写（只换词）：
The unconscious is organized similar to a language. (Lacan, 1966)

✓ 正确改写：
Lacan (1966) proposes that the structure of the unconscious follows
the same organizational principles that govern linguistic systems —
a claim that fundamentally reoriented psychoanalytic theory toward
structural linguistics.
```

### APA 格式基础（第7版）

**文内引用（In-text Citation）**：

```
单一作者：(Smith, 2021) 或 Smith (2021) argues...
两位作者：(Smith & Jones, 2021) 或 Smith and Jones (2021)
三位及以上作者：(Smith et al., 2021)
直接引用必须加页码：(Smith, 2021, p. 45)
```

**参考文献列表格式**：

期刊文章：
```
Author, A. A., & Author, B. B. (Year). Title of article.
    Title of Journal, volume(issue), page–page.
    https://doi.org/xxxxx

示例：
Twenge, J. M., Haidt, J., Joiner, T. E., & Campbell, W. K. (2020).
    Underestimating digital media harm. Nature Human Behaviour, 4(4),
    346–348. https://doi.org/10.1038/s41562-020-0839-4
```

书籍：
```
Author, A. A. (Year). Title of work: Capital letter also for subtitle.
    Publisher.

示例：
Kahneman, D. (2011). Thinking, fast and slow. Farrar, Straus and Giroux.
```

网络资源：
```
Author, A. A. (Year, Month Day). Title of web page. Website Name.
    URL

示例：
World Health Organization. (2023, March 15). Mental health and
    COVID-19: Early evidence of the pandemic's impact. WHO.
    https://www.who.int/...
```

### MLA 格式基础（第9版）

**文内引用**：使用作者姓氏 + 页码（无年份）

```
(Smith 45) 或 Smith argues that "..." (45).
```

**参考书目（Works Cited）**：

期刊文章：
```
Last, First. "Title of Article." Journal Name, vol. #, no. #, Year, pp. ##-##.

示例：
Butler, Judith. "Performative Acts and Gender Constitution." Theatre
    Journal, vol. 40, no. 4, 1988, pp. 519–531.
```

### 如何避免抄袭（Plagiarism）

| 抄袭类型 | 说明 | 预防方法 |
|---|---|---|
| 直接抄袭 | 原文照抄不标注 | 使用引号 + 引用 |
| 改写抄袭 | 换词但结构相同 | 真正理解后重写，不看原文 |
| 自我抄袭 | 提交之前的旧作业 | 每次创作新内容 |
| 马赛克抄袭 | 混合多个来源的句子 | 整合吸收后原创表达 |
| 遗忘引用 | 忘记标注来源 | 研究阶段即做好笔记 |

**实用工具**：Turnitin、Grammarly Plagiarism Checker、iThenticate（学术机构常用）

## 23.4 研究论文的写作流程

学术论文不是从第一个字写到最后一个字的线性过程，而是一个迭代循环：

```
选题 → 文献检索 → 文献综述 → 提纲 → 初稿 → 修改 → 同行评审 → 终稿
  ↑_________________反馈与修正_____________________________________|
```

### 第一阶段：选题与聚焦

好的研究题目具备三个特征：

1. **研究空白（Research Gap）**：现有研究没有充分回答的问题
2. **可行性（Feasibility）**：在时间、资源、数据获取条件内可以完成
3. **重要性（Significance）**：回答这个问题对领域有实质贡献

**研究问题的类型**：

| 类型 | 例子 |
|---|---|
| 描述性（Descriptive）| What are the current patterns of social media use among Chinese college students? |
| 解释性（Explanatory）| Why do girls experience stronger negative effects from Instagram than boys? |
| 评估性（Evaluative）| How effective is Finland's media literacy curriculum in reducing misinformation susceptibility? |
| 规范性（Normative）| Should social media platforms be regulated as public utilities? |

### 第二阶段：文献检索

**可靠的学术数据库**：

| 数据库 | 适用领域 |
|---|---|
| Google Scholar | 跨学科，免费，覆盖广 |
| JSTOR | 人文、社会科学 |
| PubMed | 医学、生命科学 |
| Web of Science | 自然科学、工程 |
| SSRN | 社会科学预印本 |
| CNKI（知网） | 中国学术文献 |

**文献筛选的PICOS框架**（适用于系统综述）：
- **P**opulation（研究对象）
- **I**ntervention（干预措施）
- **C**omparison（对比条件）
- **O**utcome（结果测量）
- **S**tudy Design（研究设计）

### 第三阶段：提纲撰写（Outline）

论文提纲的两个层次：

**一级提纲（Section Headers）**——确定章节
**二级提纲（Paragraph-level）**——每段一个主题句

```
示例提纲：
I. Introduction
   A. Hook: The paradox of connected loneliness
   B. Background: Rise of social media, simultaneous rise of loneliness
   C. Research gap: Most studies correlational; few experimental
   D. Thesis: Heavy social media use causally increases loneliness
      through three mechanisms

II. Literature Review
   A. Correlational studies (Twenge, 2017; Turkle, 2015)
   B. Experimental evidence (Hunt et al., 2018 — Facebook abstinence study)
   C. Proposed mechanisms (passive use, social comparison, FOMO)

III. Methodology
   A. Study design: 6-week randomized controlled trial
   B. Participants: 240 college students, stratified sample
   C. Measures: UCLA Loneliness Scale, ESM, screen time data
   ...
```

### 第四阶段：初稿写作

关键原则：**写初稿时关闭编辑模式**。

初稿的目标是把想法变成文字，而不是写出完美的句子。专业作者的策略：

- 设定每日写作配额（如：每天500词）
- 写之前先在提纲上完成思考
- 遇到卡壳处用 `[EXPAND LATER]` 标记，继续向前
- 引用可以用 `[CITE: Jones 2019]` 临时标记，不要因为查引用格式打断写作流

### 第五阶段：修改与润色（Revision & Editing）

修改分三个层次，必须**从宏观到微观**依次进行（不能边写边改微观细节）：

| 层次 | 关注点 | 核心问题 |
|---|---|---|
| 内容修改（Revision）| 论点、证据、结构、逻辑 | 论点是否明确？证据是否充分？结构是否合理？ |
| 段落修改（Editing）| 段落组织、过渡、连贯 | 每段是否单一主题？段落之间是否流畅？ |
| 语言润色（Proofreading）| 语法、拼写、标点、格式 | 是否有错别字、语法错误、格式不统一？ |

**修改的常见问题清单**：

- [ ] 每段是否都有清晰的主题句？
- [ ] 每个论点是否都有证据支撑？
- [ ] 是否处理了对立观点？
- [ ] 所有引用是否格式正确？
- [ ] 摘要是否准确反映论文内容？
- [ ] 标题是否具体且信息量充足？

### 第六阶段：同行评审（Peer Review）

同行评审是学术质量的核心把关机制。作为作者应：

1. **不要防御性地回应**——评审意见是礼物，不是攻击
2. **对每条意见明确回应**——接受、修改或礼貌解释为何不接受
3. **提交修改说明（Response Letter）**——条目式回应每条评审意见

## 23.5 摘要（Abstract）的写法

摘要是论文的"名片"，也是数据库中读者最先（通常也是唯一）看到的内容。大多数摘要字数限制在150-300词。

### 摘要的四段式结构（IMRaD格式）

| 要素 | 内容 | 参考字数 |
|---|---|---|
| 目的（Purpose/Objective）| 本研究的目标和研究问题 | 1-2句 |
| 方法（Methods）| 研究设计、样本、数据收集方法 | 2-3句 |
| 结果（Results/Findings）| 主要发现，包括具体数据 | 2-3句 |
| 结论（Conclusion/Implications）| 发现的意义和对领域的贡献 | 1-2句 |

### 摘要范文

```
This study investigates the relationship between passive social media
consumption and self-reported loneliness among university students in
the United Kingdom. [目的] Using a six-week randomized controlled trial,
240 first-year students were randomly assigned to either a restricted
social media condition (maximum 30 minutes per day) or a control condition
(unrestricted use). Loneliness was measured using the validated UCLA
Loneliness Scale (Version 3) at baseline, week three, and week six;
passive versus active use was tracked through screen time monitoring
software. [方法] Results indicated that participants in the restricted
condition reported significantly lower loneliness scores at week six
compared to the control group (M = 38.4 vs. 44.7; p < .001, d = 0.62).
The reduction was most pronounced among participants who had reported
high baseline levels of passive scrolling behavior. [结果] These findings
provide experimental evidence that reducing passive social media consumption
can meaningfully decrease loneliness in young adults, with implications for
both individual digital wellness interventions and platform design policy. [结论]
```

### 摘要写作的常见错误

| 错误 | 说明 |
|---|---|
| 引入背景过多 | 摘要不是引言，不需要大篇幅交代领域背景 |
| 缺少具体数据 | "significant results"不如"p < .05, d = 0.6"有信息量 |
| 泄露结论前提 | 摘要应告知读者研究了什么、发现了什么，而非评价 |
| 使用缩写不定义 | 摘要是独立文本，缩写必须在摘要内部定义 |
| 超过字数限制 | 严格遵守期刊或学校的字数要求 |

## 23.6 Literature Review（文献综述）的写法

文献综述不是参考文献列表的扩展版，也不是对每篇文章逐一描述的流水账。它是**对话**——你在与该领域的研究者对话，展示你对已有知识的把握，并论证你的研究如何填补空白或推进认识。

### 文献综述的三个核心功能

1. **展示学术背景**：让读者了解该领域已经知道了什么
2. **识别研究空白**：指出现有研究的局限、矛盾或空缺
3. **为你的研究定位**：解释你的研究如何在现有知识基础上前进

### 综述的组织方式

**① 主题组织（Thematic）**——最常见，最推荐

不按作者或发表年份，而是按主题、概念或争论组织：

```
Section 1: The correlational evidence
    → 讨论所有展示相关性的研究，并指出其局限
Section 2: Experimental evidence and causal mechanisms
    → 讨论试图建立因果关系的研究
Section 3: Moderating variables
    → 讨论影响关系强弱的变量（年龄、性别、使用类型等）
```

**② 争论组织（Debate-Based）**

当领域内存在真正的学术争论时，按立场组织：

```
Perspective A: Social media as primarily harmful (Twenge, Haidt...)
Perspective B: Social media as morally neutral or beneficial (Valkenburg...)
Synthesis: What the current evidence can and cannot support
```

### 综述的语言：综合而非罗列

**罗列式（避免）**：
```
Smith (2015) found that social media use is associated with depression.
Jones (2016) also found that social media causes loneliness. Brown (2017)
conducted a study showing similar results. Williams (2018) replicated
these findings...
```

**综合式（推荐）**：
```
A growing body of correlational research has established consistent
associations between social media use and poor mental health outcomes.
Initial cross-sectional studies (Smith, 2015; Jones, 2016) documented
significant relationships between daily social media use and depressive
symptoms, findings subsequently replicated across multiple national
contexts (Brown, 2017; Williams, 2018; Chen & Lee, 2019). However,
these correlational designs cannot rule out the possibility of reverse
causation — that individuals with pre-existing depression turn to
social media as a coping mechanism.
```

关键区别：综合式揭示了研究之间的**逻辑关系**（复制、限制、扩展、挑战），而不只是并列描述。

### 综述写作中的关键动词

| 表示支持/证实 | 表示质疑/挑战 | 表示增补 |
|---|---|---|
| demonstrates | challenges | extends |
| confirms | contests | adds to |
| corroborates | contradicts | builds upon |
| validates | questions | supplements |
| supports | refutes | complements |
| establishes | disputes | nuances |

## 23.7 学术写作中的常见错误

### 错误一：中式英语（Chinglish）思维

中国学生最常见的学术写作障碍，往往来自中文思维的直译。

| 中式英语 | 地道学术英语 |
|---|---|
| With the development of society... | As contemporary societies undergo rapid transformation... |
| In today's world, more and more people... | An increasing proportion of the population in developed nations... |
| From ancient times to now... | Historically, / Throughout history, |
| This essay will discuss... | This paper examines / This study investigates... |
| There exist both advantages and disadvantages. | This approach presents both significant benefits and notable limitations. |
| We should pay attention to... | Policymakers must / It is imperative that... |
| It is widely acknowledged that... | Prevailing scholarly consensus holds that... |

### 错误二：逻辑断裂（Non Sequitur）

论点之间缺乏明确的逻辑连接，让读者无法追踪推理过程。

```
❌ 断裂的推理：
"Finland has excellent education outcomes. Therefore, Finland should
adopt a four-day work week."

✓ 完整的推理链：
"Finland's consistently high PISA rankings have been attributed in part
to policies that prioritize student well-being, including shorter school
days and extensive recess time. This evidence that reduced working hours
can be compatible with — and may even enhance — productivity suggests
that a four-day work week is worth investigating in the Finnish context."
```

### 错误三：过度引用（Over-citation）

每句话都加引用，反而显示作者没有自己的声音。常识性知识不需要引用；被广泛接受的学科知识不需要引用；只有具体的数据、论点和研究发现才需要引用。

### 错误四：描述代替分析

描述告诉读者"是什么"，分析告诉读者"这意味着什么"。

```
❌ 纯描述：
"Smith (2019) found that 68% of participants showed improvement after the intervention."

✓ 描述 + 分析：
"Smith (2019) found that 68% of participants showed improvement after the
intervention — a result that lends credibility to the efficacy of
cognitive-behavioral approaches for this population, though the absence
of a control group limits causal inference."
```

### 错误五：段落缺乏内聚力

段落中各句子没有紧密服务于同一主题。参见第21章的段落诊断部分。

## 本章小结

| 核心主题 | 关键要点 |
|---|---|
| 四大原则 | 客观性、精确性、正式性、证据支撑 |
| 语言特点 | 正式词汇替换；对冲语言；名词化；第一人称因学科而异 |
| 引用规范 | 直接引用/改写/总结各有用途；APA与MLA格式熟悉基础规则 |
| 写作流程 | 选题→文献综述→提纲→初稿→修改→同行评审 |
| 摘要 | 目的+方法+结果+结论，150-300词，含具体数据 |
| 文献综述 | 综合而非罗列；按主题或争论组织；展示研究空白 |
| 常见错误 | 中式英语、逻辑断裂、过度引用、描述代替分析 |

学术写作是一种**思维训练**，不只是语言技能。写清楚意味着想清楚；论证严密意味着逻辑成熟。掌握学术写作，本质上是在训练批判性思维——这项能力，将在学术之外的所有领域为你带来竞争优势。
